currenti:/kəˈrɛnti/| adj.|当前的，现行的|currently existing or happening; now in progress.|例句：The currently accepted theory is that...  目前被接受的理论是……  The current situation is quite serious. 当前的情况相当严重。|近义词：present, existing, ongoing, contemporary|反义词：past, former, previous, obsolete


USAGE->
一般现在时:
The current situation is stable. = 目前局势稳定。
We are currently working on a new project. = 我们目前正在做一个新的项目。


一般过去时:
The current situation was tense last week. = 上周局势紧张。
They were not currently employed at that time. = 他们当时没有工作。


现在完成时:
The company has been currently focusing on expansion. = 公司目前一直专注于扩张。
We have currently completed the first phase of the project. = 我们目前已经完成了项目的第一个阶段。


过去完成时:
By then, the current situation had already changed dramatically. = 到那时，目前的形势已经发生了巨大的变化。
They had not been currently aware of the risk. = 他们当时没有意识到风险。


将来时:
The current situation will likely improve next year. = 明年的现状可能会好转。
They will currently be working overseas. = 他们目前将在海外工作。


虚拟语气:
If the current situation were different, we would act differently. = 如果现状不同，我们会采取不同的行动。
I wish the current situation were better. = 我希望现状更好。


被动语态:
The current system is being updated. = 当前系统正在更新中。
The current rules were being revised at that time. = 当时正在修订现行规则。


疑问句:
Is the current policy effective? = 当前政策有效吗？
What is the current status of the project? = 项目的当前状态是什么？


否定句:
The current plan is not feasible. = 当前计划不可行。
He is not currently available. = 他目前不在。
