implement:/ˈɪmplɪment/ | n., vt. | 工具；方法；执行，实施 | to put a plan or system into effect; a tool or instrument | The new marketing strategy was successfully implemented. 新的营销策略成功实施了。  The farmer used a variety of implements to till the soil. 这位农民使用了各种工具来耕地。 | 近义词：execute, carry out, enact, tool, device | 反义词：abandon, neglect


USAGE->
一般现在时:
They implement new policies every year. = 他们每年都实施新的政策。
The company implements a strict code of conduct. = 公司实行严格的行为准则。


一般过去时:
The government implemented a new tax law last month. = 政府上个月实施了一项新的税法。
She implemented her plan with great care. = 她非常仔细地执行了她的计划。


现在完成时:
We have implemented several changes to improve efficiency. = 我们已经实施了几项变革以提高效率。
They have implemented a new software system. = 他们已经实施了一个新的软件系统。


过去完成时:
They had implemented the project before the deadline. = 他们在截止日期前完成了项目。
Before the meeting, she had implemented all the necessary changes. = 会议之前，她已经实施了所有必要的改变。


将来时:
We will implement the new system next quarter. = 我们将在下个季度实施新系统。
The company will implement stricter security measures. = 公司将实施更严格的安全措施。


虚拟语气:
If we implemented this idea, it would improve our sales. = 如果我们实施这个想法，它将改善我们的销售额。
I wish they had implemented the changes sooner. = 我希望他们早点实施这些改变。


被动语态:
The new rules were implemented quickly and efficiently. = 新规迅速有效地得到了实施。
The changes will be implemented gradually over the next year. = 这些变化将在未来一年内逐步实施。


疑问句:
When will the new regulations be implemented? = 新规什么时候实施？
How will they implement the new strategy? = 他们将如何实施新的战略？


否定句:
They did not implement the plan as scheduled. = 他们没有按计划实施计划。
The company has not yet implemented the new policy. = 公司尚未实施新政策。


PHRASE->
implement a plan = 实施计划
implement a policy = 实施政策
implement a strategy = 实施战略
implement a system = 实施系统
implement changes = 实施变革
