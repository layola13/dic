netty:/ˈnɛti/| n.| 网络应用框架 | Netty is an event-driven asynchronous network application framework. It's written in Java and is used for developing high-performance, scalable network applications such as servers and clients.  | Example: We used Netty to build a high-throughput messaging server. 我们使用Netty构建了一个高吞吐量的消息服务器。 |近义词：Mina, Grizzly|反义词：无
