epel:/ˈɛpəl/| n.|艾佩尔（软件包管理器）| EPEL (Extra Packages for Enterprise Linux) is a community-maintained repository of software packages for Enterprise Linux distributions.  It provides additional packages not included in the standard repositories, often including newer versions of software and packages not offered by the vendor. |Example:  I installed the latest version of php using the EPEL repository. 我使用 EPEL 仓库安装了最新版本的 php。|近义词:  RPM Fusion,  (other similar repos)|反义词: N/A
