cacheable: /ˈkæʃəbəl/ | adj. | 可缓存的 |  Able to be stored in a cache for later retrieval. This typically refers to data that can be efficiently stored and quickly accessed from a temporary storage location (a cache) instead of retrieving it from a slower primary source.  |  The images on this website are cacheable, leading to faster loading times.  这网站上的图片可缓存，因此加载速度更快。  The database query results are cacheable to improve performance. 数据库查询结果可缓存以提高性能。 |  storable, bufferable | uncacheable
