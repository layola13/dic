linting:/ˈlɪntɪŋ/| n.|代码静态分析|The process of analyzing source code to flag programming errors, bugs, style violations, and other code quality issues without actually executing the code.  It helps improve code readability, maintainability, and reliability. |Example:  Linting my JavaScript code revealed several potential errors before I even ran the program.  代码静态分析我的 JavaScript 代码在我运行程序之前就发现了几个潜在的错误。|近义词：static analysis, code analysis|反义词：N/A|名词
