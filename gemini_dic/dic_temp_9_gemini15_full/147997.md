overright:/ˈoʊvərraɪt/| v.|覆盖；改写；推翻；压倒|To write over something already written; to override or supersede.|例句：He overwrote the original document with his new edits. 他用新的编辑内容覆盖了原始文件。|例句：The new law overwrites the previous regulations. 新法取代了之前的规章。|近义词：rewrite, supersede, override|反义词：preserve, maintain


USAGE->

一般现在时:
He overwrites the file every time he saves. = 他每次保存文件都会覆盖它。
She overwrites her old work with new ideas. = 她用新的想法覆盖了她以前的工作。


一般过去时:
I overwrote the document by accident. = 我不小心覆盖了文档。
They overwrote the existing code with a new version. = 他们用新版本覆盖了现有的代码。


现在完成时:
He has overwritten the important data. = 他已经覆盖了重要的数据。
She has overwritten the file multiple times. = 她已经多次覆盖了该文件。


过去完成时:
I had overwritten the file before I realized my mistake. = 我在意识到我的错误之前已经覆盖了文件。
They had overwritten the program before the backup was completed. = 备份完成前，他们已经覆盖了程序。


将来时:
He will overwrite the settings if you don't save them. = 如果你不保存设置，他会覆盖它们。
She will overwrite the previous results with the new data. = 她将用新的数据覆盖之前的结果。


虚拟语气:
If I had saved the file, I wouldn't have overwritten it. = 如果我保存了文件，我就不会覆盖它了。
If she hadn't overwritten the file, we would still have the original. = 如果她没有覆盖文件，我们仍然会有原始文件。


被动语态:
The file was overwritten by the new update. = 文件被新的更新覆盖了。
The data was overwritten accidentally. = 数据被意外地覆盖了。


疑问句:

否定句:
