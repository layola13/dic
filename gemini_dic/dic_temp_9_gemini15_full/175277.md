rained:/reɪnd/| v.| 下雨 | It rained heavily yesterday. 昨天雨下得很大。|近义词：showered, poured|反义词：shone, sunned


USAGE->
一般现在时:
It rains a lot in Seattle. = 西雅图经常下雨。
It rarely rains in the desert. = 沙漠很少下雨。

一般过去时:
It rained cats and dogs last night. = 昨晚下了一场倾盆大雨。
It rained steadily throughout the day. = 雨整整下了一天。

现在完成时:
It has rained all week. = 整周都在下雨。
It hasn't rained for months. = 好几个月没下雨了。

过去完成时:
It had rained all morning, so the ground was soaked. = 一上午都在下雨，所以地面湿透了。
It had rained heavily before the game started. = 比赛开始前下了一场大雨。

将来时:
It will rain tomorrow, according to the forecast. = 据预报，明天会下雨。
They predict it will rain heavily next week. = 他们预测下周会下大雨。

虚拟语气:
If it rained tomorrow, we would postpone the picnic. = 如果明天下雨，我们就推迟野餐。
I wish it rained; the garden needs watering. = 我希望下雨；花园需要浇水。

被动语态:
(The verb "rain" is rarely used in the passive voice because it is an intransitive verb.  A passive construction would be unnatural.)


疑问句:
Did it rain yesterday? = 昨天下雨了吗？
Is it raining outside? = 外面在下雨吗？

否定句:
It didn't rain yesterday. = 昨天没下雨。
It doesn't rain often here. = 这儿不常下雨。
