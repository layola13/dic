ifc:/ˈɪfsiː/|abbr.|行业基金委员会|Industry Foundation Classes,  a standardized, open specification for digital asset management in the AEC (architecture, engineering, and construction) industry. It defines a set of object types and relationships to represent building and infrastructure data in a computer-readable format.|例句：The project utilized IFC to exchange data between different software applications.|近义词：BIM standard (Building Information Modeling standard)|反义词：无
