hyperlinking:/ˈhaɪpərˌlɪŋkɪŋ/| n. |超链接，超链接技术|The process of creating a link from one digital document to another, enabling users to navigate between related information.  This is typically achieved using clickable text or images that, when selected, direct the user to a specified URL or location within the same document.|Example:  The website used hyperlinking to connect articles related to the same topic.  这个网站使用超链接来连接与同一主题相关的文章。|近义词：linking, connecting|反义词：disconnecting, separating|名词
