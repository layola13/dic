obsolete:/ˈɑːbsəliːt/ | adj. | 已过时的；废弃的；淘汰的 | No longer produced or used; out of date.  Something obsolete is no longer in use because something newer or better has replaced it.  | Example:  That technology is obsolete now.  现在这项技术已经过时了。The old system has become obsolete. 旧系统已经过时了。|近义词:outdated, outmoded, archaic, passé, defunct |反义词:current, modern, up-to-date, new

USAGE->
一般现在时:
This typewriter is obsolete. = 这台打字机已经过时了。
That software is obsolete. = 那个软件已经过时了。

一般过去时:
The factory closed because its machinery became obsolete. = 工厂倒闭是因为它的机器已经过时了。
The old method was declared obsolete. = 旧方法被宣布淘汰了。

现在完成时:
Many traditional crafts have become obsolete. = 许多传统工艺已经过时了。
That design has become obsolete since the new model was introduced. = 自从新车型推出以来，那个设计已经过时了。

过去完成时:
By the time we arrived, the technology had already become obsolete. = 我们到达时，这项技术已经过时了。
The system had become obsolete before they could upgrade it. = 他们还没来得及升级，系统就已经过时了。

将来时:
In the future, many current jobs will become obsolete. = 将来，许多当前的工作将会过时。
That model will soon become obsolete. = 那个型号很快就会过时了。

虚拟语气:
If we hadn't updated our software, it would have become obsolete. = 如果我们没有更新软件，它就会过时了。
I wish that outdated system hadn't become obsolete so quickly. = 我真希望那个过时的系统没有这么快就过时。

被动语态:
The old equipment was declared obsolete. = 旧设备被宣布淘汰。
Many traditional skills have been made obsolete by technology. = 许多传统技能已被科技淘汰。

疑问句:
Is this software obsolete? = 这个软件过时了吗？
Has that technology become obsolete? = 那项技术过时了吗？

否定句:
The information is not obsolete. = 信息没有过时。
This method is not yet obsolete. = 这个方法还没有过时。
