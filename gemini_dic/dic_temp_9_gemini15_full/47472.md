deallocated:/ˌdiːˈæləkeɪtɪd/| adj.|已释放的，已取消分配的| Released from allocation; no longer assigned or reserved.  The memory has been deallocated after the program finished. 程序结束后，内存已被释放。|近义词：freed, released, unassigned|反义词：allocated, assigned


USAGE->
一般现在时:
The system deallocates unused memory automatically. = 系统自动释放未使用的内存。
The company deallocates funds for unprofitable projects. = 公司取消对无利可图项目的资金分配。


一般过去时:
The programmer deallocated the array after use. = 程序员在使用后释放了数组。
The server deallocated the resources after the request was processed. = 服务器在请求处理后释放了资源。


现在完成时:
The operating system has deallocated all the temporary files. = 操作系统已释放所有临时文件。
The database administrator has deallocated the unused database space. = 数据库管理员已释放未使用的数据库空间。


过去完成时:
The program had deallocated the memory before it crashed. = 程序崩溃前已释放了内存。
The team had deallocated the budget before the project was cancelled. = 项目取消前，团队已取消了预算分配。


将来时:
The system will deallocate the memory when it's no longer needed. = 系统在不再需要内存时将释放它。
The manager will deallocate the resources to other projects. = 经理将把资源分配给其他项目。


虚拟语气:
If the program had deallocated the memory properly, the error wouldn't have occurred. = 如果程序正确释放了内存，就不会发生错误。
It would be better if the company deallocated funds for that failing project. = 如果公司取消对那个失败项目的资金分配就好了。


被动语态:
The memory was deallocated by the garbage collector. = 内存由垃圾收集器释放。
The resources were deallocated after the completion of the task. = 任务完成后，资源被释放。


疑问句:
Has the memory been deallocated? = 内存是否已释放？

否定句:
The system did not deallocate the memory. = 系统没有释放内存。
The resources were not deallocated properly. = 资源没有被正确释放。
