non-trivial:/ˌnɒnˈtrɪviəl/| adj. |非平凡的；重要的；不简单的|Not insignificant or commonplace; important or noteworthy.  It implies something more complex or significant than the simplest or most obvious case.|The mathematician proved a non-trivial theorem.|近义词：significant, important, substantial, complex|反义词：trivial, insignificant, simple, easy


USAGE->
一般现在时:
The problem is non-trivial to solve. =这个问题不容易解决。
This project presents a non-trivial challenge. = 这个项目提出了一个重要的挑战。


一般过去时:
The experiment yielded non-trivial results. = 实验产生了重要的结果。
The discovery was a non-trivial achievement. = 这一发现是一个重要的成就。


现在完成时:
Researchers have achieved non-trivial progress in this field. = 研究人员在这个领域取得了重要的进展。
We have encountered a non-trivial obstacle. = 我们遇到一个重要的障碍。


过去完成时:
They had already overcome several non-trivial hurdles. = 他们已经克服了几个重要的障碍。
The engineers had anticipated a non-trivial design challenge. = 工程师们预料到一个重要的设计挑战。


将来时:
The development will require non-trivial resources. = 开发将需要大量的资源。
This project will present non-trivial difficulties. = 这个项目将会带来重要的困难。


虚拟语气:
If the problem were non-trivial, it would require more time. = 如果问题很重要，它将需要更多的时间。
I wish the solution wasn't so non-trivial. = 我希望解决方案不是那么复杂。


被动语态:
The issue was considered non-trivial by the committee. = 委员会认为这个问题很重要。
The impact of the decision was deemed non-trivial. = 决定的影响被认为是重要的。


疑问句:
Is this a non-trivial matter? = 这是一个重要的问题吗？
Are the implications of this decision non-trivial? = 这个决定的影响是否重大？


否定句:
The solution was not non-trivial, it was quite simple. = 解决方案并不复杂，它很简单。
The changes were not non-trivial; they were minor adjustments. = 这些改变并不重要；它们只是微小的调整。


PHRASE->
a non-trivial amount = 相当大的数量
a non-trivial task = 一项艰巨的任务
a non-trivial problem = 一个难题
a non-trivial solution = 一个复杂的解决方案
non-trivial pursuit =  重要的追求 (通常指耗时费力的重要事情)
