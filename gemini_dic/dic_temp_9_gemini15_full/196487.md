servers:/ˈsɜːrvərz/| n.|服务器，服务员|A server is a computer or system that provides data to other computers or devices over a network.  It can also refer to a person who serves food or drinks.|例句：Our web servers are down. = 我们的网络服务器宕机了。|例句：The server brought us our appetizers. = 服务员给我们上了开胃菜。|近义词：computer, host, waiter, waitress|反义词:client, customer


USAGE->
一般现在时:
The server processes requests quickly. = 服务器快速处理请求。
The servers are always busy during peak hours. = 高峰期服务器总是很忙。


一般过去时:
The server crashed last night. = 服务器昨晚崩溃了。
The servers handled the increased traffic efficiently yesterday. = 服务器昨天有效地处理了增加的流量。


现在完成时:
The company has installed new servers. = 公司安装了新的服务器。
We have used this server for many years. = 我们已经使用这台服务器很多年了。


过去完成时:
The website was down because the server had failed. = 网站宕机是因为服务器出故障了。
Before the update, the servers had been running smoothly for months. = 更新之前，服务器已经平稳运行了几个月。


将来时:
We will upgrade our servers next month. = 我们下个月将升级我们的服务器。
The new servers will improve performance significantly. = 新服务器将显著提高性能。


虚拟语气:
If we had more servers, the website would be faster. = 如果我们有更多服务器，网站就会更快。
I wish the server were more reliable. = 我希望服务器更可靠。


被动语态:
The data is stored on the servers. = 数据存储在服务器上。
The requests are processed by the servers. = 请求由服务器处理。


疑问句:
Are the servers online? = 服务器在线吗？
How many servers do we have? = 我们有多少台服务器？


否定句:

We don't have enough servers to handle the load. = 我们没有足够的服务器来处理负载。
