installs:/ɪnˈstɔːlz/ | vt. |安装，设置|To put (software or an application) into a computer system or device so that it can be used.|The technician installs the new software. 技术员安装了新软件。|近义词：set up, implement, fix|反义词：uninstall, remove


USAGE->
一般现在时:
He installs software every week. 他每周都安装软件。
The company installs security systems for businesses. 这家公司为企业安装安全系统。


一般过去时:
She installed a new operating system yesterday. 昨天她安装了一个新的操作系统。
They installed the new printer in the office. 他们把新打印机安装在办公室里。


现在完成时:
I have installed the necessary drivers. 我已经安装了必要的驱动程序。
We have installed the new software on all the computers. 我们已经在所有电脑上安装了新软件。


过去完成时:
They had installed the program before the meeting started. 会议开始前他们已经安装了程序。
She had installed the app before she started using it. 她使用应用程序之前已经安装了它。


将来时:
He will install the game later tonight. 他今晚晚些时候会安装游戏。
They will install the new server next week. 他们下周将安装新的服务器。


虚拟语气:
If I had more time, I would install that software. 如果我有更多时间，我会安装那个软件。
It's essential that the program be installed correctly.  至关重要的是程序要正确安装。


被动语态:
The software was installed by the IT department. 软件是由IT部门安装的。
The new system is being installed this week.  新系统本周正在安装。


疑问句:
Have you installed the update yet? 你已经安装更新了吗？
Will they install the program tomorrow? 他们明天会安装程序吗？


否定句:
He didn't install the correct drivers. 他没有安装正确的驱动程序。
They haven't installed the new software yet.  他们还没有安装新软件。


PHRASE->
install software 安装软件
install hardware 安装硬件
install an application 安装应用程序
install a program 安装程序
install a driver 安装驱动程序
