fstream:/ˈfiːstrɛm/ | n. | 文件流 | A file stream is an object that allows you to read from and write to files.  It provides a way to interact with files as if they were streams of data.  It's part of the C++ standard library and provides a higher-level abstraction than lower-level file I/O functions. | Example: `std::fstream myfile("example.txt");`  (This creates a file stream object named `myfile` associated with the file "example.txt".) |近义词：file handle (less object-oriented)| 反义词：N/A | 名词


USAGE->
This word is not a verb, adjective, or adverb, so it doesn't conjugate or decline in the way verbs or adjectives do.  The usage is always as a noun, usually as an object in a sentence.  The following examples show its use in different contexts:

Declaration:
`std::fstream myFile;` =  声明一个名为myFile的文件流对象。
`std::ifstream inputFile("input.txt");` = 声明一个名为inputFile的输入文件流对象，并将其与文件"input.txt"关联。


Opening a file:
`myfile.open("example.txt", std::ios::out);` = 打开名为"example.txt"的文件以进行写入操作。
`myfile.open("data.bin", std::ios::binary | std::ios::in);` = 打开名为"data.bin"的二进制文件以进行读取操作。


Reading from a file:
`std::string line;
std::getline(myfile, line);` = 从文件中读取一行到line变量中。
`int number;
myfile >> number;` = 从文件中读取一个整数到number变量中。


Writing to a file:
`myfile << "Hello, world!" << std::endl;` = 将字符串"Hello, world!"写入文件。
`myfile.write(buffer, size);` = 将缓冲区buffer中的size个字节写入文件。


Closing a file:
`myfile.close();` = 关闭文件流。
`inputFile.close();` = 关闭输入文件流。



PHRASE->
`std::fstream myFile;`  (file stream declaration)
`myfile.open(...)` (file opening)
`myfile.close()` (file closing)
`myfile >> variable;` (reading from file)
`myfile << variable;` (writing to file)
