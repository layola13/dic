live-it-up:/lɪv ɪt ʌp/ | v. |尽情享乐，狂欢 |To enjoy oneself to the full; to indulge in pleasure and excitement.  |例句：Let's live it up this weekend!  让我们这个周末痛快玩一把！|例句：They lived it up in Las Vegas.他们在拉斯维加斯尽情享乐。|近义词：carouse, revel, celebrate, party |反义词：abstain, deny oneself, repress


USAGE->
一般现在时:
They live it up every weekend. = 他们每个周末都尽情享乐。
We live it up at the beach. = 我们在海滩上尽情玩耍。


一般过去时:
They lived it up at the concert last night. = 他们昨晚在音乐会上玩得很尽兴。
We lived it up at the party. = 我们在派对上玩得很开心。


现在完成时:
They have lived it up all summer. = 他们整个夏天都过得很开心。
We have lived it up since we won the lottery. = 自从我们中了彩票以来，我们就过着挥金如土的生活。


过去完成时:
They had lived it up for months before they ran out of money. = 他们在钱花光之前已经尽情享乐了好几个月了。
We had lived it up before the responsibilities of adulthood hit. = 在成年人的责任来临之前，我们过得很逍遥自在。


将来时:
They will live it up on their vacation. = 他们将在假期里尽情享乐。
We will live it up at the festival next week. = 我们下周将在节日上尽情狂欢。


虚拟语气:
If we had more money, we would live it up. = 如果我们有钱的话，我们会尽情享乐。
I wish I could live it up like they do. = 我真希望我能像他们那样尽情享乐。


被动语态: (此短语通常不用被动语态)
This lifestyle is not often lived it up. = 这种生活方式并不常被人们尽情享乐。


疑问句:
Will you live it up on your birthday? = 你会在生日那天尽情享乐吗？
Are they living it up tonight? = 他们今晚会尽情狂欢吗？


否定句:
They don't live it up often. = 他们不经常尽情享乐。
We didn't live it up last weekend. = 我们上周末没有尽情享乐。


PHRASE->
live it up large = 尽情享乐，大肆庆祝
live it up a bit = 小小地放纵一下
live it up in style =  奢华地享受
to really live it up = 真正地尽情狂欢
go out and live it up = 出去痛快玩耍
