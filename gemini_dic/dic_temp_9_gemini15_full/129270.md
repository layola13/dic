node:/nəʊd/|n.|节点，结点；(计算机)节点；枢纽；(植物)节；(解剖)结；|A node is a point in a network or system.  It can refer to a point in a network, a point in a structure, or a point in a process.|例句：Each computer in the network acts as a node. 网络中的每台计算机都充当一个节点。

例句：The tree has several nodes along its branches. 这棵树的树枝上有几个节。|近义词：point, junction, hub|反义词：end, terminus
