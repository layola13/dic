decompiled:/ˌdiːˈkəmˈpaɪld/ | v. |反编译|To decompile something means to translate it from a compiled form (such as machine code) back into a higher-level language (such as assembly language or a programming language like Java or C++). This process is the reverse of compilation.  It is often used to understand how a program works, to modify it, or to recover source code that has been lost.|例句：The programmer decompiled the software to fix a bug. 程序员反编译了软件来修复一个bug。|近义词：disassemble, reverse engineer|反义词：compile
