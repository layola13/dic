ost:/ɔːst/| n.|主人，东道主|The host of the party welcomed everyone warmly.|近义词：hostess, entertainer|反义词：guest


USAGE->
一般现在时:
The host greets the guests. = 主人向客人打招呼。
The host prepares the food. = 主人准备食物。

一般过去时:
The host welcomed us to his home. = 主人欢迎我们到他家做客。
The host served delicious dinner. = 主人端上了美味的晚餐。

现在完成时:
The host has already prepared the drinks. = 主人已经准备好了饮料。
The host has welcomed many guests. = 主人已经接待了许多客人。

过去完成时:
The host had already set the table before the guests arrived. = 客人到达之前，主人已经摆好了桌子。
The host had prepared a special menu for the occasion. = 主人为此场合准备了一份特别的菜单。

将来时:
The host will introduce the speaker. = 主人将介绍演讲者。
The host will provide entertainment for the guests. = 主人将为客人提供娱乐节目。

虚拟语气:
If I were the host, I would make a better speech. = 如果我是主人，我会做一个更好的演讲。
I wish the host had played some music. = 我希望主人能演奏一些音乐。

被动语态:
The guests were welcomed by the host. = 客人受到了主人的欢迎。
The food was prepared by the host. = 食物是由主人准备的。

疑问句:
Who is the host of this event? = 这个活动的主人是谁？
Is the host ready to begin? = 主人准备好开始了吗？

否定句:
The host wasn't present at the beginning. = 主人一开始没有在场。
The host didn't introduce all the guests. = 主人没有介绍所有的客人。
