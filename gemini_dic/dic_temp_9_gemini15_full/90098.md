hosts:/hoʊsts/| /həʊsts/ | n. v. | 主机；主持人；寄主；(动)主持；款待 |  A host can refer to a computer system that provides services to other computers on a network, a person who introduces a television program or other event, or an animal or plant that harbors a parasite.  |  例句：This server hosts multiple websites. 这个服务器托管多个网站。 |近义词：server, presenter, entertainer |反义词：guest, visitor


USAGE->
一般现在时:
The server hosts many websites. = 这台服务器托管许多网站。
He hosts a popular podcast. = 他主持一个很受欢迎的播客。


一般过去时:
She hosted a dinner party last night. = 她昨晚举办了一场晚宴。
The hotel hosted a conference. = 这家酒店举办了一场会议。


现在完成时:
They have hosted several successful events this year. = 他们今年已经成功举办了几场活动。
This website has hosted millions of visitors. = 这个网站已经接待了数百万访客。


过去完成时:
The company had hosted a similar event the previous year. = 公司在去年举办过类似的活动。
Before the concert, they had hosted a meet-and-greet session. = 音乐会之前，他们举办了一个见面会。


将来时:
We will host a birthday party for him next month. = 我们下个月将为他举办生日派对。
She will host the next episode of the show. = 她将主持节目的下一集。


虚拟语气:
If I had more time, I would host a party. = 如果我有更多的时间，我会举办一个派对。
I wish I could host the event. = 我希望我能主持这个活动。


被动语态:
The event was hosted by a renowned organization. = 这场活动由一个著名的组织主办。
Many websites are hosted on cloud servers. = 许多网站都托管在云服务器上。


疑问句:
Who is hosting the event? = 谁是这次活动的组织者？
Will you be hosting the meeting? = 你会主持会议吗？


否定句:
I cannot host the meeting due to a scheduling conflict. = 由于时间冲突，我无法主持会议。
They did not host the party as planned. = 他们没有按照计划举办派对。


PHRASE->
host a party: 举办派对
host a website: 托管网站
host an event: 主办活动
host a program: 主持节目
host family: 寄宿家庭
