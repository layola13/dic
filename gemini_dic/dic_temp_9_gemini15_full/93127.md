ifs:/ɪfs/ | conj. |如果，假使|  A conjunction used to introduce a conditional clause. It indicates that the clause following it describes a condition upon which the main clause depends.  |If it rains, we will stay inside. = 如果下雨，我们就待在里面。|If it snows, we will build a snowman. = 如果下雪，我们会堆雪人。|unless, supposing|then, although, because|连词


USAGE->
一般现在时:
If I study hard, I will pass the exam. = 如果我努力学习，我就能通过考试。
If she works late, she takes a taxi home. = 如果她工作到很晚，她就打车回家。


一般过去时:
If I had time, I would have gone to the party. = 如果我有时间，我就去参加聚会了。(虚拟语气)
If it rained yesterday, the ground would be wet. = 如果昨天下了雨，地面就会湿。(虚拟语气)


现在完成时:
If you have finished your homework, you can watch TV. = 如果你完成了作业，就可以看电视。
If she has seen the movie, she won't want to see it again. = 如果她看过这部电影了，她就不会想再看了。


过去完成时:
If I had known you were coming, I would have prepared a meal. = 如果我知道你要来，我就会准备饭菜了。(虚拟语气)
If they had studied harder, they would have passed the test. = 如果他们学习更努力的话，他们就通过考试了。(虚拟语气)


将来时:
If it rains tomorrow, we will stay home. = 如果明天下雨，我们就待在家里。
If she finishes her work early, she will go to the cinema. = 如果她早点完成工作，她就会去看电影。


虚拟语气:
If I were rich, I would travel the world. = 如果我富有的话，我就会环游世界。(与现在事实相反)
If I had studied harder, I would have passed the exam. = 如果我当时学习更努力的话，我就通过考试了。(与过去事实相反)


被动语态:
If the problem is solved, we will be happy. = 如果问题解决了，我们会很高兴。
If the meeting is cancelled, we will be informed. = 如果会议取消，我们会得到通知。


疑问句:
If you are hungry, what would you like to eat? = 如果你饿了，你想吃什么？
If she is ill, should we call the doctor? = 如果她病了，我们应该叫医生吗？


否定句:
If you don't hurry, you will miss the bus. = 如果你不快点，就会错过公交车。
If she hadn't been ill, she would have gone to school. = 如果她没生病，她就去上学了。(虚拟语气)


PHRASE->
If only = 但愿
If necessary = 如果有必要
If so = 如果这样
If anything = 如果有什么
As if = 好像
