compiles:/kəmˈpaɪlz/| vt.|编译，汇编|To translate (a program written in a high-level programming language) into a lower-level language (such as assembly language or machine code) that can be executed by a computer.|例句：The programmer compiles the code before running the program. 程序员在运行程序之前编译代码。|例句：The compiler successfully compiled the source code into an executable file. 编译器成功地将源代码编译成可执行文件。|近义词：translate, assemble|反义词：interpret
