kb:/keɪˈbiː/ | n. |千字节，KB|kilobyte, a unit of information equal to 1024 bytes.|例句：The file size is 2 KB.  文件大小是2千字节。|近义词：kilobyte|反义词：无


USAGE->
一般现在时:
The file size is 10 KB. = 文件大小是10 KB。
This document takes up 5 KB of space. = 这个文档占用5 KB的空间。


一般过去时:
The image was only 1 KB. = 这张图片只有1 KB。
The download took 2 KB of data. = 下载使用了2 KB的数据。


现在完成时:
We have downloaded 100 KB of data. = 我们已经下载了100 KB的数据。
The computer has processed over 1 GB, that's a lot of KBs. = 电脑已经处理了超过1 GB的数据，那就是很多KB。


过去完成时:
The program had already used 5 KB before it crashed. = 程序崩溃前已经使用了5 KB。
They had downloaded several KBs before the internet connection dropped. = 互联网连接中断之前，他们已经下载了几KB。


将来时:
This video will be around 10 KB. = 这个视频大约会有10 KB。
The software update will require at least 2 KB of free space. = 软件更新至少需要2 KB的可用空间。


虚拟语气:
If the file were only 1 KB, it would download faster. = 如果文件只有1 KB，下载速度会更快。
I wish the image was smaller, maybe only 1 KB. = 我希望图片更小，也许只有1 KB。


被动语态:
5 KB of memory was used by the application. = 应用程序使用了5 KB的内存。
Several KBs are being processed by the server. = 服务器正在处理几KB的数据。


疑问句:
How many KB is the document? = 这个文档有多大（KB）？
What is the KB size of this video? = 这个视频的大小是多少KB？


否定句:
The file is not 2 KB; it's much larger. = 文件不是2 KB；它大得多。
The image doesn't take up more than 1 KB. = 这张图片不占用超过1 KB的空间。
