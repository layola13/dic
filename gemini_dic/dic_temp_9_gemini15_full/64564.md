error-message:/ˈɛrər ˈmɛsɪdʒ/| n. |错误信息，错误提示|A message displayed by a computer program or system indicating that an error has occurred.  It typically describes the nature of the error and may offer suggestions for resolving it.|例句：The program displayed an error message indicating a file not found.  程序显示了一个文件未找到的错误信息。|例句：An error message popped up on the screen, saying "System failure".屏幕上弹出错误信息，显示“系统故障”。|近义词：error report, warning message, fault indication|反义词：success message, confirmation message
