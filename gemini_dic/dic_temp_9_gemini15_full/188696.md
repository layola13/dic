run-throughs:/ˈrʌnθruːz/| n.|彩排，预演|A run-through is a rehearsal, typically a full rehearsal of a play, concert, or other performance before a public performance.|例句：The band had a run-through of their new songs before the concert.乐队在音乐会之前对他们的新歌进行了一次彩排。|例句：The actors did several run-throughs before opening night.演员们在开幕之夜之前进行了几次彩排。|近义词：rehearsal, practice, dry run|反义词：performance, premiere


USAGE->
一般现在时:
The director runs through the scenes every day. = 导演每天排练场景。
We run through the presentation before the big meeting. = 我们在大型会议之前排练演示文稿。


一般过去时:
They ran through the script several times yesterday. = 他们昨天排练了剧本好几次。
The orchestra ran through the symphony before the recording. = 管弦乐队在录音前排练了交响乐。


现在完成时:
We have run through the choreography countless times. = 我们已经排练了无数次的舞蹈动作。
They have run through the entire play already. = 他们已经排练了整出戏。


过去完成时:
They had run through the song many times before the audition. = 他们在试镜前已经排练过这首歌很多次了。
The actors had run through their lines repeatedly before the performance. = 演员们在演出前反复排练过他们的台词。


将来时:
We will run through the entire presentation tomorrow. = 我们明天将排练整个演示文稿。
The band will run through their setlist one last time before the show. = 乐队将在演出前最后一次排练他们的歌单。


虚拟语气:
If we had more time, we would run through the play again. = 如果我们有更多的时间，我们会再次排练这出戏。
I wish we could run through the music one more time. = 我希望我们能够再排练一次音乐。


被动语态:
The scenes were run through by the actors many times. = 这些场景被演员们排练了很多次。
The script was run through thoroughly before the premiere. = 剧本在首映前被仔细地排练过。


疑问句:
Have they run through the new material yet? = 他们已经排练了新素材了吗？
Will we run through the entire act before the curtain rises? = 在幕布升起之前我们会排练整个剧目吗？


否定句:
We didn't run through the dialogue properly. = 我们没有正确地排练对话。
They haven't run through the dance routine yet. = 他们还没有排练舞蹈动作。


PHRASE->
a full run-through = 全面彩排
a final run-through = 最后一次彩排
a quick run-through = 快速排练
a dress run-through = 带妆彩排
a technical run-through = 技术彩排
