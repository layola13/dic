object-oriented:/ˈɑːbdʒɪktˈɔːriɛntɪd/|/ˌɑːbdʒɛktˈɔːriɛntɪd/| adj.|面向对象的|Relating to a style of programming that organizes software design around data, or objects, rather than functions and logic.  Object-oriented programming (OOP) is a programming paradigm based on the concept of "objects", which can contain data and code: data in the form of fields (often known as attributes or properties), and code, in the form of procedures (often known as methods).|Example:  Java is an object-oriented programming language.  Java是一种面向对象的编程语言。


Example:  Many modern software applications are built using object-oriented principles.  许多现代软件应用程序都是使用面向对象原则构建的。


近义词：class-based (in the context of programming paradigms)


反义词：procedural (in the context of programming paradigms)

词性：形容词
