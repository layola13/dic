les:/lez/| pron.| 他们（宾格）|The pronoun "les" is the object pronoun form of "they" in French. It is used when "they" is the object of a verb or preposition.|Example: Je les vois. (I see them.)|近义词:eux (masculine plural object pronoun), elles (feminine plural object pronoun)|反义词:N/A


USAGE->
一般现在时:
Je les aime. = 我爱他们。
Nous les voyons. = 我们看见他们。

一般过去时:
Je les ai vus. = 我看见了他们。(masculine)
Je les ai vues. = 我看见了她们。(feminine)

现在完成时:
J'ai déjà les lus. = 我已经读过它们了。(masculine or feminine)
Nous les avons rencontrés. = 我们已经遇到他们了。(masculine)

过去完成时:
J'avais déjà les vus. = 我之前已经见过他们了。(masculine)
Nous les avions rencontrées. = 我们之前已经遇到她们了。(feminine)


将来时:
Je les verrai. = 我将会看见他们。
Nous les aiderons. = 我们将会帮助他们。

虚拟语气:
Si je les connaissais, je leur parlerais. = 如果我认识他们，我会和他们说话。
Je voudrais qu'ils les aident. = 我希望他们能帮助他们。

被动语态:
Ils sont vus par les autres. = 他们被其他人看见了。
Elles sont aimées par tous. = 她们被所有人爱着。

疑问句:
Les voyez-vous? = 你看见他们了吗？
Qui les a vus? = 谁看见了他们？

否定句:
Je ne les aime pas. = 我不爱他们。
Nous ne les connaissons pas. = 我们不认识他们。


PHRASE->
parmi les = 在...之中
avec les = 和...一起
entre les = 在...之间
chez les = 在...家（复数）
pour les = 为了...
