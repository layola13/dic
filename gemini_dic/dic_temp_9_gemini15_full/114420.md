make sure:/meɪk ʃʊr/| v. |确保，确定 |To take action to ensure that something happens or is the case. It often expresses a strong intention to prevent something bad from happening.|例句：Make sure you lock the door before you leave.  确保你离开前锁上门。|例句：Please make sure the report is submitted by Friday. 请确保在周五之前提交报告。|近义词：ensure, ascertain, verify |反义词：neglect, disregard


USAGE->
一般现在时:
I make sure the lights are off before I go to bed. = 我睡觉前确保灯都关了。
We make sure to arrive early for the meeting. = 我们确保早到会议。

一般过去时:
He made sure the package was securely sealed. = 他确保包裹被牢固地封好。
She made sure to call her parents before leaving on vacation. = 她出发去度假前确保给父母打了电话。

现在完成时:
I have made sure all the files are backed up. = 我已经确保所有文件都备份了。
They have made sure everything is ready for the event. = 他们已经确保一切为活动做好了准备。

过去完成时:
I had made sure the car was parked before I went inside. = 我进去之前已经确保车停好了。
She had made sure all the invitations were sent out before the deadline. = 她在截止日期之前已经确保所有请柬都发出去了。


将来时:
I will make sure to pick you up from the airport. = 我会确保去机场接你。
We will make sure the project is completed on time. = 我们会确保项目按时完成。

虚拟语气:
I would make sure to bring an umbrella if it were raining. = 如果下雨，我会确保带伞。
It would be best if you made sure to check your email regularly. = 最好你确保定期查看你的邮件。

被动语态:
The safety of the passengers must be made sure. = 必须确保乘客的安全。
The accuracy of the data needs to be made sure. = 需要确保数据的准确性。

疑问句:
Did you make sure the oven was turned off? = 你确保烤箱关了吗？
Will you make sure the children are asleep? = 你会确保孩子们睡着了吗？

否定句:
I didn't make sure the door was locked. = 我没有确保门锁上了。
He didn't make sure to save his work. = 他没有确保保存他的工作。


PHRASE->
make sure to do sth = 务必做某事
make sure of sth = 确保某事
make sure that... = 确保...
make sure about sth = 弄清楚某事
be sure to do sth = 一定要做某事
