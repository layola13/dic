debian:/ˈdɛbiən/| n. |戴比安（Linux发行版）|A Debian is a free and open-source operating system based on the Linux kernel.  It is known for its stability, security, and large software repository. |Example: I installed Debian on my server for its reliability.  我为了它的可靠性在我的服务器上安装了Debian。|近义词：Linux distribution, operating system|反义词：无


USAGE->
(Debian itself doesn't have verb conjugations.  The usage examples below apply to sentences *about* Debian.)

一般现在时:
Debian provides a stable platform. = Debian 提供了一个稳定的平台。
Many users prefer Debian. = 许多用户更喜欢 Debian。

一般过去时:
I installed Debian last week. = 我上周安装了 Debian。
The server ran Debian successfully. = 这台服务器成功运行了 Debian。

现在完成时:
I have used Debian for many years. = 我已经使用 Debian 多年了。
They have upgraded to the latest Debian version. = 他们已经升级到最新的 Debian 版本。

过去完成时:
I had already installed Debian before the deadline. = 我在截止日期之前就已经安装了 Debian。
The system had been running Debian for months. = 该系统已经运行 Debian 数月了。

将来时:
I will install Debian on my new computer. = 我将在我的新电脑上安装 Debian。
They will migrate to Debian next year. = 他们明年将迁移到 Debian。

虚拟语气:
If I had more time, I would install Debian. = 如果我有更多的时间，我会安装 Debian。
It would be better if we used Debian. = 如果我们使用 Debian 会更好。

被动语态:
Debian is used by many organizations. = Debian 被许多组织使用。
The system was configured with Debian. = 该系统使用 Debian 配置。

疑问句:
Is Debian suitable for my needs? = Debian 是否适合我的需求？
Does Debian support this hardware? = Debian 是否支持此硬件？

否定句:
Debian is not compatible with this software. = Debian 与此软件不兼容。
I did not install Debian on that machine. = 我没有在那台机器上安装 Debian。


PHRASE->
