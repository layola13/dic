prob:/prɒb/ /prɑːb/| n. |可能性；概率；问题；难题|The probability of success is high.成功的可能性很高。 There are many problems to solve.有很多问题需要解决。|近义词：likelihood, chance, issue, difficulty |反义词：certainty, impossibility


USAGE->
一般现在时:
The probability of rain is high. = 下雨的可能性很高。
There is a problem with the computer. = 电脑出现了一个问题。


一般过去时:
The probability of success was low. = 成功的可能性很低。
We had a problem with the car yesterday. = 昨天我们的车出了问题。


现在完成时:
There have been many problems this year. = 今年已经出现了很多问题。
We have had several problems with the project. = 我们在这个项目中遇到了一些问题。


过去完成时:
We had already encountered problems before the meeting. = 我们在会议之前就已经遇到问题了。
They had solved the problem before I arrived. = 我到达之前他们已经解决了这个问题。


将来时:
We will probably have problems with the new system. = 我们很可能在新系统中遇到问题。
There will be many challenges to overcome. = 将会有许多挑战需要克服。


虚拟语气:
If there were no problems, the project would be finished on time. = 如果没有问题，项目就能按时完成。
I wish there weren't so many problems. = 我希望没有这么多问题。


被动语态:
The problem is being addressed by the team. = 这个问题正在由团队处理。
The problem was solved quickly. = 这个问题很快就被解决了。


疑问句:
What is the problem? = 什么问题？
What's the probability of that happening? = 那件事发生的概率是多少？


否定句:
There is no problem. = 没有问题。
There is no probability of that happening. = 那件事发生的概率为零。


PHRASE->
in all probability = 很可能
more than likely = 很可能
the problem is that = 问题是...
no problem = 没问题
a problem child = 问题儿童
