enumerator:/ɪˈnjuːməreɪtər/ | n. |枚举器，列举者 |A person or thing that enumerates;  a program object that iterates over a collection, yielding one item at a time.  In computer science, an enumerator is an object that allows you to traverse a collection of items sequentially, accessing each item one by one.  It's often used with data structures like arrays, lists, or sets. |Example:  The database query returned an enumerator that I used to process each record individually.  数据库查询返回了一个枚举器，我用它来逐个处理每条记录。|近义词：iterator, counter, lister |反义词：aggregator (in some contexts)
