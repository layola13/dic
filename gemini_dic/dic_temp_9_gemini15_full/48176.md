decompiling:/ˌdiːkəmˈpaɪlɪŋ/ | v. |反编译|The process of converting compiled machine code back into a higher-level programming language, such as Java or C++.  This allows developers to examine the source code of a program, understand its functionality, or potentially modify it. |例句：The security team is decompiling the malware to understand how it works. 安全团队正在反编译恶意软件以了解其工作原理。|近义词：reverse engineering, disassembling |反义词：compiling
