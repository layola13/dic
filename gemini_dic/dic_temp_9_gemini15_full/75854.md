funcs:/fʌŋks/| n.|函数，功能|A function, especially a computer program that performs a specific task.  Functions are a fundamental building block in programming.|Example:  The program contains several functions to handle user input.  这个程序包含几个处理用户输入的函数。|近义词：routine, procedure, subroutine|反义词：N/A


USAGE->
一般现在时:
The function calculates the area. = 这个函数计算面积。
These functions handle data processing. = 这些函数处理数据处理。


一般过去时:
The function returned an error code. = 该函数返回了一个错误代码。
