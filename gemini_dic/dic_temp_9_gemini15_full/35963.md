/koʊd/ | n. |代码；密码；编码 |A code is a system of words, letters, figures, or symbols used to represent others, especially for secrecy.  |Example: The message was encoded using a complex code. 这条消息是用复杂的密码编码的。|近义词：cipher, cypher, encryption |反义词：plaintext, decoded message
