non-mutable: /ˌnɒnˈmjuːtəbl/ | adj. | 不可变的 |  Not able to be changed; fixed.  This term is frequently used in computer science to describe data structures whose values cannot be altered after creation.  | Example:  In many programming languages, strings are immutable.  This means that once a string is created, its value cannot be changed; instead, a new string must be created if a modification is needed. |近义词:unchangeable, invariable, constant, permanent |反义词:mutable, changeable, variable, alterable
