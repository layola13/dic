getline:/ˈɡɛtˌlaɪn/| vt.| 获取一行 |  To read a line of text from an input stream (usually a file or standard input).  The function typically reads characters until a newline character ('\n') is encountered, or the end of the file is reached. | Example:  `getline(cin, myString);` //Reads a line from standard input (cin) and stores it in the string variable myString. |近义词：read line, input line | 反义词：N/A
