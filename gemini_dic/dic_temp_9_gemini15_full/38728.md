conditionals:/kənˈdɪʃənəlz/ | n. | 条件句，条件式 |  Conditional sentences express a result that depends on a condition. They are used to discuss possibilities, hypothetical situations, and consequences.  | Example:  If it rains, the game will be cancelled. = 如果下雨，比赛将被取消。  If I had won the lottery, I would have bought a house. = 如果我中了彩票，我就买了一栋房子。|近义词：hypotheticals, provisos | 反义词：unconditionals | 名词


USAGE->
一般现在时:
If it rains, we will stay inside. = 如果下雨，我们就待在家里。
If you study hard, you will pass the exam. = 如果你努力学习，你就会通过考试。


一般过去时:
If it rained, we stayed inside. = 如果下雨，我们就待在家里。（表示过去事实）
If you studied hard, you passed the exam. = 如果你努力学习，你就会通过考试。（表示过去事实）


现在完成时:
If it has rained, the ground will be wet. = 如果下雨了，地面就会湿。
If you have studied hard, you will be well-prepared for the exam. = 如果你努力学习了，你就会为考试做好充分准备。


过去完成时:
If it had rained, the ground would have been wet. = 如果当时下雨了，地面就会湿。（表示与过去事实相反的假设）
If you had studied hard, you would have passed the exam. = 如果你当时努力学习了，你就会通过考试。（表示与过去事实相反的假设）


将来时:
If it rains tomorrow, we will cancel the picnic. = 如果明天下雨，我们就取消野餐。
If you finish your work early, you will have time to relax. = 如果你早点完成工作，你就会有时间放松。


虚拟语气(与现在事实相反):
If I were rich, I would travel the world. = 如果我有钱，我就会环游世界。
If she were here, she would help us. = 如果她在这里，她就会帮助我们。


虚拟语气(与过去事实相反):
If I had studied harder, I would have passed the exam. = 如果我当时学习更努力，我就会通过考试。
If he had known, he would have told us. = 如果他知道的话，他会告诉我们的。


虚拟语气(与将来事实相反):
If it were to rain tomorrow, we would stay home. = 如果明天要下雨，我们就待在家里。
If I were to win the lottery, I would donate some money to charity. = 如果我中彩票了，我会向慈善机构捐款。


被动语态:
If the car is repaired, it will be ready for pickup. = 如果汽车修好了，就可以取车了。
If the project is completed on time, the bonus will be given. = 如果项目按时完成，奖金就会发放。


疑问句:
If it is raining, should we go out? = 如果正在下雨，我们应该出去吗？
If he is coming, will you meet him? = 如果他要来，你会去见他吗？


否定句:
If it doesn't rain, we will have a picnic. = 如果不下雨，我们就去野餐。
If you don't study hard, you won't pass the exam. = 如果你不努力学习，你就会考试不及格。



PHRASE->
on condition that = 以…为条件
under the condition that = 在…条件下
in the event that = 如果…发生
provided that = 假如；条件是
assuming that = 假设
