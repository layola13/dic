return:/rɪˈtɜːrn/|/rɪˈtɝːn/| vt. vi. n. |返回；归还；回报；收益|To go or come back to a previous place or condition.  To give something back to someone. The profit from an investment.|例句：He returned to his hometown after many years. 他多年后回到了家乡。

The library requires the return of all borrowed books. 图书馆要求归还所有借阅的书籍。


近义词：revert, restore, give back, repay, yield |反义词：depart, leave, stay, keep


USAGE->
一般现在时:
I return to my hometown every summer. = 我每年夏天都回故乡。
They return the borrowed books to the library. = 他们把借来的书还给图书馆。


一般过去时:
He returned home late last night. = 他昨晚很晚才回家。
She returned the money to its owner. = 她把钱还给了失主。


现在完成时:
I have returned the book to the library. = 我已经把书还给图书馆了。
They have returned from their trip. = 他们已经从旅行中回来了。


过去完成时:
He had returned home before the rain started. = 下雨开始之前他已经回家了。
She had returned the call before leaving for work. = 她上班前已经回过电话了。


将来时:
I will return to work next week. = 我下周将返回工作岗位。
They will return the car tomorrow. = 他们明天将把车还回来。


虚拟语气:
If I had returned earlier, I would have seen him. = 如果我早点回来，我就看到他了。
I wish I had returned the book on time. = 我真希望我按时还书。


被动语态:
The book was returned to the library. = 书被还给了图书馆。
The call was returned promptly. = 电话很快就被回拨了。


疑问句:
Did he return the money? = 他还钱了吗？
When will you return? = 你什么时候回来？


否定句:
He did not return my call. = 他没有回我的电话。
She has not yet returned from her vacation. = 她还没有从假期回来。


PHRASE->
return a call = 回电话
return to normal = 恢复正常
return a favor = 帮忙；报答
return fire = 还击
safe return = 平安归来
