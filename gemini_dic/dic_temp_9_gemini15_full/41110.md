coprocessing:/ˌkoʊˈprɑːsɛsɪŋ/| n. |协同处理，并发处理 |The simultaneous execution of tasks by multiple processors, often involving a main processor and a specialized coprocessor. This enhances performance by offloading specific tasks to the coprocessor, freeing up the main processor for other tasks.  |Example:  The system uses coprocessing to accelerate graphics rendering.  这个系统使用协同处理来加速图形渲染。

The new server utilizes coprocessing to handle large data sets more efficiently. 这台新服务器利用协同处理更有效地处理大型数据集。|近义词：parallel processing, concurrent processing |反义词：serial processing, sequential processing
