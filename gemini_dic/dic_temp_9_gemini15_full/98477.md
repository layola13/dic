instal: /ɪnˈstɔːl/ | vt. |安装，安置，任命 |To put (something) in place for use; to establish (someone) in a position or office. |例句：The technician installed the new software on my computer.  技术人员在我的电脑上安装了新软件。

例句：The company installed her as the new CEO. 公司任命她为新任首席执行官。

近义词：fit, fix, set up, appoint | 反义词：remove, uninstall, dismiss


USAGE->
一般现在时:
They install new software every month. = 他们每月安装新软件。
The company installs new employees carefully. = 公司谨慎地录用新员工。


一般过去时:
He installed the new printer yesterday. = 他昨天安装了新的打印机。
They installed him as the head of the department. = 他们任命他为部门主管。


现在完成时:
We have installed the security system. = 我们已经安装了安全系统。
She has been installed as the new president. = 她已经就任新总统。


过去完成时:
They had installed the equipment before the deadline. = 他们已在截止日期前安装好设备。
The committee had already installed him in his new position before the announcement. = 在公告发布之前，委员会已经将他安置在了他的新职位上。


将来时:
We will install the new air conditioner next week. = 我们下周将安装新的空调。
The company will install her as a managing director. = 公司将任命她为董事总经理。


虚拟语气:
If they installed the right software, the problem would be solved. = 如果他们安装了正确的软件，问题就能解决。
I wish they had installed the security cameras sooner. = 我真希望他们早点安装监控摄像头。


被动语态:
The software was installed quickly and efficiently. = 软件安装迅速高效。
He was installed as the chairman of the board. = 他被任命为董事会主席。


疑问句:
Have they installed the new operating system? = 他们安装了新的操作系统了吗？
When will the new equipment be installed? = 新设备什么时候安装？


否定句:
They did not install the correct drivers. = 他们没有安装正确的驱动程序。
She was not installed as the CEO because of a conflict of interest. = 由于利益冲突，她没有被任命为首席执行官。


PHRASE->
install a program = 安装程序
install an application = 安装应用程序
install a driver = 安装驱动程序
install a system = 安装系统
install an update = 安装更新
