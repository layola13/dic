interface:/ˈɪntərfeɪs/ | n. |接口；界面；交互界面 | An interface is a point where two systems, subjects, organizations, etc. meet and interact.  It can refer to a physical connection point or a shared boundary across which information is exchanged. In computing, it's a way for different programs or parts of a program to communicate. | Example: The user interface of this software is very intuitive.  这个软件的用户界面非常直观。 The computer's interface with the printer is via USB.  电脑与打印机的接口是通过USB实现的。|近义词：boundary, connection, point of contact,  interaction |反义词：separation, disconnection |名词
