downloading:/ˈdaʊnˌloʊdɪŋ/ | v. |下载 | The act of receiving data from a remote system, typically over a network, and saving it to a local device.  | Example: I'm downloading a large file, so it will take some time. 我正在下载一个大文件，所以需要一些时间。  |近义词: transferring, receiving |反义词: uploading


USAGE->
一般现在时:
I am downloading a movie. = 我正在下载一部电影。
We download music every week. = 我们每周都下载音乐。

一般过去时:
I downloaded a new game last night. = 我昨晚下载了一个新的游戏。
She downloaded all the necessary files. = 她下载了所有必需的文件。

现在完成时:
I have downloaded the latest software update. = 我已经下载了最新的软件更新。
They have downloaded several documents this morning. = 他们今天早上下载了好几份文件。

过去完成时:
I had downloaded the file before the meeting started. = 会议开始前我已经下载了文件。
She had downloaded the program before she started working on the project. = 在她开始做这个项目之前，她已经下载了程序。

将来时:
I will download the album tonight. = 我今晚会下载这张专辑。
He will download the necessary files tomorrow. = 他明天会下载必要的文件。

虚拟语气:
If I had a faster internet connection, I would download that movie faster. = 如果我的网络连接速度更快，我会更快地下载那部电影。
I wish I could download this game without any errors. = 我希望我能无错误地下载这个游戏。

被动语态:
The file is being downloaded. = 文件正在下载。
The software was downloaded successfully. = 软件成功下载。

疑问句:
Are you downloading anything right now? = 你现在正在下载什么吗？
Is the file still downloading? = 文件还在下载吗？

否定句:
I am not downloading anything at the moment. = 我目前没有下载任何东西。
He didn't download the file because of the slow internet. = 由于网络缓慢，他没能下载该文件。
