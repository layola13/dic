set-asides:/ˈsɛt əˈsaɪdz/| n. v.|拨款；留出；暂缓；搁置|A sum of money or other resources that have been allocated for a specific purpose, often to support a particular group or project.  The act of setting something aside, or the thing set aside.|例句：The government set aside $1 million for the project. 政府为该项目拨款100万美元。

The company set aside a portion of its profits for research and development. 公司将其部分利润留作研发之用。|近义词：reserve, allocate, earmark, postpone, defer|反义词：use, spend, utilize|词性: n. v.


USAGE->
一般现在时:
They set aside funds for emergencies. = 他们留出资金以备不时之需。
The committee sets aside time each month for reviewing proposals. = 委员会每月留出时间来审查提案。


一般过去时:
She set aside her work to help her friend. = 她放下工作去帮助她的朋友。
The judge set aside the verdict due to procedural errors. = 由于程序错误，法官撤销了判决。


现在完成时:
We have set aside enough money for the down payment. = 我们已经为首付存够了钱。
They have set aside their differences to work together. = 他们已经放下分歧，一起工作。


过去完成时:
He had set aside his personal feelings to make a fair decision. = 他已经放下个人感情，做出了公平的决定。
They had set aside the original plan and created a new one. = 他们已经放弃了原计划并制定了一个新的计划。


将来时:
She will set aside an hour each day for exercise. = 她每天将留出一个小时进行锻炼。
The company will set aside a portion of its profits for charitable donations. = 公司将把部分利润留作慈善捐款。


虚拟语气:
If they had set aside more money, they could have bought the house. = 如果他们当时存更多的钱，他们就能买下房子了。
I wish I had set aside more time to study. = 我希望我当时留出更多时间学习。


被动语态:
A special fund was set aside for the victims of the disaster. = 为灾难受害者设立了一个特别基金。
The meeting was set aside because of the storm. = 由于暴风雨，会议被取消了。


疑问句:
Have you set aside enough time for the presentation? = 你为演示留出足够的时间了吗？
Was money set aside for this project? = 这个项目拨款了吗？


否定句:
They didn't set aside any money for a rainy day. = 他们没有为将来留出任何钱。
The government did not set aside enough resources to complete the project. = 政府没有拨出足够的资源来完成该项目。
