photostatically:/ˌfoʊtoʊˈstætɪkli/| adv.|用影印方式地，通过影印|Adverb describing something done by means of a photostat machine.  It implies a reproduction of a document or image using photocopying technology.|The document was photostatically reproduced for distribution.|近义词：by photocopying, by photocopier|反义词：originally, manually


USAGE->
This word is an adverb, so it doesn't conjugate like verbs.  It describes the *manner* in which an action is performed.

Example 1: The blueprints were photostatically copied before being sent to the construction site. = 蓝图在被送往建筑工地之前被影印复制了。
Example 2:  The evidence was photostatically reproduced for the court proceedings. = 证据被影印复制，用于法庭程序。


PHRASE->
There are no common phrases specifically using "photostatically."  Its usage is quite straightforward and limited to describing the method of reproduction.  Phrases would involve the verb it modifies, such as:

photostatically reproduced document
photostatically copied file
photostatically created image
photostatically duplicated form
photostatically obtained evidence
