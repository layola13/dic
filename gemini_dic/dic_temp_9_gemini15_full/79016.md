gitlab:/ˈɡɪtlæb/| n. |吉特拉布（代码托管平台）|GitLab is a web-based DevOps lifecycle tool that provides a Git-repository manager providing wiki, issue-tracking and CI/CD pipeline features, using an open-source license.| Example: We use GitLab for version control and project management. 我们使用 GitLab 进行版本控制和项目管理。|近义词：GitHub, Bitbucket|反义词：无
