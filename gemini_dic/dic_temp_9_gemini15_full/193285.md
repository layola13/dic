seemed:/siːmd/| v.|似乎，好像|It appeared to be true; it gave the impression of being true.  The verb "seem" suggests an impression or appearance rather than a certainty.|The weather seemed nice. 天气看起来不错。

It seemed like a good idea at the time. 那时看起来是个好主意。|appeared, looked, appeared to be|certain, definite.|动词

USAGE->
一般现在时:
It seems that he is happy. = 他似乎很开心。
It seems like it will rain. = 看起来要下雨了。

一般过去时:
It seemed that he was happy yesterday. = 他昨天似乎很开心。
It seemed like it would rain yesterday. = 昨天看起来要下雨了。

现在完成时:
It has seemed that he has been happy. = 他似乎一直很开心。
It has seemed like it would rain all week. = 整周看起来都要下雨了。

过去完成时:
It had seemed that he was happy before the accident. = 事故前他似乎很开心。
It had seemed like it would rain before we left. = 我们离开前看起来要下雨了。

将来时:
It will seem that he is happy. = 他似乎会很开心。
It will seem like it will rain tomorrow. = 明天看起来要下雨了。


虚拟语气:
It seemed as if he were happy. = 他似乎很开心。(虚拟语气，表示非真实情况)
It seemed as though it might rain. = 看起来好像要下雨。(虚拟语气，表示可能性)


被动语态:
It is seemed that the problem was solved. = 问题似乎解决了。(被动语态很少用于seem)
(被动语态不常用，因为seem本身就是不及物动词)


疑问句:
Does it seem that he is happy? = 他看起来开心吗？
Does it seem like it will rain? = 看起来要下雨吗？


否定句:
It didn't seem that he was happy. = 他看起来并不开心。
It didn't seem like it would rain. = 看起来不会下雨。


PHRASE->
It seems to me that... = 我觉得...
It seems likely that... = 似乎有可能...
It seems unlikely that... = 似乎不大可能...
It seems to be that... = 看起来...
It would seem that... = 看来...
