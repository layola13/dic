reindexes:/ˌriːˈɪndɛksɪz/| v. |重新编制索引；重新建立索引 |To create a new index for (data). This usually involves rebuilding the index from scratch, often to improve search performance or to correct errors.  |例句：The database administrator reindexes the table nightly to maintain optimal query speed.  数据库管理员每晚重新编制索引以保持最佳查询速度。

例句：The search engine reindexes websites regularly to keep its results up-to-date. 搜索引擎定期重新编制网站索引以保持其结果最新。


近义词：re-index, rebuild index |反义词：无


词源：re-(again) + index (to create an alphabetical list of items) + -es (third-person singular present tense)
