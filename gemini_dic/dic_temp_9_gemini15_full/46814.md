dans:/dɑ̃z/ | prep. | 在…之中；在…期间 |  Indicates location or time within a larger context.  It is often used to express being inside, among, or during something. | 例句：The book is dans la bibliothèque. = 书在图书馆里。  The concert will take place dans une semaine. = 音乐会将在一个星期后举行。 | 近义词：in, among, during, within | 反义词：outside, beyond, after


USAGE->
一般现在时:
The cat is dans la maison. = 猫在房子里。
The meeting is dans la salle de conférence. = 会议在会议室里。

一般过去时:
The book was dans mon sac. = 书在我的包里。
We were dans le parc. = 我们在公园里。

现在完成时:
The child has been dans le jardin toute la journée. = 孩子一整天都在花园里。
The documents have been dans mon bureau. = 文件一直都在我的办公室里。

过去完成时:
The letter had been dans la boîte aux lettres. = 信已经在邮箱里了。
They had been dans la ville toute la semaine. = 他们整个星期都在城里。

将来时:
The party will be dans le jardin. = 派对将在花园里举行。
We will be dans la montagne demain. = 我们明天将在山里。

虚拟语气:
If the key were dans ma poche, I would open the door. = 如果钥匙在我的口袋里，我会开门。
I wish the money were dans mon compte. = 我希望钱在我的账户里。

被动语态:
The painting was placed dans le musée. = 这幅画被放在博物馆里。
The package is being kept dans un endroit sûr. = 包裹被保存在安全的地方。

疑问句:
Où est la clé ? Elle est dans la boîte. = 钥匙在哪里？它在盒子里。
Où sont les enfants ? Ils sont dans le jardin. = 孩子们在哪里？他们在花园里。

否定句:
The dog is not dans la maison. = 狗不在房子里。
The book is not dans la bibliothèque. = 书不在图书馆里。
