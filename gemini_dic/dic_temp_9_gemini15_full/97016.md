infeasible:/ɪnˈfiːzəbəl/ | adj. |不可行的，不切实际的 |  impossible to do easily or successfully; impractical.  |The plan was infeasible because of the high cost.  这个计划因为成本太高而不可行。|  impractical, unworkable, impossible | feasible, possible, practicable |


USAGE->
一般现在时:
The project is infeasible due to funding constraints. = 由于资金限制，该项目不可行。
This approach is infeasible given the current technology. = 鉴于目前的科技水平，这种方法不可行。

一般过去时:
The initial plan proved infeasible. = 最初的计划证明是不可行的。
The proposal was deemed infeasible by the committee. = 该提案被委员会认为是不可行的。

现在完成时:
They have found that the solution is infeasible. = 他们发现该解决方案不可行。
We have determined the proposed strategy to be infeasible. = 我们已经确定拟议的策略不可行。


过去完成时:
It had become clear that the project was infeasible. = 很明显，该项目已经变得不可行。
They realized that their previous approach had been infeasible. = 他们意识到他们之前的方法是不可行的。


将来时:
The new regulations will make the current system infeasible. = 新的法规将使目前的系统不可行。
The rising costs will likely make the project infeasible. = 上涨的成本可能会使该项目不可行。


虚拟语气:
If the resources were available, the project would not be infeasible. = 如果资源可用，该项目就不会不可行。
If we had more time, the task would not be infeasible. = 如果我们有更多的时间，这项任务就不会不可行。


被动语态:
The proposal was deemed infeasible by the board. = 该提案被董事会认为不可行。
The plan was found to be infeasible after careful consideration. = 在仔细考虑之后，该计划被发现不可行。


疑问句:
Is this approach really infeasible? = 这种方法真的不可行吗？
Is it infeasible to complete the project within the deadline? = 在截止日期内完成项目是否不可行？


否定句:
The solution is not infeasible; it just requires more effort. = 该解决方案并非不可行；它只是需要更多努力。
The task is not infeasible, but it will be challenging. = 这项任务并非不可行，但它将极具挑战性。


PHRASE->
infeasible solution = 不可行的解决方案
infeasible goal = 不切实际的目标
infeasible plan = 不可行的计划
prove infeasible = 证明不可行
render infeasible = 使…不可行


词源:
In- (否定前缀) + feasible (可行的)  feasible 源于拉丁语 *facere (to do, make)
