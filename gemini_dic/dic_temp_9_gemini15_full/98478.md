install:/ɪnˈstɔːl/| /ɪnˈstɑːl/ | vt. |安装，设置，安置 |To put (something) in place for use; to establish or set up (a system, etc.).|例句：The technician installed the new software on my computer. 技术员在我的电脑上安装了新软件。|例句：They installed a new security system in the building. 他们在建筑物里安装了一个新的安全系统。|近义词：set up, fix, put in |反义词：remove, uninstall, delete |动词


USAGE->
一般现在时:
They install new software every month. = 他们每月安装新软件。
We install the equipment carefully. = 我们小心地安装设备。

一般过去时:
He installed the printer yesterday. = 他昨天安装了打印机。
She installed a new app on her phone. = 她昨天在手机上安装了一个新应用程序。

现在完成时:
I have installed the game. = 我已经安装了游戏。
They have installed the security cameras. = 他们已经安装了监控摄像头。

过去完成时:
We had installed the system before the deadline. = 我们在截止日期之前已经安装好了系统。
She had installed the program before the meeting. = 她在会议之前已经安装了程序。

将来时:
They will install the air conditioning tomorrow. = 他们明天将安装空调。
He will install the new operating system next week. = 他下周将安装新的操作系统。

虚拟语气:
If I had more time, I would install that program. = 如果我有更多时间，我会安装那个程序。
It would be better if they installed a new firewall. = 如果他们安装一个新的防火墙会更好。

被动语态:
The software was installed quickly. = 软件安装很快。
The new system is being installed now. = 新系统正在安装中。

疑问句:
Did you install the drivers? = 你安装驱动程序了吗？
Have they installed the updates? = 他们安装了更新吗？

否定句:
I didn't install the update because of the error. = 我因为错误没有安装更新。
