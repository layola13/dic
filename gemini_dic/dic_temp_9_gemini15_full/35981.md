coder:/ˈkoʊdər/| n. |程序员，编码员 |A person who writes computer code.  |She's a talented coder who specializes in Python. 她是一位擅长 Python 的天才程序员。|programmer, developer, software engineer | non-coder


USAGE->
一般现在时:
He codes every day. = 他每天都编程。
She codes in Java and Python. = 她使用Java和Python编程。

一般过去时:
I coded a website last week. = 我上周编写了一个网站。
They coded the game in C++. = 他们用C++编写了这个游戏。

现在完成时:
We have coded the entire application. = 我们已经编写了整个应用程序。
She has coded for over ten years. = 她已经编程超过十年了。

过去完成时:
They had coded the module before the deadline. = 他们在截止日期前已经编写了该模块。
He had coded the program before the meeting. = 他在会议前已经编写了程序。

将来时:
I will code a new function tomorrow. = 我明天将编写一个新函数。
They will code the next version of the software. = 他们将编写软件的下一个版本。

If I were a coder, I would build my own app. = 如果我是一个程序员，我会开发我自己的应用程序。
I wish I had coded that feature earlier. = 我希望我早点编写那个功能。
