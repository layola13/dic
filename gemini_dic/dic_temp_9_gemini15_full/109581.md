leted:/ˈliːtɪd/ | adj. | 已删除的，已去除的 | Deleted refers to something that has been removed or erased, often from a digital system or document.  | Example: The email was deleted from the server.  这封邮件已从服务器中删除。  |近义词：removed, erased, expunged | 反义词：added, inserted, included |


USAGE->
一般现在时:
The file is deleted. = 文件已被删除。
The message is deleted from the inbox. = 消息已从收件箱中删除。


一般过去时:
I deleted the file yesterday. = 我昨天删除了文件。
She deleted the email before reading it. = 她没读就删掉了邮件。


现在完成时:
He has deleted all the old photos. = 他已经删除了所有旧照片。
They have deleted their accounts. = 他们已经删除了他们的账户。


过去完成时:
I had deleted the document before the meeting. = 我在会议之前已经删除了文档。
She had deleted the message before I could see it. = 我还没看到她就删掉了信息。


将来时:
I will delete the files tomorrow. = 我明天将删除这些文件。
They will delete the account after the trial period. = 试用期结束后，他们将删除账户。


虚拟语气:
If I deleted that file, I would lose my work. = 如果我删除了那个文件，我会丢失我的工作。
I wish I hadn't deleted that photo. = 我真希望我没有删除那张照片。


被动语态:
The file was deleted by mistake. = 文件被错误地删除了。
The messages are being deleted automatically. = 消息正在自动删除。


疑问句:
Was the file deleted? = 文件被删除了吗？
Have the emails been deleted? = 邮件被删除了吗？


否定句:
The file was not deleted. = 文件没有被删除。
I didn't delete the file. = 我没有删除文件。
