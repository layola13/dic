externs: /ˈɛkstɜːrnz/ | n. |外部程序，外部变量| External procedures or variables, typically in programming contexts, that are declared outside the current scope or module but are accessible within it.  They allow modularity and code reuse. | 例句：The program uses externs to access functions defined in a separate library. = 该程序使用外部程序来访问在单独的库中定义的函数。

The externs are declared in a header file for use by multiple source files.= 外部变量在头文件中声明，以便多个源文件使用。


近义词：external references, external declarations|反义词：internals, locals|名词
