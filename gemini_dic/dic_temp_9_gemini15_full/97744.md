initializer:/ɪˈnɪʃəlaɪzər/ | n. | 初始化器，初始化程序 | A piece of code or data that sets the initial values of variables or objects.  It prepares an object for use by giving it starting values. | Example:  The initializer sets the default color of the button to blue. 初始化器将按钮的默认颜色设置为蓝色。|近义词：initialiser (British spelling), starter, setter | 反义词：finalizer, terminator


USAGE->
This word is primarily a noun and doesn't conjugate like a verb.  Therefore, typical verb tenses are not applicable.  Instead, here are examples showing its usage in different sentence structures:

Declarative Sentence:
The program uses a complex initializer for its main data structure.  程序使用一个复杂初始化器来初始化其主要数据结构。


Interrogative Sentence:
What type of initializer is best for this array?  哪种初始化器最适合这个数组？


Exclamatory Sentence:
What a clever initializer! 多么巧妙的初始化器！


Relative Clause:
The class contains an initializer that sets the initial state. 类包含一个设置初始状态的初始化器。


Comparative Sentence:
This initializer is more efficient than the previous one. 这个初始化器比之前的更有效率。


PHRASE->
class initializer: 类初始化器
