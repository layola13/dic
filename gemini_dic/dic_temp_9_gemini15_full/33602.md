cin:/sɪn/| n. |标准输入流|C++中的标准输入流对象，用于从标准输入设备（通常是键盘）读取数据。它是一个预定义的对象，不需要显式声明。|Example: `cin >> myVariable;`  (This reads input from the keyboard and stores it in `myVariable`)  = 这从键盘读取输入并将其存储在myVariable中。|近义词:stdin (standard input)|反义词:cout (standard output)|名词


USAGE->
(由于cin本身不是一个动词，因此无法按照常规动词时态进行分类。其用法主要在于输入操作。)

Example 1:
`int age; cin >> age;` =  声明一个整型变量age，并从标准输入流中读取一个整数赋值给age。

Example 2:
`string name; cin >> name;` = 声明一个字符串变量name，并从标准输入流中读取一个字符串赋值给name。


PHRASE->
(cin本身不构成词组，其用法通常与输入操作符>>结合使用。)

Example 1:
`cin >> x >> y;` = 从标准输入流中读取两个值，分别赋给变量x和y。

Example 2:
`if (cin >> num) { ... }` = 检查标准输入流是否成功读取一个数字到变量num中。如果成功，则执行{...}中的代码。
