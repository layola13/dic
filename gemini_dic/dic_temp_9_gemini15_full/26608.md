caches:/kæʃɪz/| n. |缓存；隐藏处；贮藏处 |Caches are temporary storage areas used by computers to speed up data access. 缓存是计算机用来加快数据访问速度的临时存储区域。|近义词：store, hoard, reserve |反义词：discard, delete, remove |名词


USAGE->
一般现在时:
The browser caches images to load web pages faster. = 浏览器缓存图像以更快地加载网页。
My computer caches frequently accessed files. = 我的电脑缓存经常访问的文件。


一般过去时:
The system cached the data before processing it. = 系统在处理数据之前缓存了数据。
The program cached the results for later use. = 该程序缓存了结果以供以后使用。


现在完成时:
The website has cached the page for improved performance. = 该网站已缓存页面以提高性能。
The application has cached user preferences. = 该应用程序已缓存用户偏好设置。


过去完成时:
The server had cached the files before the request. = 服务器在请求之前已缓存文件。
The application had cached the images before displaying them. = 该应用程序在显示图像之前已缓存图像。


将来时:

虚拟语气:

被动语态:
