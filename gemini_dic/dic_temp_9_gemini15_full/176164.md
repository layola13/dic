USAGE->
一般现在时:
There's a book on the table. = 桌子上有一本书。
There are many stars in the sky. = 天空中有很多星星。

一般过去时:
There was a party last night. = 昨晚有一个派对。
There were many people at the concert. = 音乐会上有很多的人。

现在完成时:
There have been many changes. = 已经发生了许多变化。
There has been an accident. = 发生了一起事故。

过去完成时:
There had been several attempts before. = 之前已经有过几次尝试。
There had been a storm the night before. = 前一天晚上下了一场暴风雨。

将来时:
There will be a meeting tomorrow. = 明天将会有一个会议。
There will be sunshine tomorrow. = 明天会有阳光。

虚拟语气:
If there were a chance, I would try. = 如果有机会，我会尝试。
I wish there were more time. = 我希望有更多的时间。

被动语态:
There was a decision made. = 做出了一个决定。(被动语态中"there be"结构较少见，此例为特例)
There is a problem to be solved. = 有一个问题需要解决。(被动语态中"there be"结构较少见，此例为特例)

疑问句:
Is there anything I can do? = 我能做些什么吗？
Are there any questions? = 有什么问题吗？

否定句:
There isn't enough time. = 时间不够。
There weren't any tickets left. = 没有剩余的票了。


PHRASE->
There you go! = 好了！/给你！
There it is! = 看！/找到了！
There's no way! = 没门！/不可能！
There's more to it than that. = 这其中还有更多的东西。
There's nothing to it. = 这很容易。
