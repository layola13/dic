jerry's:/ˈdʒɛriz/| n.|杰里的（所有格）|Jerry's belongings were scattered everywhere.  杰里的东西散落得到处都是。|近义词：Jerry's possessions|反义词：无


USAGE->
(Jerry's is a possessive noun and doesn't have verb conjugations.  The following examples show how it's used in different sentence structures.)

一般现在时:
Jerry's car is red. = 杰里的车是红色的。
This is Jerry's house. = 这是杰里的房子。


一般过去时:
Jerry's dog ran away yesterday. = 杰里的狗昨天跑掉了。
We visited Jerry's farm last week. = 我们上周参观了杰里的农场。


现在完成时:
Jerry's work has been completed. = 杰里的工作已经完成了。
I've seen Jerry's new painting. = 我看过杰里的新画了。


过去完成时:
Jerry's project had already begun before I arrived. = 在我到达之前，杰里的项目已经开始了。
We learned that Jerry's departure had been unexpected. = 我们获悉杰里的离开是出乎意料的。


将来时:
Jerry's presentation will be tomorrow. = 杰里的演示将在明天进行。
We believe Jerry's efforts will pay off. = 我们相信杰里的努力会得到回报。


虚拟语气:
If Jerry's plan worked, we would be very happy. = 如果杰里的计划成功了，我们会非常高兴。
I wish Jerry's advice had been heeded. = 我真希望杰里的建议被采纳了。


被动语态:
Jerry's suggestion was rejected. = 杰里的建议被否决了。
Jerry's book is being reviewed. = 杰里的书正在被审阅。


疑问句:
Is this Jerry's phone? = 这是杰里的手机吗？
Whose is this?  Is it Jerry's? = 这是谁的？是杰里的吗？


否定句:
That's not Jerry's coat. = 那不是杰里的外套。
This isn't Jerry's handwriting. = 这不是杰里的笔迹。
