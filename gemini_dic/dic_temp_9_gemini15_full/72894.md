following:/ˈfɒləʊɪŋ/ /ˈfɑːloʊɪŋ/| adj., n., prep.|下列的；接着的；跟随；跟随者|The following is a list of the participants. 下面是参与者的名单。  Following the accident, the police arrived. 事故发生后，警察来了。|近义词：subsequent, succeeding, next, after|反义词：preceding, prior


USAGE->
一般现在时:
The following instructions are very important. = 下面的指示非常重要。
What are the following steps? = 接下来的步骤是什么？


一般过去时:
The following day, it rained heavily. = 第二天，下起了大雨。
Following the meeting, we had lunch. = 会议结束后，我们吃了午饭。


现在完成时:
Following all the instructions, I have completed the task. = 按照所有的指示，我已经完成了任务。
Everything has gone smoothly following the initial hiccups. = 在最初的几次小故障之后，一切都很顺利。


过去完成时:
Following the completion of the project, we had a celebration. = 项目完成后，我们举行了一次庆祝活动。
Following the announcement, many people had already left. = 公告发布后，许多人已经离开了。


将来时:
Following the presentation, we will have a Q&A session. = 演示结束后，我们将有一个问答环节。
Following the completion of the training, you will be assigned to a project. = 培训结束后，你将被分配到一个项目。


虚拟语气:
Following the instructions carefully, we would have avoided the mistake. = 如果仔细按照指示操作，我们就不会犯这个错误。
If I had been following the news, I would have known about the incident. = 如果我一直关注新闻，我就知道这件事了。


被动语态:
The following information is being updated regularly. = 下面的信息正在定期更新。
The case was closed following a thorough investigation. = 在彻底调查后，案件被结案。


疑问句:
What is following the introduction? = 介绍之后是什么？
What is following the main course? = 主菜之后是什么？


否定句:
The following information is not available. = 下面的信息不可用。
The instructions were not followed correctly. = 指示没有被正确执行。
