dropdown:/ˈdrɒpdaʊn/| n. |下拉菜单，下拉列表|A dropdown is a graphical user interface element that allows the user to select one option from a list of choices.  It typically appears as a box with a currently selected item displayed and an arrow that, when clicked, reveals the full list of options.|Example:  The website uses a dropdown menu to select your country.  这个网站使用下拉菜单来选择你的国家。|近义词：drop-down list, pull-down menu |反义词：None  |名词
