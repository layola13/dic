serverless:/ˈsɜːrvərləs/| adj.|无服务器的；无服务器计算的|A serverless architecture is a cloud computing execution model where the cloud provider dynamically manages the allocation of computing resources.  In a serverless system, developers don't manage servers. Instead, they write and deploy code, and the cloud provider handles the underlying infrastructure, including server provisioning, scaling, and maintenance.|Example:  Serverless functions are becoming increasingly popular for building scalable and cost-effective applications.  无服务器函数正日益流行，用于构建可扩展且经济高效的应用程序。|近义词：cloud-based (in some contexts), function-as-a-service (FaaS)|反义词：server-based, traditional
