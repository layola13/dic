event:/ɪˈvɛnt/ | n. |事件，大事，比赛项目 | An event is something that happens, especially something important.  |Example: The party was a memorable event.  这次派对是一次难忘的事件。

The earthquake was a major event.  那场地震是一件大事。|近义词：occurrence, happening, incident |反义词：non-event


USAGE->
一般现在时:
This event happens every year. = 这个活动每年都会举行。
What events are planned for the weekend? = 周末计划了哪些活动？


一般过去时:
The event took place last night. = 活动昨晚举行。
What event did you attend yesterday? = 你昨天参加了什么活动？


现在完成时:
Many significant events have occurred this century. = 本世纪发生过许多重大的事件。
Have you been to any interesting events recently? = 你最近参加过什么有趣的活动吗？


过去完成时:
By the time the event started, everyone had arrived. = 活动开始时，大家都到了。
What had happened before the event started? = 活动开始之前发生了什么？


将来时:
The event will be held next month. = 活动将于下个月举行。
What events will take place during the festival? = 节日期间将举行哪些活动？


虚拟语气:
If the event hadn't been canceled, I would have gone. = 如果活动没有取消，我就去了。
I wish I had attended that event. = 我真希望我参加了那个活动。


被动语态:
The event was organized by the committee. = 这个活动是由委员会组织的。
The event is being filmed for television. = 这个活动正在被拍摄成电视节目。


疑问句:
What event are you referring to? = 你指的是哪个事件？
What was the most important event of the year? = 今年最重要的事件是什么？


否定句:
The event did not go as planned. = 活动没有按计划进行。
There was no significant event that day. = 那天没有发生什么重大的事件。
