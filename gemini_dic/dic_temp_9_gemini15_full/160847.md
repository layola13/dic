possible:/ˈpɒsəbl/ /ˈpɑːsəbl/| adj.|可能的，也许的，可行的| capable of being done, happening, or existing; likely; feasible.  It expresses the potential for something to occur or exist. |例句：It's possible to learn a new language in a year. = 一年内学会一门外语是有可能的。

例句：Is it possible to borrow your car? = 我可以借你的车吗？


近义词：likely, probable, feasible, potential|反义词：impossible, improbable, infeasible


USAGE->
一般现在时:
It is possible to finish the work today. = 今天完成这项工作是有可能的。
It's possible that he will call later. = 他可能晚些时候打电话。


一般过去时:
It was possible to see the stars last night. = 昨晚可以看到星星。
It was possible that she had already left. = 她可能已经离开了。


现在完成时:
It has been possible to contact him by email. = 一直可以通过电子邮件联系他。
It's possible that she has already arrived. = 她可能已经到了。


过去完成时:
It had been possible to avoid the accident. = 本可以避免这次事故。
It was possible that they had already eaten. = 他们可能已经吃过了。


将来时:
It will be possible to travel to Mars one day. = 总有一天能够去火星旅行。
It is possible that it will rain tomorrow. = 明天可能下雨。


虚拟语气:
It would be possible to solve the problem if we had more time. = 如果我们有更多的时间，就能解决这个问题。
It's possible that she would have agreed if you had asked nicely. = 如果你客气点问，她可能会答应。


被动语态:
The problem is possible to be solved. = 这个问题是可以解决的。(较不自然，主动语态更常用)
The solution is possible to be implemented. = 这个方案是可以实施的。(较不自然，主动语态更常用)


疑问句:
Is it possible to get a refund? = 可以退款吗？
Is it possible that you made a mistake? = 你可能犯了个错误吗？


否定句:
It's not possible to complete the project on time. = 不可能按时完成这个项目。
It's not possible that he didn't know. = 他不可能不知道。


PHRASE->
as possible = 尽可能地
every possible = 各种可能的
make possible = 使成为可能
as far as possible = 尽可能地
the possible = 可能的
