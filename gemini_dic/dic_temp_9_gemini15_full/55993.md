dns:/ˈdiːɛnˈɛs/| n. |域名系统|Domain Name System; a hierarchical decentralized naming system for computers, services, or any resource connected to the Internet or a private network.|例句：The DNS server translates domain names into IP addresses.  DNS服务器将域名转换为IP地址。|近义词：domain name resolution|反义词：无


USAGE->
DNS本身作为名词，没有动词形式，因此不适用各种时态变化。  以下例句展示DNS在不同语境下的用法：

例句1：Our website is down because of a DNS problem. 我们的网站因为DNS问题而瘫痪了。
例句2：The DNS records need to be updated to point to the new server. DNS记录需要更新以指向新服务器。


例句3：I configured the DNS settings on my router. 我在我的路由器上配置了DNS设置。
例句4:  A faulty DNS server can cause slow website loading times.  有故障的DNS服务器会导致网站加载缓慢。


例句5:  He used a public DNS server for faster browsing. 他使用公共DNS服务器来加快浏览速度。
例句6: The company uses its own internal DNS server. 这家公司使用自己的内部DNS服务器。


PHRASE->
DNS server: DNS服务器
DNS record: DNS记录
DNS lookup: DNS查询
DNS caching: DNS缓存
DNS propagation: DNS传播
