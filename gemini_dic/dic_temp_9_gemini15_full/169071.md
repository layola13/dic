programming:/ˈproʊɡræmɪŋ/ | n. |程序设计，编程 |The act of writing computer programs; the process of creating a set of instructions that a computer can follow to perform a specific task.  |Example:  Learning programming is essential for many tech jobs. 学习编程对于许多科技工作至关重要。 |近义词：coding, software development |反义词：N/A


USAGE->
一般现在时:
He programs in Python. = 他用Python编程。
She programs every day. = 她每天都编程。

一般过去时:
I programmed a game last week. = 我上周编写了一个游戏。
They programmed the robot to move. = 他们编写程序让机器人移动。

现在完成时:
We have programmed the website. = 我们已经编写了网站程序。
She has programmed many apps. = 她已经编写了许多应用程序。

过去完成时:
He had programmed the software before the deadline. = 他在截止日期前已经编写了软件。
They had programmed the system for years. = 他们已经为该系统编程多年了。

将来时:
I will program a new application next month. = 我下个月将编写一个新的应用程序。
They will program the AI to learn. = 他们将编写程序让AI学习。

虚拟语气:
If I knew how to program, I would build a website. = 如果我会编程，我会建立一个网站。
I wish I had programmed that feature earlier. = 我真希望我早点编写那个功能。

被动语态:
The software was programmed by a team of experts. = 该软件由一个专家团队编写。
The robot is being programmed to follow instructions. = 机器人正在被编程以遵循指令。

疑问句:
Do you program in Java? = 你用Java编程吗？
Can you program this function? = 你能编写这个函数吗？

否定句:

He didn't program the game correctly. = 他没有正确地编写游戏程序。
