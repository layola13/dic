scoped:/skoʊpt/| adj.| 有范围的；限定范围的；作用域的 |Having a limited or defined scope or application; relating to or determined by a scope.|The project has a clearly defined scoped budget. 这个项目有一个明确限定范围的预算。|近义词：limited, restricted, confined |反义词：unlimited, unrestricted, broad


USAGE->
一般现在时:
The project is scoped to meet specific client needs.  =  项目范围限定为满足客户的特定需求。
This feature is scoped for the next release. = 这个功能限定在下一个版本中发布。


一般过去时:
The initial scope was too broad, so we rescoped the project. = 最初的范围太广，所以我们重新界定了项目范围。
The team carefully scoped the problem before starting the solution. = 团队在开始解决方案之前仔细界定了问题的范围。


现在完成时:
We have scoped out the major features for the new software. = 我们已经规划出了新软件的主要功能。
The developers have scoped the entire project for efficiency. = 开发人员为了效率已经对整个项目进行了范围界定。


过去完成时:
They had scoped the project before the meeting, leading to a productive discussion. = 他们在会议之前已经界定了项目范围，从而进行了富有成效的讨论。
The team had already scoped the tasks before assigning them to individuals. = 团队在将任务分配给个人之前就已经确定了任务范围。


将来时:
We will scope the project next week. = 我们下周将确定项目范围。
The requirements will be scoped by the end of the month. = 到月底将确定需求范围。


虚拟语气:
If the project were better scoped, it would be completed on time. = 如果项目范围界定得更好，它就能按时完成。
I wish the requirements had been more clearly scoped at the beginning. = 我希望一开始需求范围能更清晰地界定。


被动语态:
The project was scoped by the project manager. = 项目范围由项目经理确定。
The budget was carefully scoped to avoid overspending. = 预算范围被仔细限定以避免超支。


疑问句:
How is the project scoped? = 项目范围是如何界定的？
What aspects of the project are not scoped? = 项目的哪些方面没有包含在范围内？


否定句:
The project is not adequately scoped. = 项目范围界定不足。
The features were not properly scoped in the initial design. = 在最初的设计中，功能没有得到适当的范围界定。


PHRASE->
scoped out =  规划；勘察
in scope = 在范围内
out of scope = 超出范围
fully scoped = 完全限定范围
scoped definition = 范围定义
