circumscriptively:/ˌsɜːrkəmˈskrɪptɪvli/ | adv. |  限制地；限定地；间接地 | In a manner that is restricted or limited; indirectly.  The author described the events circumscriptively, avoiding explicit details.  作者间接地描述了这些事件，避免了明确的细节。 |  indirectly, restrictively, vaguely | openly, directly, explicitly | 副词


USAGE->
This word is an adverb, so it doesn't have tenses like verbs.  Instead, it modifies verbs, adjectives, or other adverbs.

Example 1 (modifying a verb):
He spoke circumscriptively about the sensitive topic. = 他含糊其辞地谈论了这个敏感话题。

Example 2 (modifying an adjective):
The evidence was only circumscriptively presented. = 证据只是间接地呈现。


PHRASE->
There aren't common idiomatic phrases directly using "circumscriptively."  The word itself is used to describe the *manner* of something, not to form common phrases.  Phrases would need to be built around the word.  For example:

"to speak circumscriptively" = 含糊其辞地说话
"a circumscriptively worded statement" = 一份措辞含糊的声明
"to act circumscriptively" = 行事谨慎
"to describe something circumscriptively" = 间接地描述某事
"a circumscriptively defined area" = 一个界定模糊的区域
