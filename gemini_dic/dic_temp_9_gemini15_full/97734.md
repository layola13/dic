initialise:/ɪˈnɪʃəlaɪz/| /ɪˈnɪʃəlʌɪz/ | vt.|初始化，启动|To set up (something, such as a program or device) at the beginning of operation; to give an initial value to (a variable).  It often involves setting variables to default values or preparing resources. |例句：You need to initialise the printer before you can use it. 你需要先初始化打印机才能使用它。

例句：The program automatically initialises the memory when it starts. 程序启动时会自动初始化内存。


近义词：initiate, start up, begin, activate |反义词：terminate, end, close, deactivate |动词
