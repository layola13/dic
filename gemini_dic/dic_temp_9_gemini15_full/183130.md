repository: /rɪˈpɒzɪtɔːri/ /rɪˈpɑːzɪtɔːri/ | n. | 仓库；储藏室；知识库；代码库 | A place, especially a building, for storing things.  It can also refer to a digital store of information, such as a database or a code repository. |  例句：The museum's repository holds thousands of historical artifacts.  博物馆的仓库收藏着数千件历史文物。

例句：GitHub is a popular repository for software developers. GitHub 是软件开发者常用的代码库。


近义词：storehouse, archive, depot, database, collection


反义词：disposal, destruction


词性：名词


USAGE->

The museum uses its repository to store artifacts. = 博物馆用它的仓库来储存文物。
This repository contains valuable research data. = 这个知识库包含宝贵的科研数据。
