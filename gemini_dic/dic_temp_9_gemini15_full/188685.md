run: /rʌn/ /rʌn/ | v., n. | 跑，运行，经营 | To move swiftly on foot; to operate or function; to manage or control a business. | 例句：The dog ran across the field.  狗跑过田野。  The program is running smoothly. 程序运行流畅。 | 近义词：jog, sprint, operate, manage | 反义词：walk, stop, stand still


USAGE->
一般现在时:
He runs every morning. = 他每天早上跑步。
The company runs efficiently. = 公司运作效率很高。

一般过去时:
She ran a marathon last year. = 她去年跑了一场马拉松。
The machine ran for hours without stopping. = 机器连续运行了数小时而没有停止。

现在完成时:
I have run five miles today. = 我今天跑了五英里。
They have run the business for twenty years. = 他们经营这家公司已经二十年了。

过去完成时:
He had run a mile before he stopped. = 他跑了一英里之后才停下来。
The factory had run at full capacity for a week. = 工厂已经满负荷运转了一周。

将来时:
I will run in the race next week. = 我下周将参加比赛。
The project will run for another six months. = 这个项目将再持续六个月。

虚拟语气:
If I ran faster, I would win the race. = 如果我跑得更快，我就能赢得比赛。
I wish I had run more often. = 我希望我以前经常跑步。

被动语态:
The company is run by a CEO. = 这家公司由一位首席执行官管理。
The marathon was run in difficult conditions. = 马拉松比赛在艰难的条件下进行。

疑问句:
Does he run every day? = 他每天都跑步吗？
How long has the program been running? = 程序运行了多久？

否定句:
I didn't run yesterday. = 我昨天没跑步。
The engine wasn't running properly. = 发动机运转不正常。


PHRASE->
run out (of): 用完，耗尽  (The milk has run out.)
run into: 偶然遇见 (I ran into an old friend.)
run away: 逃跑 (The thief ran away.)
run down: 撞倒；(机器)停止运转 (The car ran down the pedestrian. The battery ran down.)
run over:  碾过；匆匆过一遍 (The car ran over the cat. Let's run over the plan again.)
