multi-threading:/ˌmʌltiˈθredɪŋ/| n. |多线程|The ability of a computer program or operating system to execute multiple threads concurrently.  This allows multiple tasks to be performed seemingly at the same time, improving efficiency and responsiveness.|Example:  Multi-threading is crucial for modern applications to handle multiple user requests simultaneously.  多线程对于现代应用程序同时处理多个用户请求至关重要。|近义词：concurrent processing, parallel processing|反义词：single-threading|名词
