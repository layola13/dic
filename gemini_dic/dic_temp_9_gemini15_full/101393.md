ip:/ˈaɪpiː/| n. |互联网协议，IP地址|Internet Protocol, a set of rules governing the format of data sent via the internet or local network.|Example: My computer's IP address is 192.168.1.100. 我的电脑的IP地址是192.168.1.100。|近义词：internet address, network address|反义词：无


USAGE->
一般现在时:
What is your IP address? = 你的IP地址是多少？
What IP address does this server use? = 这台服务器使用哪个IP地址？


一般过去时:
What was the server's IP address yesterday? = 昨天服务器的IP地址是什么？
What IP address did you use last week? = 你上周用的是哪个IP地址？


现在完成时:
What IP addresses have you used? = 你用过哪些IP地址？
What has changed in the IP address configuration? = IP地址配置发生了什么变化？


过去完成时:

What IP address had the server used before the failure? = 服务器故障前使用的是哪个IP地址？
