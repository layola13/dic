exe:/ˈɛɡzɪ/ | n. | 可执行文件 | An executable file is a computer file that contains instructions that can be executed directly by a computer's processor.  These files typically have extensions like .exe (on Windows), .com, or .app.  They are the result of compiling or assembling source code written in programming languages. | Example:  Double-clicking an .exe file will launch the associated program.  双击一个.exe文件将会启动相关的程序。|近义词：application, program | 反义词:source code, script


USAGE->
exe本身作为名词，不具备动词属性，因此无法进行时态变化。它主要用于描述一种文件类型。  以下例句展示了exe文件在不同语境下的用法：


Example 1: I downloaded the game's .exe file and installed it. 我下载了游戏的.exe文件并安装了它。
Example 2: The antivirus software detected a suspicious .exe file. 反病毒软件检测到一个可疑的.exe文件。
Example 3:  Make sure you run the .exe file as administrator. 确保你以管理员身份运行.exe文件。
Example 4: The installation package includes several .exe files. 安装包包含几个.exe文件。
Example 5: He tried to open the corrupted .exe file, but it wouldn't run. 他试图打开损坏的.exe文件，但是它无法运行。


PHRASE->
exe文件本身很少与其他词构成固定短语，其用法主要在于与其他名词或动词搭配，例如：


Example 1:  executable file 可执行文件
Example 2:  run the .exe file 运行.exe文件
Example 3:  delete the .exe file 删除.exe文件
Example 4:  install the .exe file 安装.exe文件
Example 5:  virus in the .exe file .exe文件中的病毒
