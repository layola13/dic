if:/ɪf/| conj.|如果，假如|It is a conjunction used to introduce a conditional clause, indicating a condition upon which something else depends.|例句：If it rains, we will stay home.  如果下雨，我们就待在家里。|例句：If I win the lottery, I will travel the world.如果我中了彩票，我将环游世界。|近义词：provided that, supposing, in case|反义词:unless


USAGE->
一般现在时:
If it rains, I will stay home. = 如果下雨，我就待在家里。
If she studies hard, she will pass the exam. = 如果她努力学习，她就会通过考试。

一般过去时:
If it rained, we stayed home. = 如果下雨，我们就待在家里。
If he worked hard, he succeeded. = 如果他努力工作，他就成功了。

现在完成时:
If it has rained, the ground will be wet. = 如果下雨了，地面就会湿。
If she has finished her work, she can go home. = 如果她完成了工作，她就可以回家了。

过去完成时:
If it had rained, the plants would have grown. = 如果下雨了，植物就会生长。
If he had studied harder, he would have passed the exam. = 如果他学习更努力，他就能通过考试了。

将来时:
If it will rain tomorrow, we will cancel the picnic. = 如果明天下雨，我们将取消野餐。
If she will be free, she will come to the party. = 如果她有空，她就会来参加聚会。

虚拟语气:
If I were you, I would not do that. = 如果我是你，我不会那样做。
If she had studied harder, she would have passed the exam. = 如果她学习更努力，她就能通过考试了。(与过去事实相反)

被动语态:
If the project is completed on time, we will get a bonus. = 如果项目按时完成，我们将获得奖金。
If the problem is solved quickly, the damage will be minimized. = 如果问题迅速得到解决，损失将会最小化。

疑问句:
If it is raining, why are you outside? = 如果正在下雨，你为什么在外面？
If she is coming, when will she arrive? = 如果她要来，她什么时候到？

否定句:
If it doesn't rain, we will go for a walk. = 如果不下雨，我们就去散步。
If she isn't busy, she will help us. = 如果她不忙，她会帮助我们的。


PHRASE->
if only = 但愿；要是…就好了
even if = 即使；纵然
as if = 好像；仿佛
if anything = 如果有什么不同的话
if necessary = 如果必要的话
