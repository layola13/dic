code:/koʊd/| n. |代码，密码，编码 |A code is a system of words, letters, figures, or symbols used to represent others, especially for the purposes of secrecy.  |例句：The secret code was finally broken.  秘密代码最终被破译了。|例句：The programmer wrote clean and efficient code. 程序员编写了简洁高效的代码。|近义词：cipher, cypher, encryption |反义词：plaintext, decode
