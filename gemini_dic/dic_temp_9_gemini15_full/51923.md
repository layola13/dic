/dɪz/ | pron. | 这；这些 |  This refers to something near the speaker.  It can be singular or plural. | This is my car. = 这是我的车。| Those | That |  pronoun


USAGE->
一般现在时:
This is a book. = 这是一本书。
These are apples. = 这些是苹果。

一般过去时:
This was a good movie. = 这是一部好电影。
These were difficult problems. = 这些是难题。

现在完成时:
This has been a long day. = 今天过得很长。
These have been helpful suggestions. = 这些都是有帮助的建议。

过去完成时:
This had already happened. = 这已经发生了。
These had been discussed before. = 这些之前已经被讨论过了。

将来时:
This will be exciting. = 这将会令人兴奋。
These will be finished soon. = 这些很快就会完成。

虚拟语气:
If this were true, I would be surprised. = 如果这是真的，我会很惊讶。
If these were available, I would buy them. = 如果这些货有货，我会买下它们。

被动语态:
This is being considered. = 正在考虑这个。
These are being repaired. = 这些正在被修理。

疑问句:
Is this yours? = 这是你的吗？
Are these the keys? = 这些是钥匙吗？

否定句:
This isn't mine. = 这不是我的。
These aren't the right ones. = 这些不是正确的。


PHRASE->
This is it! = 就这样！/ 就是它了！
This way, please. = 请这边走。
This much is certain. = 这一点是肯定的。
This time tomorrow... = 明天这个时候...
Think of this. = 想想这个。
