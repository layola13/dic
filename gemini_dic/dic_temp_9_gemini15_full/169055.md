programming:/ˈproʊɡræmɪŋ/ | n. |程序设计，编程|The act of writing computer programs.  It involves designing, writing, testing, debugging, and maintaining the source code of computer programs.|例句：She's learning programming to become a software engineer. = 她正在学习编程以成为一名软件工程师。|例句：The programming of this software is quite complex. = 这个软件的编程相当复杂。|近义词：coding, software development|反义词：N/A
