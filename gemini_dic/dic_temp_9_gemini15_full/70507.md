file:/faɪl/| n.|文件，档案|A file is a container for storing data.|例句：I saved the document as a PDF file.  我将文档保存为PDF文件。|近义词：document, record|反义词:None


USAGE->
一般现在时:
The program creates a new file. = 程序创建一个新文件。
This file contains important information. = 此文件包含重要信息。

一般过去时:
I deleted the file yesterday. = 我昨天删除了该文件。
She opened the file and read its contents. = 她打开文件并阅读了其内容。

现在完成时:
I have saved the file to the cloud. = 我已将文件保存到云端。
Have you updated the file? = 你更新了文件了吗？

过去完成时:
I had already closed the file before the computer crashed. = 电脑崩溃之前我已经关闭了文件。
She had saved the file several times before submitting it. = 她在提交之前已经保存了几次文件。

将来时:
I will send you the file tomorrow. = 我明天会把文件发给你。
We will need to create a new file for this project. = 我们将需要为这个项目创建一个新文件。

虚拟语气:
If I had the file, I would send it to you. = 如果我有这个文件，我会发给你。
I wish I had backed up the file. = 我真希望我备份了这个文件。

被动语态:
The file was deleted by mistake. = 文件被错误地删除了。
The file is being processed by the server. = 文件正在由服务器处理。

疑问句:
What type of file is this? = 这是什么类型的文件？
Where is the file located? = 文件在哪里？

否定句:
I don't have the file you requested. = 我没有你请求的文件。
He didn't open the file. = 他没有打开文件。
