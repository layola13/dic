os's:/ˈɒzɪz/|abbr.|操作系统|operating system的缩写，指管理和控制计算机硬件及软件资源的程序|例句：Windows and macOS are popular operating systems.  Windows 和 macOS 是流行的操作系统。|近义词：OS, operating system|反义词:无


USAGE->
一般现在时:
The OS manages the computer's memory. = 操作系统管理计算机的内存。
My OS is running smoothly. = 我的操作系统运行流畅。


一般过去时:
The old OS crashed frequently. = 旧的操作系统经常崩溃。
We upgraded the OS last week. = 我们上周升级了操作系统。


现在完成时:
I have updated my OS several times this year. = 今年我已经更新了几次操作系统。
The company has installed a new OS on all its computers. = 公司已在其所有计算机上安装了新的操作系统。


过去完成时:
The application wouldn't run because the OS had not been updated. = 应用程序无法运行，因为操作系统尚未更新。
Before the crash, the OS had been running perfectly. = 在崩溃之前，操作系统运行得完美无缺。


将来时:
We will install a new OS next month. = 我们下个月将安装一个新的操作系统。
The company will upgrade its OS to the latest version. = 公司将把操作系统升级到最新版本。


虚拟语气:
If the OS were more stable, we wouldn't have so many problems. = 如果操作系统更稳定，我们就不会有这么多问题。
I wish the OS had more features. = 我希望操作系统有更多功能。


被动语态:

The computer's settings were changed by the OS. = 计算机的设置被操作系统更改了。


疑问句:
