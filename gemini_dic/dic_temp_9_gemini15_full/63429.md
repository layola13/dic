entre:/ˈɑ̃trə/ | prep. |在…之间；在…之中|Between; among; amidst.  The restaurant is situated entre deux rues. 这家餐厅位于两条街之间。|近义词：between, among, amidst |反义词：outside, beyond


USAGE->
一般现在时:
The book lies entre les pages. = 书夹在书页之间。
He sits entre deux chaises. = 他坐在两把椅子之间。


一般过去时:
The ball rolled entre les arbres. = 球滚到了树木之间。
She hid entre les rideaux. = 她躲在窗帘后面。


现在完成时:
The secret has remained entre nous. = 这个秘密一直被我们保守着。
The agreement has been kept entre les deux parties. = 协议一直被双方遵守着。


过去完成时:
The letter had been placed entre les pages du livre. = 信被放在书页之间。
The treasure had been hidden entre les rochers. = 宝藏被藏在岩石之间。


将来时:
The meeting will be held entre les deux chefs d'état. = 会议将在两位国家元首之间举行。
The decision will be made entre les membres du jury. = 决定将由陪审团成员做出。


虚拟语气:
If the agreement were kept entre nous, it would be better. = 如果协议在我们之间保守，那就更好了。
I wish the truth were known only entre nous. = 我希望只有我们才知道真相。


被动语态:
The secret was kept entre les deux amis. = 秘密被这两个朋友保守着。
The decision was made entre les membres du conseil. = 决定是由理事会成员做出的。


疑问句:
Est-ce que le secret est entre nous? = 这个秘密在我们之间吗？
Où est la différence entre les deux? = 两者之间有什么区别？


否定句:
Le secret n'est pas entre nous. = 这个秘密不在我们之间。
Il n'y a pas de différence entre les deux. = 两者之间没有区别。


PHRASE->
entre autres = among other things
entre temps = meanwhile
entre nous = between ourselves
entre parenthèses = parenthetically
entre chien et loup = twilight
