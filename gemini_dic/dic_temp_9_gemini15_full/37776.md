compatibles:/kəmˈpætəblz/ | n. |兼容的；可兼容的物品 |Things that are compatible; things that can be used together without problems.  |Example:  The new software is compatible with all versions of Windows. 新软件与所有版本的Windows兼容。


The printer and computer are compatibles. 打印机和电脑是兼容的。|近义词：compatible, interchangeable |反义词：incompatible, conflicting |名词


USAGE->
一般现在时:
These systems are compatible. = 这些系统是兼容的。
The two programs are compatible with each other. = 这两个程序彼此兼容。

一般过去时:
The old system was compatible with the new one. = 旧系统与新系统兼容。
Those components were not compatible. = 那些组件不兼容。

现在完成时:
We have found a compatible solution. = 我们找到了一个兼容的解决方案。
They have ensured the new software is compatible with the old. = 他们确保新软件与旧软件兼容。

过去完成时:
The previous version had been compatible with older models. = 上一个版本与旧型号兼容。
We had discovered that the parts were not compatible. = 我们发现零件不兼容。

将来时:
The next update will be compatible with all devices. = 下次更新将与所有设备兼容。
Future versions will be compatible with current hardware. = 未来版本将与当前硬件兼容。

虚拟语气:
If the software were compatible, we could use it. = 如果软件兼容，我们就可以使用它。
It would be better if the two systems were compatible. = 如果这两个系统兼容就好了。

被动语态:
The software is being checked for compatibility. = 软件正在进行兼容性检查。
