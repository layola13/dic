/ɪkˈspɛktɪd/ | adj. | 预期的，期待的 | Expected arrival time is 3 pm. 预计到达时间是下午3点。|近义词：anticipated, foreseen, projected | 反义词：unexpected, unforeseen


USAGE->
一般现在时:
The expected outcome is positive. = 预期结果是积极的。
We expect a lot of visitors this weekend. = 我们预计这个周末会有很多访客。


一般过去时:
The meeting was expected to last an hour. = 预计会议持续一个小时。
They expected better results from the experiment. = 他们期望实验有更好的结果。


现在完成时:
The project has exceeded our expected completion date. = 项目已经超过了我们预期的完成日期。
We have expected this problem for some time. = 我们已经预料到这个问题有一段时间了。


过去完成时:
We had expected the rain to stop by then. = 我们当时预计雨到那时会停。
They had expected him to arrive earlier. = 他们原以为他会早点到。


将来时:
The flight is expected to arrive at 6 pm. = 预计航班将在下午6点到达。
We expect to finish the project by next week. = 我们预计下周完成项目。


虚拟语气:
If he had expected this, he would have prepared better. = 如果他预料到这一点，他会做得更好。
I wish I had expected the traffic to be so bad. = 我真希望我预料到交通会这么糟糕。


被动语态:
The results were expected to be published soon. = 预计结果很快就会公布。
The increase in sales was expected by the company. = 公司预计销售额会增长。


疑问句:
Is the package expected to arrive today? = 包裹预计今天到达吗？
What is the expected lifespan of this product? = 这个产品的预期寿命是多少？


否定句:
The outcome was not expected. = 结果出乎意料。
They did not expect such a strong reaction. = 他们没有预料到如此强烈的反应。


PHRASE->
as expected = 正如预期
expected value = 期望值
expected return = 预期收益
highly expected = 高度预期
unexpectedly expected = 出乎意料地被期待
