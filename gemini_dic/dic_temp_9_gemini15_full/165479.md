preload:/ˈpriːloʊd/| vt.|预先加载，预载|To load (something) beforehand, typically data or software, to improve performance or speed.|例句：The game preloads assets to reduce loading times. 游戏预加载资源以减少加载时间。例句：We preloaded the website with images to speed up the page load. 我们预先将图像加载到网站中以加快页面加载速度。|近义词：pre-load, load beforehand|反义词：unload, delete

The browser preloads images. = 浏览器预加载图像。
They preload data before the application starts. = 他们在应用程序启动之前预加载数据。
