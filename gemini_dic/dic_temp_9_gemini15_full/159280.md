point-and-click:/pɔɪnt ænd klɪk/|/ˌpɔɪnt ænd ˈklɪk/| adj. n.|点击式；指向点击|Describes a type of computer interface where the user interacts by pointing and clicking a mouse or other pointing device on screen elements.| 例句：Many adventure games utilize a point-and-click interface.  许多冒险游戏都使用点击式界面。

例句： This software uses a simple point-and-click system. 这个软件使用简单的点击式系统。


近义词：click-based, graphical user interface (GUI)|反义词：command-line interface (CLI)|词性: adjective, noun
