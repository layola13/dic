reconfigure:/ˌriːkənˈfɪɡjər/ | vt. |重新配置；重新构造 | To change the structure or organization of something.  |例句：The software allows you to reconfigure the network settings.  该软件允许您重新配置网络设置。 |例句：We need to reconfigure the production line to accommodate the new product. 我们需要重新配置生产线以适应新产品。|近义词：rearrange, reorganize, restructure |反义词：maintain, preserve


USAGE->
一般现在时:
They reconfigure the system every year. = 他们每年都重新配置系统。
The technician reconfigures the server regularly. = 技术员定期重新配置服务器。


一般过去时:
He reconfigured the software after the update. = 他在更新后重新配置了软件。
The engineer reconfigured the network yesterday. = 工程师昨天重新配置了网络。


现在完成时:
They have reconfigured the database for improved performance. = 他们已经重新配置了数据库以提高性能。
We have reconfigured the security settings. = 我们已经重新配置了安全设置。


过去完成时:
The team had reconfigured the equipment before the test. = 团队在测试前已经重新配置了设备。
She had reconfigured the settings before starting the program. = 她启动程序之前已经重新配置了设置。


将来时:
They will reconfigure the system next week. = 他们下周将重新配置系统。
The company will reconfigure its supply chain. = 公司将重新配置其供应链。


虚拟语气:
If we reconfigured the server, it might work better. = 如果我们重新配置服务器，它可能会运行得更好。
It would be better if they reconfigured the software. = 如果他们重新配置软件，会更好。


被动语态:
The system is being reconfigured at the moment. = 系统目前正在重新配置中。
The network was reconfigured last night. = 网络昨晚被重新配置了。


疑问句:
Will you reconfigure the settings for me? = 你能帮我重新配置设置吗？
Have they reconfigured the server yet? = 他们已经重新配置服务器了吗？


否定句:
They did not reconfigure the system. = 他们没有重新配置系统。
I cannot reconfigure the settings without the password. = 没有密码我无法重新配置设置。



PHRASE->
reconfigure the network = 重新配置网络

词源:
re- (prefix indicating again or anew) + configure (from Latin *configurāre, "to shape, form")
