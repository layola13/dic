multiplexors:/ˈmʌltiplɛksərz/| n. |多路复用器，多路选择器 |A multiplexor (MUX) is a device that selects one of several input signals and forwards the selected input into a single line.  It's essentially an electronic switch that allows you to choose which signal to pass through.  | Example: The router uses multiplexors to combine data streams from multiple users onto a single network line. 路由器使用多路复用器将来自多个用户的數據流组合到单个网络线路中。|近义词：selector, switch |反义词：demultiplexer
