hardcode:/ˈhɑːrdkoʊd/ | vt. |硬编码，直接写入代码 |To write a value directly into the source code of a program instead of calculating it or retrieving it from a configuration file or database.  This makes the code less flexible and harder to maintain. |Example:  The developers hardcoded the database connection string into the application. 开发人员将数据库连接字符串硬编码到应用程序中。|近义词：embed, fix, set |反义词：parameterize, configure


USAGE->
一般现在时:
They hardcode the API key into the client application. = 他们将API密钥硬编码到客户端应用程序中。
The program hardcodes the user's name. = 该程序硬编码了用户名。


一般过去时:
We hardcoded the default settings yesterday. = 我们昨天硬编码了默认设置。
The developers hardcoded the server address. = 开发人员硬编码了服务器地址。


现在完成时:
They have hardcoded the values into the script. = 他们已经将值硬编码到脚本中。
The team has hardcoded several critical parameters. = 团队已经硬编码了几个关键参数。


过去完成时:

The system had hardcoded the date and time. = 系统已经硬编码了日期和时间。

虚拟语气:

被动语态:
