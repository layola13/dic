exolete:/ˈɛksəʊleɪt/| adj.|过时的；废弃的；淘汰的|Something that is obsolete is no longer used or needed because something newer and better has been invented.  It often refers to technology, practices, or systems that have been replaced.|Example:  That software is obsolete; we need a newer version.  那个软件过时了；我们需要一个更新的版本。|近义词：outdated, outmoded, archaic, passé|反义词：modern, up-to-date, current, new


USAGE->
一般现在时:
This technology is obsolete. = 这项技术已经过时了。
The old system is obsolete. = 旧系统已经过时了。


一般过去时:
The typewriter became obsolete. = 打字机已经过时了。
That method was obsolete. = 那种方法已经过时了。


现在完成时:
Many traditional methods have become obsolete. = 许多传统方法已经过时了。
The old machine has become obsolete. = 旧机器已经过时了。


过去完成时:
By the time the new model was released, the old one had already become obsolete. = 当新车型发布时，旧车型早已过时了。
Before the internet became widespread, many communication methods had become obsolete. = 在互联网普及之前，许多通讯方式已经过时了。


将来时:
This technology will soon become obsolete. = 这项技术很快就会过时。
In the near future, this system will be obsolete. = 在不久的将来，这个系统将被淘汰。


虚拟语气:
If we don't innovate, our products will become obsolete. = 如果我们不创新，我们的产品将会过时。
I wish this software wasn't already obsolete. = 我希望这个软件还没有过时。


被动语态:
The old system was rendered obsolete by the new one. = 旧系统被新系统淘汰了。
This process has been made obsolete by automation. = 这个过程已经被自动化取代了。


疑问句:
Is this software obsolete? = 这个软件过时了吗？
Has that technology become obsolete? = 那项技术过时了吗？


否定句:
That equipment is not obsolete. = 那套设备并没有过时。
The information is not obsolete. = 这信息没有过时。


PHRASE->
declare obsolete = 宣布…已过时
render obsolete = 使…过时
become obsolete = 变得过时
make obsolete = 使…过时
obsolete technology = 过时的技术
