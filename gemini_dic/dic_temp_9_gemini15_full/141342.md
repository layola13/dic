operable:/ˈɒpərəbl/ /ˈɑːpərəbl/| adj.| 可手术的，可操作的 | Capable of being operated on or worked on; able to be used or handled.  |例句：The tumor is operable.  肿瘤是可以手术的。

The machine is operable after the repair. 修理后机器可以使用了。|近义词：treatable, manageable, workable |反义词：inoperable, unworkable, unmanageable


USAGE->
一般现在时:
The tumor is operable. = 肿瘤是可以手术的。
This software is operable on all platforms. = 这个软件在所有平台上都可以运行。

一般过去时:
The machine was operable before the accident. = 事故发生前机器是可以使用的。
The problem was operable using a simple solution. = 用一个简单的解决方案就可以解决这个问题。


现在完成时:
The machine has been operable since the repair. = 修理后机器一直可以使用。
The situation has been operable with careful management. = 通过细致的管理，局面已经可以控制了。


过去完成时:
The bridge had been operable for fifty years before it collapsed. = 那座桥在倒塌之前已经可以使用五十年了。
The system had been operable until the recent power outage. = 直到最近停电之前，系统一直都可以运行。


将来时:
The equipment will be operable after the upgrade. = 设备升级后就可以使用了。
The new system will be operable by next month. = 新系统下个月就能投入使用了。


虚拟语气:
If the tumor were operable, we would proceed with surgery immediately. = 如果肿瘤可以手术，我们将立即进行手术。
If the software were operable on my old computer, I would use it. = 如果这个软件在我的旧电脑上可以使用，我就会用它。


被动语态:
The problem can be operated on easily. = 这个问题很容易解决。
The equipment was operated on by the technician. = 设备由技师操作。


疑问句:
Is the machine operable? = 机器可以使用吗？

否定句:
The tumor is inoperable. = 肿瘤无法手术。
The software is not operable on this system. = 这个软件在这个系统上无法运行。
