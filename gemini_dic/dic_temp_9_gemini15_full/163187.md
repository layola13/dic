precompiled:/ˌpriːkəmˈpaɪld/ | adj. | 预编译的 | Already compiled; translated into machine code before the main program is compiled.  Precompiled code is often used to improve performance or to reuse code in multiple projects.  | Example: The precompiled libraries significantly reduced the compilation time of the project. 预编译的库大大缩短了项目的编译时间。 |近义词：prebuilt, compiled |反义词：uncompiled, interpreted
