interpreter:/ɪnˈtɜːprɪtər/| n. |口译员，解释者，翻译器|A person who translates spoken language from one language into another.  An interpreter can also refer to a program that translates code from one programming language to another.|例句：She works as an interpreter at the United Nations. 她作为口译员在联合国工作。

例句：The software uses an interpreter to run the script. 该软件使用解释器运行脚本。

|近义词：translator, translator, explicator |反义词：N/A |词性 n.
