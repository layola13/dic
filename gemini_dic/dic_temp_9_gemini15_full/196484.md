server:/ˈsɜːrvər/| n. |服务器，服务员，侍者 |A server is a computer that provides data to other computers.  It can also refer to a person who serves food or drinks. |例句：The web server is down. = 服务器宕机了。  The server brought us our food. = 服务员给我们送来了食物。|近义词：computer, waiter, waitress, attendant |反义词：client, customer


USAGE->
一般现在时:
The server runs continuously. = 服务器持续运行。
He serves food in a restaurant. = 他在餐馆里做服务员。

一般过去时:
The server crashed last night. = 服务器昨晚崩溃了。
She served us tea and biscuits. = 她给我们端来了茶点。

现在完成时:
The server has been running for 24 hours. = 服务器已经运行了24小时。
He has served many customers today. = 他今天已经服务了很多顾客。

过去完成时:
The server had already crashed before we could access it. = 我们还没来得及访问服务器，它就已经崩溃了。
The waiter had served all the guests before the main course. = 在上主菜之前，服务员已经服务了所有客人。

将来时:
The server will be upgraded next week. = 服务器下周将升级。
She will serve us at our table. = 她会在我们的桌旁为我们服务。

虚拟语气:
If the server were faster, the website would load quicker. = 如果服务器更快，网站加载速度就会更快。
If I were a server, I would try to be more efficient. = 如果我是服务员，我会努力提高效率。

被动语态:
The data is served by the server. = 数据由服务器提供。
The customers were served promptly. = 顾客得到了及时的服务。

疑问句:
Is the server online? = 服务器在线吗？
Does he serve alcohol? = 他有提供酒水吗？

否定句:
The server isn't responding. = 服务器没有响应。
He doesn't serve breakfast. = 他不提供早餐。
