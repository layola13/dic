runner:/ˈrʌnər/| n. |跑步者，赛跑者；（机器等的）活动部件；（尤指）信使|A person who runs, especially in a race.  Also, a sliding or moving part in a machine; or a person who carries messages or delivers goods.|例句：He is a marathon runner. 他是一个马拉松运动员。|例句：The runner in the washing machine is broken. 洗衣机的滚筒坏了。|近义词：racer, sprinter, courier, messenger|反义词：walker, driver


USAGE->
一般现在时:
He runs every morning. = 他每天早上跑步。
She runs a successful business. = 她经营着一家成功的企业。


一般过去时:
I ran a 5k last weekend. = 我上周末跑了5公里。
They ran to catch the bus. = 他们跑着去赶公共汽车。


现在完成时:
He has run several marathons. = 他已经跑过几次马拉松了。
She has run the company for ten years. = 她经营这家公司十年了。


过去完成时:
I had run five miles before I stopped. = 在我停下来之前，我已经跑了五英里。
They had run out of money by the end of the month. = 到月底他们已经钱花光了。


将来时:
I will run a race next week. = 我下周将参加一场比赛。
She will run the meeting tomorrow. = 她明天将主持会议。


虚拟语气:
If I were younger, I would run a marathon. = 如果我年轻一些，我会跑马拉松。
I wish I could run faster. = 我希望我能跑得更快。


被动语态:
The company is run by a young CEO. = 这家公司由一位年轻的首席执行官管理。
The race was run in terrible weather. = 比赛在恶劣的天气下进行。


疑问句:
Does he run every day? = 他每天都跑步吗？
Do they run the business together? = 他们一起经营这家企业吗？


否定句:
I don't run often. = 我不经常跑步。
She doesn't run that company anymore. = 她不再经营那家公司了。
