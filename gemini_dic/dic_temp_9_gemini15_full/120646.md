mis-routed:/ˌmɪsˈraʊtɪd/| adj.|错误路由的，错误发送的|The email was mis-routed and didn't reach the intended recipient.|近义词：misdirected, wrongly sent|反义词：correctly routed, properly delivered|


USAGE->
一般现在时:
The package is mis-routed. = 这个包裹被错误路由了。
The emails are mis-routed to the wrong department. = 邮件被错误地发送到错误的部门。


一般过去时:
The message was mis-routed yesterday. = 信息昨天被错误路由了。
The delivery was mis-routed to a different address. = 快递被错误地发送到不同的地址。


现在完成时:
The data has been mis-routed several times. = 数据已经被错误路由了几次。
The system has mis-routed numerous transactions. = 系统已经错误路由了许多交易。


过去完成时:
The files had been mis-routed before the problem was discovered. = 在问题被发现之前，文件已经被错误路由了。
The shipment had been mis-routed, causing a significant delay. = 货物已经被错误路由，导致了严重的延误。


将来时:
The update will be mis-routed if the server is down. = 如果服务器宕机，更新将会被错误路由。
The new software might mis-route the traffic. = 新软件可能会错误地路由流量。


虚拟语气:
If the address were correct, the package wouldn't have been mis-routed. = 如果地址正确，包裹就不会被错误路由了。
I wish the information hadn't been mis-routed. = 我希望信息没有被错误路由。


被动语态:
The call was mis-routed by the operator. = 电话被接线员错误路由了。
The request was mis-routed to the wrong department. = 请求被错误地发送到错误的部门。


疑问句:
Was the message mis-routed? = 信息被错误路由了吗？
Were the payments mis-routed to the wrong account? = 付款被错误地发送到错误的账户了吗？


否定句:
The email was not mis-routed; it simply arrived late. = 邮件没有被错误路由；它只是到达晚了。
The network did not mis-route the packets. = 网络没有错误路由数据包。

词源:
mis- (前缀，表示错误、坏) + routed (过去分词，来自route，表示路线、途径)
