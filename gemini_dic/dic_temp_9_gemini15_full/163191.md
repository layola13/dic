precompiling:/ˌpriːkəmˈpaɪlɪŋ/| v. |预编译|The process of compiling a program's source code before the main compilation process. This can improve performance by reducing the amount of work the main compiler needs to do.  Precompiling often involves generating intermediate code or translating parts of the code into a more efficient form. |Example: The software uses precompiling techniques to optimize the execution speed.  该软件使用预编译技术来优化执行速度。The game engine performs precompiling of shaders to improve rendering performance. 游戏引擎对着色器进行预编译以提高渲染性能。|近义词：precompilation, ahead-of-time compilation|反义词：post-compilation, just-in-time compilation
