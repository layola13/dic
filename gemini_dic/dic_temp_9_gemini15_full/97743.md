initialized:/ɪˈnɪʃəlaɪzd/ | adj. | 已初始化的；已设置初始值的 |  Having been given initial values or settings; set up for the first time.  |  The program initialized successfully.  程序已成功初始化。 |  initiated, started, configured | uninitialized, unconfigured | adj.


USAGE->
一般现在时:
The system initializes automatically upon startup. = 系统在启动时会自动初始化。
The program initializes variables before execution. = 程序在执行前初始化变量。


一般过去时:
The database was initialized yesterday. = 数据库昨天已初始化。
The software initialized without errors. = 软件初始化无错误。


现在完成时:
The variables have been initialized. = 变量已初始化。
The application has successfully initialized. = 应用程序已成功初始化。


过去完成时:
The program had initialized before the data was processed. = 数据处理前，程序已初始化。
After the system had initialized, the user logged in. = 系统初始化后，用户登录。


将来时:
The system will initialize automatically. = 系统将自动初始化。
The software will initialize when you click the button. = 点击按钮后，软件将初始化。


虚拟语气:
If the system were initialized correctly, it would work properly. = 如果系统正确初始化，它将正常工作。
It is important that the variables be initialized before use. = 变量在使用前必须初始化。


被动语态:

疑问句:

否定句:
