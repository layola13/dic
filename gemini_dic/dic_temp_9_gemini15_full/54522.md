disk:/dɪsk/| n.|磁盘，圆盘|A flat, circular object, typically made of plastic or metal, used for storing data or playing recorded sound or video.  It can refer to both hard disk drives and optical discs.|例句：I saved the file to the hard disk. 我把文件保存到硬盘上了。|例句：The old vinyl disk was scratched. 那张旧黑胶唱片划花了。|近义词：disc, platter|反义词：None


USAGE->

一般现在时:
The disk spins rapidly inside the computer. = 磁盘在电脑内部高速旋转。
This disk contains important data. = 这个磁盘包含重要数据。


一般过去时:
I copied the files from the disk. = 我从磁盘复制了文件。
The disk crashed and lost all my data. = 磁盘崩溃了，丢失了我所有的数据。


现在完成时:
I have backed up all my data to the disk. = 我已经将所有数据备份到磁盘上了。
She has already written the data onto the disk. = 她已经将数据写入磁盘了。


过去完成时:
Before the crash, I had copied the important files to a different disk. = 在崩溃之前，我已经将重要文件复制到另一个磁盘。
He had erased the disk before giving it away. = 他在送出磁盘之前已经擦除了它。


将来时:
I will copy the data to the external disk tomorrow. = 我明天将数据复制到外部磁盘。
We will need a new disk for the project. = 我们这个项目需要一个新的磁盘。


虚拟语气:
If I had a spare disk, I would back up my files immediately. = 如果我有一个备用磁盘，我会立即备份我的文件。
I wish I had known about the disk failure earlier. = 我真希望早点知道磁盘故障。


被动语态:
The data was saved to the disk automatically. = 数据自动保存到了磁盘。
The disk is being formatted now. = 磁盘现在正在格式化。


疑问句:
Is the disk full? = 磁盘满了？
Which disk should I use? = 我应该使用哪个磁盘？


否定句:

I don't have enough space on the disk. = 我的磁盘空间不足。
