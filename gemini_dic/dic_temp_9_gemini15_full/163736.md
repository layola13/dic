predefined:/ˌpriːdɪˈfaɪnd/| adj.|预定义的，预先定义的|Already defined; existing before the current program or process begins.  A predefined function or variable is one that has already been created and is available for use.|例句：The programming language has several predefined functions for common tasks.  这门编程语言有几个预定义的函数来完成常见任务。

例句：The system uses a predefined set of rules for data validation. 系统使用一组预定义的规则来验证数据。


近义词：predetermined, preset, pre-established, set in advance

反义词：undefined, unspecified, customized, user-defined
