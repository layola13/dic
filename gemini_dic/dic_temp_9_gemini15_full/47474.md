deallocating: /ˌdiːˈæləˌkeɪtɪŋ/ | v. | 释放，解除分配 | The process of releasing or freeing up allocated resources, such as memory or other system resources.  This is often done when an object or data structure is no longer needed. | Example: The program is deallocatng memory before exiting. 程序在退出之前正在释放内存。 The operating system is deallocatng the file handle. 操作系统正在释放文件句柄。|近义词：releasing, freeing, deallocating memory | 反义词：allocating, assigning


USAGE->
一般现在时:
The system deallocates unused memory. = 系统释放未使用的内存。
The application deallocates resources efficiently. = 该应用程序有效地释放资源。

一般过去时:
The program deallocated the buffer after use. = 程序在使用后释放了缓冲区。
The function deallocated the memory block successfully. = 该函数成功释放了内存块。


现在完成时:
The garbage collector has deallocated many objects. = 垃圾收集器已经释放了许多对象。
