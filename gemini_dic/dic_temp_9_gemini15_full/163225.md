precomputing:/ˌpriːkəmˈpjuːtɪŋ/| n. |预计算，预先计算 |The process of calculating values beforehand to speed up later computations.  It involves performing computations in advance, typically when computational resources are readily available, and storing the results for later use. This can significantly improve performance when the same calculations need to be repeated multiple times. |Example:  Precomputing the inverse of a matrix can significantly speed up subsequent calculations involving that matrix. 预先计算矩阵的逆矩阵可以显著加快后续涉及该矩阵的计算。

The game engine used precomputing to render the complex 3D environment more efficiently. 游戏引擎使用预计算来更有效地渲染复杂的3D环境。|近义词：precalculation, advance calculation |反义词：on-demand calculation, real-time calculation
