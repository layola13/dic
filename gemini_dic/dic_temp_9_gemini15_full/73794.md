format:/ˈfɔːrmæt/ /ˈfɔːrmæt/| n. |格式，样式，版式|The format of the document is easy to read.  文档的格式易于阅读。|近义词：style, layout, arrangement |反义词：chaos, disorder


USAGE->
一般现在时:
The document uses a standard format. = 这份文件使用标准格式。
This software supports various file formats. = 这个软件支持各种文件格式。

一般过去时:
The book was formatted differently in the second edition. = 这本书在第二版中格式有所不同。
They formatted the report according to company guidelines. = 他们根据公司指南对报告进行了格式化。

现在完成时:
We have formatted the presentation for better readability. = 我们已经对演示文稿进行了格式化，以提高可读性。
They have formatted all the documents consistently. = 他们已经对所有文档进行了统一的格式化。

过去完成时:
The editor had formatted the manuscript before sending it to the printer. = 编辑在将稿件发送给印刷厂之前已经对其进行了格式化。
They had formatted the database before importing the new data. = 他们在导入新数据之前已经对数据库进行了格式化。

将来时:
We will format the data before analysis. = 我们将在分析之前对数据进行格式化。
