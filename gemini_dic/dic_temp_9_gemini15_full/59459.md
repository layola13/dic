eas:/iːz/|adj.|容易的，简单的|easy means not difficult to do or understand.  It implies a lack of difficulty or complexity.|例句：The exam was easy. = 考试很简单。|例句：This task is easy to complete. = 这个任务很容易完成。|近义词：simple, effortless, straightforward|反义词：difficult, hard, challenging


USAGE->
一般现在时:
The problem is easy to solve. = 这个问题很容易解决。
This task is easy for her. = 这个任务对她来说很容易。

一般过去时:
The test was easy yesterday. = 昨天的考试很简单。
It was easy to find the house. = 找到那所房子很容易。

现在完成时:
The job has been easy so far. = 到目前为止，这项工作很容易。
Everything has been easy for him recently. = 最近对他来说一切都很顺利。

过去完成时:
The instructions had been easy to follow. = 说明书很容易理解。
The puzzle had been easy to solve before. = 这个谜题以前很容易解决。

将来时:
The journey will be easy. = 这段旅程将会很容易。
The game will be easy to win. = 这场比赛很容易获胜。


虚拟语气:
If the task were easy, I would do it myself. = 如果任务容易，我会自己做。
It would be easy to succeed if you tried harder. = 如果你更努力，成功就会很容易。

被动语态:
The problem is easily solved. = 这个问题很容易解决。
The question was easily answered. = 这个问题很容易回答。

疑问句:
Is the task easy? = 这个任务容易吗？
Is the book easy to read? = 这本书容易读吗？

否定句:
The test was not easy. = 考试并不容易。
The instructions were not easy to understand. = 说明书不容易理解。
