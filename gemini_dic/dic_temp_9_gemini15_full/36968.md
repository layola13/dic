cols:/kɒlz/ /kɔlz/| n.|栏；列；圆柱；柱状物|Columns are vertical structures that support a building.  柱子是支撑建筑物的垂直结构。|近义词：column, row, pillar|反义词：row (in the sense of a horizontal line)|名词


USAGE->
一般现在时:
The table has five columns. = 这张桌子有五列。
This document is divided into several columns. = 这份文件被分成几列。


一般过去时:
The architect designed elegant columns for the temple. = 建筑师为寺庙设计了优雅的柱子。
The old columns were crumbling. = 那些旧柱子正在崩塌。


现在完成时:
They have added two new columns to the spreadsheet. = 他们已经在这个电子表格中添加了两列。
We have finally finished aligning the columns. = 我们终于完成了对齐列的工作。


过去完成时:
The columns had already been erected before the roof was installed. = 屋顶安装之前，柱子就已经竖立起来了。
They had removed the damaged columns before the restoration began. = 在修复开始之前，他们已经移除了损坏的柱子。


将来时:
We will add more columns to the database next week. = 我们下周将向数据库添加更多列。
The new building will have impressive marble columns. = 新建筑将拥有令人印象深刻的大理石柱子。


虚拟语气:

被动语态:
