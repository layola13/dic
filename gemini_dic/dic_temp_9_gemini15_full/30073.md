cdk:/ˈsiːdiːkeɪ/| n. |云开发工具包|Cloud Development Kit; a software development framework to define your cloud application resources using familiar programming languages. It allows developers to model and provision cloud infrastructure using code, rather than through a graphical user interface.|Example:  I used the AWS CDK to deploy my serverless application. 我用 AWS CDK 部署了我的无服务器应用程序。|近义词：infrastructure-as-code tool, cloud provisioning tool|反义词：manual cloud provisioning|名词
