sdk:/ɛsdiːˈkeɪ/ | n. | 软件开发工具包 | A software development kit (SDK) is a collection of software development tools in one installable package.  It provides everything needed to create applications for a specific platform or operating system. This typically includes libraries, header files, documentation, sample code, and tools such as debuggers and compilers. | Example:  The Android SDK allows developers to build apps for Android devices.  安卓SDK允许开发者为安卓设备构建应用程序。 |近义词：development kit, software library, API (Application Programming Interface) |反义词：无
