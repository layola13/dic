precompile:/ˌpriːkəmˈpaɪl/| vt.|预编译|To compile (a program or part of a program) before the main compilation process. This is often done to improve performance or to handle parts of a program that are platform-specific.|例句：The precompiler handles the inclusion of header files before the main compilation stage. 预编译器在主编译阶段之前处理包含头文件。

例句：We precompiled the database queries to optimize the application's speed. 我们预编译了数据库查询以优化应用程序的速度。


近义词：preprocess, compile beforehand


反义词：postprocess


词性:动词
