minify:/ˈmɪnɪfaɪ/| vt.|缩小（文件大小）,精简|To reduce the size of (a file or other data) by removing unnecessary elements or using more efficient encoding.  This is often done with JavaScript, CSS, and HTML files to improve website loading speed.|例句：We minified the JavaScript files to improve the website's performance. 我们缩小了JavaScript文件的大小以提高网站的性能。
例句：The CSS was minified to reduce the file size. CSS文件被缩小以减小文件大小。|近义词：compress, reduce, shrink|反义词：expand, enlarge|动词
