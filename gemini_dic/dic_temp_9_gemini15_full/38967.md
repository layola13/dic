configuring:/kənˈfɪɡərɪŋ/ | v. |配置；组态；设定 | The act of setting up or adjusting the parameters or settings of a system, device, or software.  | Example:  We are configuring the network for optimal performance. 我们正在配置网络以获得最佳性能。  |近义词：setting up, adjusting, installing | 反义词：dismantling, uninstalling


USAGE->
一般现在时:
They are configuring the server. = 他们正在配置服务器。
I am configuring my new router. = 我正在配置我的新路由器。


一般过去时:
He configured the software yesterday. = 他昨天配置了软件。
We configured the firewall last week. = 我们上周配置了防火墙。


现在完成时:
She has configured the database. = 她已经配置了数据库。
They have configured the system for remote access. = 他们已经配置了远程访问系统。


过去完成时:
I had configured the printer before the meeting. = 我在会议前已经配置好了打印机。
He had configured the network settings prior to the installation. = 他在安装之前已经配置好了网络设置。


将来时:
We will configure the application next week. = 我们下周将配置该应用程序。
They will configure the security settings tomorrow. = 他们明天将配置安全设置。


虚拟语气:
If I configured the system correctly, it would work. = 如果我正确配置了系统，它就能工作。
It would be better if we configured the firewall more securely. = 如果我们更安全地配置防火墙会更好。


被动语态:
The system is being configured by the technician. = 系统正在由技术人员配置。
The software was configured according to the instructions. = 该软件是根据说明进行配置的。


疑问句:
Are you configuring the new software? = 你正在配置新的软件吗？
Has he configured the network yet? = 他已经配置好网络了吗？


否定句:
I did not configure the printer. = 我没有配置打印机。
They haven't configured the security settings yet. = 他们还没有配置安全设置。


PHRASE->
configure a network = 配置网络

configure a device = 配置设备
