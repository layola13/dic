malfunctioned:/mælˈfʌŋkʃənd/| v.| 发生故障；失灵 |  To fail to function correctly.  |例句：The engine malfunctioned and the car stopped.  发动机发生故障，车停了。|例句：My computer malfunctioned during the presentation. 我的电脑在演示过程中发生了故障。|近义词：failed, broke down, stopped working |反义词：functioned correctly, worked perfectly


USAGE->
一般现在时:
The machine malfunctions frequently. = 这台机器经常发生故障。
The system malfunctions occasionally. = 系统偶尔会发生故障。

一般过去时:
The engine malfunctioned during the race. = 发动机在比赛中发生了故障。
The software malfunctioned and crashed. = 软件发生故障并崩溃了。

现在完成时:
The printer has malfunctioned several times this week. = 这台打印机本周已经发生故障好几次了。
The lift has malfunctioned, and we're stuck. = 电梯出故障了，我们被困住了。

过去完成时:
The plane had malfunctioned before it crashed. = 飞机在坠毁前就已发生故障。
The computer had malfunctioned several times that day. = 那天电脑已经发生故障好几次了。

将来时:
The system might malfunction if overloaded. = 如果超载，系统可能会发生故障。
I fear the device will malfunction in extreme temperatures. = 我担心该设备在极端温度下会发生故障。

虚拟语气:
If the brakes had malfunctioned, the accident would have been worse. = 如果刹车失灵，事故会更严重。
It would be disastrous if the power grid malfunctioned. = 如果电网发生故障，那将是灾难性的。

被动语态:
The machine was malfunctioning. = 机器当时正在发生故障。
The system was found to have malfunctioned. = 系统被发现发生了故障。

疑问句:
Did the equipment malfunction? = 设备发生故障了吗？
Is the software malfunctioning again? = 软件又发生故障了吗？

否定句:
The equipment did not malfunction. = 设备没有发生故障。
The system hasn't malfunctioned yet. = 系统还没有发生故障。


PHRASE->
system malfunctioned = 系统故障
equipment malfunctioned = 设备故障
severe malfunction = 严重故障
mechanical malfunction = 机械故障
electrical malfunction = 电气故障
