serialize:/ˈsɪəriˌaɪz/ | vt. |序列化，连续刊登|To convert an object into a sequence of bytes (or other data type) so that it can be stored or transmitted and later reconstructed.  This process is often used to store complex data structures in files or databases, or to transmit them over a network.|例句：The program serializes the data before saving it to the file. = 程序在将数据保存到文件之前对其进行序列化。

The application serializes the user's preferences to a configuration file. = 应用程序将用户的偏好序列化到配置文件中。|近义词：marshal, encode|反义词：deserialize|动词
