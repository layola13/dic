glitch:/ɡlɪtʃ/| n. v.|小故障；小毛病；短暂的失误|A glitch is a small problem or fault that prevents something from working perfectly.  It is often temporary and easily fixed. |例句：There was a glitch in the system that prevented me from logging in. 系统出现一个小故障，导致我无法登录。

例句：The rocket launch was delayed due to a minor glitch in the countdown sequence.  由于倒计时序列中出现了一个小故障，火箭发射被推迟了。


近义词：fault, flaw, malfunction, hiccup, bug |反义词：perfection, flawlessness


USAGE->
一般现在时:
A glitch occurs frequently in this software. = 这个软件经常出现故障。
This machine glitches sometimes. = 这台机器有时会出故障。


一般过去时:
A glitch occurred during the live broadcast. = 直播过程中出现了一个小故障。
The system glitched and crashed. = 系统出现故障并崩溃了。


现在完成时:
Several glitches have occurred since the update. = 更新后发生了一些故障。
The program has glitched several times today. = 该程序今天已经出现了几次故障。


过去完成时:
The game had glitched before I saved my progress. = 在我保存进度之前，游戏已经出现故障了。
The server had glitched multiple times, causing delays. = 服务器多次出现故障，导致延迟。


将来时:
The new software may glitch occasionally. = 新软件可能会偶尔出现故障。
We expect the system to glitch during peak hours. = 我们预计系统在高峰时段会发生故障。


虚拟语气:
If the system glitched, we would have to reboot it. = 如果系统出现故障，我们就必须重新启动它。
I wish the software didn't glitch so often. = 我希望软件不会经常出现故障。


被动语态:
The problem was caused by a glitching hard drive. = 问题是由硬盘故障引起的。
The game is being affected by a recurring glitch. = 游戏正受到反复出现的故障的影响。


疑问句:
Is the system glitching again? = 系统又出现故障了吗？
Did the program glitch during the presentation? = 演示过程中程序出现故障了吗？


否定句:
The system didn't glitch during the test. = 系统在测试过程中没有出现故障。
The software hasn't glitched at all today. = 该软件今天根本没有出现故障。


PHRASE->
