shebang:/ˈʃeɪbæŋ/| n. |Shebang(首行解释器指示符)|A shebang is a line at the beginning of a script, usually a Unix shell script, that specifies the interpreter to use to execute the script.  It starts with `#!` followed by the path to the interpreter.  For example, `#!/bin/bash` indicates that the script should be executed using the Bash shell.|Example: `#!/bin/bash` This line tells the operating system to use Bash to run the script.  This script will print "Hello, world!": `#!/bin/bash echo "Hello, world!"` |近义词：interpreter directive, hashbang|反义词：无|名词
