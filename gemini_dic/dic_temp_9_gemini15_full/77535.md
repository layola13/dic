gcc:/ˌdʒiːsiːˈsiː/|/ˈdʒiːsiːsiː/| n. |GNU编译器集合|GNU Compiler Collection, a compiler system produced by the GNU Project supporting various programming languages, primarily C, C++, Objective-C, Fortran, Ada, Go, and D.|Example: I compiled my C++ code using gcc. 我用gcc编译了我的C++代码。|近义词：compiler, build system|反义词：interpreter


USAGE->
This word is a noun and does not conjugate like a verb.  Therefore, grammatical tenses, voice and mood are not applicable.  The usage focuses on its function as a compiler.

Example 1:  The gcc compiler is widely used in Linux systems. gcc编译器在Linux系统中被广泛使用。

Example 2:  We need to install gcc before compiling the project.  在我们编译项目之前需要安装gcc。
