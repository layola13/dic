dummying:/ˈdʌmiɪŋ/| v.|虚拟化；模拟；用虚拟数据填充|The act of creating dummy data or using dummy variables in a process or system.  It's often used for testing, training, or concealing sensitive information.|例句：We're dummying the database for the training session. 我们正在为培训课程虚拟化数据库。|例句：The developers are dummying the user interface to test the backend functionality. 开发人员正在模拟用户界面以测试后端功能。|近义词：simulating, mocking, faking,  placeholdering |反义词：authenticating, verifying, using real data
