delete:/dɪˈliːt/| vt.|删除，去除|To remove or erase something.|Please delete that file. 请删除那个文件。|近义词：erase, remove, expunge|反义词：add, insert, create


USAGE->
一般现在时:
I delete unnecessary files regularly. = 我定期删除不必要的文件。
He deletes his browsing history every day. = 他每天都删除他的浏览历史记录。

一般过去时:
I deleted the email by mistake. = 我不小心删除了那封邮件。
She deleted all the photos from her phone. = 她从手机上删除了所有照片。

现在完成时:
I have deleted all the old documents. = 我已经删除了所有旧文件。
He has deleted his account on that platform. = 他已经删除了他在那个平台上的帐户。

过去完成时:
I had deleted the file before the system crashed. = 系统崩溃之前我已经删除了该文件。
They had deleted the evidence before the police arrived. = 警察到达之前他们已经销毁了证据。

将来时:
I will delete these files later. = 我稍后会删除这些文件。
She will delete the message as soon as she reads it. = 她读完信息后就会删除它。

虚拟语气:
If I deleted that file, I would lose important data. = 如果我删除了那个文件，我会丢失重要数据。
I wish I hadn't deleted those photos. = 我真希望我没有删除那些照片。

被动语态:
The file was deleted accidentally. = 该文件被意外删除了。
All the messages were deleted by the administrator. = 所有信息都被管理员删除了。

疑问句:
Did you delete the file? = 你删除了那个文件吗？
Have you deleted your account yet? = 你删除你的账户了吗？

否定句:
I didn't delete the file. = 我没有删除那个文件。
She didn't delete any of the messages. = 她没有删除任何信息。


PHRASE->
delete a file = 删除文件
delete a message = 删除信息
delete a user = 删除用户
delete an account = 删除账户
delete a scene = 删除场景
