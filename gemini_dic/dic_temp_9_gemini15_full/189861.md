same:/seɪm/| adj.|相同的，同样的|Identical or alike in every respect.  It refers to things that are exactly alike or equivalent.  |例句：The two houses are almost the same. 这两座房子几乎一模一样。

例句：They wore the same dress.  她们穿着同样的衣服。


近义词：identical, alike, equal, equivalent |反义词：different, unlike, dissimilar


词源:  Old English *sama, from Proto-Germanic *samaz (“same”), from Proto-Indo-European *sem- (“one”).


USAGE->
一般现在时:
The cars are the same color. = 这些车颜色相同。
This shirt is the same as that one. = 这件衬衫和那件一样。


一般过去时:
The results were the same as last year. = 结果和去年一样。
Her answer was the same as mine. = 她的答案和我的一样。


现在完成时:
The conditions have remained the same. = 条件保持不变。
They have been the same since then. = 从那时起他们就一直一样。


过去完成时:
The situation had been the same for years. =  这种情况已经持续多年了。
Everything had been the same as before. = 一切都和以前一样。


将来时:
The rules will remain the same. = 规则将保持不变。
Things will be the same next year. = 明年情况将保持不变。


虚拟语气:
If it were the same, I would be happy. = 如果情况一样，我会很高兴。
I wish the situation were the same. = 我希望情况一样。


被动语态:
The same method was used. = 使用了相同的方法。
The same result was achieved. = 达到了相同的结果。


疑问句:
Is this the same as that? = 这和那一样吗？
Are the two reports the same? = 两份报告一样吗？


否定句:
The two situations are not the same. = 这两种情况不一样。
The answers are not the same. = 答案不一样。


PHRASE->
at the same time = 同时
the same as = 和...一样
all the same = 尽管如此；依然
one and the same = 同一个人或事物
in the same way = 以同样的方式
