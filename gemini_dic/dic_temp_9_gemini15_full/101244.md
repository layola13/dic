invocable:/ˈɪnvoʊkəbl/ | adj. | 可调用的，可引用的 | Capable of being invoked or called upon; readily available or accessible.  | Example: The function is invocable only after the database connection is established.  该函数只有在数据库连接建立后才能调用。 | 近义词：callable, accessible, usable | 反义词： inaccessible, unavailable |  形容词


USAGE->
一般现在时:
The function is invocable. = 该函数是可调用的。
This method is invocable from anywhere in the code. = 此方法可在代码的任何地方调用。


一般过去时:
The subroutine was invocable only under specific conditions. = 该子程序只有在特定条件下才能调用。
The library function was not invocable due to a missing dependency. = 由于缺少依赖项，库函数不可调用。


现在完成时:
The API has been invocable since the last update. = 自上次更新以来，API 一直是可调用的。
The feature has been made invocable through a new command-line interface. = 通过新的命令行界面，该功能已变得可调用。


过去完成时:
The method had been invocable before the recent security patch. = 在最近的安全补丁之前，该方法是可调用的。
The script had already been made invocable when the error occurred. = 错误发生时，脚本已经可以调用了。


将来时:
The module will be invocable after the build process completes. = 构建过程完成后，模块将可调用。
The service will be invocable from any client application. = 该服务将可从任何客户端应用程序调用。


虚拟语气:
If the library were invocable, the program would work correctly. = 如果库可调用，程序将正常工作。
It would be helpful if the function were invocable from the main thread. = 如果该函数可以从主线程调用，那将很有帮助。


被动语态:
The function is invoked by the main program. = 该函数由主程序调用。
The method was invoked unexpectedly. = 该方法被意外调用。


疑问句:
Is the function invocable from this context? = 该函数可以从此上下文调用吗？
Are all the necessary libraries invocable? = 所有必要的库都可调用吗？


否定句:
The object is not invocable at this stage. = 此阶段对象不可调用。
The method was not invocable because of a runtime error. = 由于运行时错误，该方法不可调用。
