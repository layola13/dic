incremented:/ˈɪŋkrɪmɛntɪd/ | v. |递增的；增加的 |To increase the value of a variable or quantity by a certain amount.  |Example: The counter was incremented by one each time the button was pressed.  每次按下按钮，计数器都会递增一。|近义词：increased, raised, augmented |反义词：decremented, reduced, diminished |动词


USAGE->
一般现在时:
The program increments the value automatically. = 程序会自动递增该值。
The counter increments every second. = 计数器每秒递增。


一般过去时:
The value was incremented after the calculation. = 计算之后，该值被递增了。
The score incremented after each correct answer. = 每次答对后，分数都会增加。


现在完成时:
The total has been incremented several times. = 总数已经被递增了好几次。
The counter has incremented to 100. = 计数器已经递增到100。


过去完成时:
The value had been incremented before the error occurred. = 错误发生之前，该值已经被递增了。
The counter had incremented to 50 before the system crashed. = 系统崩溃前，计数器已经递增到50。


将来时:
The program will increment the variable tomorrow. = 程序明天将递增该变量。
The counter will increment automatically. = 计数器将自动递增。


虚拟语气:
If the value were incremented, the result would be different. = 如果该值被递增，结果将会不同。
I wish the counter had been incremented properly. = 我希望计数器已被正确递增。


被动语态:
The variable is incremented by the function. = 变量由函数递增。
The counter is being incremented continuously. = 计数器正在持续递增。


疑问句:
Is the value incremented correctly? = 该值是否被正确递增？
Has the counter been incremented? = 计数器是否已被递增？


否定句:
The value was not incremented. = 该值没有被递增。
The counter didn't increment properly. = 计数器没有正确递增。
