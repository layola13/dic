decompilers:/ˌdiːˈkɒmpaɪləz/ | n. |反编译器 |A decompiler is a computer program that translates low-level code (such as machine code or assembly language) into a higher-level programming language, making it more readable and understandable.  It is essentially the reverse process of a compiler.  Decompilers are often used in reverse engineering, software analysis, and debugging.  However, they don't always perfectly reconstruct the original source code, and the resulting code might not be fully functional or identical to the original. |Example:  The software engineer used a decompiler to examine the compiled program and understand its functionality.  反编译器被软件工程师用来检查已编译的程序并理解其功能。

  The cybersecurity expert employed a decompiler to analyze the malware sample.  网络安全专家使用反编译器来分析恶意软件样本。


近义词:reverse translator, disassembler (for lower-level code)
反义词:compiler
