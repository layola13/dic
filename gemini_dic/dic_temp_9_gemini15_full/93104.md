if's:/ɪf/| conj.|如果，假如|if引导条件状语从句，表示假设或条件。它可以引导多种类型的条件状语从句，例如真实条件句、非真实条件句等。|If it rains, we will stay home.  如果下雨，我们就待在家里。|
If you study hard, you will pass the exam. 如果你努力学习，你就会通过考试。|unless, provided that, supposing|unless (除非), because|conj.


USAGE->
一般现在时:
If it rains tomorrow, we will stay home. = 如果明天下雨，我们就待在家里。
If she studies hard, she will pass the exam. = 如果她努力学习，她就能通过考试。


一般过去时:
If it rained, we would stay home. = 如果下雨，我们就待在家里。(虚拟语气)
If she studied hard, she would pass the exam. = 如果她努力学习，她就能通过考试。(虚拟语气)


现在完成时:
If it has rained a lot, the river will flood. = 如果雨下得很多，河水就会泛滥。
If she has studied hard, she will pass the exam. = 如果她已经努力学习了，她就能通过考试。


过去完成时:
If it had rained the day before, the ground would have been wet. = 如果前一天下雨了，地面就会湿。
If she had studied hard, she would have passed the exam. = 如果她已经努力学习了，她就能通过考试。


将来时:
If it will rain tomorrow, we will stay home. = 如果明天下雨，我们就待在家里。(不太常用，通常用一般现在时代替)
If she will study hard, she will pass the exam. = 如果她将努力学习，她就能通过考试。(不太常用，通常用一般现在时代替)


虚拟语气:
If I were you, I would accept the offer. = 如果我是你，我会接受这个提议。
If she had worked harder, she would have got a better grade. = 如果她学习更努力的话，她就能得到更好的成绩。


被动语态:
If the plan is approved, it will be implemented immediately. = 如果计划获得批准，它将立即实施。
If the problem were solved, the situation would improve. = 如果问题得到解决，情况就会好转。(虚拟语气)


疑问句:
If it is raining, should we go out? = 如果正在下雨，我们应该出去吗？
If she is coming, will you tell me? = 如果她要来，你会告诉我吗？


否定句:
If it doesn't rain, we will go to the park. = 如果不下雨，我们就去公园。
If she doesn't study hard, she won't pass the exam. = 如果她不努力学习，她就不会通过考试。


PHRASE->
if only = 但愿；要是…就好了
even if = 即使；纵然
as if = 好像；仿佛
if anything = 如果有什么不同的话
if so = 如果这样的话
