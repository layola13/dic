slidder:/ˈslɪdər/| n. |滑块，滑块开关|A slidder is a control element in a graphical user interface that allows the user to select a value from a range by sliding a thumb along a track.|Example:  The volume control on this device is a slidder. 这设备的音量控制是一个滑块。|近义词：slider, control, regulator |反义词:  None
