remapped:/ˌriːˈmæpt/ | vt. |重新映射，重新规划|To change the way something is mapped or organized.  This often refers to changing the way data is organized, a computer program's functionality, or a physical system's configuration.  For example, remapping a computer keyboard changes which keys perform which functions.|例句：The software engineers remapped the controls to make the game more intuitive. 软件工程师重新映射了控件，使游戏更直观。

The car's engine control unit (ECU) was remapped to improve fuel efficiency. 汽车的发动机控制单元 (ECU) 进行了重新映射以提高燃油效率。

近义词：reconfigure, reorganize, reprogram |反义词：preserve, maintain the original configuration
