compatible:/kəmˈpætəbl/ | adj. | 兼容的；相容的；和谐的 |  Compatible means able to exist or be used together without problems or conflict. It suggests a harmonious relationship or a lack of conflict between two or more things.  |  The new software is compatible with older operating systems. 新软件与旧操作系统兼容。|  consistent, agreeable, harmonious | incompatible, conflicting, discordant | 形容词


USAGE->
一般现在时:
This printer is compatible with my computer. = 这台打印机与我的电脑兼容。
These two ideas are compatible with each other. = 这两个想法彼此兼容。

一般过去时:
The old system was compatible with the new one. = 旧系统与新系统兼容。
That software was not compatible with my operating system. = 那软件与我的操作系统不兼容。

现在完成时:
We have found a solution that is compatible with all our needs. = 我们找到一个能满足我们所有需求的解决方案。
They have ensured that all their products are compatible with each other. = 他们确保所有产品彼此兼容。

过去完成时:
The previous version had been compatible with all previous releases. = 之前的版本与所有之前的版本兼容。
We realized that the components had not been compatible. = 我们意识到这些组件不兼容。

将来时:
The next update will be compatible with all current devices. = 下一个更新将与所有当前设备兼容。
Future versions will likely be compatible with this system. = 未来版本可能会与该系统兼容。

虚拟语气:
If the software were compatible, I would use it. = 如果软件兼容，我会使用它。
It would be ideal if all the systems were compatible. = 如果所有系统都兼容，那就理想了。

被动语态:
The new software is considered compatible with most devices. = 新软件被认为与大多数设备兼容。
The system was found to be incompatible with the older models. = 发现该系统与旧型号不兼容。

疑问句:
Is this software compatible with Windows 10? = 这个软件与Windows 10兼容吗？
Are these two approaches compatible? = 这两种方法兼容吗？

否定句:
The two programs are not compatible. = 这两个程序不兼容。
His personality is not compatible with hers. = 他的性格与她的性格不相容。
