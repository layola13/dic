init:/ɪˈnɪt/ | vt. | 初始化，启动 | To initialize; to set up or start something.  | Example:  The program automatically initializes the variables.  程序自动初始化变量。 | 近义词：initialize, start, begin | 反义词：terminate, end


USAGE->
一般现在时:
The system initializes the database every morning. = 系统每天早上初始化数据库。
The computer automatically initializes the hard drive on startup. = 电脑在启动时自动初始化硬盘。


一般过去时:
The technician initialized the server before the test. = 技术员在测试前初始化了服务器。
They initialized the project with a detailed plan. = 他们用详细的计划启动了项目。


现在完成时:
The program has initialized all the necessary files. = 程序已初始化所有必要的文 件。
She has initialized the software and is ready to proceed. = 她已经初始化了软件，准备继续进行。


过去完成时:
After the system had initialized, the application started running. = 系统初始化后，应用程序开始运行。
The programmer had initialized the variables before the loop began. = 程序员在循环开始之前已经初始化了变量。


将来时:
The operating system will initialize the graphics card upon boot. = 操作系统将在启动时初始化显卡。
We will initialize the network connection before the data transfer. = 我们将在数据传输前初始化网络连接。


虚拟语气:
If I initialized the database correctly, the program would work. = 如果我正确地初始化了数据库，程序就会运行。
It is important that the system be initialized before use. = 使用前必须初始化系统。


被动语态:
The variables were initialized by the program. = 变量由程序初始化。
The database is being initialized at this moment. = 数据库正在初始化中。


疑问句:
Has the software been initialized? = 软件已经初始化了吗？
Did you initialize the printer correctly? = 你正确地初始化打印机了吗？


否定句:
The program did not initialize properly. = 程序没有正确初始化。
He failed to initialize the network interface. = 他未能初始化网络接口。


PHRASE->
