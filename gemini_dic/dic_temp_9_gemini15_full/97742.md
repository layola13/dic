initialize:/ɪˈnɪʃəlaɪz/| /ɪˈnɪʃəlʌɪz/| vt.|初始化，起始|To set something to its initial state or starting values.  This often involves setting variables to default values, clearing memory, or preparing a system for operation.|例句：You need to initialize the database before running the application.  你需要在运行应用程序之前初始化数据库。|例句：The program initializes several variables at the start. 程序在启动时初始化多个变量。|近义词：initiate, start, begin, set up|反义词：terminate, end, finish
