non-urgent:/ˌnɒnˈɜːdʒənt/| adj. |非紧急的，不紧急的|Not requiring immediate attention or action.|The email marked "non-urgent" can be dealt with later.|近义词：unimportant, less important, secondary|反义词：urgent, critical, pressing


USAGE->
一般现在时:
This task is non-urgent. = 这项任务不紧急。
That problem is non-urgent for now. = 那个问题目前不紧急。


一般过去时:
The issue was non-urgent yesterday. = 昨天这个问题不紧急。
The project seemed non-urgent last week. = 上周这个项目看起来不紧急。


现在完成时:
The situation has remained non-urgent. = 局势一直不紧急。
Everything has been non-urgent so far. = 到目前为止，一切都不紧急。


过去完成时:
The request had been non-urgent until now. = 之前的请求一直不紧急。
The matter had remained non-urgent before the incident. = 在事件发生之前，此事一直不紧急。


将来时:
The repair will be non-urgent next week. = 下周的维修不紧急。
The meeting will remain non-urgent until further notice. = 会议将持续保持非紧急状态，直至另行通知。


虚拟语气:
If the matter were non-urgent, we could postpone it. = 如果这件事不紧急，我们可以推迟它。
I wish this task were non-urgent. = 我希望这项任务不紧急。


被动语态:
The problem is considered non-urgent by the manager. = 经理认为这个问题不紧急。
The request was deemed non-urgent by the team. = 团队认为该请求不紧急。


疑问句:
Is this request non-urgent? = 这个请求不紧急吗？
Are these matters non-urgent? = 这些事情不紧急吗？


否定句:
This is not a non-urgent matter. = 这不是一件不紧急的事情。
That is not a non-urgent issue. = 那不是一个不紧急的问题。


PHRASE->
mark as non-urgent  标记为非紧急
deal with non-urgent tasks 处理非紧急任务
non-urgent requests 非紧急请求
non-urgent emails 非紧急邮件
