cursor:/ˈkɜːrsər/| n.|光标，游标|A cursor is a symbol on a computer screen that shows where the next character will be inserted.|例句：The cursor blinked on the empty screen. 光标在空白屏幕上闪烁。|近义词：pointer, indicator|反义词:None


USAGE->
一般现在时:
The cursor blinks steadily. = 光标稳定地闪烁。
The cursor indicates the current position. = 光标指示当前位置。

一般过去时:
The cursor moved to the next line. = 光标移动到下一行。
The cursor was positioned at the end of the document. = 光标位于文档末尾。

现在完成时:
The cursor has moved several times. = 光标已经移动了好几次。
The cursor has remained in the same spot. = 光标一直停留在同一个位置。

过去完成时:
The cursor had already moved before I clicked. = 我点击之前光标已经移动了。
The cursor had been flashing for minutes. = 光标已经闪烁了几分钟了。

将来时:
The cursor will move to the right. = 光标将向右移动。
The cursor will be visible after the program loads. = 程序加载后光标将可见。

虚拟语气:
If the cursor were blinking, I would know the computer was on. = 如果光标闪烁，我就知道电脑开着。
I wish the cursor would stop moving. = 我希望光标能停止移动。

被动语态:
The cursor is controlled by the mouse. = 光标由鼠标控制。
The cursor was accidentally moved by the user. = 光标被用户意外移动了。

疑问句:
Where is the cursor located? = 光标位于何处？
Is the cursor blinking correctly? = 光标闪烁正常吗？

否定句:
The cursor is not visible. = 光标不可见。
The cursor didn't move when I pressed the key. = 我按下按键时光标没有移动。
