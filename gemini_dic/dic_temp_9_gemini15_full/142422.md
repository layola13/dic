os:/ˈɒs/ /ɑːs/| n. |操作系统；骨；口|Operating system; bone; mouth (archaic)|The new operating system is faster and more stable.  新的操作系统更快更稳定。|  The dog broke its leg bone. 狗的腿骨断了。|近义词：system, software, skeleton |反义词：hardware, software (in contrast)


USAGE->
一般现在时:
The OS runs smoothly. = 操作系统运行流畅。
This OS supports multiple languages. = 这个操作系统支持多种语言。


一般过去时:
The old OS crashed frequently. = 旧的操作系统经常崩溃。
We upgraded the OS yesterday. = 我们昨天升级了操作系统。


现在完成时:
We have installed the new OS. = 我们已经安装了新的操作系统。
They have been using this OS for years. = 他们已经使用这个操作系统多年了。


过去完成时:
The program failed because the OS had not been updated. = 程序失败是因为操作系统没有更新。
Before the crash, the OS had been running perfectly. = 在崩溃之前，操作系统运行完美。


将来时:
We will install the new OS next week. = 我们下周将安装新的操作系统。
They will be using a different OS soon. = 他们很快就会使用不同的操作系统。


虚拟语气:
If the OS were more stable, we wouldn't have so many problems. = 如果操作系统更稳定，我们就不会有这么多问题。
I wish the OS supported more applications. = 我希望操作系统支持更多应用程序。


被动语态:
The OS was updated automatically. = 操作系统自动更新了。
The problem is being solved by the OS developers. = 问题正在由操作系统开发者解决。


疑问句:
What OS are you using? = 你使用什么操作系统？
Which OS is better for gaming? = 哪个操作系统更适合游戏？


否定句:
The OS doesn't support this feature. = 此操作系统不支持此功能。
We haven't updated the OS yet. = 我们还没有更新操作系统。
