configured:/kənˈfɪɡərd/ | adj. | 配置好的；已设置的 | Arranged or set up in a particular way.  | The system is configured to automatically back up data.  系统被配置为自动备份数据。 | established, set up, arranged | unconfigured, disorganized |


USAGE->
一般现在时:
The network is configured correctly. = 网络配置正确。
The software is configured for optimal performance. = 软件被配置为获得最佳性能。


一般过去时:
The server was configured yesterday. = 服务器昨天配置好了。
The database was configured to meet our specific needs. = 数据库被配置以满足我们的特定需求。


现在完成时:
The system has been configured for remote access. = 系统已配置为远程访问。
The application has been configured to integrate with other systems. = 该应用程序已被配置为与其他系统集成。


过去完成时:
The program had been configured before the test began. = 测试开始前程序已配置好。
The settings had been configured according to the manual's instructions. = 设置已按照手册的说明进行了配置。


将来时:
The network will be configured next week. = 网络将在下周配置。
The new software will be configured by the IT team. = 新软件将由IT团队配置。


虚拟语气:
If the system were configured properly, it would work perfectly. = 如果系统配置正确，它将完美运行。
It is essential that the software be configured correctly. = 软件必须正确配置。


被动语态:
The settings are configured by the administrator. = 设置由管理员配置。
The application is being configured for the new hardware. = 该应用程序正在为新硬件进行配置。


疑问句:
Is the printer configured correctly? = 打印机配置正确吗？
How is the firewall configured? = 防火墙是如何配置的？


否定句:
The device is not configured properly. = 设备配置不正确。
The software hasn't been configured yet. = 软件尚未配置。


PHRASE->
configured correctly = 正确配置
configured for = 为...而配置
configured to = 被配置为
configured with = 用...配置
highly configured = 高度配置的
