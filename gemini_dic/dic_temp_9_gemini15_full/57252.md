download: /ˈdaʊnˌloʊd/ | vt. n. | 下载 | To copy (a file, program, etc.) from a computer network to a local computer system.  |例句：I downloaded the software from the company website. 我从公司网站下载了软件。  |例句：The download speed is very slow. 下载速度很慢。|近义词：transfer, obtain |反义词：upload


USAGE->
一般现在时:
I download music every day. = 我每天都下载音乐。
He downloads videos from YouTube. = 他从YouTube下载视频。

一般过去时:
I downloaded a new game last night. = 我昨晚下载了一个新游戏。
She downloaded the document before the meeting. = 她在会议前下载了文件。

现在完成时:
I have downloaded all the necessary files. = 我已经下载了所有必要的文档。
He has downloaded that movie several times. = 他已经下载那部电影好几次了。

过去完成时:
I had downloaded the software before he called. = 他打电话之前我已经下载好了软件。
She had downloaded the presentation before the conference began. = 会议开始前她已经下载好了演示文稿。

将来时:
I will download the update later. = 我稍后会下载更新。
He will download the movie tonight. = 他今晚会下载这部电影。

虚拟语气:
If I had a faster internet connection, I would download the movie faster. = 如果我的网速更快，我会更快地下载电影。
I wish I could download that software for free. = 我真希望可以免费下载那个软件。

被动语态:
The file is being downloaded. = 文件正在下载中。
The software was downloaded successfully. = 软件成功下载。

疑问句:
Can you download this file for me? = 你能帮我下载这个文件吗？
Have you downloaded the latest version? = 你下载了最新版本了吗？

否定句:
I didn't download the program. = 我没有下载那个程序。
He couldn't download the file because of the slow internet speed. = 由于网速慢，他无法下载该文件。
