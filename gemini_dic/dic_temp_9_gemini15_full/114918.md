manageable:/ˈmænɪdʒəbəl/| adj.|易于管理的，可控制的，可应付的|easily controlled or dealt with|The workload is manageable, even with the new project.|近义词：controllable, tractable,  feasible|反义词：unmanageable, overwhelming, insurmountable


USAGE->
一般现在时:
The project is manageable with the right resources. =  有了合适的资源，这个项目是可以应付的。
The task is manageable for her. = 这项任务对她来说是能够胜任的。


一般过去时:
The problem was manageable before the deadline. = 截止日期之前这个问题是可以处理的。
The situation was manageable until the storm hit. = 直到暴风雨来临之前，情况都是可以控制的。


现在完成时:
The team has made the situation manageable. = 团队已经使情况变得可控。
We have found a manageable solution to the crisis. = 我们已经找到了一个可行的解决方案来应对危机。


过去完成时:
The situation had been manageable until the unexpected delay. = 直到意外延误之前，情况都是可以控制的。
The project had become manageable after we hired more staff. = 在我们雇佣更多员工之后，项目变得易于管理了。


将来时:
The workload will be manageable once we automate some processes. = 一旦我们自动化一些流程，工作量就会容易管理。
The situation will be manageable if we take preventative measures. = 如果我们采取预防措施，情况是可以控制的。


虚拟语气:
If we had more staff, the workload would be manageable. = 如果我们有更多员工，工作量就会容易管理。
It would be manageable if the project was simplified. = 如果项目简化了，它就会容易管理。


被动语态:
The problem was easily managed by the team. = 这个问题很容易被团队解决。
The situation is being managed effectively. = 情况正在得到有效控制。


疑问句:
Is the workload manageable? = 工作量是可以应付的吗？
Is this project manageable within the given timeframe? = 在规定的时间内，这个项目是可以完成的吗？


否定句:
The situation is not manageable without additional support. = 没有额外的支持，情况是无法控制的。
The task is not manageable given the current constraints. = 鉴于目前的限制，这项任务是无法完成的。
