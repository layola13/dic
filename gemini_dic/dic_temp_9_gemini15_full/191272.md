scatterplots:/ˈskætə(r)plɒts/| n. |散点图 |A scatter plot (or scatter graph) is a type of plot or mathematical diagram using Cartesian coordinates to display values for typically two variables for a set of data.  The data is displayed as a collection of points, each having the value of one variable determining the position on the horizontal axis and the value of the other variable determining the position on the vertical axis.  Scatter plots are used to observe and show the relationship between two variables. | Example: The scatter plot clearly shows a positive correlation between hours studied and exam scores. 散点图清楚地显示了学习时间与考试分数之间存在正相关关系。|近义词：scatter diagram, correlation chart |反义词：N/A
