prebuild:/ˈpriːbɪld/| adj. |预先构建的；预编译的|Already built or compiled before the main process.  It refers to something that is prepared or constructed in advance.|The software includes a prebuild library of functions.|近义词：prefabricated, pre-assembled, pre-compiled|反义词：post-build, custom-built


USAGE->
一般现在时:
The company prebuilds many components for its products. = 公司预先构建其产品的许多组件。
This software prebuilds several essential modules. = 此软件预先构建了几个必要的模块。


一般过去时:
They prebuilt the house's frame before installing the walls. = 他们在安装墙壁之前预先搭建了房屋的框架。
The developers prebuilt the application's core functionality. = 开发人员预先构建了应用程序的核心功能。


现在完成时:
We have prebuilt the database for faster access. = 我们已经预先构建了数据库以加快访问速度。
The team has prebuilt the entire system architecture. = 团队已经预先构建了整个系统架构。


过去完成时:
They had prebuilt the website's template before adding the content. = 他们在添加内容之前已经预先构建了网站的模板。
The engineers had prebuilt the engine components before assembly. = 工程师在组装之前已经预先构建了发动机组件。


将来时:
We will prebuild the project's foundation next week. = 我们下周将预先构建项目的框架。
They will prebuild the software's user interface before testing. = 他们在测试之前将预先构建软件的用户界面。


虚拟语气:
If we had prebuilt the system, it would have saved time. = 如果我们预先构建了系统，就会节省时间。
It would be better if they prebuilt the components beforehand. = 如果他们事先预先构建组件会更好。


被动语态:
The modules were prebuilt by the development team. = 模块是由开发团队预先构建的。
The entire structure was prebuilt using advanced software. = 整个结构是使用高级软件预先构建的。


疑问句:

否定句:
The system was not prebuilt; it was built from scratch. = 该系统不是预先构建的；它是从头开始构建的。
They did not prebuild the database; they built it live. = 他们没有预先构建数据库；他们是在线构建的。
