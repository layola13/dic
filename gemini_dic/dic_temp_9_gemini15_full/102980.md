jedis:/ˈdʒɛdɪs/| n. |杰迪斯（Java的Redis客户端）|A Java client for Redis, an in-memory data structure store, used as a database, cache, and message broker.|Example:  We used Jedis to connect to our Redis server and retrieve the cached data. 我们使用Jedis连接到我们的Redis服务器并检索缓存的数据。|近义词：Redisson (another Java Redis client)|反义词：无


USAGE->
This word is a proper noun and doesn't have typical verb conjugations like "what".  Therefore, grammatical tenses don't apply directly to "Jedis". The usage focuses on its application in Java code.

Example 1 (Instantiation):
`Jedis jedis = new Jedis("localhost", 6379);` = 创建一个连接到本地Redis服务器的Jedis实例。

Example 2 (Setting a value):
`jedis.set("mykey", "myvalue");` = 将值 "myvalue" 设置到键 "mykey"。
