machin: /məˈʃiːn/ | n. | 机器，机械装置 | A device with moving parts that performs a specific task; a complex system of interacting parts.  | Example: The washing machine is broken. 这台洗衣机坏了。 |近义词：device, apparatus, engine, mechanism | 反义词：handicraft, manual labor


USAGE->
一般现在时:
The machine runs smoothly. = 这台机器运转平稳。
This machine produces cars. = 这台机器生产汽车。

一般过去时:
The old machine broke down yesterday. = 那台旧机器昨天坏了。
The machine malfunctioned during the test. = 机器在测试过程中出现故障。

现在完成时:
The engineer has repaired the machine. = 工程师已经修理好了机器。
They have installed a new machine. = 他们安装了一台新机器。

过去完成时:
The machine had already stopped before we arrived. = 我们到达之前机器就已经停了。
We had used that machine for years before it broke. = 我们用了那台机器好几年才坏。

将来时:
They will replace the machine next week. = 他们下周将更换这台机器。
We will need a new machine soon. = 我们很快就会需要一台新机器。

虚拟语气:
If the machine worked properly, we would finish on time. = 如果机器正常工作，我们会按时完成。
I wish the machine were more efficient. = 我希望这台机器更高效。

被动语态:
The machine is being repaired. = 机器正在修理中。
The parts were manufactured by a specialized machine. = 零件是由一台特种机器制造的。

疑问句:
Is this machine working? = 这台机器工作吗？
What kind of machine is that? = 那是什么样的机器？

否定句:
The machine doesn't work. = 这台机器不工作。
That machine wasn't designed for this purpose. = 那台机器并非为此目的而设计的。
