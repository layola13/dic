onprogress:/ˈɑːnˈprɑːɡrɛs/ | n. | 进度，进行中 |  An event that fires periodically during the loading process of an object, such as an image or a video.  It provides information about the progress of the loading. | Example: The `onprogress` event allows us to display a progress bar while a file is uploading.  = `onprogress` 事件允许我们在文件上传时显示进度条。


The `onprogress` event is triggered repeatedly while the download or upload is in progress.  Each time it's triggered, it provides information such as the bytes loaded so far, the total bytes expected, and the current loading speed.  This information can be used to update a progress bar or provide other feedback to the user. = `onprogress`事件在下载或上传过程中会重复触发。每次触发时，它都会提供一些信息，例如到目前为止加载的字节数、预期加载的总字节数和当前加载速度。这些信息可以用来更新进度条或向用户提供其他反馈。

| 近义词:in progress, ongoing | 反义词:completed, finished | 词性:noun
