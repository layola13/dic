re-indexes:/ˌriːˈɪndɛksɪz/ | v. |重新建立索引 | To create a new index for (a database or similar system) or to update an existing index.  This usually involves rebuilding the data structure used for fast searching. | Example: The database administrator re-indexes the customer database nightly to ensure optimal query performance.  数据库管理员每晚都会重新索引客户数据库以确保最佳查询性能。 |近义词：reindex, rebuild index |反义词：delete index
