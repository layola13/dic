easy:/ˈiːzi/| adj.|容易的，简单的，舒适的 |Easy refers to something that is not difficult to do or understand, requiring little effort. It can also describe something comfortable or relaxed. |例句：The exam was easy.  考试很简单。|例句：This chair is easy to sit in. 这把椅子坐起来很舒服。|近义词：simple, effortless, uncomplicated, comfortable |反义词：difficult, hard, challenging, arduous


USAGE->
一般现在时:
The task is easy. = 这项任务很容易。
This problem is easy to solve. = 这个问题很容易解决。

一般过去时:
The test was easy yesterday. = 昨天的考试很简单。
The journey was easy. = 旅途很轻松。

现在完成时:
The work has been easy so far. = 到目前为止工作都很轻松。
It has been easy to learn this. = 学习这个很容易。

过去完成时:
The puzzle had been easy until the last piece. = 直到最后一块拼图之前，拼图都很容易。
Everything had been easy up to that point. = 到那时为止，一切都很顺利。

将来时:
The project will be easy to complete. = 这个项目将很容易完成。
The game will be easy to win. = 这场比赛很容易赢。

虚拟语气:
If the instructions were easy, I would have finished earlier. = 如果说明书容易理解，我就会早点完成了。
It would be easy if we had more help. = 如果我们有更多帮助，那就容易多了。

被动语态:
The questions were made easy for the students. = 这些问题对学生来说很容易。
The path was made easy by the workers. = 工人们把路修得很平坦。

疑问句:
Is the assignment easy? = 作业容易吗？
Is it easy to understand? = 这容易理解吗？

否定句:
The task isn't easy. = 这项任务不容易。
It wasn't easy to convince him. = 说服他并不容易。
