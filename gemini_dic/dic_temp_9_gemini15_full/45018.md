csv:/ˌsiːˌesˈviː/|abbr.|逗号分隔值|Comma Separated Values, a simple file format used to store tabular data.  Each line in a CSV file represents a row, and each value within a row is separated by a comma.  CSV files are commonly used for exchanging data between different software applications.|例句：I imported the customer data from a CSV file.我从CSV文件中导入了客户数据。|近义词：spreadsheet data, tabular data|反义词：N/A
