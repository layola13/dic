hashmaps:/ˈhæʃmæps/| n. |哈希映射，哈希表 |A hash map (also called a hash table) is a data structure that implements an associative array abstract data type, a structure that can map keys to values.  It uses a hash function to compute an index into an array of buckets or slots, from which the desired value can be found. | Example:  We used a hashmap to store and quickly retrieve user data. 我们用哈希映射来存储和快速检索用户数据。|近义词：hash table, associative array |反义词：无
