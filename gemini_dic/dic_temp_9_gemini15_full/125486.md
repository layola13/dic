multiprogrammed:/ˌmʌltiˈprəʊɡræmd/| adj. |多道程序的，多任务的 |  Relating to or denoting a computer system that can execute multiple programs concurrently.  This is achieved by rapidly switching between different programs, giving the illusion that they are running simultaneously.  This improves efficiency by maximizing the use of the CPU and other resources. | Example: The operating system supports multiprogrammed environments, allowing users to run multiple applications at once.  = 这个操作系统支持多道程序环境，允许用户同时运行多个应用程序。  The server is multiprogrammed to handle numerous requests concurrently. = 服务器采用多道程序设计来同时处理大量请求。|近义词：concurrent, parallel, multitasking |反义词：single-programmed, single-tasking
