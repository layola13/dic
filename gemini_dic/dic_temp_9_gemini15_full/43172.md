USAGE->
(C++本身不是一个动词，因此没有时态变化。下面的例子展示的是使用C++进行编程的动词的时态变化。)

一般现在时:
I write C++ code every day. = 我每天都编写C++代码。
He uses C++ for his projects. = 他在他的项目中使用C++。


一般过去时:
I wrote a C++ program yesterday. = 昨天我写了一个C++程序。
She used C++ in her previous job. = 她以前的工作中使用过C++。


现在完成时:
I have written several C++ applications. = 我已经编写了几个C++应用程序。
He has used C++ for over five years. = 他已经使用C++五年多了。


过去完成时:
I had written the code before the meeting. = 我在会议之前已经写好了代码。
She had used C++ extensively in her research. = 她在她的研究中广泛地使用了C++。


将来时:
I will write a C++ library next week. = 我下周将编写一个C++库。
