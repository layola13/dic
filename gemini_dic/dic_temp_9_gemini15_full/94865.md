in-action:/ɪnˈækʃən/| adj.| 实施中的；正在进行中的 |  Describes something that is currently happening or being done. It emphasizes the active process rather than a completed state. | Example: The new policy is in-action, and we're seeing positive results.  这项新政策正在实施中，我们看到了积极的结果。|近义词：active, underway, ongoing, operational|反义词：inactive, dormant, suspended, complete


USAGE->
一般现在时:
The plan is in action. = 计划正在实施中。
The project is in action now. = 项目现在正在进行中。

一般过去时:
The operation was in action last night. = 昨晚行动正在进行中。
The system was in action throughout the day. = 系统全天都在运行。

现在完成时:
The reforms have been in action for several months. = 改革已经实施了几个月了。
The new rules have been in action since January. = 新规自一月起就开始实施了。

过去完成时:
The emergency procedures had been in action before the fire broke out. = 火灾发生之前，紧急程序就已经启动了。
The backup system had been in action for hours before the main system failed. = 主系统故障前，备份系统已经运行了数小时。


将来时:
The new strategy will be in action next quarter. = 新战略将在下个季度实施。
The changes will be in action by the end of the year. = 到年底这些变化将全面实施。

虚拟语气:
If the plan were in action, we would see immediate results. = 如果计划正在实施，我们将看到立竿见影的效果。
I wish the new system were in action already. = 我希望新系统已经投入运行了。

被动语态:
The measures are being put in action. = 措施正在被实施。
The plan is being put in action gradually. = 计划正在逐步实施。

疑问句:
Is the emergency plan in action? = 紧急计划正在实施吗？
Is the new software in action yet? = 新软件投入运行了吗？

否定句:
The new rules are not in action yet. = 新规尚未生效。
The project is not in action at the moment. = 项目目前尚未启动。


PHRASE->
in action:  行动中；运作中
take action: 采取行动
call to action:号召性用语
put in action:付诸实施
course of action: 行动方案
