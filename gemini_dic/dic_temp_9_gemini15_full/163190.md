precompiles:/ˌpriːkəmˈpaɪlz/ | vt. |预编译|To precompile means to compile (translate source code into machine code) before the main compilation process.  This is often done to improve performance or to handle dependencies.  Precompilation can generate intermediate code that is then used in a later stage of compilation. |Example: The compiler precompiles the header files before compiling the main source code. 预编译器在编译主源代码之前预编译头文件。|近义词：preprocess, compile beforehand |反义词：post-compile  |动词
