multiprogram:/ˈmʌltiˌproʊɡræm/ | n. |多道程序,多程序设计 |The ability of an operating system to execute multiple programs concurrently.  This allows for better utilization of system resources and improved responsiveness.  | Example:  Modern operating systems are designed for multiprogramming, allowing users to run multiple applications simultaneously. = 现代操作系统设计用于多道程序设计，允许用户同时运行多个应用程序。|近义词：multitasking, parallel processing |反义词：single programming, sequential processing |名词

词源:
The word "multiprogram" is a combination of "multi-" (meaning many or multiple) and "program" (referring to a set of instructions executed by a computer).  It emerged with the development of operating systems capable of handling multiple programs concurrently.
