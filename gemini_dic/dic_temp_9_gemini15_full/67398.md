extern:/ˈɛkˌstɜːrn/ | adj. | 外部的；外部的；外来的 |  An `extern` declaration specifies that a variable or function is defined elsewhere, typically in another source file.  It tells the compiler that the variable or function exists but doesn't define its storage location or implementation. |  
Example 1: `extern int count;`  (Declares that an integer variable named `count` is defined elsewhere.)  外部声明一个名为count的整型变量。
Example 2: `extern void print_message(const char *msg);` (Declares that a function named `print_message` is defined elsewhere.) 外部声明一个名为print_message的函数。
|近义词:external, outside, foreign | 反义词:internal, intrinsic, local | adj.
