glitched:/ɡlɪtʃt/| adj. v.|故障的；卡住了；出现小故障|Something that has malfunctioned briefly or unexpectedly.  A temporary disruption or malfunction in a system, process, or piece of equipment.|例句：The game glitched and I lost all my progress. 游戏卡住了，我所有的进度都丢失了。 My computer glitched and froze. 我的电脑出现故障并冻结了。|近义词：malfunctioned, malfunction, bugged, flawed|反义词：functioning, working, operating


USAGE->
一般现在时:
The system glitches occasionally. = 系统偶尔会出故障。
The software glitches frequently. = 软件经常出现故障。


一般过去时:
The game glitched during the final level. = 游戏在最后一关时出现了故障。
The website glitched yesterday. = 网站昨天出故障了。


现在完成时:
The program has glitched several times today. = 程序今天已经出现过几次故障了。
My phone has glitched since the update. = 自从更新后，我的手机就一直出故障。


过去完成时:
The computer had glitched before I could save my work. = 在我保存工作之前，电脑已经出现故障了。
The printer had glitched repeatedly, causing delays. = 打印机反复出现故障，导致延误。


将来时:
The system might glitch during peak hours. = 系统在高峰时段可能会出现故障。
I think the software will glitch again. = 我认为软件还会再次出现故障。


虚拟语气:
If the network glitched again, we would lose connection. = 如果网络再次出现故障，我们将失去连接。
I wish the program hadn't glitched during the presentation. = 我希望程序在演示期间没有出现故障。


被动语态:
The game was glitched by a recent update. = 游戏因最近的更新而出现故障。
The system is often glitched by power surges. = 系统经常因电源波动而出现故障。


疑问句:
Did the software glitch? = 软件出故障了吗？
Has the computer glitched again? = 电脑又出故障了吗？


否定句:
The software didn't glitch during the test. = 软件在测试期间没有出现故障。
The system hasn't glitched all day. = 系统整天都没有出现故障。
