interpolatable: /ɪnˈtɜːrpəleɪtəbl/ | adj. | 可内插的，可内推的 | Capable of being interpolated; able to have intermediate values inserted.  This usually refers to data points or functions where values can be estimated between known data points.  | Example: The data set is interpolatable, allowing us to estimate values for the missing years.  这个数据集是可内插的，允许我们估计缺失年份的值。|近义词: estimable, inferable | 反义词: extrapolatable (in some contexts, as it refers to estimating values *outside* known data)
