onely:/ˈoʊnli/| adj.| 唯一的，仅有的，仅仅，只不过|only means being the single one of its kind; alone means being without the company of others.  Only emphasizes singularity or exclusivity, while alone emphasizes solitude. |例句：She is the only girl in her class who plays football.  她是她班里唯一一个踢足球的女孩。


例句：He only ate a small piece of cake. 他只吃了一小块蛋糕。


近义词：sole, single, unique


反义词：many, numerous, multiple


词性：形容词


USAGE->
一般现在时:
He is only a child. = 他只是一个孩子。
This is the only way to do it. = 这是唯一的方法。


一般过去时:
She was only joking. = 她只是开玩笑。
That was the only solution. = 那时唯一的解决办法。


现在完成时:
I have only just arrived. = 我刚刚到达。
They have only recently moved here. = 他们最近才搬到这里。


过去完成时:
I had only just finished when the phone rang. = 我刚做完电话就响了。
She had only seen him once before. = 她以前只见过他一次。


将来时:
We will only be there for a short time. = 我们只会在那里待一小会儿。
He will only know the truth later. = 他以后才会知道真相。


虚拟语气:
If only I had known! = 要是我早知道就好了！
I wish I only had more time. = 我真希望我有更多的时间。


被动语态:
Only a small portion was eaten. = 只吃了一小部分。
This problem is only solved in a few cases. = 这个问题只在少数情况下得到解决。


疑问句:
Is this the only option? = 这是唯一的选择吗？
Are you only going for a walk? = 你只是去散步吗？


否定句:
It's not only expensive, but it's also useless. = 它不仅昂贵，而且没用。
He is not only intelligent, but also kind. = 他不仅聪明，而且善良。


PHRASE->
only if = 只有如果...
not only...but also... = 不仅...而且...
only just = 刚刚
if only = 但愿
the only one = 唯一一个
