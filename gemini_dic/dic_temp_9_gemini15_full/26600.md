cache:/kæʃ/| n. v.|高速缓存，缓存；隐藏，贮藏|A cache is a temporary storage area used to speed up access to frequently used data.  It can refer to a storage location in a computer's memory or a hidden storage place.|例句：The web browser uses a cache to store frequently visited websites for faster loading.  网页浏览器使用缓存来存储经常访问的网站以加快加载速度。 The robbers hid the stolen jewels in a secret cache. 抢劫犯把偷来的珠宝藏在一个秘密的地方。|近义词：store, hoard, buffer, repository|反义词：discard, delete, remove|名词，动词


USAGE->
一般现在时:
The browser caches web pages. = 浏览器缓存网页。
He caches data for faster access. = 他缓存数据以加快访问速度。

一般过去时:
The system cached the files yesterday. = 系统昨天缓存了这些文件。

现在完成时:

过去完成时:
The server had cached the request before the failure. = 服务器在故障前已经缓存了请求。
They had cached the information prior to the presentation. = 他们在演示之前已经缓存了信息。

将来时:
