I'd:/aɪd/| pron. | 我会，我想，我将| I'd is a contraction of "I would" or "I had".  It is used in various grammatical contexts depending on the intended meaning.|例句：I'd love to go to the party. = 我很想参加派对。

I'd like some coffee. = 我想要一些咖啡。


近义词:I would, I had (in certain contexts)|反义词: N/A |词性:pronoun


USAGE->
一般现在时:
N/A  ("I'd" is not used in simple present tense)


一般过去时:
I'd seen that movie before. = 我以前看过那部电影。
I'd already eaten when you called. = 你打电话的时候我已经吃过了。


现在完成时:
I'd finished my work by then. = 到那时我已经完成了我的工作。
I'd been waiting for hours. = 我已经等了几个小时了。


过去完成时:
I'd already left when she arrived. = 她到达时我已经离开了。
I'd never seen anything like it before. = 我以前从未见过这样的事情。


将来时:
I'd go if I had the time. = 如果我有时间，我会去的。(Conditional)
I'd rather not go. = 我宁愿不去。(Conditional)


虚拟语气:
I'd rather you didn't tell anyone. = 我宁愿你谁也不告诉。(Conditional)
I'd have gone if I'd known. = 如果我知道的话，我就去了。(Past Conditional)


被动语态:
N/A ("I'd" itself is not a passive construction, but it can be used within passive sentences.)
The work would have been finished by now, I'd think. = 我想，工作现在应该已经完成了。(Conditional Passive)


疑问句:
Would you like some tea? (I'd like some tea is a response, not a question directly using "I'd") = 你想喝点茶吗？
What would you do if...? (I'd... is a response) = 如果你……你会怎么做？


否定句:
I wouldn't go there if I were you. = 如果我是你，我不会去那里。
I'd never do such a thing. = 我决不会做那样的事。


PHRASE->
I'd rather... = 我宁愿……
I'd better... = 我最好……
I'd like to... = 我想……
I'd love to... = 我非常乐意……
I'd prefer to... = 我宁愿……
