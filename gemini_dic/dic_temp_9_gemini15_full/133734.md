nonfunctional:/ˌnɒnˈfʌŋkʃənəl/| adj.|非功能性的，非功能的|Not related to the primary function or purpose; not working properly.|The nonfunctional requirements of the system include security and usability.|近义词：dysfunctional, ineffective, inoperative|反义词：functional, operational


USAGE->
一般现在时:
The software is nonfunctional. = 软件无法运行。
This button is nonfunctional. = 这个按钮失灵了。


一般过去时:
The website was nonfunctional yesterday. = 网站昨天无法运行。
The machine was nonfunctional due to a power outage. = 机器由于断电而无法运行。


现在完成时:
The system has been nonfunctional for hours. = 系统已经瘫痪数小时了。
The printer has been nonfunctional since last week. = 打印机从上周开始就无法使用了。


过去完成时:
The server had been nonfunctional before the technician arrived. = 技术人员到达之前，服务器就已经无法运行了。
The software had been nonfunctional for days before we discovered the problem. = 我们发现问题之前，软件已经无法运行好几天了。


将来时:
The network will be nonfunctional during the maintenance. = 网络在维护期间将无法使用。
The application will be nonfunctional unless the bug is fixed. = 除非修复错误，否则应用程序将无法运行。


虚拟语气:
If the system were nonfunctional, we would use the backup. = 如果系统无法运行，我们将使用备份。
I wish the software were not nonfunctional. = 我希望软件不会无法运行。


被动语态:
The feature was found to be nonfunctional. = 该功能被发现无法运行。
The network is reported to be nonfunctional. =据报道网络无法运行。


疑问句:
Is the application nonfunctional? = 应用程序无法运行吗？
Is this part of the system nonfunctional? = 系统的这部分是否无法运行？


否定句:
The system is not nonfunctional; it's just slow. = 系统并非无法运行；只是速度慢。
The device is not nonfunctional, it just needs a new battery. = 设备没有坏，只是需要新电池。


PHRASE->
nonfunctional requirements = 非功能性需求
nonfunctional testing = 非功能性测试
nonfunctional specification = 非功能性规范
nonfunctional design = 非功能性设计
nonfunctional aspect = 非功能性方面
