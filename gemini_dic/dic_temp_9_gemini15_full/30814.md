cfg:/ˈsiːˌdʒiːˈɛf/ | n. | 配置文件，配置；配置文件格式 | A configuration file (cfg) is a text file that contains settings or parameters for a program or system. It specifies how the software should behave, what options it should use, and other relevant information.  Different programs use different cfg file formats.  | Example:  The game's cfg file allows you to adjust graphics settings. 游戏的cfg文件允许你调整图形设置。 |近义词：configuration file, parameter file, settings file | 反义词：无 | 名词
