reducible:/rɪˈdjuːsəbl/| adj.| 可约化的；可简化的 | capable of being reduced; able to be simplified or made smaller or less complex.  | Example: The problem is reducible to a simpler equation.  这个问题可以简化为一个更简单的方程式。  |近义词：simplifiable,  decreasable |反义词：irreducible, unreducible


USAGE->
一般现在时:
The equation is reducible. = 这个方程式是可以约化的。
This problem is reducible to a simpler form. = 这个问题可以简化成更简单的形式。

一般过去时:
The complex data was reducible to a manageable size. = 复杂的数据被简化到可管理的规模。
The argument was reducible to a single point. = 论点可以简化为一个要点。

现在完成时:
The scientists have reduced the data to a manageable set. = 科学家们已将数据简化为可管理的集合。
We have reduced the complexity of the system. = 我们已经降低了系统的复杂性。

过去完成时:
They had reduced the problem to its essential components before the presentation. = 他们在演示之前已将问题简化为其基本组成部分。
The engineers had reduced the design to a more efficient model. = 工程师们已将设计简化为更有效的模型。

将来时:
The software will reduce the image size. = 该软件将减小图像大小。
They will reduce the cost of production. = 他们将降低生产成本。(Note:  While "reduce" is used, the adjective "reducible" focuses on the *potential* for reduction)

虚拟语气:
If the problem were reducible, it would be easily solved. = 如果问题可以简化，它将很容易解决。
I wish the workload were reducible. = 我希望工作量可以减少。

被动语态:
The data can be reduced by using this algorithm. = 使用此算法可以简化数据。
The complexity of the task was reduced by a new approach. = 通过一种新的方法降低了任务的复杂性。

疑问句:
Is the fraction reducible? = 这个分数可以约分吗？
Is the system reducible to a simpler model? = 这个系统可以简化为更简单的模型吗？

否定句:
The problem is not reducible to a simple solution. = 这个问题无法简化为简单的解决方案。
The equation is irreducible. = 这个方程式不可约。


PHRASE->
highly reducible =高度可约化的
easily reducible =易于简化的
reducible to = 可简化为
reducible in complexity = 在复杂性上可简化的
reducible to absurdity = 可归谬的
