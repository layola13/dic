running: /ˈrʌnɪŋ/ | v. adj. n. | 跑步；正在运行；连续不断 |  to move swiftly on foot; operating; continuous | 近义词：jogging, sprinting, operating, ongoing | 反义词：standing, still, stopped


USAGE->
一般现在时:
He runs every morning. = 他每天早上跑步。
The machine is running smoothly. = 机器运转平稳。


一般过去时:
She ran a marathon last year. = 她去年跑了一场马拉松。
The program ran without errors. = 程序运行无误。


现在完成时:
I have been running for an hour. = 我已经跑了1个小时了。
The factory has been running continuously for 10 years. = 这家工厂已经连续运转了10年。


过去完成时:
They had been running for miles before they stopped. = 他们在停下来之前已经跑了数英里。
The system had been running smoothly until the power outage. = 直到停电，系统运行才出现故障。


将来时:
I will run a 5k next week. = 下周我将跑一个5公里。
The software will be running on all new computers. = 该软件将在所有新电脑上运行。


虚拟语气:
If I ran faster, I would win. = 如果我跑得更快，我会赢。
I wish the project were running more efficiently. = 我希望这个项目运行效率更高。


被动语态:
The company is being run by a new CEO. = 公司由一位新任首席执行官管理。
The marathon is run annually. = 马拉松每年举行一次。


疑问句:
Is the engine running? = 发动机启动了吗？
Are you running late? = 你迟到了吗？


否定句:
The car isn't running. = 车子发动不了。
He doesn't run often. = 他不经常跑步。
