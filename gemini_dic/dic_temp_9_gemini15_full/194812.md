semaphoring:/ˈsɛməˌfɔːrɪŋ/ | n. |信号传递；信号控制；使用信号量 | The act of using semaphores to control access to shared resources in a concurrent system.  Semaphore is a synchronization primitive used to coordinate concurrent processes. Semaphores control access to a shared resource by allowing only a specified number of processes to access it at any one time.  | Example:  The program uses semaphoring to prevent race conditions when multiple threads access the database simultaneously. = 该程序使用信号量来防止多个线程同时访问数据库时发生竞争条件。
