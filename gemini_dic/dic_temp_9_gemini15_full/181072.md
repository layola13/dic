reindexed:/ˌriːˈɪndɛkst/| v.|重新编制索引|To reindex means to create a new index for something, usually a database or a set of data.  This often involves rebuilding the index from scratch or updating an existing one to reflect changes in the data.|例句：The database was reindexed to improve search performance. 数据库被重新编制索引以提高搜索性能。
The website was reindexed after a major content update. 网站在重大内容更新后被重新编制索引。|近义词：reorganize, rebuild, update|反义词：delete, remove
