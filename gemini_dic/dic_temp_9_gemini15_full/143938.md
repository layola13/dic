output:/ˈaʊtˌpʊt/| n. |输出，产量，结果 |The output of the factory has increased this year.  今年工厂的产量增加了。|近义词：production, result, yield |反义词：input


USAGE->
一般现在时:
The factory's output is high. = 工厂的产量很高。
Their output meets the demand. = 他们的产量满足了需求。

一般过去时:
The computer's output was incorrect. = 电脑的输出结果不正确。
Last year's output was lower. = 去年的产量更低。

现在完成时:
The company has increased its output significantly. = 公司已大幅提高了产量。
We have improved our output this quarter. = 本季度我们提高了产量。

过去完成时:
The output had already been reduced before the meeting. = 会议之前产量已经被降低了。
They had exceeded their output target. = 他们已经超过了他们的产量目标。

将来时:
The new machine will increase output. = 新机器将提高产量。
We will monitor the output closely next week. = 下周我们将密切监控产量。

虚拟语气:
If we had better equipment, the output would be higher. = 如果我们有更好的设备，产量就会更高。
I wish the output were more consistent. = 我希望产量更稳定一些。

被动语态:
The data is outputted to the screen. = 数据输出到屏幕上。
The results were outputted in a report. = 结果以报告的形式输出。

疑问句:
What is the output of this program? = 这个程序的输出是什么？
What was the total output for the month? = 这个月的总产量是多少？

否定句:
The system did not output the correct data. = 系统没有输出正确的数据。
There was no output from the machine. = 机器没有任何输出。
