setTimeout: /ˌsɛtˈtaɪmaʊt/ | v. | 设置超时 |  To schedule the execution of a function or piece of code after a specified delay (in milliseconds). This is a fundamental method in JavaScript and other programming languages for asynchronous operations. It doesn't block the main thread while waiting, allowing other tasks to continue executing. |  `setTimeout(function(){ alert("Hello!"); }, 3000);` This code will display an alert box saying "Hello!" after a 3-second delay (3000 milliseconds).  |  setInterval |  clearTimeout (for canceling a scheduled timeout) | v.
