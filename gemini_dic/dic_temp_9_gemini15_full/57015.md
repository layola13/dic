double-clicked:/ˈdʌbəlklɪkt/| v. |双击|To click a computer mouse button twice in quick succession.|例句：I double-clicked the icon to open the file. 我双击图标打开文件。  例句：She double-clicked the link, and the webpage opened. 她双击链接，网页就打开了。|近义词：click twice, tap twice|反义词:single-click


USAGE->
一般现在时:
He double-clicks the mouse frequently. = 他经常双击鼠标。
She double-clicks on every link. = 她双击每一个链接。

一般过去时:
I double-clicked the icon yesterday. = 我昨天双击了那个图标。
He double-clicked the button to submit the form. = 他双击按钮提交表单。

现在完成时:
I have double-clicked the file multiple times. = 我已经多次双击了这个文件。
She has double-clicked the image to enlarge it. = 她已经双击了图片来放大它。

过去完成时:
I had double-clicked the link before the page loaded. = 在页面加载之前，我已经双击了链接。
He had double-clicked the icon several times before it responded. = 在它响应之前，他已经多次双击了图标。

将来时:
I will double-click the button to start the program. = 我将双击按钮启动程序。
She will double-click the icon when she’s ready. = 当她准备好时，她会双击图标。

虚拟语气:
If I double-clicked the icon, the file would open. = 如果我双击图标，文件就会打开。
I wish I had double-clicked the save button. = 我真希望我双击了保存按钮。

被动语态:
The icon was double-clicked by the user. = 图标被用户双击了。
The button needs to be double-clicked to activate the function. = 需要双击按钮才能激活该功能。

疑问句:
Did you double-click the mouse button? = 你双击了鼠标按钮吗？
Have you ever double-clicked a link accidentally? = 你曾经不小心双击过链接吗？

否定句:
I didn't double-click the icon; I only single-clicked it. = 我没有双击图标；我只是单击了它。
She didn't double-click the button, so the program didn't start. = 她没有双击按钮，所以程序没有启动。
