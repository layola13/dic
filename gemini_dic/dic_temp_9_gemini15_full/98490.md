/ɪnˈstɔːlz/ | vt. |安装，设置|To put (software) into a computer system so that it is ready to be used.|The technician installed the new software on my computer. 技术员在我的电脑上安装了新软件。|近义词：fit, set up, put in|反义词：uninstall, remove


USAGE->
一般现在时:
He installs security software every month. 他每月安装安全软件。
They install new apps on their phones regularly.  他们定期在手机上安装新应用。


一般过去时:
I installed the game last night. 我昨晚安装了游戏。
She installed a new operating system on her laptop. 她在她笔记本电脑上安装了一个新的操作系统。


现在完成时:
We have installed the latest updates. 我们已经安装了最新的更新。
They have installed a new firewall system. 他们已经安装了一个新的防火墙系统。


过去完成时:
He had installed the program before the meeting started. 会议开始前他已经安装了程序。
She had installed the drivers before connecting the printer. 她连接打印机之前已经安装了驱动程序。


将来时:
I will install the printer tomorrow. 我明天会安装打印机。
They will install the new software next week. 他们下周会安装新软件。


虚拟语气:
If I had time, I would install that new software. 如果我有时间，我会安装那个新软件。
It would be better if we installed a more secure system. 如果我们安装一个更安全的系统会更好。


被动语态:
The software was installed quickly. 软件安装得很快。
The new system is being installed now. 新系统正在安装中。


疑问句:
Did you install the program correctly? 你正确安装程序了吗？
Have they installed the necessary drivers? 他们安装了必要的驱动程序了吗？


否定句:
I didn't install the update yet. 我还没安装更新。
He hasn't installed the security software. 他还没有安装安全软件。


PHRASE->
install a program 安装程序
install an application 安装应用程序
install a driver 安装驱动程序
