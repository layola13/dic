dto:/ˈdiːtiːˈoʊ/|/ˈdiːˌtiːˈoʊ/|abbr.|数据传输对象|Data Transfer Object, a simple object that carries data between processes.  It is often used in software architecture to transfer data between layers or services.  The DTO is typically a plain old Java object (POJO) or a similar construct in other programming languages.|Example:  The application uses a DTO to transfer user information from the database to the presentation layer.|近义词：data object, value object|反义词：无
