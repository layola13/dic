falsey:/ˈfɔːlsi/| adj.|虚假的，不真实的；无效的 |  A falsey value is a value that is considered false in a boolean context.  In many programming languages, this includes 0, 0.0, "", null, undefined, and NaN.  It's important to note that the specific falsey values can vary slightly across different programming languages. | Example: In JavaScript, if (0) evaluates to false because 0 is a falsey value.  In Python, an empty string "" is a falsey value. |近义词：falsy, untrue, invalid|反义词：truthy, true, valid|
