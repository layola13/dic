hell:/hɛl/ | n., interj. |地狱，极度痛苦；该死，胡说| A place or state of suffering after death; an expression of anger, frustration, or disbelief. |例句：The fire was hell.  这火简直是地狱。|例句： What the hell are you doing? 你到底在干什么？|近义词：inferno, damnation,  (interj.) darn, damn |反义词：heaven, paradise


USAGE->
一般现在时:
The heat is hell. = 这热度简直是地狱。
This job is hell. = 这份工作糟透了。


一般过去时:
It was hell getting there. = 到达那里真是太难了。
That party was hell. = 那次聚会糟糕透顶。


现在完成时:
This project has been hell. = 这个项目一直很糟糕。
Driving in this traffic has been hell. = 在这种交通状况下开车简直是折磨。


过去完成时:
Getting there had been hell. = 到达那里之前已经很艰难了。
The whole process had been hell. = 整个过程都非常糟糕。


将来时:
This will be hell to clean up. = 这将很难清理。
Tomorrow will be hell if it rains. = 如果明天下雨，那就糟糕了。


虚拟语气:
It would be hell if we missed the deadline. = 如果我们错过最后期限，那就糟了。
If I had to do it again, it would be hell. = 如果让我重做，那将会非常糟糕。


被动语态:
The situation was made hell by the bad weather. = 恶劣的天气使情况变得糟糕。
The project was described as hell by the team. = 项目组将项目形容为糟糕透顶。


疑问句:
What the hell is going on? = 到底发生了什么事？
What the hell are you talking about? = 你到底在说什么？


否定句:
It wasn't hell, it was just difficult. = 这不是地狱，只是有点困难。
It wasn't all hell; there were some good parts. = 也并非一无是处；也有一些好的方面。


PHRASE->
What the hell? = 搞什么鬼？
Hell yes! = 当然！
Go to hell! = 去死吧！（粗俗）
Hell of a = 非常，极其（例如：hell of a job）
Like hell! = 才不呢！（表示强烈否定）
