columns:/ˈkɑːləmz/ /ˈkɒləmz/| n. |列，栏，柱，专栏 |A vertical section of a table or page.  Columns are often used to organize information clearly. |例句：The spreadsheet is divided into several columns.  这个电子表格被分成几列。|例句：He writes a regular column for the newspaper. 他为报纸定期撰写专栏文章。|近义词：rows, lines, pillars|反义词：rows (in the context of tables)


USAGE->
一般现在时:
The table has five columns. = 这张桌子有五列。
This newspaper has a sports column. = 这份报纸有一个体育专栏。


一般过去时:
The architect designed beautiful columns for the building. = 建筑师为这座建筑设计了美丽的柱子。
The report was organized into clear columns and rows. = 报告被组织成清晰的列和行。


现在完成时:
They have added a new column to the database. = 他们已经向数据库添加了一列。
We have consistently read this newspaper's opinion columns. = 我们一直都在读这份报纸的评论专栏。


过去完成时:
The writer had already submitted his column before the deadline. = 作者在截止日期前已经提交了他的专栏文章。
They had rearranged the columns in the spreadsheet before printing. = 他们在打印前已经重新排列了电子表格中的列。


将来时:
We will add more columns to the table next week. = 我们下周将向表格添加更多列。
He will write a column about the election next month. = 他下个月将撰写一篇关于选举的专栏文章。


虚拟语气:
If the columns were wider, the data would be easier to read. = 如果列更宽，数据就更容易阅读。
I wish the columns were more evenly spaced. = 我希望列的间距更均匀一些。


被动语态:
The data is organized into columns and rows. = 数据被组织成列和行。
The columns were carefully measured before installation. = 在安装之前，这些柱子被仔细测量过。


疑问句:
How many columns are there in the spreadsheet? = 电子表格中有多少列？
Which column contains the sales figures? = 哪一列包含销售数据？


否定句:
The table doesn't have enough columns. = 这张桌子没有足够的列。
The report did not include a separate column for comments. = 该报告没有包含单独的评论列。
