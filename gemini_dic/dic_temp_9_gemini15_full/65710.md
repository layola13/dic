eventh:/ɪˈvɛnt/| n. |事件，大事；比赛项目 |An event is something that happens, especially something important.  It can also refer to a planned occasion or competition. |Example: The wedding was a joyous event. 婚礼是一场快乐的盛事。  The Olympic Games are a major sporting event. 奥运会是一项主要的体育赛事。|近义词：occurrence, happening, incident, affair |反义词：nothingness, non-event


USAGE->
一般现在时:
This event happens every year. = 这个事件每年都会发生。
What event is scheduled for tomorrow? = 明天安排了什么活动？


一般过去时:
A major event occurred last night. = 昨晚发生了一件大事。
What event did you attend yesterday? = 你昨天参加了什么活动？


现在完成时:
Many events have taken place this week. = 本周发生了许多事件。
Have you ever been to a similar event? = 你以前参加过类似的活动吗？


过去完成时:
The event had already started when I arrived. = 我到达时活动已经开始了。
Before the event, they had made thorough preparations. = 活动之前，他们做了充分的准备。


将来时:
A significant event will happen next month. = 下个月将发生一件重要的事情。
What event will you be attending next week? = 下周你将参加什么活动？


虚拟语气:
If the event hadn't been canceled, I would have gone. = 如果活动没有取消，我就去了。
I wish the event had been more exciting. = 我希望这个活动更令人兴奋。


被动语态:
The event was organized by the committee. = 这个活动是由委员会组织的。
The event is being filmed for television. = 这个活动正在被拍摄成电视节目。


疑问句:
What event are you talking about? = 你在说什么事件？
What event is most popular this year? = 今年最受欢迎的活动是什么？


否定句:
There was no significant event last week. = 上周没有发生什么重大的事件。
The event didn't go as planned. = 活动没有按计划进行。


PHRASE->
major event = 重大事件

current event = 时事
recent event = 近期事件
future event = 未来事件
