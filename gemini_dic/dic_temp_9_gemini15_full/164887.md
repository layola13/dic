prefunded:/ˌpriːˈfʌndɪd/| adj.|预先资助的，预付资金的|Previously funded; having funds allocated in advance.  This adjective describes something that has already received funding before a project or activity begins.|The prefunded research project allowed scientists to begin work immediately.|近义词：prepaid, advance-funded|反义词：unfunded, post-funded


USAGE->

一般现在时:
The program is prefunded by the government. = 这项计划由政府预先资助。
This project is prefunded, so we can start immediately. = 这个项目已经预先资助，所以我们可以立即开始。


一般过去时:
The project was prefunded last year. = 该项目去年已获得预先拨款。
The research was prefunded by a private foundation. = 这项研究由一个私人基金会预先资助。


现在完成时:
The initiative has been prefunded for the next five years. = 该倡议已获得未来五年的预先资助。
They have prefunded the entire operation. = 他们已经为整个行动预先拨款。


过去完成时:
The project had been prefunded before the construction began. = 在施工开始之前，项目已经获得预先资助。
We had prefunded the trip, so we didn't have to worry about expenses. = 我们已经预付了旅行费用，所以不用担心开支。


将来时:
The program will be prefunded by the university. = 该计划将由大学预先资助。
The project will be prefunded next quarter. = 该项目将在下个季度预先获得资金。


虚拟语气:
If the project were prefunded, we could start sooner. = 如果该项目已预先获得资金，我们就能早点开始。
I wish the grant had been prefunded; we're short on cash now. = 我希望这笔拨款已经预先拨付了；我们现在资金短缺。


被动语态:
The costs were prefunded by a generous donation. = 成本由一笔慷慨的捐款预先支付。
The expenses will be prefunded by the company. = 费用将由公司预先支付。


疑问句:
Is the project prefunded? = 这个项目是预先资助的吗？
Are the materials prefunded for the construction? = 建设的材料是预先资助的吗？


否定句:
The project is not prefunded; we need to find more funding. = 该项目没有预先资助；我们需要寻找更多资金。
The trip was not prefunded, so we had to pay as we went. = 旅行没有预先资助，所以我们不得不边走边付钱。
