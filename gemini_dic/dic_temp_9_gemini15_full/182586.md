reoccurred:/ˌriːəˈkɜːrd/| v. |再次发生；反复出现 |To happen or occur again.  This implies a previous occurrence. |The pain reoccurred after a few hours.  几小时后疼痛又复发了。|The symptoms reoccurred several weeks later.  症状几周后又复发了。|recur, happen again |cease, end |动词


USAGE->
一般现在时:
The problem reoccurs frequently. = 这个问题经常复发。
This pattern reoccurs throughout the book. = 这种模式贯穿全书。

一般过去时:
The earthquake reoccurred last night. = 地震昨晚再次发生。
The same mistake reoccurred again and again. = 同样的错误一次又一次地重演。

现在完成时:
The symptoms have reoccurred since the surgery. = 手术后症状反复出现。
The issue has reoccurred several times this year. = 今年这个问题已经反复出现几次了。

过去完成时:
The pain had reoccurred before he reached the hospital. = 他到达医院之前疼痛就复发了。
The problem had reoccurred several times before they found a solution. = 他们在找到解决方案之前，这个问题已经反复出现过好几次了。

将来时:
The doctor fears the illness will reoccur. = 医生担心这种疾病会复发。
We hope the problem won't reoccur. = 我们希望这个问题不会再次发生。


被动语态:
The event was reported to have reoccurred. = 据报道，事件再次发生。
It is feared that the situation will be reoccurred. = 人们担心这种情况会再次发生。


疑问句:
Will the problem reoccur? = 这个问题还会再次发生吗？
Has the pain reoccurred since your last visit? = 自上次就诊以来，你的疼痛是否复发了？


否定句:
The problem did not reoccur. = 问题没有再次发生。
The symptoms have not reoccurred since the treatment. = 治疗后症状没有再次出现。


PHRASE->
reoccur frequently = 频繁复发
reoccur unexpectedly = 出乎意料地再次发生
reoccur periodically = 定期复发
reoccur with alarming frequency = 以惊人的频率复发
reoccur in a different form = 以不同的形式再次出现
