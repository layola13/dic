instantiations:/ˌɪnstænʃiˈeɪʃənz/| n. |实例化，具体化，具体表现|The process of creating a specific instance of a class or template.  It refers to the creation of a concrete object from an abstract definition. This is common in object-oriented programming and other fields where abstract concepts are made real.|例句：The different instantiations of the algorithm showed varying performance. 算法的不同实例化显示出不同的性能。


例句：The program allows for multiple instantiations of the same object. 该程序允许对同一对象进行多次实例化。


近义词：realizations, embodiments, examples, instances|反义词：abstractions, generalizations|词性：名词
