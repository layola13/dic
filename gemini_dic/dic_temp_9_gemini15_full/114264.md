main:/meɪn/| adj. |主要的，最重要的|The main point is that we need more time.  主要的问题是我们需要更多时间。|近义词：principal, chief, primary |反义词：secondary, minor, subsidiary

USAGE->
一般现在时:
The main problem is the lack of funding. = 主要问题是资金不足。
This is the main reason for our success. = 这是我们成功的主要原因。

一般过去时:
The main challenge was overcoming the language barrier. = 主要挑战是克服语言障碍。
Her main concern was her family's well-being. = 她最关心的是家人的福祉。

现在完成时:
The main achievement has been the reduction in crime rates. = 主要成就是犯罪率的下降。
My main focus has been on completing this project. = 我的主要精力都放在完成这个项目上。

过去完成时:
The main difficulty had been securing the necessary permits. = 主要困难是获得必要的许可证。
His main regret had been not spending more time with his children. = 他最大的遗憾是没有花更多时间陪伴孩子。

将来时:
The main task will be to implement the new strategy. = 主要任务是实施新战略。
Our main priority will be ensuring the safety of our employees. = 我们的首要任务是确保员工的安全。

虚拟语气:
If the main argument were stronger, the essay would be better. = 如果主要论点更强有力，文章就会更好。
I wish the main character were more likeable. = 我希望主角更讨人喜欢。

被动语态:
The main points were summarized in the conclusion. = 主要要点在结论中进行了总结。
The main decisions are being made by the board of directors. = 主要决定是由董事会做出的。

疑问句:
What is the main ingredient in this dish? = 这道菜的主要配料是什么？
What is your main concern about this project? = 你对这个项目的主要担忧是什么？

否定句:
The main problem is not lack of funds, but lack of planning. = 主要问题不是资金不足，而是缺乏规划。
This is not the main issue we need to address right now. = 这不是我们现在需要解决的主要问题。
