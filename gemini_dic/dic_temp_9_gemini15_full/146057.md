overdue:/ˌoʊvərˈduː/ | adj. | 过期的；逾期的；早就该做的 |  Past the due date; delayed; long awaited.  |例句：The library book is overdue. 这本书已经过期了。

The payment is overdue.  付款已经过期了。|近义词：late, delinquent, delayed |反义词：timely, prompt


USAGE->
一般现在时:
The rent is overdue. = 房租逾期了。
This project is overdue. = 这个项目已经过期了。


一般过去时:
The bill was overdue. = 账单过期了。
The deadline was overdue. = 截止日期已经过了。


现在完成时:
My library books have been overdue for weeks. = 我的图书馆书籍已经过期几周了。
The report has been overdue for a long time. = 报告早就该交了。


过去完成时:
The project was already overdue when the new manager arrived. = 新经理来的时候，项目已经过期了。
The payment was overdue before the reminder was sent. = 在发出催款通知之前，付款就已经逾期了。


将来时:
The deadline will be overdue if we don't finish the project soon. = 如果我们不尽快完成项目，截止日期就会过期。
The book will be overdue by next week. = 这本书下周就要过期了。


虚拟语气:
If the report had been submitted on time, it wouldn't be overdue now. = 如果报告按时提交，现在就不会过期了。
I wish the payment wasn't overdue. = 我希望付款没有过期。


被动语态:
The task was overdue. = 任务已经过期了。
The payment was found to be overdue. = 付款被发现已经过期了。


疑问句:
Is the assignment overdue? = 作业过期了吗？
Is your library card overdue? = 你的借书证过期了吗？


否定句:
The payment is not overdue. = 付款没有过期。
The project is not yet overdue. = 项目还没过期。
