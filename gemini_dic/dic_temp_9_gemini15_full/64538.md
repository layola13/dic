errands:/ˈerəndz/ | n. |差事；杂务 | Short trips taken to do something, such as shopping or visiting someone.  |例句：I have a few errands to run this afternoon.  我今天下午有一些杂事要办。

例句：She spent the morning running errands for her mother. 她上午帮她母亲跑腿办事。


近义词：chores, tasks, commissions | 反义词：None


USAGE->
一般现在时:
I run errands every Saturday. = 我每个星期六都跑腿办事。
He does errands for his elderly neighbor. = 他为年迈的邻居跑腿。


一般过去时:
I ran errands yesterday. = 我昨天跑腿办事了。
She did some errands before lunch. = 她在午餐前办了一些杂事。


现在完成时:
I have run several errands this morning. = 我今天早上已经办了几件杂事。
She has done all her errands for the week. = 她已经完成了本周所有的杂事。


过去完成时:
I had run all my errands before the storm hit. = 在暴风雨来临之前，我已经办完了所有的差事。
She had done her errands before the shop closed. = 商店关门之前，她已经办完了她的差事。


将来时:
I will run errands tomorrow. = 我明天将要去办事。
He will do his errands after work. = 他下班后将要去办事。


虚拟语气:
If I had more time, I would run all my errands. = 如果我有更多时间，我会把所有的差事都办了。
I wish I had time to run my errands today. = 我希望今天有时间去办事。


被动语态:
The errands were run by my assistant. = 差事由我的助理去办。
The errands are being run by my assistant. = 差事正在由我的助理去办。


疑问句:
Have you run your errands yet? = 你办完差事了吗？
What errands do you need to run? = 你需要办哪些差事？


否定句:
I didn't run any errands today. = 我今天没有跑腿办事。
She hasn't run her errands yet. = 她还没去办事。


PHRASE->
run errands = 跑腿办事
go on errands = 去办事
do errands = 办差事
have errands to run = 有差事要办
send someone on errands = 派人去办事

词源:
The word "errand" comes from the Old English word "erend," meaning "message" or "journey."  Over time, its meaning shifted to refer to a short journey undertaken to perform a task or deliver something.
