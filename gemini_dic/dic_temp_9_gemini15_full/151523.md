parseInt: /pɑːrsɪnt/ | v. | 解析整数 |  Parses a string and returns an integer.  It attempts to convert a string into an integer. If the conversion fails, it returns `NaN` (Not a Number).  |  `let num = parseInt("10"); // num will be 10`  `let num2 = parseInt("10px"); // num2 will be 10` `let num3 = parseInt("ten"); // num3 will be NaN` |  Number, parseFloat |  String  |
