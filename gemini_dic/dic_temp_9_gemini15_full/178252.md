recompiles:/ˌriːˈkɑːmpaɪlz/| vt.|重新编译|To compile (a program or code) again. This is often done after making changes to the source code or when there are errors detected in the compilation process.  |例句：The programmer recompiled the code after fixing the bug. 程序员修复错误后重新编译了代码。
The system automatically recompiles the application whenever a change is detected.  系统在检测到更改时会自动重新编译应用程序。|近义词：recompile, rebuild|反义词：compile (in the sense of compiling for the first time)


USAGE->
一般现在时:
The compiler recompiles the source code every time it runs. = 编译器每次运行时都会重新编译源代码。
He recompiles the program frequently to check for errors. = 他经常重新编译程序以检查错误。


一般过去时:
She recompiled the program after making several changes. = 她修改了几处后重新编译了程序。
They recompiled the software to fix a critical bug. = 他们重新编译了软件来修复一个严重的错误。


现在完成时:
The developer has recompiled the application several times today. = 开发人员今天已经重新编译了应用程序几次。
We have recompiled the code and found the source of the problem. = 我们已经重新编译了代码并找到了问题根源。


过去完成时:
They had recompiled the code before the final testing. = 他们在最终测试之前已经重新编译了代码。
