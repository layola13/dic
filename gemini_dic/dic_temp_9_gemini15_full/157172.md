pinged:/pɪŋd/| v.|发出轻微的撞击声；发出（短促的）声音；|发出轻微的撞击声，发出（短促的）声音，（尤指通过网络）收到（信息）|The ball pinged off the wall.  球撞击墙壁发出轻微的声响。|My phone pinged, indicating a new message.我的手机发出提示音，表示收到新消息。|近义词：bounced, clicked, rang|反义词：thudded, crashed


USAGE->

一般现在时:
The bell pings every hour. = 钟每小时都会响一下。
The phone pings when you receive a message. = 收到信息时手机会发出提示音。


一般过去时:
The stone pinged against the window. = 石子打在窗户上发出清脆的响声。
The notification pinged, startling me. = 通知提示音响起，吓了我一跳。


现在完成时:
My phone has pinged several times this morning. = 我手机今天早上已经响了好几次了。
The server has pinged back the requested data. = 服务器已经返回请求的数据。


过去完成时:
The email had pinged just before the meeting started. = 邮件在会议开始前刚刚发出提示音。
The system had pinged an error message earlier. = 系统之前已经发出错误信息提示。


将来时:
The alarm will ping at 6:00 AM. = 闹钟将在早上6点发出提示音。
The system will ping you when the process is complete. = 流程完成后，系统会给你发送提示。


虚拟语气:
If the system pinged, I would know there was a problem. = 如果系统发出提示音，我就知道有问题了。
I wish the notification would ping more quietly. = 我希望通知提示音能更轻一些。


被动语态:
The user was pinged by the server. = 用户被服务器提示。
The bell was pinged by the caretaker. = 看门人敲响了钟。


疑问句:
Did the server ping? = 服务器有提示吗？
Has your phone pinged yet? = 你的手机响了吗？


否定句:
The system didn't ping. = 系统没有发出提示音。
The ball didn't ping when it hit the ground. = 球落地时没有发出响声。


PHRASE->
ping someone =  给某人发消息
ping a server =  探测服务器
ping the network =  探测网络
ping back = 回复
ping off = 反弹
