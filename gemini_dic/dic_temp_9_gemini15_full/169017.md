program:/ˈproʊɡræm/|/ˈprəʊɡræm/| n. v. |程序；节目；规划；方案；训练；编写程序|A set of instructions that a computer follows to perform a specific task; a plan or schedule of activities; a broadcast on television or radio; a course of study or training.|例句：The computer program crashed.  电脑程序崩溃了。|例句：The television program is very popular. 这个电视节目很受欢迎。|近义词：plan, schedule, scheme, project, course|反义词：chaos, disorder


USAGE->
一般现在时:
He programs computers. = 他编写电脑程序。
The school programs activities for students. = 学校为学生安排活动。

一般过去时:
She programmed the VCR to record the show. = 她设定录像机录制节目。
They programmed the robot to perform specific tasks. = 他们为机器人编写程序执行特定任务。

现在完成时:
I have programmed this software for years. = 我已经编写这个软件很多年了。
They have programmed the new features into the system. = 他们已经将新功能编入系统。

过去完成时:
He had programmed the robot before the demonstration. = 他在演示之前已经编写了机器人的程序。
She had already programmed the meeting into her schedule. = 她已经把会议安排到她的日程表里了。


将来时:
We will program the new game next month. = 我们下个月将编写新的游戏程序。
They will program the lights to change color. = 他们将编程使灯光变色。


虚拟语气:
If I had programmed the robot correctly, it wouldn't have malfunctioned. = 如果我正确地编写了机器人的程序，它就不会发生故障了。
I wish I had programmed the system earlier. = 我希望我早点编写程序。

被动语态:
The computer is programmed to run automatically. = 电脑被编程为自动运行。
The event was programmed by a professional organizer. = 这个活动是由专业的组织者策划的。

疑问句:
What programs are on television tonight? = 今晚电视上有什么节目？
How is the robot programmed? = 机器人是怎样编程的？

否定句:
I cannot program this software. = 我无法编写这个软件。
They did not program the event properly. = 他们没有妥善策划这个活动。
