ran:/ræn/| v.|跑，奔；运转；延伸；持续|to move swiftly on foot; to operate or function; to extend or stretch; to continue without interruption|例句：The dog ran across the street.  狗跑过了街道。

例句：The machine ran smoothly. 机器运转顺利。

近义词：run, dash, sprint, jog | 反义词：walk, crawl, stop

词性：动词


USAGE->
一般现在时:
He runs every morning. 他每天早上跑步。
The river runs to the sea. 这条河奔向大海。


一般过去时:
She ran a marathon last year. 她去年跑了一场马拉松。
The engine ran for hours. 发动机运转了好几个小时。


现在完成时:
I have run five miles today. 我今天已经跑了五英里。
The company has run successfully for ten years. 这家公司已经成功运营了十年。


过去完成时:
They had run out of money before the project finished.  项目完成前他们已经钱花光了。
The clock had run down. 钟停了。


将来时:
I will run in the race next week. 我下周将参加比赛。
The program will run automatically. 程序将自动运行。


虚拟语气:
If I ran faster, I would win. 如果我跑得更快，我会赢。
I wish I had run further. 我希望我跑得更远。


被动语态:
The company is run by my father. 公司由我父亲经营。
The race was run fairly. 比赛进行得很公平。


疑问句:
Does he run every day? 他每天都跑步吗？
How long did the machine run? 机器运转了多久？


否定句:
I didn't run today. 我今天没跑步。
The car didn't run well. 车子运转不好。
