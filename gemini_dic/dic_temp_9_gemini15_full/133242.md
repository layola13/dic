nonexecutable: /ˌnɒnˈɛksɪkjuːtəbl/ | adj. | 不可执行的 | Not capable of being executed or run as a program.  It refers to a file or code that cannot be directly processed by a computer's processor.  | Example: The downloaded file is nonexecutable; you need to install it first. 下载的文件不可执行；您需要先安装它。 | 近义词:  ineffective, inoperable, unusable | 反义词: executable, runnable


USAGE->
This word is primarily an adjective and doesn't have verb conjugations like "what".  Therefore, typical verb tenses are not applicable.  The usage focuses on describing the state of a file or code.


PHRASE->
nonexecutable file: 不可执行文件
nonexecutable code: 不可执行代码
nonexecutable script: 不可执行脚本
nonexecutable application: 不可执行应用程序
mark as nonexecutable: 标记为不可执行
