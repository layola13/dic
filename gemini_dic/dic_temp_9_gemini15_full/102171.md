it:/ɪt/| pron.|它，这，那|It refers to something previously mentioned or easily understood from the context. It can also be used as a placeholder for an unspecified thing or event.  |例句：It is a beautiful day.  这是一个美好的日子。|例句：I saw a bird; it was blue. 我看到一只鸟；它是蓝色的。|近义词：this, that, he, she|反义词：无


USAGE->
一般现在时:
It is raining. = 正在下雨。
It is a sunny day. = 今天阳光明媚。

一般过去时:
It rained yesterday. = 昨天下了雨。
It was cold last night. = 昨晚很冷。

现在完成时:
It has rained a lot this month. = 这个月雨下得很多。
It has been a long day. = 今天很漫长。

过去完成时:
It had rained before we left. = 我们离开之前下过雨。
It had been a difficult week. = 那是一个艰难的星期。

将来时:
It will rain tomorrow. = 明天会下雨。
It will be hot this summer. = 今年夏天会很热。

虚拟语气:
If it were sunny, we would go to the beach. = 如果天气晴朗，我们就去海滩。
I wish it were Friday. = 我希望是星期五。

被动语态:
It was said that he was innocent. = 有人说他是无辜的。
It is believed that he will win. = 人们相信他会赢。

疑问句:
Is it raining? = 正在下雨吗？
Is it far from here? = 离这里远吗？

否定句:
It is not raining. = 没有下雨。
It wasn't cold yesterday. = 昨天不冷。


PHRASE->
It's up to you. = 由你决定。
It's a pity. = 真可惜。
It's a shame. = 真可惜。
It depends. = 看情况。
It doesn't matter. = 没关系。
