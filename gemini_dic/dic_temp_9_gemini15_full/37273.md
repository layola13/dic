command-driven:/kəˈmændˌdrɪvən/| adj. | 命令驱动的；命令行驱动的 |  Adjective describing a system, program, or interface that is controlled primarily by user-entered commands.  Unlike graphical user interfaces (GUIs), command-driven interfaces rely on text-based commands to execute actions.  This typically involves typing specific keywords or phrases into a command prompt or terminal. | Example:  Many early computer operating systems were command-driven, requiring users to type commands to perform tasks.  Many server management tasks are still often command-driven for efficiency and precision. |近义词：command-line, text-based, console-based | 反义词：graphical user interface (GUI), menu-driven
