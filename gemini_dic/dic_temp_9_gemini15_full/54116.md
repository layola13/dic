disenable:/dɪsɪˈneɪbl/| vt.|使失效；使不能；禁用|To make something unable to function or be used; to deactivate.|例句：The administrator disabled the user account. 管理员禁用了用户帐户。  The virus disabled my computer's firewall. 病毒禁用了我的电脑防火墙。|近义词：deactivate, disable, invalidate|反义词：enable, activate|动词


USAGE->
一般现在时:
He disables the alarm system every night. 他每晚都禁用报警系统。
The company disables employee accounts after termination. 公司在员工离职后会禁用其帐户。


一般过去时:
They disabled the emergency brakes during the test. 他们在测试期间禁用了紧急制动器。
She disabled the software update notification. 她禁用了软件更新通知。


现在完成时:
The IT department has disabled access to the server. IT 部门已禁用对服务器的访问。
The authorities have disabled several suspicious websites. 当局已禁用几个可疑网站。


过去完成时:
They had disabled the old system before installing the new one. 他们在安装新系统之前已禁用了旧系统。
The technician had disabled the network connection before starting repairs. 技术人员在开始维修之前已禁用网络连接。


将来时:
The update will disable the old features. 更新将禁用旧功能。
The security team will disable the compromised account. 安全团队将禁用被入侵的帐户。


虚拟语气:
If I disabled the firewall, my computer would be vulnerable. 如果我禁用了防火墙，我的电脑就会变得脆弱。
I wish I had disabled the automatic updates earlier. 我真希望我早点禁用了自动更新。


被动语态:
The account was disabled due to suspicious activity. 由于可疑活动，该帐户已被禁用。
The feature will be disabled in the next software release. 此功能将在下一个软件版本中被禁用。


疑问句:
Was the printer disabled by accident? 打印机是不是意外被禁用了？
Has the security system been disabled? 安全系统已被禁用了吗？


否定句:
The software did not disable the ad blocker. 该软件没有禁用广告拦截器。
They did not disable the automatic backup function. 他们没有禁用自动备份功能。


PHRASE->
disable access 禁用访问
disable a function 禁用功能
disable an account 禁用帐户
disable a device 禁用设备
disable a program 禁用程序
