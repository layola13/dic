default's:/dɪˈfɔːlts/| n. | 默认值，违约 | The defaults of the software are usually acceptable.  软件的默认设置通常是可以接受的。|近义词：setting, standard, presetting |反义词：customization, specification


USAGE->
一般现在时:
The software defaults to a specific font. = 软件默认为某种特定字体。
What are the default settings for this printer? = 这台打印机的默认设置是什么？


一般过去时:
The system defaulted to the backup file. = 系统默认为备份文件。
The payment defaulted, leading to penalties. = 付款违约，导致罚款。


现在完成时:
The program has defaulted to the previous version. = 程序已恢复到之前的版本（默认为之前的版本）。
He has defaulted on his loan payments. = 他拖欠了贷款。


过去完成时:
The computer had defaulted to safe mode. = 电脑已默认进入安全模式。
Before the trial, the defendant had defaulted on several court appearances. = 在审判之前，被告缺席了几次出庭。


将来时:
The application will default to the user's last saved settings. = 应用程序将默认为用户上次保存的设置。
If payment isn't received, the account will default. = 如果收不到付款，账户将进入违约状态。


虚拟语气:
If the system defaulted, we would have lost all our data. = 如果系统出现默认值错误，我们就会丢失所有数据。
I wish the program hadn't defaulted to this option. = 我希望程序没有默认为这个选项。


被动语态:
The settings were defaulted to the factory presets. = 设置被恢复为出厂预设值。
The account was defaulted on due to non-payment. = 由于未付款，账户进入了违约状态。


疑问句:
What does the system default to if the internet connection fails? = 如果互联网连接失败，系统会默认什么？
Does this software default to English? = 这个软件默认语言是英语吗？


否定句:
The program didn't default to the expected behaviour. = 程序并没有按照预期行为进行默认操作。
The client did not default on their contract. = 客户没有违反合同。

in default = 违约
