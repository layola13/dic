jogged:/dʒɑːɡd/ | v. |慢跑；轻推；轻撞；|to run at a steady, gentle pace.|I jogged around the park this morning.  我今天早上在公园里慢跑了。|He jogged his memory to recall the details. 他努力回忆细节。|run, trot, sprint |  walk, stop |动词


USAGE->
一般现在时:
I jog every morning. = 我每天早上慢跑。
He jogs in the park. = 他在公园里慢跑。

一般过去时:
I jogged yesterday. = 我昨天慢跑了。
She jogged to the store. = 她慢跑到商店。

现在完成时:
I have jogged three times this week. = 我这周已经慢跑了三次。
He has jogged a marathon before. = 他以前跑过马拉松。

过去完成时:
I had jogged five kilometers before I stopped. = 我停下来之前已经慢跑了五公里。
They had jogged for an hour when it started to rain. = 他们慢跑了一个小时后开始下雨了。

将来时:
I will jog tomorrow. = 我明天会慢跑。
We will jog along the beach. = 我们会在海滩上慢跑。

虚拟语气:
If I had time, I would jog every day. = 如果我有时间，我每天都会慢跑。
I wish I could jog faster. = 我希望我能跑得更快。

被动语态:
(Jog is rarely used in the passive voice.  A more appropriate example using a related verb would be needed.) The race was run quickly. = 比赛跑得很快。(替代例句)
The ball was bumped by the player.  球被球员轻撞了一下。(替代例句)


疑问句:
Do you jog regularly? = 你经常慢跑吗？
Does he jog before work? = 他上班前慢跑吗？

否定句:
I don't jog very often. = 我不经常慢跑。
She didn't jog yesterday. = 她昨天没慢跑。


PHRASE->
jog along = 慢慢地进行；进展顺利
jog memory = 唤起记忆
jog the memory = 提醒某人想起某事
jog someone's memory = 提醒某人某事
to jog somebody's memory = 提醒某人
