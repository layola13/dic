offline: /ˌɒfˈlaɪn/ /ˌɔːfˈlaɪn/ | adj., adv. | 离线的；脱机的；不在线的 | Not connected to a computer network or the internet.  |  例句：I'm working offline today because the internet is down. 今天我离线工作，因为网络瘫痪了。


例句：The database is currently offline for maintenance.  数据库目前离线维护中。


近义词：disconnected, unconnected, unavailable


反义词：online, connected


词性：形容词，副词


USAGE->
一般现在时:
The system is offline. = 系统离线了。
We work offline sometimes. = 我们有时离线工作。


一般过去时:
The server was offline yesterday. = 服务器昨天离线了。
They worked offline during the power outage. = 停电期间他们离线工作。


现在完成时:
The website has been offline for hours. = 网站已经离线数小时了。
We have worked offline several times this week. = 本周我们已经几次离线工作了。


过去完成时:
The system had been offline before the technician arrived. = 技师到达之前系统已经离线了。
They had worked offline for days before the internet was restored. = 互联网恢复之前，他们已经离线工作了几天。


将来时:
The system will be offline for updates tomorrow. = 系统明天将离线进行更新。
We will work offline if the internet connection is unstable. = 如果互联网连接不稳定，我们将离线工作。


虚拟语气:
If the internet were offline, we would use the offline version. = 如果互联网离线，我们将使用离线版本。
I wish the software had an offline mode. = 我希望该软件有离线模式。


被动语态:
The application was taken offline by the developers. = 该应用程序被开发者下线了。
The system is being taken offline for scheduled maintenance. = 系统正在被下线进行计划维护。


疑问句:
Is the server offline? = 服务器离线了吗？
Are you working offline today? = 你今天离线工作吗？


否定句:
The system is not offline; it's just slow. = 系统没有离线；只是速度慢。
We are not working offline; we have a good internet connection. = 我们没有离线工作；我们的互联网连接良好。


PHRASE->
go offline = 离线
offline mode = 离线模式
offline backup = 离线备份
offline payment = 离线支付
work offline = 离线工作


词源:  The word "offline" is a compound word formed from "off" (away from) and "line" (a connection, in this context a communication line). It originally referred to being disconnected from a telephone line, but the meaning extended to computer networks and the internet.
