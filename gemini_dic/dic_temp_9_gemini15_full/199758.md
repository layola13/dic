siri:/ˈsɪri/| Proper Noun|Siri|Siri是苹果公司开发的智能个人助理，它能够通过自然语言理解用户指令并执行相应的任务，例如拨打电话、发送短信、设置提醒、播放音乐、搜索信息等。Siri使用语音识别技术和人工智能算法来理解用户的请求，并提供相应的反馈。|Example: I asked Siri to set an alarm for tomorrow morning. 我让Siri设置了明天的闹钟。|近义词：Alexa, Google Assistant, Cortana|反义词：无


USAGE->
This word is a proper noun and doesn't have verb conjugations like "What".  The usage examples below demonstrate how Siri is used in sentences.

Declarative Sentence:
Siri is a virtual assistant. = Siri是一个虚拟助手。

Siri is on my iPhone. = Siri在我的iPhone上。

Interrogative Sentence:
Is Siri available offline? = Siri离线可用吗？

Can Siri translate languages? = Siri可以翻译语言吗？


PHRASE->
There are no common phrases directly using "Siri" as a core component;  phrases involving Siri would be contextual commands given to Siri itself.  For instance:

"Siri, set a timer for 10 minutes." = Siri，设置一个10分钟的计时器。
"Siri, what's the weather like today?" = Siri，今天天气怎么样？
"Siri, play my favorite song." = Siri，播放我最喜欢的歌。
"Siri, call Mom." = Siri，打电话给妈妈。
"Siri, navigate to the nearest coffee shop." = Siri，导航到最近的咖啡店。
