cached:/kæʃt/| adj. |缓存的，高速缓存的|Stored in a cache (a temporary storage area) for quick retrieval.  This typically refers to data stored temporarily to speed up access.  |Example: The cached image loaded instantly.  缓存的图像立即加载。|近义词：stored, buffered,  |反义词：unstored, fresh


USAGE->
一般现在时:
The browser caches images for faster loading. = 浏览器缓存图像以加快加载速度。
This system caches frequently accessed data. = 此系统缓存频繁访问的数据。


一般过去时:
The program cached the results of the calculation. = 程序缓存了计算结果。

现在完成时:

I have cached all the necessary files. = 我已经缓存了所有必要的文。


过去完成时:
