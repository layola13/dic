metalanguages:/ˈmɛtəlæŋɡwɪdʒɪz/ | n. |元语言；超语言 | A language used to describe or analyze another language.  It provides a framework for discussing the structure, meaning, and use of the object language. For example, linguistic theory is a metalanguage used to describe the grammatical rules of natural languages. | Example:  Linguistics uses a metalanguage to describe the rules of grammar.  语言学使用元语言来描述语法规则。|近义词:formal language, descriptive language |反义词:object language
