editable:/ˈedɪtəbl/| adj.| 可编辑的，可修改的 | capable of being edited or modified.  |  This document is editable; you can change the text. 这个文档是可编辑的；你可以更改文本。 | modifiable, changeable, alterable | uneditable, fixed, immutable | 形容词


USAGE->
一般现在时:
This document is editable. = 这个文档是可编辑的。
The settings are editable. = 这些设置是可编辑的。

一般过去时:
The file was editable before the update. = 更新之前，该文件是可编辑的。
The document wasn't editable yesterday. = 昨天这个文档不可编辑。

现在完成时:
The document has been editable since the last update. = 自上次更新以来，该文档一直可编辑。
The settings have become editable. = 设置已变得可编辑。


过去完成时:
The form had been editable until the deadline. = 表格在截止日期之前一直是可编辑的。
The document hadn't been editable before the software upgrade. = 在软件升级之前，文档不可编辑。

将来时:
The software will be editable next week. = 软件下周将可编辑。
The document will not be editable after submission. = 提交后文档将不可编辑。

虚拟语气:
If the document were editable, I would change it. = 如果文档可编辑，我会更改它。
I wish this image were editable. = 我希望这张图片是可编辑的。


被动语态:
The file is being edited. = 文件正在被编辑。(Although this uses "being edited", it's describing the state of something that *is* editable)
The document can be edited by anyone with access. = 任何有权限的人都可以编辑该文档。


疑问句:
Is this image editable? = 这张图片可编辑吗？
Are the settings editable in this version? = 在这个版本中设置可编辑吗？

否定句:
This section is not editable. = 本节不可编辑。
The document is currently uneditable. = 文档当前不可编辑。


PHRASE->
editable text = 可编辑文本
editable field = 可编辑字段
editable content = 可编辑内容
editable document = 可编辑文档
editable region = 可编辑区域
