instances:/ˈɪnstənsɪz/| n.|例子，事例，实例|Examples or occurrences of something.|The teacher gave several instances of student plagiarism. 老师举了几个学生剽窃的例子。|examples, cases, occurrences|non-examples, abstractions|名词


USAGE->
一般现在时:
This software provides many instances of error handling. = 这个软件提供了许多错误处理的例子。
These instances show a clear trend. = 这些例子显示了一个清晰的趋势。


一般过去时:
Several instances of fraud were uncovered during the audit. = 审计过程中发现了数起欺诈行为。
Past instances show this is not an unusual occurrence. = 过去的例子表明，这不是一件不寻常的事情。


现在完成时:
There have been numerous instances of cyberattacks recently. = 最近发生了多次网络攻击事件。
We have seen many instances where this approach has failed. = 我们已经看到很多这种方法失败的例子。


过去完成时:
Before the meeting, there had been several instances of disagreement. = 会议之前，曾发生过几次分歧。
By the end of the year, there had been few instances of customer complaints. = 到年底为止，客户投诉的案例很少。


将来时:
In the future, we anticipate fewer instances of this problem. = 未来，我们预计这类问题会减少。
There will be many instances where you need to use this skill. = 将会有很多情况需要你运用这项技能。


虚拟语气:
If there were more instances of cooperation, the project would succeed. = 如果有更多合作的例子，项目就会成功。
I wish there were fewer instances of violence in the world. = 我希望世界上暴力事件少一些。


被动语态:
Many instances of the bug were reported by users. = 许多错误实例由用户报告。
These instances of misconduct were investigated thoroughly. = 这些不当行为的例子受到了彻底的调查。


疑问句:
Are there any instances of this happening before? = 以前有过这样的例子吗？
How many instances of this error have you encountered? = 你遇到过多少次这种错误？


否定句:
There were no instances of this problem in the previous version. = 在之前的版本中没有出现这个问题。
There are no known instances of this disease being transmitted through the air. = 目前尚无已知的这种疾病通过空气传播的例子。
