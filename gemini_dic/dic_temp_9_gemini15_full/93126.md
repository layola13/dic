iframes:/ˈaɪfreɪmz/| n. |内嵌框架|An iframe (inline frame) is an HTML element that allows you to embed another HTML document within the current HTML document.  It's essentially a window within a window, displaying content from a different source.  This is useful for embedding videos, interactive maps, or other web content from a different domain or server. |Example:  This website uses iframes to display videos from YouTube.  这个网站使用内嵌框架来显示来自YouTube的视频。|近义词: embedded frame, subframe |反义词:  N/A
