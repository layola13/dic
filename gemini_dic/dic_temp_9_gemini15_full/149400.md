package:/ˈpækɪdʒ/ | n. v. |包裹；套件；打包；捆扎 | A collection of things wrapped or boxed together; a bundle; a set of related items offered as a unit.  |例句：I received a package in the mail today. 我今天收到一个包裹。  He packaged the software with useful tutorials. 他将软件与有用的教程打包在一起。|近义词：bundle, parcel, kit, set |反义词：unpack, unbundle


USAGE->
一般现在时:
He packages the goods carefully. = 他仔细地包装货物。
She packages the software for distribution. = 她打包软件以便分发。


一般过去时:
They packaged the furniture for shipping. = 他们打包家具以便运输。
I packaged all my belongings before moving. = 我搬家前打包了所有我的东西。


现在完成时:
He has packaged and shipped the order. = 他已经打包并发货了订单。
She has packaged her presentation slides neatly. = 她已经把演示文稿的幻灯片整理好打包了。


过去完成时:
They had packaged all the gifts before the party started. = 派对开始前，他们已经把所有的礼物都打包好了。
She had packaged the documents for the meeting. = 她已经把会议的文件打包好了。


将来时:
We will package the products tomorrow. = 我们明天将打包产品。
They will package the software with updated features. = 他们将把软件与更新的功能一起打包。


虚拟语气:
If I had more time, I would package the items more carefully. = 如果我有更多时间，我会更仔细地包装这些物品。
It would be better if they packaged the products in stronger boxes. = 如果他们用更结实的箱子包装产品会更好。


被动语态:
The goods are packaged securely for transport. = 货物被安全地包装好以便运输。
The software is packaged with a user manual. = 软件与用户手册一起打包。


疑问句:
How are the products packaged? = 产品是如何包装的？
Are the items packaged individually or together? = 这些物品是单独包装还是一起包装？


否定句:
They didn't package the fragile items properly. = 他们没有正确地包装易碎物品。
She didn't package the order in time for delivery. = 她没有及时打包订单以便送货。
