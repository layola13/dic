in-operation:/ɪn ˌɑːpəˈreɪʃən/| adj.|正在运行的，在运转中|in operation means functioning, working, or being used.  It describes something that is currently active and performing its intended purpose.|例句：The new system is in operation and running smoothly.  新的系统正在运行，运转顺利。|例句：The factory has been in operation for over 50 years. 这家工厂已经运营了50多年。|近义词：functioning, running, active, working|反义词：out of operation, inactive, broken, defunct|词性：形容词


USAGE->
一般现在时:
The machine is in operation. = 机器正在运行。
The new policy is in operation. = 新政策正在实施。

一般过去时:
The system was in operation yesterday. = 系统昨天还在运行。
The old engine was in operation for many years. = 旧引擎运行了很多年。

现在完成时:
The factory has been in operation since 1970. = 这家工厂自1970年以来一直在运营。
The software has been in operation for several months. = 该软件已经运行了几个月。

过去完成时:
The bridge had been in operation for decades before it collapsed. = 那座桥在倒塌之前已经运行了几十年。
The program had been in operation for a year before the bugs were discovered. = 该程序运行一年后才发现错误。

将来时:
The new airport will be in operation next year. = 新机场将于明年投入运营。
The upgraded system will be in operation by the end of the month. = 升级后的系统将在月底投入运行。

虚拟语气:
If the system were in operation, we could process the data. = 如果系统正在运行，我们就能处理数据。
I wish the new software were in operation already. = 我希望新软件已经投入运行了。

被动语态:
The equipment is being kept in operation. = 设备正在保持运行状态。
The plan is being put into operation. = 计划正在实施。

疑问句:
Is the power plant in operation? = 发电厂正在运行吗？
Is the new rule in operation yet? = 新规章生效了吗？

否定句:
The machine is not in operation. = 机器没有运行。
The system is not yet in operation. = 系统尚未运行。
