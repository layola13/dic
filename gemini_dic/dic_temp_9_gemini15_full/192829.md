searchable:/ˈsɜːrtʃəbl/ | adj. | 可搜索的，可查阅的 |  Able to be searched or looked through systematically; easily found or accessed through a search.  | Example: This database is searchable by keyword. 这个数据库可以通过关键词搜索。  The information is easily searchable online.  这些信息很容易在网上搜索到。|近义词：accessible, findable, retrievable | 反义词：inaccessible, unfindable, hidden


USAGE->
一般现在时:
The database is searchable by keyword. = 这个数据库可以通过关键词搜索。
This website is searchable using a variety of methods. = 这个网站可以使用多种方法进行搜索。


一般过去时:
The document was searchable before the update. = 更新之前，该文档是可以搜索的。
The old system wasn't searchable efficiently. = 旧系统搜索效率低下。


现在完成时:
The website has been searchable since last week. = 该网站自上周以来一直可以搜索。
The information has become searchable through a new index.  = 通过新的索引，信息变得可搜索了。


过去完成时:
The database had been searchable for years before the crash. = 数据库在崩溃之前已经可以搜索多年了。
The data had not been searchable until the new software was installed. = 直到安装了新软件，数据才变得可搜索。


将来时:
The new system will be searchable next month. = 新系统下个月将可以搜索。
The updated website will be fully searchable by the end of the year. = 更新后的网站将在年底前完全可搜索。


虚拟语气:
If the data were searchable, we could find the answer. = 如果数据可搜索，我们就能找到答案。
It would be helpful if the files were searchable. = 如果文件可搜索，那就太好了。


被动语态:
The data is being made searchable by the developers. = 开发人员正在使数据可搜索。
The information was being searched using the new system. = 信息正在使用新系统进行搜索。


疑问句:
Is the document searchable? = 这个文档可搜索吗？
Are all the files searchable? = 所有文件都可搜索吗？


否定句:
The website is not searchable at the moment. = 目前该网站不可搜索。
The information is not easily searchable. = 信息不容易搜索。
