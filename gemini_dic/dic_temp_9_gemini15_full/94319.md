impetrative:/ɪmˈpɛrətɪv/| adj. n.|命令的；强制的；必要的；祈使的 |  Expressing a command; of vital importance.  Something that is absolutely necessary. |例句：It is imperative that you finish this report by Friday.  你必须在周五之前完成这份报告。

例句：The doctor stressed the imperative nature of immediate treatment. 医生强调了立即治疗的必要性。


近义词：mandatory, obligatory, essential, crucial, vital

反义词：optional, unnecessary, unimportant, inconsequential

词性：形容词，名词


词源：源自拉丁语imperātivus，来自imperāre（命令）。


USAGE->
一般现在时:
The imperative mood is used to give orders. = 祈使语气用于发出命令。
It is imperative that we act now. = 我们现在必须采取行动。


一般过去时:
It was imperative that we act quickly. = 我们必须迅速行动。（过去时态强调过去情况下的必要性）
The general issued imperative orders. = 将军发出了强制性的命令。（描述过去的行为）


现在完成时:
It has been imperative that we maintain a strong defense. = 保持强大的防御一直是必要的。（强调持续到现在的必要性）
The situation has demanded imperative action. = 局势要求采取强制性行动。（强调过去行为对现在造成的影响）


过去完成时:
It had been imperative that we arrive early. = 我们必须早到。（强调过去某个时间点之前的必要性）
It had become imperative to find a solution. = 找到解决方案已经变得刻不容缓。（强调过去情况发展到一定程度的必要性）


将来时:
It will be imperative to address this issue immediately. = 立即解决这个问题将是必要的。（强调未来必要性）
It will be imperative that we prepare for all possibilities. = 我们必须为所有可能性做好准备。（强调未来必要性）


虚拟语气:
It is imperative that he be present. = 他必须在场。（虚拟语气，表示建议或要求）
It is imperative that we should have acted sooner. = 我们应该早点采取行动。（虚拟语气，表示对过去行为的遗憾）


被动语态:
Immediate action was imperative. = 立即采取行动是必要的。（被动语态，强调行动本身的必要性）
The need for immediate action was deemed imperative. = 立即采取行动的必要性被认为是必要的。（被动语态，更强调“必要性”这一概念）


疑问句:
Is immediate action imperative? = 立即采取行动有必要吗？（疑问句，询问必要性）
Isn't it imperative that we proceed with caution? = 我们谨慎行事难道不必要吗？（反问句，强调必要性）


否定句:
It is not imperative that you attend. = 你不必出席。（否定句，表示没有必要）
It wasn't imperative to rush. = 没有必要匆忙。（否定句，表示过去没有必要）


PHRASE->
categorical imperative: 绝对命令 (Kant's philosophy)
moral imperative: 道德义务
practical imperative: 实用原则
imperative need: 迫切的需要
imperative sentence: 祈使句
