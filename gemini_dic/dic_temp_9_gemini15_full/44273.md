cron:/krɒn/| n. |定时任务调度程序，克隆|A time-based job scheduler in Unix-like computer operating systems.|Example:  The cron job runs the backup script every night at midnight.  这个cron作业每天午夜运行备份脚本。|近义词：scheduler, timer |反义词：manual execution 手动执行


USAGE->
一般现在时:
The cron job runs the backup script daily. = cron 作业每天运行备份脚本。
This cron task executes at 3 AM. = 这个 cron 任务在凌晨 3 点执行。

一般过去时:
The cron job ran successfully last night. = cron 作业昨晚成功运行。
The system administrator checked the cron logs. = 系统管理员检查了 cron 日志。

现在完成时:
The cron job has completed its tasks. = cron 作业已完成其任务。
The system has successfully scheduled many cron jobs. = 系统已成功调度了许多 cron 作业。

过去完成时:
The cron job had already finished before the server restarted. = 服务器重启之前，cron 作业已经完成了。
The administrator had reviewed the cron settings before the deployment. = 管理员在部署之前已经审查了 cron 设置。

将来时:
The cron job will run the script tomorrow. = cron 作业明天将运行脚本。
The new cron task will start next week. = 新的 cron 任务将于下周开始。

虚拟语气:
If the cron job were configured correctly, it would run without errors. = 如果 cron 作业配置正确，它将无错误运行。
It would be better if the cron job had more logging. = 如果 cron 作业有更多日志记录会更好。

被动语态:
The backup script is run by a cron job. = 备份脚本由 cron 作业运行。
The cron tasks are managed by the system administrator. = cron 任务由系统管理员管理。

疑问句:
Does this cron job run every hour? = 这个 cron 作业每小时运行吗？
What time are the cron jobs scheduled to run? = cron 作业计划运行的时间是什么时候？

否定句:
The cron job did not run successfully. = cron 作业没有成功运行。
The cron job is not configured correctly. = cron 作业配置不正确。


PHRASE->
cron job = 定时任务
cron expression = cron 表达式
cron schedule = cron 计划
cron daemon = cron 守护进程
