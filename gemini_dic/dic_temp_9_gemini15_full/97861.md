inline:/ˈɪnlaɪn/| adj.|内联的；嵌入式的|In computer programming, inline refers to a method of embedding code directly into the calling function, rather than calling a separate function.  This can improve performance by reducing function call overhead.|例句：The compiler optimized the code by inlining the small functions.  编译器通过内联小函数优化了代码。|近义词：embedded, integrated |反义词：external, separate
