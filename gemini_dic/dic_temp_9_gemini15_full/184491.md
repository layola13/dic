restar:/riˈstɑːr/| vt.|重启，重新启动|To start (a machine or process) again after it has stopped.|例句：Please restart your computer. 请重启你的电脑。
例句：We need to restart the project from scratch. 我们需要从头开始重启这个项目。|近义词：reboot, relaunch, resume|反义词：shutdown, halt, stop|动词


USAGE->
一般现在时:
The system restarts automatically after a power failure. = 系统在断电后会自动重启。
I restart my computer every morning. = 我每天早上都会重启我的电脑。


一般过去时:
He restarted the engine after it stalled. = 发动机熄火后，他重新启动了它。
She restarted the washing machine because it stopped unexpectedly. = 洗衣机意外停止后，她重新启动了它。


现在完成时:
I have restarted my computer three times today. = 我今天已经重启了三次电脑。
They have restarted the negotiations after a long break. = 经过长时间的休整后，他们已经重启了谈判。


过去完成时:
The computer had restarted itself before I could save my work. = 在我能够保存我的工作之前，电脑已经自行重启了。
He had restarted the experiment after the initial failure. = 初次实验失败后，他重新开始了实验。


将来时:
I will restart the server after the maintenance is complete. = 维修完成后，我将重启服务器。
We will restart the project next week. = 我们下周将重启项目。


虚拟语气:
If the system failed, I would restart it immediately. = 如果系统失败了，我会立即重启它。
I wish I had restarted the computer sooner; I lost my unsaved work. = 我真希望我早点重启电脑；我丢失了未保存的工作。


被动语态:
The computer was restarted by the technician. = 电脑由技术人员重启。
The program is being restarted because of an error. = 程序由于错误正在重新启动。


疑问句:
Should I restart my computer now? = 我现在应该重启电脑吗？
Have you ever had to restart a project completely? = 你曾经需要完全重启一个项目吗？


否定句:
I did not restart the application. = 我没有重启应用程序。
The machine did not restart after the power surge. = 电涌后机器没有重启。


PHRASE->
restart the game = 重启游戏
restart the process = 重启进程
restart the server = 重启服务器
restart the computer = 重启电脑
restart negotiations = 重启谈判
