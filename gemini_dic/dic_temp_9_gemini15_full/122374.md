mkdir:/ˈmækˌdɪr/| vt.|创建目录|To make a new directory (folder) in a file system.|例句：I used the command `mkdir new_folder` to create a new directory. 我使用命令`mkdir new_folder`创建了一个新目录。|近义词：makedir, createdir |反义词：rmdir
