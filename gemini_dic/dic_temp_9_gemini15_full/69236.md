feasible:/ˈfiːzəbəl/| adj.|可行的；可能的；切实可行的|capable of being done or achieved; possible.|The plan seems feasible, but we need to do more research before we can be certain.  这个计划看起来可行，但我们需要做更多研究才能确定。|It is not feasible to complete the project within the given timeframe. 在规定的时间内完成该项目是不可行的。|practicable, possible, viable|impossible, impractical, unachievable


USAGE->

一般现在时:
The project is feasible. = 这个项目是可行的。
This solution seems feasible to me. = 这个方案对我来说似乎可行。


一般过去时:
The proposal was feasible at the time. = 那时这个提案是可行的。
We thought the idea was feasible, but we were wrong. = 我们认为这个想法是可行的，但我们错了。


现在完成时:
The plan has proven feasible. = 该计划已被证明是可行的。
It has become feasible to travel to Mars.  去火星旅行已经成为可能。


过去完成时:
The engineers had determined that the design was feasible.  工程师们已经确定该设计是可行的。
We had thought the plan feasible, but new information changed our minds. 我们曾认为该计划可行，但新的信息改变了我们的想法。


将来时:
The new technology will make many things feasible that were previously impossible. 新技术将使许多以前不可能的事情成为可能。
This project will be feasible once we secure additional funding. 一旦我们获得额外资金，这个项目就可行了。


虚拟语气:
If the budget were larger, the project would be feasible. 如果预算更大，这个项目就可行了。
It would be feasible to implement the changes if we had more staff. 如果我们有更多员工，实施这些改变是可行的。


被动语态:
The proposal was deemed feasible by the committee.  该提案被委员会认为是可行的。
The plan has been proven feasible after extensive testing.  经过广泛测试后，该计划已被证明是可行的。


疑问句:
Is the project feasible?  这个项目可行吗？
Is it feasible to complete the work by Friday?  周五之前完成工作可行吗？


否定句:
The idea is not feasible.  这个想法不可行。
It is not feasible to finish the report by tomorrow. 明天之前完成报告是不可能的。


PHRASE->

a feasible solution 一个可行的方案
a feasible option 一个可行的选择
a feasible plan 一个可行的计划
technically feasible 技术上可行
economically feasible 经济上可行
