reset:/ˌriːˈsɛt/ | vt. n. | 重置；恢复默认设置 | To return (something) to a previous state or condition, especially to a zero or default setting.  |例句：I had to reset my password after forgetting it. 我忘记密码后不得不重置密码。

例句：The computer automatically resets after a power failure. 电脑断电后会自动重置。


近义词：reboot, restart, reinitialize | 反义词：set, establish


USAGE->
一般现在时:
He resets his alarm clock every morning. = 他每天早上都会重置闹钟。
She resets her goals after each failure. = 她每次失败后都会重置目标。


一般过去时:
I reset the router after the internet went down. = 网断了之后我重置了路由器。
They reset the experiment after detecting an error. = 他们在检测到错误后重置了实验。


现在完成时:
She has reset her computer several times today. = 她今天已经多次重启电脑了。
He has reset his expectations after the disappointing results. = 在令人失望的结果之后，他已经调整了他的期望。


过去完成时:
The system had reset itself before the technician arrived. = 技术人员到达之前，系统已经自行重置了。
They had reset the database before running the new program. = 他们在运行新程序之前已经重置了数据库。


将来时:
I will reset the printer if it jams again. = 如果打印机再次卡纸，我会重置它。
We will reset the password for all users next week. = 我们下周将重置所有用户的密码。


虚拟语气:
If the program crashed, I would reset it. = 如果程序崩溃了，我会重置它。
It would be better if we reset the counter and start over. = 如果我们重置计数器并重新开始会更好。


被动语态:
The settings were reset by the administrator. = 设置由管理员重置了。
The password must be reset by the user himself. = 密码必须由用户本人重置。


疑问句:
Can you reset the application for me? = 你能帮我重置一下应用程序吗？
Should we reset the system before installing the update? = 我们应该在安装更新之前重置系统吗？


否定句:
I cannot reset my phone; it's frozen. = 我无法重置我的手机；它冻住了。
They did not reset the timer before the race began. = 比赛开始前他们没有重置计时器。
