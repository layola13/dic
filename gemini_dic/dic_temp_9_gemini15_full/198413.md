shp:/ʃp/| n. |形状文件 |A shapefile is a popular geospatial vector data format for geographic information systems (GIS) software.  It is not a single file but a collection of files with a .shp extension along with other related files (.shx, .dbf, .prj, etc.) that together store geographic data. |This map was created using a shapefile of the city's roads. 这张地图是用城市道路的形状文件创建的。|近义词：geospatial data, vector data |反义词：raster data
