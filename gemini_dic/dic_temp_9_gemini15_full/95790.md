inconvenience:/ˌɪnkənˈviːniəns/| n. |不便；麻烦 |The inconvenience caused by the road closure was considerable. 道路封闭造成的诸多不便相当可观。|近义词：trouble, difficulty, bother |反义词：convenience, ease


USAGE->
一般现在时:
The construction causes much inconvenience. = 这项工程造成很多不便。
This schedule brings little inconvenience to me. = 这个时间安排对我来说几乎没有不便。


一般过去时:
The delay caused considerable inconvenience to passengers. = 延误给乘客带来了相当大的不便。
The power outage caused great inconvenience last night. = 昨晚的停电造成了很大的不便。


现在完成时:
The frequent changes have caused much inconvenience. = 频繁的变动已经造成了很多不便。
This new system hasn't caused any inconvenience so far. = 这个新系统到目前为止还没有造成任何不便。


过去完成时:
They had anticipated the inconvenience, so they prepared for it. = 他们预料到了不便，所以做了准备。
We had already experienced the inconvenience of the old system. = 我们已经体验过旧系统的诸多不便了。


将来时:
The new rules will cause some inconvenience initially. = 新规一开始会造成一些不便。
The traffic jam will cause significant inconvenience to commuters. = 交通堵塞将给通勤者造成很大的不便。


虚拟语气:
If I knew this would cause so much inconvenience, I wouldn't have done it. = 如果我知道这会造成这么大的不便，我不会这么做的。
I wish the new policy wouldn't cause so much inconvenience to everyone. = 我希望这项新政策不会给所有人带来这么大的不便。


被动语态:
Much inconvenience was caused by the bad weather. = 恶劣天气造成了很多不便。
The inconvenience was minimized by efficient planning. = 通过有效的规划，不便被降到了最低。


疑问句:
What inconvenience will this change cause? = 这一变化会造成什么不便？
Is there any inconvenience this process will cause? = 这个过程会造成什么不便吗？


否定句:
The new software doesn't cause any inconvenience. = 新软件没有任何不便。
This method should not cause significant inconvenience. = 这种方法不应造成重大不便。
