get-set:/ˈɡɛtˌsɛt/| phr.|准备，起跑|A phrase used to signal the beginning of a race or other competitive event, or to indicate readiness to begin an activity.|例句：Get set, go! 准备，跑！|近义词：ready, prepare, |反义词:stop, halt.


USAGE->
This phrase doesn't have verb conjugations in the traditional sense. It's a fixed phrase.


PHRASE->
get set go = 准备，开始
get set to do sth = 准备好做某事
all set = 一切就绪
get the set = 获得装备/设置
be set to do sth = 准备做某事
