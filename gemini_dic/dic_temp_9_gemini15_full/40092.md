contains:/kənˈteɪnz/ | vt. |包含；容纳；含有 |To include or hold within itself.  |The box contains toys.  盒子装着玩具。|The book contains many interesting stories. 这本书包含许多有趣的故事。|includes, encompasses, comprises |excludes, omits, lacks |动词


USAGE->
一般现在时:
This box contains many books. = 这个盒子包含许多书。
The letter contains important information. = 这封信包含重要信息。


一般过去时:
The jar contained pickles. = 罐子里装着泡菜。
The speech contained many powerful ideas. = 演讲包含许多有力的观点。


现在完成时:
The package has contained fragile items, handle with care. = 包裹里装着易碎物品，请小心轻放。
The study has contained evidence of a link between the two. = 这项研究已经包含了这两个之间存在联系的证据。


过去完成时:
The room had contained a painting before it was moved. = 这间屋子之前曾挂着一幅画。
The report had contained several errors. = 报告中包含了几处错误。


将来时:
The capsule will contain the medication. = 胶囊将包含药物。
The museum will contain artifacts from various cultures. = 博物馆将收藏来自不同文化的文物。


虚拟语气:
If the box contained gold, we would be rich. = 如果盒子里面装着金子，我们就会富有。
I wish the letter contained good news. = 我希望这封信里包含好消息。


被动语态:
The information was contained in a secret document. = 信息包含在一份秘密文件中。
The box is contained in a larger crate. = 箱子装在一个更大的板条箱里。


疑问句:
Does the envelope contain a letter? = 信封里装有信吗？
What does the bag contain? = 包里装有什么？


否定句:
The solution does not contain any harmful chemicals. = 该溶液不含任何有害化学物质。
The report did not contain the necessary data. = 报告中没有包含必要的数据。
