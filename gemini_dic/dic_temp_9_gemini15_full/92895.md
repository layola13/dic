ideally:/aɪˈdiːəli/| adv.|理想地，最好地，理论上|In an ideal way; perfectly.  It suggests a situation that is perfect or the best possible, but may not be realistically achievable.|例句：Ideally, we should finish the project by Friday.  = 理想情况下，我们应该在周五之前完成项目。

例句：Ideally, we'd like to have a bigger house, but it's not realistic right now. = 理想情况下，我们想要一套更大的房子，但这现在不现实。

近义词：perfectly, optimally, preferably, theoretically

反义词：unrealistically, impractically, poorly


USAGE->
一般现在时:
Ideally, we work together on this project. = 理想情况下，我们一起合作完成这个项目。
Ideally, he finishes his work before the deadline. = 理想情况下，他在截止日期前完成工作。

一般过去时:
Ideally, we finished the project early. = 理想情况下，我们提前完成了项目。
Ideally, she completed her report on time. = 理想情况下，她按时完成了报告。

现在完成时:
Ideally, we have already completed the first phase. = 理想情况下，我们已经完成了第一阶段。
Ideally, he has already submitted his application. = 理想情况下，他已经提交了申请。

过去完成时:
Ideally, we had already prepared everything before the meeting. = 理想情况下，我们在会议之前已经准备好了一切。
Ideally, she had already studied the materials before the exam. = 理想情况下，她在考试前已经学习了资料。

将来时:
Ideally, we will finish the project next week. = 理想情况下，我们下周将完成该项目。
Ideally, he will graduate with honors. = 理想情况下，他将以优异成绩毕业。

虚拟语气:
Ideally, we would have a bigger house. = 理想情况下，我们会有一所更大的房子。
Ideally, she would have won the competition. = 理想情况下，她会赢得比赛。

被动语态:
Ideally, the problem would be solved quickly. = 理想情况下，问题会被迅速解决。
Ideally, the report would be completed before the deadline. = 理想情况下，报告会在截止日期前完成。

疑问句:
Ideally, when would we meet? = 理想情况下，我们什么时候见面？
Ideally, where should we hold the meeting? = 理想情况下，我们应该在哪里举行会议？


否定句:
Ideally, we shouldn't have any more delays. = 理想情况下，我们不应该再有延误了。
Ideally, he shouldn't have made that mistake. = 理想情况下，他不应该犯那个错误。


PHRASE->

Ideally positioned = 位置极佳
Ideally speaking = 理论上来说
Ideally speaking, this is the best solution. = 从理论上讲，这是最好的解决方案。
