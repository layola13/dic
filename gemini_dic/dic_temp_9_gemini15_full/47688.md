debugs:/dɪˈbʌɡz/| v.|调试，排除故障|To identify and remove errors from (computer hardware or software).|例句：The programmer spent hours debugging the code. 程序员花了数小时调试代码。|近义词：troubleshoot, fix, correct|反义词：introduce bugs, break


USAGE->
一般现在时:
He debugs the software daily. = 他每天调试软件。
She debugs the code to find the errors. = 她调试代码以查找错误。

一般过去时:
They debugged the program last night. = 他们昨晚调试了程序。
The technician debugged the faulty machine. = 技术员调试了故障机器。

现在完成时:
We have debugged the system. = 我们已经调试了系统。
She has debugged all the known errors. = 她已经调试了所有已知的错误。

过去完成时:
They had debugged the software before the release. = 他们发行前已经调试好软件了。
The team had debugged the code for hours. = 团队已经调试代码数小时了。

将来时:
We will debug the application next week. = 我们下周将调试该应用程序。
He will debug the program after the meeting. = 会议结束后，他将调试程序。

虚拟语气:
If I had more time, I would debug the code more thoroughly. = 如果我有更多时间，我会更彻底地调试代码。
It would be helpful if someone could debug this for me. = 如果有人能帮我调试一下，那就太好了。

被动语态:
The software is being debugged by a team of experts. = 该软件正在由专家团队进行调试。
The problem was debugged quickly. = 问题很快得到解决。

疑问句:
Are you debugging the application right now? = 你现在正在调试应用程序吗？
Have you debugged the program completely? = 你已经完全调试程序了吗？

否定句:
He didn't debug the software properly. = 他没有正确调试软件。
They haven't finished debugging the code yet. = 他们还没有完成代码的调试。
