hard-code:/ˌhɑːrdˈkoʊd/ | vt. | 硬编码，直接嵌入代码 | To hard-code something means to write it directly into the source code of a program, rather than storing it in a configuration file or database where it could be easily changed.  This makes the code less flexible and harder to maintain. | Example: The developer hard-coded the database connection string into the application.  开发人员将数据库连接字符串硬编码到应用程序中。

The company decided to hard-code the tax rate into the software instead of making it configurable. 公司决定将税率硬编码到软件中，而不是使其可配置。|近义词：embed, fix, set  | 反义词：parameterize, configure | 动词

词源:  The term "hard-code" combines "hard" (meaning fixed or inflexible) and "code" (referring to computer programming).  It reflects the idea that the data is permanently fixed within the code itself, making it difficult to modify without altering the source code.
