key-in: /kiːˈɪn/ | vt. |键入，输入|To enter (data) into a computer or other device using a keyboard.  |例句：Please key in your password. 请输入您的密码。|例句：I keyed in the address and hit enter. 我输入了地址并按下了回车键。|近义词：type, enter, input |反义词：delete, erase


USAGE->
一般现在时:
He keys in his password every morning. = 他每天早上输入他的密码。
She keys in the data carefully. = 她仔细地输入数据。

一般过去时:
I keyed in the wrong information yesterday. = 我昨天输入了错误的信息。
They keyed in the details of the project. = 他们输入了项目的细节。

现在完成时:
She has keyed in all the data. = 她已经输入了所有数据。
He has keyed in the report. = 他已经输入了报告。

过去完成时:
I had keyed in the address before I realized it was wrong. = 我在意识到地址错误之前已经输入了地址。
They had keyed in all the numbers before the system crashed. = 系统崩溃之前他们已经输入了所有数字。

将来时:
I will key in the information later. = 我稍后会输入信息。
We will key in the data tomorrow. = 我们明天将输入数据。

虚拟语气:
If I had keyed in the correct password, I would have accessed the system. = 如果我输入了正确的密码，我就能访问系统了。
I wish I had keyed in the address more carefully. = 我希望我当时更仔细地输入地址。

被动语态:
The data was keyed in by the assistant. = 数据是由助理输入的。
The information is being keyed in right now. = 信息正在被输入。

疑问句:
Have you keyed in the address correctly? = 你是否正确地输入了地址？
Did he key in the password? = 他输入密码了吗？

否定句:
I didn't key in the correct number. = 我没有输入正确的数字。
She hasn't keyed in the report yet. = 她还没有输入报告。

PHRASE->

key in the code = 输入代码
