pre-install:/ˌpriːɪnˈstɔːl/| adj. v.|预安装的；预先安装|Pre-installed software refers to software that is already installed on a device before it is sold or used for the first time.|例句：My new laptop came with pre-installed antivirus software. 我的新笔记本电脑预装了杀毒软件。|近义词：installed, built-in|反义词:uninstalled, post-installed

USAGE->
一般现在时:
The company pre-installs several applications on each new phone. = 公司在每部新手机上预装多个应用程序。
They pre-install the operating system before shipping the computers. = 他们在发货前预先安装操作系统。


一般过去时:
The factory pre-installed the software on all the devices. = 工厂在所有设备上预装了软件。
We pre-installed the necessary drivers before the launch. = 我们在发布前预安装了必要的驱动程序。


现在完成时:
The manufacturer has pre-installed a suite of productivity apps. = 制造商已经预装了一套生产力应用程序。
They have pre-installed the security updates for improved protection. = 他们已经预安装了安全更新以增强保护。


过去完成时:
The technician had pre-installed the software before the client arrived. = 技术人员在客户到达之前已经预安装了软件。
The team had pre-installed all the apps by the deadline. = 团队在截止日期前已经预安装了所有应用程序。


将来时:
We will pre-install the latest version of the software on all new devices. = 我们将在所有新设备上预安装最新版本的软件。
They will pre-install the game before the official release date. = 他们将在正式发布日期前预安装游戏。


虚拟语气:
If they had pre-installed the updates, the system would have been more secure. = 如果他们预安装了更新，系统就会更安全。
It would be helpful if the vendor pre-installed all required drivers. = 如果供应商预安装所有必要的驱动程序，那就太好了。


被动语态:

The updates are being pre-installed on all the units. = 所有单元都在预安装更新。


疑问句:
Was the operating system pre-installed? = 操作系统是预装的吗？

否定句:
The software was not pre-installed on my new computer. = 我的新电脑上没有预装该软件。
They did not pre-install the necessary applications. = 他们没有预安装必要的应用程序。
