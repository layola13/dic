database:/ˈdeɪtəbeɪs/ | n. |数据库 |A structured set of data organized and accessed electronically from a computer system.  A database is usually controlled by a database management system (DBMS).|例句：The company's database contains millions of customer records. 这家公司的数据库包含数百万条客户记录。|例句：We need to update the database with the latest information. 我们需要用最新信息更新数据库。|近义词：data store, repository, collection |反义词：None

This database manages all the company's data. = 这个数据库管理公司所有的数据。
