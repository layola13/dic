great:/ɡreɪt/ | adj. |伟大的；极好的；大量的 |  Excellent; of a large size or extent; important.  |例句：It's a great day!  这是一天好日子！
例句：He did a great job. 他做得很好。

近义词：wonderful, excellent, superb, magnificent, immense, vast

反义词：terrible, awful, poor, small, insignificant


USAGE->
一般现在时:
This is a great idea. = 这是一个好主意。
The view is great from here. = 从这里看景色很棒。


一般过去时:
The party was great. = 派对很棒。
We had a great time. = 我们玩得很开心。


现在完成时:
We've had great success. = 我们取得了巨大的成功。
She's done a great job. = 她做得很好。


过去完成时:
I had already had a great day before the party. = 在聚会之前我已经度过了美好的一天。
They had already achieved great things by that time. = 到那时他们已经取得了巨大的成就。


将来时:
The concert will be great. = 音乐会将会很棒。
We will have a great vacation. = 我们将拥有一个美好的假期。


虚拟语气:
It would be great if you could come. = 如果你能来就好了。
I wish I had a great car. = 我真希望有一辆好车。


被动语态:
The project was considered great. = 该项目被认为很棒。
The book is praised as great. = 这本书被赞誉为伟大的作品。


疑问句:
Isn't this great? = 这不是很棒吗？
Is that great?  = 那很棒吗？


否定句:
It's not a great idea. = 这不是一个好主意。
The food wasn't great. = 食物不太好。


PHRASE->
great deal: 大量，很多
great many: 许许多多
to a great extent: 在很大程度上
great effort: 巨大的努力
to one's great surprise: 令某人非常惊讶的是

词源:
The word "great" comes from Old English *grēat, meaning "large," "big," or "great."  It is related to the Old High German *grōz and Old Norse *gróðr, all sharing a common Proto-Germanic ancestor.  The modern sense of "excellent" developed later.
