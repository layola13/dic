precedent:/ˈpresɪdənt/| n. |先例；判例；前例 |A precedent is an earlier event or action that is regarded as an example or guide to be considered in subsequent similar circumstances.  It sets a standard or pattern for future situations. |The court's decision set a precedent for future cases involving similar issues. 法院的判决为未来涉及类似问题的案件树立了先例。  The company's success served as a precedent for other startups. 这家公司的成功为其他初创公司树立了榜样。 |example, model, instance |novelty, innovation |名词


USAGE->

一般现在时:
This case sets a precedent for future actions. = 这个案例为将来的行动树立了先例。
The company's policy serves as a precedent for others. = 该公司的政策为其他公司树立了先例。


一般过去时:
The 1960s civil rights movement set many precedents for social change. = 20世纪60年代的民权运动为社会变革树立了许多先例。
That decision established a dangerous precedent. = 那个决定树立了一个危险的先例。


现在完成时:
This ruling has set an important precedent in environmental law. = 这一裁决在环境法中树立了一个重要的先例。
Their actions have created a precedent that others will follow. = 他们的行为创造了一个其他人将效仿的先例。


过去完成时:
They had already established a precedent for dealing with such cases. = 他们已经为处理此类案件确立了先例。
The previous government had set a precedent of fiscal irresponsibility. = 前一届政府已经树立了财政不负责任的先例。


将来时:
This will set a precedent for future negotiations. = 这将为未来的谈判树立先例。
This decision will likely create a precedent that is difficult to overturn. = 这个决定可能会创造一个难以推翻的先例。


虚拟语气:
If we followed this precedent, it would be a disaster. = 如果我们遵循这个先例，那将是一场灾难。
If they had not set this precedent, things would be different now. = 如果他们没有设定这个先例，情况现在就会不同了。


被动语态:
A precedent was set by the Supreme Court’s ruling. = 最高法院的裁决树立了一个先例。
A dangerous precedent is being set by these actions. = 这些行动正在树立一个危险的先例。


疑问句:
Does this action set a precedent? = 这个行动是否树立了一个先例？
What precedent does this decision follow? = 这个决定遵循什么先例？


否定句:
This case does not set a precedent. = 这个案例没有树立先例。
There is no precedent for such behavior. = 这种行为没有先例。



PHRASE->
set a precedent  =  树立先例
create a precedent = 创造先例
follow a precedent = 遵循先例
establish a precedent = 确立先例
overturn a precedent = 推翻先例
