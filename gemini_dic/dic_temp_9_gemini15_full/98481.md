installed:/ɪnˈstɔːld/ | vt. |安装，设置，安置 |To place (something) in position for use; to establish (something) firmly.  |例句：The mechanic installed a new engine in my car.  机械师在我的车里安装了一个新引擎。|例句：They installed a new security system in the building. 他们在建筑物里安装了一个新的安保系统。|近义词：set up, fix, put in |反义词：remove, uninstall, dismantle


USAGE->
一般现在时:
They install new software every month. = 他们每月安装新软件。
He installs the app on his phone. = 他在他的手机上安装这个应用。

一般过去时:
We installed the new kitchen cabinets yesterday. = 我们昨天安装了新的厨房橱柜。
She installed a new security system last week. = 她上周安装了一个新的安保系统。

现在完成时:
I have installed the printer. = 我已经安装了打印机。
They have installed solar panels on their roof. = 他们已经在屋顶安装了太阳能电池板。

过去完成时:
He had installed the program before the meeting started. = 会议开始前他已经安装了程序。
She had installed the app before she went on vacation. = 她度假前已经安装了应用程序。

将来时:
We will install the new software next week. = 我们下周将安装新的软件。
They will install the air conditioner tomorrow. = 他们明天将安装空调。

虚拟语气:
If I had time, I would install the new operating system. = 如果我有时间，我会安装新的操作系统。
It would be better if you installed a firewall. = 如果你安装防火墙会更好。

被动语态:
The software was installed by the technician. = 该软件由技术人员安装。
The new system will be installed next month. = 新系统将于下个月安装。

疑问句:
Have you installed the updates yet? = 你已经安装更新了吗？
Will the program be installed automatically? = 程序会自动安装吗？

否定句:
I didn't install the drivers correctly. = 我没有正确安装驱动程序。
They haven't installed the new software yet. = 他们还没有安装新的软件。
