coding:/ˈkoʊdɪŋ/| n. |编码，编程|The act of writing code to create a program or software.  It involves translating human-readable instructions into a language that a computer can understand and execute.| Example:  He spent hours on coding the new game. 他花了几个小时编写这个新游戏。|programming, programming, scripting|decoding, writing in plain language|名词


USAGE->
一般现在时:
He is coding a new website. = 他正在编写一个新的网站。
She codes every day. = 她每天都编程。

一般过去时:
They coded the program last night. = 他们昨晚编写了这个程序。
I coded a simple script for the task. = 我为这项任务编写了一个简单的脚本。

现在完成时:
We have coded the major features of the app. = 我们已经编写了应用程序的主要功能。
She has coded for over five years. = 她已经编程五年多了。

过去完成时:

He had coded most of the game before taking a break. = 他在休息之前已经编写了大部分游戏。

将来时:
They will code the next module tomorrow. = 他们明天将编写下一个模块。
I will be coding all weekend. = 我整个周末都会在编程。
