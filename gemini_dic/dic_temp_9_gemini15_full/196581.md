set-aside:/setˈsaɪd/| vt.|留出，拨出；搁置；暂不考虑|To put something aside for a specific purpose or to postpone dealing with something.|例句：We've set aside a sum of money for our vacation. 我们留出一笔钱用于度假。|例句：Let's set aside our differences and work together. 让我们搁置分歧，一起工作。|近义词：reserve, allocate, postpone, shelve|反义词：use, employ, consider, address


USAGE->
一般现在时:
They set aside some land for a park. = 他们留出一块地作为公园。
We set aside time each week for family dinner. = 我们每周留出时间吃家庭晚餐。

一般过去时:
She set aside her worries and focused on the task. = 她把担忧放在一边，专注于任务。
The company set aside funds for research and development. = 公司拨出资金用于研发。

现在完成时:
They have set aside enough money for the down payment. = 他们已经攒够首付的钱了。
We have set aside our differences and reached an agreement. = 我们已经搁置分歧，达成了协议。

过去完成时:
He had set aside his personal feelings before making the decision. = 他在做决定之前已经抛开了个人情感。
They had set aside a special room for the guests. = 他们已经为客人准备了一个特别的房间。

将来时:
We will set aside an hour to discuss the proposal. = 我们将留出一个小时讨论这个提案。
The government will set aside funds for disaster relief. = 政府将拨款用于救灾。

虚拟语气:
If we set aside our prejudices, we could have a better understanding. = 如果我们抛开偏见，就能更好地理解。
I wish I had set aside more time for my studies. = 我真希望我当时留出更多时间学习。

被动语态:
A special fund was set aside for the project. = 为该项目拨出了专项资金。
Time should be set aside for relaxation and recreation. = 应该留出时间用于放松和娱乐。

疑问句:
Have you set aside enough time for the journey? = 你为旅程留出足够的时间了吗？
Will they set aside funds for the new initiative? = 他们会为这项新举措拨款吗？

否定句:
They didn't set aside any money for emergencies. = 他们没有为紧急情况留出任何钱。
We haven't set aside enough time to complete the project. = 我们没有留出足够的时间来完成项目。
