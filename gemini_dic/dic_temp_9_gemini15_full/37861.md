compile:/kəmˈpaɪl/ | vt. |编译，汇编|To collect and combine (written material) into a volume or anthology; to assemble (a program) from source code into an executable file.  |例句：The editor compiled the author's best short stories into a single volume. 编辑将作者最好的短篇小说汇编成一卷。  The programmer compiled the code before running the program. 程序员在运行程序之前编译了代码。|近义词：collect, assemble, gather |反义词：disassemble, decompile |动词


USAGE->
一般现在时:
He compiles data for his research. = 他为他的研究收集数据。
She compiles a list of names. = 她编制了一个名字列表。


一般过去时:
They compiled a report on the accident. = 他们编写了一份关于事故的报告。
The team compiled the software last week. = 团队上周编译了软件。


现在完成时:
She has compiled a comprehensive bibliography. = 她已经编制了一个全面的参考书目。
We have compiled all the necessary documents. = 我们已经收集了所有必要的文档。


过去完成时:
They had compiled the data before the presentation. = 他们在演示之前已经编译了数据。
The writer had compiled his notes before starting the manuscript. = 作者在开始写手稿之前已经整理了他的笔记。


将来时:
