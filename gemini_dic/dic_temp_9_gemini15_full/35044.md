closeable: /ˈkloʊzəbl/ | adj. | 可关闭的，可封闭的 | Able to be closed.  Something that is closeable can be shut, fastened, or sealed.  | 例句：The window is closeable.  这扇窗户是可以关闭的。  The lid on this container is closeable. 这个容器的盖子是可以关闭的。 | 近义词：closable, shuttable, sealable | 反义词：uncloseable, unsealable


USAGE->
一般现在时:
The window is closeable. = 这扇窗户是可以关闭的。
The file is closeable after saving. = 保存后该文件可以关闭。


一般过去时:
The door was closeable before it broke. = 门坏掉之前是可以关闭的。
The application was closeable after the update. = 更新后该应用程序是可以关闭的。


现在完成时:
The system has been closeable since the last patch. = 自从上次补丁以来，系统一直是可以关闭的。
The software has become closeable after the bug fix. =  修复bug后，该软件可以关闭了。


过去完成时:
The program had been closeable until the power outage. = 直到停电之前，程序都是可以关闭的。
The valve had been closeable before the pressure increased. = 在压力增加之前，阀门是可以关闭的。


将来时:
The new feature will make the app closeable. = 新功能将使该应用程序可以关闭。

虚拟语气:

被动语态:
The application is designed to be closeable. = 该应用程序设计为可关闭的。

疑问句:
