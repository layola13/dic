/skrɪpt/ | n. | 剧本；手稿；笔迹 | A written text for a play, film, or broadcast.  | Example: The script for the movie was excellent.  这个电影的剧本很棒。 |近义词：manuscript, screenplay | 反义词：improvisation


USAGE->
一般现在时:
The playwright writes a new script. = 剧作家正在写一个新的剧本。
She always carefully reviews the script before filming. = 她总是在拍摄前仔细审查剧本。

一般过去时:
He wrote the script in three weeks. = 他在三周内写完了剧本。
They adapted the script from a novel. = 他们把一部小说改编成了剧本。

现在完成时:
She has written several award-winning scripts. = 她已经写了好几部获奖的剧本。
They have finished the first draft of the script. = 他们已经完成了剧本的第一稿。

过去完成时:
He had already written the script before the meeting. = 他在会议之前就已经写好了剧本。
The director had rewritten the script several times. = 导演已经重写过几次剧本了。

将来时:
She will write a new script for the next season. = 她将为下一季写一个新的剧本。
They will finish the script by the end of the month. = 他们将在月底完成剧本。

虚拟语气:
If I had more time, I would write a better script. = 如果我有更多时间，我会写一个更好的剧本。
I wish I had written a more compelling script. = 我希望我写了一个更引人入胜的剧本。

被动语态:
The script was written by a famous author. = 剧本是由一位著名的作家写的。
The script is being revised for the upcoming production. = 剧本正在为即将到来的演出修改。

疑问句:
Who wrote the script? = 谁写的剧本？
Is the script ready yet? = 剧本准备好了吗？

否定句:
He didn't write the script himself. = 他没有自己写剧本。
The script wasn't very original. = 剧本不是很有创意。
