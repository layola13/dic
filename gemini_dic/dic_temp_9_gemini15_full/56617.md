donet:/doʊˈnɛt/| adj.|完成的，完毕的|done 的过去分词形式，表示完成或结束的状态。|例句：The project is donet. 项目完成了。|近义词：completed, finished, accomplished |反义词：unfinished, incomplete |形容词


USAGE->
一般现在时:
The work is done. = 工作完成了。
The task is done. = 任务完成了。

一般过去时:
The work was done yesterday. = 工作昨天完成了。
The painting was done quickly. = 画很快完成了。

现在完成时:
The work has been done. = 工作已经完成了。
The report has been done. = 报告已经完成了。

过去完成时:
The work had been done before he arrived. = 他到达之前工作已经完成了。
The preparations had been done by then. = 到那时准备工作已经完成了。

将来时:
The work will be done tomorrow. = 工作明天将完成。
The project will be done by next week. = 项目下周将完成。

虚拟语气:
If the work were done, we could leave. = 如果工作完成了，我们就可以离开了。
I wish the report were done. = 我希望报告已经完成了。

被动语态:
The work is being done now. = 工作正在进行中。
The problem is being done. = 问题正在解决中。

疑问句:
Is the work done? = 工作完成了吗？
Is the project done yet? = 项目完成了吗？

否定句:
The work is not done yet. = 工作还没完成。
The task is not done. = 任务还没完成。
