operational:/ˌɒpəˈreɪʃənəl/ /ˌɑːpəˈreɪʃənəl/ | adj. |可操作的；运作的；运行中的；业务上的|

英文解析:Relating to or concerned with the functioning of a system or process.  Able to be used or put into effect.

例句:

The new software is now fully operational.  新的软件现在完全可以运行了。

The emergency services are operational despite the bad weather.  尽管天气恶劣，紧急服务仍在运作。


近义词:functional, working, running, active, in operation

反义词:inoperative, nonfunctional, broken, defunct, idle


词性:形容词


USAGE->
一般现在时:
The system is operational. = 系统正在运行。
The factory is operational 24/7. = 工厂24/7全天候运作。

一般过去时:
The machine was operational until yesterday. = 这台机器直到昨天还能运行。
The plan was operational for only a short time. = 这个计划只运行了很短时间。

现在完成时:
The system has been operational since last week. = 系统自上周以来一直在运行。
The project has been operational for several months. = 该项目已经运行了几个月。

过去完成时:
The equipment had been operational for years before it failed. = 设备在故障前已经运行多年了。
The system had been operational until the power outage. = 直到停电系统才停止运行。

将来时:
The new line will be operational next month. = 新生产线将于下个月投入运行。
The software will be operational by the end of the year. = 该软件将于年底投入运行。

虚拟语气:
If the system were operational, we could proceed. = 如果系统能运行，我们就能继续进行。
It would be better if all systems were operational. = 如果所有系统都能运行就好了。

被动语态:
The machine is being made operational. = 机器正在被调试运行。
The system is being kept operational. = 系统正在被保持运行。

疑问句:
Is the equipment operational? = 设备能运行吗？
Is the plan operational yet? = 计划现在可以运行了吗？

否定句:
The server is not operational. = 服务器无法运行。
The system is currently not operational. = 系统目前无法运行。
