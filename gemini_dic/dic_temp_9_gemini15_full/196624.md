setter:/ˈsɛtər/| n. |设置器；设定者；猎犬；|A setter is a person or thing that sets something.  In computer programming, a setter is a method used to assign a value to a property of an object. In hunting, a setter is a type of dog trained to locate game birds.|例句：The software uses a setter method to update the user's profile.  这个软件用设置器方法更新用户的个人资料。 The setter flushed a covey of quail. 这只猎犬惊起了一群鹌鹑。|近义词：adjuster, configurer, programmer (in programming context); pointer, retriever (in hunting context)|反义词：getter (in programming context)
