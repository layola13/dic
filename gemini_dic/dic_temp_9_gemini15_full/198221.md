shortener:/ˈʃɔːrtənər/ | n. | 缩短器，缩写器 | A tool or service that shortens URLs or other long pieces of text.  | Example:  Bitly is a popular URL shortener.  比特链接是一个流行的网址缩短器。 |近义词：URL shortener, link shortener |反义词：expander (in the context of text)


USAGE->
一般现在时:
The website uses a URL shortener. = 这个网站使用网址缩短器。
This tool shortens long links. = 这个工具缩短长链接。

一般过去时:
I shortened the link before sharing it. = 我在分享之前缩短了链接。
They used a shortener to create a shorter URL. = 他们使用缩短器创建了一个较短的网址。

现在完成时:
I have shortened several links today. = 我今天已经缩短了好几个链接。
She has used that shortener many times. = 她已经多次使用过那个缩短器。

过去完成时:
I had already shortened the link before sending the email. = 我在发送邮件之前已经缩短了链接。
They had shortened all the URLs before the presentation. = 他们在演示文稿之前已经缩短了所有网址。

将来时:
I will shorten the link later. = 我稍后会缩短链接。
We will use a different shortener next time. = 下次我们将使用不同的缩短器。

虚拟语气:
If I had a better shortener, I would use it. = 如果我有一个更好的缩短器，我会使用它。
I wish I had known about this shortener earlier. = 我真希望我早点知道这个缩短器。

被动语态:
The link was shortened automatically. = 链接被自动缩短了。
The URLs are shortened for better readability. = 网址被缩短以提高可读性。

疑问句:
What shortener do you recommend? = 你推荐哪个缩短器？
Does this shortener track clicks? = 这个缩短器是否追踪点击量？

否定句:
I didn't shorten the link. = 我没有缩短链接。
That shortener doesn't support custom URLs. = 那个缩短器不支持自定义网址。
