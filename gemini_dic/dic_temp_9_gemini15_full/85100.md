hardcoding:/ˈhɑːrdˌkoʊdɪŋ/| n. |硬编码，硬性编码 |The act of directly embedding values or data into the source code of a program instead of retrieving them from a configuration file or database.  This makes the code less flexible and harder to modify. |例句：Hardcoding values into the application makes it difficult to change them without recompiling the code.  硬编码值到应用程序中使得更改它们而不重新编译代码变得困难。
例句：We avoided hardcoding the database connection string; instead, we used a configuration file. 我们避免了硬编码数据库连接字符串；相反，我们使用了配置文件。|近义词：embedding, integrating, fixing|反义词：configuring, parameterizing
