protograph:/ˈproʊtəɡræf/ | n. |原型图，原型；母图 |A protograph is a graphical representation of a code, often used in the context of low-density parity-check (LDPC) codes. It serves as a template for constructing larger, more complex codes.  It's a basic building block that can be repeated and modified to create different code structures.  |例句：The protograph was designed to achieve good performance with iterative decoding. 原型图的设计是为了在迭代解码中获得良好的性能。|近义词：template, model, archetype |反义词：final product, finished code |名词
