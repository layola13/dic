prog:/prɒɡ/|ˈprɑːɡ/| n. |程序，进程；节目；课程|A sequence of operations or instructions designed to perform a specific task, especially one executed by a computer.|例句：The program is running smoothly.  程序运行流畅。|例句：The TV program started late. 电视节目开播晚了。|近义词：process, routine, schedule|反义词：end, conclusion


USAGE->
一般现在时:
The program runs well. = 程序运行良好。
This program teaches English. = 这个程序教英语。


一般过去时:
The program ran yesterday. = 程序昨天运行过。
The program finished early. = 节目提前结束了。


现在完成时:
The program has finished. = 程序已结束。
We have completed the program. = 我们已经完成了这个程序。


过去完成时:
The program had already started before I arrived. = 我到达之前节目已经开始了。
They had run the program several times. = 他们已经运行了该程序几次。


将来时:
The program will start soon. = 程序很快就会开始。
The program will be updated next week. = 程序下周将更新。


虚拟语气:
If the program were running, we would see the results. = 如果程序正在运行，我们会看到结果。
I wish the program had worked better. = 我希望程序运行得更好。


被动语态:
The program is being developed. = 程序正在开发中。

疑问句:

Which program is better? = 哪一个程序更好？


否定句:

The program didn't run correctly. = 程序没有正确运行。
