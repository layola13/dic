ran-up:/ræn ʌp/| v.| 积累；迅速增加；追赶；(债务)累积 |To increase rapidly; to accumulate; to catch up with someone or something; to accumulate debts. |例句：Their debts ran up to a huge amount.他们的债务累积到了一个巨大的数额。 |例句：He ran up the stairs to answer the phone. 他跑上楼去接电话。|近义词：accumulate, increase, amass, build up |反义词：decrease, diminish, reduce


USAGE->
一般现在时:
The company's profits run up steadily. = 公司的利润稳步增长。
The debt runs up every month. = 每月债务都在增加。


一般过去时:
They ran up a huge bill at the restaurant. = 他们在餐厅刷了一笔巨款。
The runner ran up the hill quickly. = 赛跑者迅速跑上山。


现在完成时:
We have run up against a serious problem. = 我们遇到了一个严重的问题。
She has run up a large credit card debt. = 她欠下了巨额信用卡债务。


过去完成时:
By the end of the year, they had run up significant losses. = 到年底，他们已经累计了大量的损失。
He had run up the stairs before I could stop him. = 我还没来得及阻止他，他已经跑上楼了。


将来时:
They will run up against more difficulties in the future. = 未来他们会遇到更多困难。
The total cost will run up to $10,000. = 总成本将达到1万美元。


虚拟语气:
If they continued to spend like that, they would run up a massive debt. = 如果他们继续那样花钱，他们会欠下巨额债务。
If I had run up the stairs, I might have missed the phone call. = 如果我跑上楼，我可能会错过电话。


被动语态:
A large debt was run up by the company. = 公司欠下了巨额债务。（这句被动语态略显不自然，主动语态更常用）
The score was run up by the opposing team. = 比分被对方球队迅速拉开了。（此处的“run up”指迅速增加比分）


疑问句:
How much debt have they run up? = 他们欠了多少钱？
Did they run up the stairs? = 他们跑上楼了吗？


否定句:
They didn't run up the stairs. = 他们没有跑上楼。
The company didn't run up any losses this year. = 公司今年没有亏损。


PHRASE->
run up against:  遇到（困难）
run up a bill:  欠下一笔账
run up a flag: 升起旗帜
run up the score:  迅速增加比分
run up to:  达到（数量）
