cout:/kaʊt/| n.|标准输出流|  C++中的标准输出流对象，用于将数据输出到控制台（通常是屏幕）。它预定义在iostream库中，可以直接使用。| 例句：`cout << "Hello, world!";`  (输出 "Hello, world!" 到控制台) |近义词:  stdout (在其他编程语言中)|反义词:  cin (标准输入流)|名词


USAGE->
(cout本身没有时态变化，它是一个对象，使用方法是通过操作符<<进行输出。)

输出字符串：
`cout << "This is a string.";` = 输出 "This is a string." 到控制台
`cout << "The value of x is: " << x;` = 输出 "The value of x is: " 加上变量x的值到控制台


输出数值：
`cout << 123;` = 输出数字 123 到控制台
`cout << 3.14159;` = 输出数字 3.14159 到控制台


输出变量：
`int age = 30; cout << age;` = 输出变量 age 的值 (30) 到控制台
`double price = 99.99; cout << price;` = 输出变量 price 的值 (99.99) 到控制台


输出格式化数据：
`cout << std::setw(10) << 123;` = 使用setw设置输出宽度为10，输出123，在前面添加空格。
`cout << std::setprecision(2) << 3.14159;` = 使用setprecision设置精度为两位小数，输出3.14


输出换行：
`cout << "Line 1\nLine 2";` = 输出 "Line 1" 换行后输出 "Line 2"
`cout << endl;` = 输出一个换行符


PHRASE->
(cout本身不构成短语，但它与其他C++元素结合使用)

`cout << std::hex << 255;` = 以十六进制形式输出255 (输出ff)
`cout << std::oct << 255;` = 以八进制形式输出255 (输出377)
`cout << std::boolalpha << true;` = 输出true的布尔值 (输出true)
`cout << std::fixed << 3.14159;` = 以定点表示法输出3.14159
`cout << std::scientific << 1e-5;` = 以科学计数法输出1e-5
