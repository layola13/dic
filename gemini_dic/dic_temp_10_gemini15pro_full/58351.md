dto:/ˌdɪ tiː ˈoʊ/ | n. | Data Transfer Object | A data transfer object (DTO) is an object that carries data between processes.  It's a simple object with no behavior, used primarily for data exchange. |  We use DTOs to transfer data between the database and the front-end. 我们使用 DTO 在数据库和前端之间传输数据。| synonym: Value Object, POJO | antonym: Entity, Business Object | noun
