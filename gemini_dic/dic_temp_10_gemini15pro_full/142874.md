out-of-line:/ˌaʊt əv ˈlaɪn/| adj.|（代码）外联的，非内联的|In computer programming, referring to code that is not part of the main flow of execution, such as a function call or a separate block of code. |The function call was out-of-line, which improved performance. 函数调用是外联的，这提高了性能。|Out-of-line code can make the program easier to understand. 外联代码可以使程序更容易理解。|近义词: external, separate |反义词: inline, embedded|词性:形容词
