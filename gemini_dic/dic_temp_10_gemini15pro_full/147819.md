overprovokes:/ˌoʊvərprəˈvoʊks/
|v.|过度激怒，过分挑衅|To provoke excessively; to incite beyond reasonable limits.|His constant insults overprovoked her, leading to a heated argument. 他不断的侮辱过分地激怒了她，导致了一场激烈的争论。|The comedian overprovoked the audience with his controversial jokes.  这位喜剧演员用他备受争议的笑话过度激怒了观众。|近义词:inflame, incite, aggravate, exasperate|反义词:soothe, calm, placate|词性:动词

over-|prefix|过度，过分|From Old English *ofer, from Proto-Germanic *uberi. Related to over. | | | | |


provokes:/prəˈvoʊks/
|v.|激怒，挑衅，引发|To stimulate or incite (someone) to do or feel something, especially by arousing anger in them.|His remarks provoked laughter. 他的话引起了笑声。|The government's actions provoked protests. 政府的行为引发了抗议。|近义词:incite, stimulate, anger, irritate|反义词:soothe, calm, appease|词性:动词

pro-|prefix|向前，提前|From Latin pro, meaning "forward," "forth," "in front of," "before," "on behalf of," "instead of."| | | | |


voke:|voʊk/
|v.|激怒，挑衅|From Latin vocare "to call," originally "to call upon (a god), invoke." Meaning shifted to "challenge, provoke" in Late Latin. | | | | |
