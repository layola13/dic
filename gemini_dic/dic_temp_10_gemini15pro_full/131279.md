nonce:/nɑːns/| n.|一次性使用；随机数；临时使用的值；（为特定场合而造的）临时词|A number or string used only once. A value generated for a particular use, typically related to cryptography or authentication.  A word coined for a particular occasion.|The system generates a new nonce for each transaction to prevent replay attacks.  系统为每笔交易生成一个新的随机数，以防止重放攻击。|synonyms:  one-off, single-use, ephemeral|antonyms: permanent, persistent|noun
