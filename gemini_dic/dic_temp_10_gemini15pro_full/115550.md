mappings:/ˈmæpɪŋz/| n. |映射，映射关系，制图|A mapping is a relationship between two sets of data, where each element of the first set is related to one or more elements of the second set.  It's often used in computer science and mathematics to describe how data is transformed or connected.  In cartography, it refers to the creation of maps.|例句：The mappings between the database and the user interface are complex. 数据库和用户界面之间的映射关系很复杂。

例句：The software creates mappings between different file formats.  该软件创建不同文件格式之间的映射。|近义词：correspondence, association, relation, transformation|反义词: N/A |词性: noun
