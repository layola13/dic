decompiling:/ˌdiːkəmˈpaɪlɪŋ/
|n.|反编译|The process of converting executable code (like .exe or .class files) back into a higher-level, more readable source code (like Java or C++).|Example: Decompiling the program revealed how the encryption algorithm worked. 反编译程序揭示了加密算法的工作原理。|Example:  Decompiling can be useful for understanding how a program works, but it can also be used for malicious purposes. 反编译有助于理解程序的工作原理，但也可能被用于恶意目的。|Synonyms: reverse engineering, disassembly|Antonyms: compiling|noun

decompile:/ˌdiːkəmˈpaɪl/
|v.|反编译|To convert executable code back into a more readable source code format.|Example: I decompiled the application to understand its functionality. 我反编译了应用程序以了解其功能。|Example: He tried to decompile the game to access its hidden levels. 他试图反编译游戏以访问其隐藏关卡。|Synonyms: reverse engineer, disassemble|Antonyms: compile|verb
