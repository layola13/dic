bootloader:/ˈbuːtˌləʊdə(r)/ (US) /ˈbuːtˌloʊdər/ (UK)| n.|引导加载程序，启动加载程序|A program that loads an operating system or other system software into a computer's main memory.  It is the first software program that runs when a computer starts. |Example 1: The bootloader is essential for starting the operating system. 引导加载程序对于启动操作系统至关重要。

Example 2:  A damaged bootloader can prevent the computer from booting. 损坏的引导加载程序可能会阻止计算机启动。|Synonym: boot manager, boot program|Antonym: N/A|Noun
