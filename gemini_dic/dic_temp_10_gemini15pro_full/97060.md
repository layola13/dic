PHRASE->
There are not many common set phrases using "inferentially" because it's a formal word. It's more commonly used within a sentence to describe the way something is concluded.  However, you can use it in constructions like:

1.  "It can be inferentially argued that..."  可以推断地认为...
2. "One can inferentially conclude that..." 可以推断地得出结论...
3. "Inferentially speaking, ..."  从推断的角度来说，...
4.  "Based on the evidence, we can inferentially suggest that..."  根据证据，我们可以推断地建议...
5. "The data inferentially points to..." 数据推断地指向...
