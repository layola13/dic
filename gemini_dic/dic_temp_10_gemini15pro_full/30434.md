centralizer:/ˈsɛntrəˌlaɪzər/| n.|中心化装置，集中器，扶正器，中心化器，（群论中的）中心化子|A device that centers something, or in mathematics, the set of elements that commute with a given element.|The centralizer ensured the drill bit remained aligned.  扶正器确保钻头保持对齐。|The centralizer of an element 'a' in a group 'G' is the set of all elements in 'G' that commute with 'a'. 在群 'G' 中元素 'a' 的中心化子是 'G' 中所有与 'a' 交换的元素的集合。|Synonyms: stabilizer, normalizer|Antonyms: decentralizer|Noun
