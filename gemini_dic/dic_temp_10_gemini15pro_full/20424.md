bitmapped:/ˈbɪtˌmæpt/| adj.|位图的，点阵图的|An image represented as a two-dimensional array of pixels, each of which is assigned a color value.  It is contrasted with vector graphics, which use mathematical formulas to describe shapes.|The logo was saved as a bitmapped image.  该标识被保存为位图图像。|近义词：raster, pixelated|反义词：vector|词性:形容词

bitmapped:/ˈbɪtˌmæpt/| n.|位图|A bitmapped image. |The program can convert vector graphics to bitmaps. 该程序可以将矢量图形转换为位图。|近义词：raster image, pixel image|反义词：vector graphic|词性:名词
