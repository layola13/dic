callback: /ˈkɔːlbæk/ (US) /ˈkɔːlbæk/ (UK) | n. | 回调，回叫 | A function that is passed as an argument to another function and is executed later. |  Example: The `sort` function uses a callback to compare elements. `sort`函数使用回调来比较元素。

| Example: We use callbacks to handle asynchronous operations. 我们使用回调来处理异步操作。 | Synonyms: handler, return function | Antonyms: N/A | noun
