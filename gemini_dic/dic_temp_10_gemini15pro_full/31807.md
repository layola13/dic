checkpoint:/ˈtʃɛkˌpɔɪnt/| n.|检查站，关卡，控制点|A place where people or vehicles are stopped and inspected. |例句：The police set up a checkpoint on the highway. 警察在高速公路上设立了检查站。|近义词：control point, inspection point, roadblock|反义词: None|noun

checkpoint:/ˈtʃɛkˌpɔɪnt/| n.|（计算机）检查点|A point in a computer program where data is saved so that processing can be restarted from that point if there is a problem. |例句：The program automatically creates checkpoints every few minutes. 程序每隔几分钟自动创建检查点。|近义词：save point, restore point|反义词: None|noun
