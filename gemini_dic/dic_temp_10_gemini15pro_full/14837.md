axios:/ˈæksiɒs/ (US) /ˈæksiɒs/ (UK)| n.|Axios|A promise-based HTTP client for the browser and Node.js.  It is used to make HTTP requests (like GET, POST, PUT, DELETE) from your JavaScript code to a server or API.|

Example:
axios.get('/user?ID=12345')
  .then(function (response) {
    console.log(response.data);
  })
  .catch(function (error) {
    console.log(error);
  });
中文：使用axios发起GET请求获取用户信息，如果成功则打印响应数据，如果失败则打印错误信息。


Example:
axios.post('/user', {
    firstName: 'Fred',
    lastName: 'Flintstone'
  })
  .then(function (response) {
    console.log(response.data);
  })
  .catch(function (error) {
    console.log(error);
  });
中文：使用axios发起POST请求提交用户数据，如果成功则打印响应数据，如果失败则打印错误信息。

|近义词：fetch, XMLHttpRequest, jQuery.ajax|反义词: None|词性: Noun (in practice often used as if it were a function)|
