ftps:/ˈɛf tiː piː ɛs/| n. |文件传输协议安全(File Transfer Protocol Secure)|A secure extension of the File Transfer Protocol (FTP) that offers secure communication channels for transferring files over a network. It uses SSL/TLS encryption to protect data integrity and confidentiality.|例句：We use FTPS to transfer sensitive financial documents. 我们使用FTPS传输敏感的财务文件。

例句：The server supports both FTP and FTPS for file uploads. 服务器支持FTP和FTPS文件上传。|近义词：SFTP, HTTPS|反义词:FTP|abbr.
