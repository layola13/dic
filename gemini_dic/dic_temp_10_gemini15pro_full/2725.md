adt:/ˈeɪ diː tiː/
noun
抽象数据类型 (Abstract Data Type)
An Abstract Data Type (ADT) is a theoretical model of a data structure that specifies the operations that can be performed on the data, but doesn't specify how those operations are implemented.  It focuses on "what" the data structure does rather than "how" it does it.
The List ADT supports operations like insert, delete, and search, regardless of whether it's implemented as an array or a linked list.  这个列表ADT支持插入、删除和搜索等操作，而不管它是用数组还是链表实现的。

The Stack ADT can be implemented using either an array or a linked list.  堆栈ADT可以使用数组或链表来实现。

近义词:data structure, data type
反义词:concrete data type
词性:n.
