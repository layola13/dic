combinator:/ˈkɒmbɪneɪtə(r)/ (US) /ˈkɑːmbɪneɪtər/| n. |组合子，组合器|A function that manipulates other functions without referencing any external values.  It's a higher-order function that takes one or more functions as arguments and returns a new function.  Used in functional programming and lambda calculus.|The Y combinator is a famous example in computer science. Y组合子是计算机科学中的一个著名例子。|近义词:function, operator|反义词: N/A|词性:noun
