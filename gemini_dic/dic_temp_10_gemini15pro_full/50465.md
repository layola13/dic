depredicate:/dɛprɪkeɪt/ | v. | 弃用，不赞成使用 | To mark (something, especially a software feature) as deprecated, meaning it is still functional but expected to be removed in future versions and should not be used in new code.  It indicates that the feature is obsolete and a better alternative exists. | Example 1: The developers have deprecated the old API and encourage developers to switch to the new one.  开发人员已经弃用了旧的API，并鼓励开发人员切换到新的API。

Example 2:  Although the function is deprecated, it will remain available for backward compatibility. 虽然该函数已被弃用，但为了向后兼容，它仍将可用。 | Synonyms:  obsolete, discontinue, retire, phase out | Antonyms:  support, maintain, introduce, promote | Verb
