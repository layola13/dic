focusable:/ˈfoʊkəsəbl/| adj.|可聚焦的，可获得焦点的|Able to be brought into focus; able to receive focus.  It often refers to elements in a user interface that can be selected using a keyboard or other input device.|The button is focusable, allowing users to navigate to it using the tab key.  这个按钮是可聚焦的，允许用户使用Tab键导航到它。|synonyms: selectable, accessible, targetable|antonyms: unfocusable, inaccessible|adjective
