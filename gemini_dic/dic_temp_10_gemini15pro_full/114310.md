maintainable:/meɪnˈteɪnəbl/
adjective
可维护的
able to be maintained in good condition; able to be supported or sustained.
The system must be designed to be easily maintainable.  这个系统必须设计成易于维护。
This code is not very maintainable, we need to refactor it. 这段代码可维护性很差，我们需要重构它。
近义词:supportable, sustainable, repairable, serviceable
反义词:unmaintainable, unsustainable, irreparable
词性:adjective
