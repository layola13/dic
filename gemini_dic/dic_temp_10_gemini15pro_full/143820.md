outpath:/ˈaʊtˌpæθ/| n. |输出路径|The location where output files or data are written.  It specifies the directory and/or filename where the results of a process or program are stored.|The program saved the results to the specified outpath. 程序将结果保存到指定的输出路径。|例句：Please specify the outpath for the generated report. 请指定生成报告的输出路径。|近义词：output directory, output location, destination path|反义词: input path|noun
