checkpoint:/ˈtʃekpɔɪnt/| n.|检查站，关卡，控制点|A point at which progress is checked.|The police set up checkpoints on the roads leading out of the city. 警方在通往城外的道路上设立了检查站。|近义词:waystation, stop, station, control point|反义词:destination|名词

checkpoint:/ˈtʃekpɔɪnt/| n.|（尤指电脑工作的）检查点，存档点|A point in a computer program at which information is recorded so that the program can be restarted from that point if there is a problem.|The program automatically creates checkpoints every few minutes. 程序每隔几分钟自动创建检查点。|近义词:save point, restore point|反义词: (No direct antonym)|名词
