hashmaps:/ˈhæʃmæps/| n. |哈希映射，哈希表|A hashmap (or hash table) is a data structure that implements an associative array abstract data type, a structure that can map keys to values. A hash table uses a hash function to compute an index, also called a hash code, into an array of buckets or slots, from which the desired value can be found.|We used a hashmap to store the user data for quick retrieval. 我们使用哈希映射来存储用户数据以便快速检索。|近义词：hash table, dictionary, associative array|反义词: N/A|名词
