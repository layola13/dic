PHRASE->
`std::cin >> x;`  读取用户输入并存储到变量x。
Read user input and store it in variable x.

`std::cin.get();`  读取一个字符，包括空格和换行符。
Read a single character, including whitespace and newline.

`std::cin.ignore();`  忽略输入流中的下一个字符。
Ignore the next character in the input stream.

`std::cin.clear();`  清除输入流的错误状态标志。
Clear the error flags of the input stream.

`std::cin.fail();`  检查输入流是否处于错误状态。
Check if the input stream is in an error state.


注意事项:

`cin` is specific to C++. Other languages have different mechanisms for handling standard input.  
`cin` operations are usually synchronous, meaning the program waits for user input before proceeding.
