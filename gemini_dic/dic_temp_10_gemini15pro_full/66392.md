executables:/ˌɛksɪˈkjuːtəblz/ | n. | 可执行文件 | Files that can be run by a computer's operating system.  These files contain instructions that the computer can directly understand and execute. |  The executables for this program are located in the 'bin' directory.  这个程序的可执行文件位于 'bin' 目录中。 | synonyms: binaries, programs, applications | antonyms: data files, source code | noun

executable:/ˌɛksɪˈkjuːtəbl/ | adj. | 可执行的 | Capable of being executed, or put into effect; performable. | The file is executable, so you can double-click it to run it. 该文件是可执行的，所以你可以双击运行它。 | synonyms: runnable, operable | antonyms: non-executable, inoperable | adjective
