decompiled:/ˌdiːkəmˈpaɪld/ | adj. | 反编译的 | Converted from machine code back into a higher-level programming language.  It refers to the process of reversing the compilation process, turning executable code back into more readable source code. | The decompiled code was difficult to understand.  反编译后的代码很难理解。 | The programmer decompiled the software to analyze its functionality. 程序员反编译了软件来分析它的功能。 |近义词: reverse-engineered, disassembled |反义词: compiled | 词性:形容词
