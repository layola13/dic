huffman:/ˈhʌfmən/| n. adj.|霍夫曼，霍夫曼编码，霍夫曼树|A variable-length, prefix-free code used for lossless data compression. Named after David A. Huffman, who developed the algorithm in 1952. It assigns shorter codes to more frequent symbols and longer codes to less frequent symbols.|The image file was compressed using Huffman coding.  图像文件使用霍夫曼编码进行压缩。| The Huffman tree is a binary tree used in the algorithm. 这颗霍夫曼树是算法中使用的二叉树。|近义词:coding, compression|反义词:decompression, expansion|词性:名词, 形容词
