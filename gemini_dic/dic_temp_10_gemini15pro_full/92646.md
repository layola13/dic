ibidem:/ɪˈbaɪdɛm/ /ɪˈbiːdɛm/| adv.|同上，同前|In the same place (used in citations to refer to a previously mentioned work).|Example 1:  The information can be found in Smith (2023, p. 12) and ibidem.| 例1：该信息可以在Smith（2023，第12页）和同前找到。

Example 2: See ibidem for further details.| 例2：更多细节见同前。|近义词：id., ibid., op. cit.|反义词:None|词性:副词

一般现在时 (Simple Present):
Example 1: He reads the same book every day. 他每天都读同一本书。
Example 2:  She goes to the library every week. 她每周都去图书馆。

现在进行时 (Present Continuous):
Example 1: He is reading the same book again. 他正在重读同一本书。
Example 2: She is studying in the library now. 她现在正在图书馆学习。

现在完成时 (Present Perfect):
Example 1: He has read that book twice. 他已经读过那本书两次了。
Example 2: She has been to the library many times. 她去过图书馆很多次了。

现在完成进行时 (Present Perfect Continuous):
Example 1: He has been reading the same book all day. 他一整天都在读同一本书。
Example 2: She has been studying in the library for hours.  她在图书馆学习好几个小时了。

一般过去时 (Simple Past):
Example 1: He read the same book yesterday. 他昨天读了同一本书。
Example 2: She went to the library last week. 她上周去了图书馆。


过去进行时 (Past Continuous):
Example 1: He was reading the same book when I called. 我打电话时他正在读同一本书。
Example 2:  She was studying in the library when the fire alarm went off. 火灾警报响起时，她正在图书馆学习。

过去完成时 (Past Perfect):
Example 1: He had already read the book before I recommended it. 在我推荐之前，他已经读过那本书了。
Example 2: She had left the library before it closed. 在图书馆关门之前，她已经离开了。


过去完成进行时 (Past Perfect Continuous):
Example 1: He had been reading the same book for hours before he fell asleep. 他睡着前已经读了几个小时的同一本书。
Example 2: She had been studying in the library all day before she went home. 在她回家之前，她已经在图书馆学习了一整天。


一般将来时 (Simple Future):
Example 1: He will read the same book tomorrow. 他明天会读同一本书。
Example 2:  She will go to the library next week. 她下周会去图书馆。


将来进行时 (Future Continuous):
Example 1: He will be reading the same book this time tomorrow. 明天这个时候他将会在读同一本书。
Example 2: She will be studying in the library at this time next week. 下周这个时候她将会在图书馆学习。


将来完成时 (Future Perfect):
Example 1: He will have read the book by next week. 他下周之前会读完这本书。
Example 2: She will have finished her research by the end of the month. 她会在月底前完成她的研究。


将来完成进行时 (Future Perfect Continuous):
Example 1: By next month, he will have been reading the same book for a year. 到下个月，他读同一本书就满一年了。
Example 2:  By next year, she will have been studying at the library for five years. 到明年，她在图书馆学习就满五年了。


虚拟语气:
Example 1: If I had the same book, I would read it. 如果我有同一本书，我会读它。
Example 2: If she were here, she would go to the library with us. 如果她在这里，她会和我们一起去图书馆。



被动语态:
Example 1: The same book was read by many students.  许多学生读了同一本书。
Example 2: The library is used by many people every day.  每天都有很多人使用图书馆。



疑问句:
Example 1: Did he read the same book?  他读了同一本书吗？
Example 2: Does she go to the library often?  她经常去图书馆吗？



否定句:
Example 1: He didn't read the same book. 他没有读同一本书。
Example 2: She doesn't go to the library often. 她不经常去图书馆。
