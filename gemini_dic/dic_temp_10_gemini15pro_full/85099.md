hardcoded:/ˌhɑːrdˈkoʊdɪd/| adj.|硬编码的，硬连接的|Embedded directly into the source code of a program or device, rather than being obtained from external sources or calculated at runtime.  This makes the value fixed and difficult to change without modifying the code itself.|The database password was hardcoded into the application, which is a security risk.|近义词：fixed, embedded, built-in|反义词:configurable, dynamic, flexible|词性:形容词
