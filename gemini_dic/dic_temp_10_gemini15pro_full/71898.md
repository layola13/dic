flexbox:/ˈflɛksˌbɒks/| n. |弹性盒子|A layout model in CSS that provides a more efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown and/or dynamic.|The website uses flexbox for its responsive design. 该网站使用flexbox进行响应式设计。|近义词: flexible box layout, flex layout, grid layout (partially)|反义词: block layout, float layout|noun
