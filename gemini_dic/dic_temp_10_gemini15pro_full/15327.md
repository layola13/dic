backslashes:/ˈbækˌslæʃɪz/ | n. | 反斜杠 | A backslash is a typographical mark ( \ ) used primarily in computing.  It is used to escape special characters in strings and regular expressions, and to specify file paths in some operating systems. |  The file path is C:\Users\Documents.  文件路径是 C:\用户\文档. | synonyms: reverse solidus, backslash character | antonyms: (no direct antonym, perhaps forward slash /) | noun

ETYMOLOGY->
From *back* + *slash*.  So named because it leans *back*wards, in contrast to the forward slash (/).
