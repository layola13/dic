nonprimitives:/nɒnˈprɪmɪtɪvz/ | n. | 非基本类型，非原始类型 | Data types that are more complex than primitive types, such as arrays, objects, and strings.  They are often built using primitive types. |  Java uses nonprimitives like Strings and Arrays. Java使用像字符串和数组这样的非基本类型。| synonyms: composite types, reference types | antonyms: primitives, basic types | noun

nonprimitive:/nɒnˈprɪmɪtɪvz/ | adj. | 非基本类型的，非原始类型的 | Not a primitive data type.  Describing data types that are composed of or based on primitive types. | Many programming languages support nonprimitive data types. 许多编程语言支持非基本数据类型。| synonyms: composite, reference | antonyms: primitive, basic | adjective
