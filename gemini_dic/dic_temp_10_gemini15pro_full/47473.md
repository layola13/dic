deallocates:/diːˈæləˌkeɪts/
verb
解除分配，释放（内存）
To release (memory) that has been allocated.
The system deallocates the memory assigned to the variable when the program ends.  系统在程序结束时释放分配给变量的内存。
The operating system deallocates memory when a program finishes. 操作系统在程序结束后会释放内存。
synonyms: releases, frees, unallocates
antonyms: allocates, reserves, assigns
词性: 动词
