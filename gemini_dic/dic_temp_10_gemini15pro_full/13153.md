async: /ˈeɪsɪŋk/ /eɪˈsɪŋk/ | adj. | 异步的 |  Not occurring at the same time.  Relating to operations that do not block the execution of other operations. | Example:  The `async` keyword allows you to write asynchronous code in JavaScript.  `async`函数允许你在 JavaScript 中编写异步代码。


| Example: We use async requests to fetch data from the server without blocking the user interface. 我们使用异步请求从服务器获取数据，而不会阻塞用户界面。 | 近义词: non-blocking, concurrent | 反义词: synchronous, blocking | 词性: adjective
