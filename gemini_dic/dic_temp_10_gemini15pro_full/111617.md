localhost:/ˈloʊkəlˌhoʊst/| n.|本地主机|A standard hostname given to the address of the loopback network interface (127.0.0.1).  It refers to the current computer itself.|Accessing the website via localhost allows developers to test it before deploying it to a live server.|Synonym: loopback address|Antonym: remote host|noun
