bindable:/ˈbaɪndəbl/| adj.|可绑定的，可装订的|Suitable for binding, capable of being bound.|The report was over 500 pages long, so we needed to make it bindable.  这份报告超过500页，所以我们需要把它装订起来。|近义词：attachable, securable|反义词：unbound, loose|词性:形容词

bindable:/ˈbaɪndəbl/| adj.|（在编程中）可绑定的|In programming, refers to a property or data element that can be connected to a user interface element, so that changes in one automatically affect the other.|The "text" property of the text box is bindable to the "name" variable. 文本框的“文本”属性可以绑定到“名称”变量。|近义词：linkable, connectable|反义词：unbindable, unconnected|词性:形容词
