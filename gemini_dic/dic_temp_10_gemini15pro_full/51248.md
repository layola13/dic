destructor:/dɪˈstrʌktər/| n.|析构函数，解构器|A special method in object-oriented programming that is automatically called when an object is destroyed or goes out of scope, used to release resources like memory allocated to the object.|The destructor releases the memory allocated to the object. 析构函数释放分配给对象的内存。|synonyms: finalizer|antonyms: constructor|noun
