morse-alphabets:/ˈmɔːrs ˈælfəˌbɛts/ | n. | 摩尔斯电码 | A system of representing letters, numbers, and punctuation marks by dots, dashes, and spaces, used for transmitting messages by telegraph, radio, or other means. |  The message was sent using Morse alphabets. 这条消息是用摩尔斯电码发送的。 | synonyms: Morse code | antonyms: None | noun

morse-alphabet:/ˈmɔːrs ˈælfəˌbɛt/ | n. | 摩尔斯字母表 |  A table showing the representation of letters, numbers, and punctuation marks in Morse code.  | He consulted the Morse alphabet to decode the message. 他查阅了摩尔斯字母表来解码这条消息。 | synonyms: Morse code chart, Morse code table | antonyms: None | noun
