how-so:/haʊˈsoʊ/| adv.|怎么会这样；怎么；为什么如此|Used to ask someone to explain something more clearly, especially something surprising or unexpected.| "He's leaving?" "How so?"  “他要走了？” “怎么会这样？”|Synonyms: How come?, Why?, In what way?, How?, Really?|Antonyms:  (No direct antonyms, but consider phrases like "That's expected" or "That makes sense" which express the opposite sentiment)
词性: 副词


USAGE->

一般现在时 (Simple Present):
How so is it always so cold here? = 这里为什么总是这么冷？
How so do you manage to stay so fit? = 你是怎么保持这么健康的？

现在进行时 (Present Continuous):
How so are you feeling better now? = 你现在感觉好些了吗？（如果之前感觉不好，现在好转了，问原因）
How so is she managing with all that work? = 她如何应付所有那些工作？

现在完成时 (Present Perfect):
How so have you finished the project so quickly? = 你怎么这么快就完成了这个项目？
How so has the situation changed so drastically? = 情况怎么会发生如此巨大的变化？

现在完成进行时 (Present Perfect Continuous):
How so have you been coping with the stress? = 你一直以来是如何应对压力的？
How so has he been improving his skills so rapidly? = 他怎么会进步如此神速？

一般过去时 (Simple Past):
How so did you get here so fast? = 你怎么这么快就到这里了？
How so did they manage to win the game? = 他们是怎么赢得比赛的？

过去进行时 (Past Continuous):
How so were you feeling before the operation? = 手术前你感觉如何？（如果手术是由于某种状况引起的）
How so was she dealing with the problem at that time? = 她当时是如何处理这个问题的？

过去完成时 (Past Perfect):
How so had the situation deteriorated so badly? = 情况怎么会恶化到如此地步？
How so had he managed to escape? = 他是怎么逃脱的？

过去完成进行时 (Past Perfect Continuous):
How so had you been preparing for the exam? = 你一直在如何准备考试？ (暗示考试结果出乎意料)
How so had they been working on the project for so long without success? = 他们怎么会在这个项目上工作了这么久都没有成功？


一般将来时 (Simple Future):
How so will you travel to Europe? = 你将如何前往欧洲？ (如果旅行方式比较特别或出乎意料)
How so will the new policy affect the economy? = 新政策将如何影响经济？ (如果影响的方式比较特别或出乎意料)

将来进行时 (Future Continuous):
How so will you be spending your vacation? = 你将如何度过你的假期? (如果度假方式比较特别或出乎意料)
How so will they be managing the increased workload? = 他们将如何管理增加的工作量？ (如果管理方式比较特别或出乎意料)


将来完成时 (Future Perfect):

How so will the company have achieved such growth in just five years? = 公司怎么会在短短五年内取得如此大的增长？

将来完成进行时 (Future Perfect Continuous):
How so will you have been living in this city for ten years by then? = 到那时你将如何在这个城市生活十年？(如果居住方式比较特别或出乎意料)
How so will she have been studying for so long without graduating? =  她怎么学了这么久还没毕业？

PHRASE->

How so? Really? = 真的吗？怎么会这样？
But how so? = 但是怎么会这样呢？
How so exactly? = 具体是怎么样的？
How so can you be sure? = 你怎么确定呢？
How so did that happen? = 那是怎么发生的？
