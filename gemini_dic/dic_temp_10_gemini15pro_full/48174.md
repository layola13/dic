decompiler:/ˌdiːkəmˈpaɪlər/| n. |反编译器|A program that takes executable code as input and attempts to create a high-level, more easily understandable source code equivalent.  It reverses the process of compilation.|Example 1: The decompiler successfully reconstructed the original source code from the binary file. 反编译器成功地从二进制文件中重建了原始源代码。

Example 2:  Using a decompiler can help understand the functionality of closed-source software. 使用反编译器可以帮助理解闭源软件的功能。|近义词:reverse engineer, disassembler|反义词:compiler|词性:noun


decompilers:/ˌdiːkəmˈpaɪlərz/| n. |反编译器（复数）|Plural form of "decompiler," referring to multiple programs that reverse the compilation process.|Example 1: Several decompilers were used to analyze the different versions of the software.  几个反编译器被用来分析不同版本的软件。

Example 2: Modern decompilers can handle a variety of programming languages and platforms. 现代反编译器可以处理各种编程语言和平台。|近义词:reverse engineering tools|反义词:compilers|词性:noun
