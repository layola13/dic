IIFE:/ˈɪfi/| n.|Immediately Invoked Function Expression (立即执行函数表达式)|A JavaScript function that runs as soon as it is defined.|
例句：
(function() { console.log("Hello from an IIFE!"); })(); 
(function(){ 控制台输出("来自IIFE的问候！"); })();

例句：
let result = (function(x, y) { return x + y; })(5, 3);
let result = (function(x, y) { return x + y; })(5, 3); //result will be 8  //result 的值将为 8

近义词：self-executing anonymous function|
反义词:|
词性:noun
