in the event of:/ɪn ði ɪˈvɛnt əv/
conjunction
如果；万一；倘若
In case of; should something happen.  This phrase introduces a possibility.
In the event of fire, break the glass. 万一发生火灾，打破玻璃。
Synonyms: in case of, if, should
Antonyms: unless, except if
词性: 连词 (conjunction)

in the event of:/ɪn ði ɪˈvɛnt əv/
preposition
如果；万一；倘若
In case of; should something happen. This phrase introduces a possibility.
The game will be postponed in the event of rain.  如果下雨，比赛将延期。
Synonyms: in case of, if, should
Antonyms: unless, except if
词性: 介词 (preposition)
