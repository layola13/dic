mappers:/ˈmæpərz/| n.|映射器，制图员|People who create maps, or software developers who write code to transform data from one format to another.  They might work on geographical maps, or data maps used in computing.|Cartographers are mappers who create detailed maps of the Earth's surface.  Software mappers create code to transform data for different applications.|近义词：cartographers, coders, programmers, transformers|反义词: N/A|noun
