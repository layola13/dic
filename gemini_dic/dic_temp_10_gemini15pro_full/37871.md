compiling:/kəmˈpaɪlɪŋ/
| v. (present participle)|编译；汇编；编纂|The process of transforming source code written in a programming language into machine-readable code that a computer can execute.  It involves several steps, including lexical analysis, syntax analysis, semantic analysis, optimization, and code generation.|例句：The program is compiling slowly due to its large size.  由于程序很大，编译速度很慢。

例句：He is compiling a dictionary of slang. 他正在编纂一本俚语词典。|近义词：assembling, building, composing|反义词:decompiling, disassembling|verb

compile:/kəmˈpaɪl/
| v.|编译；汇编；编纂|To convert (a program) into a machine-language form.  Also, to gather (information) into a single work. |例句：The compiler transformed the code into an executable file.  编译器将代码转换为可执行文件。

例句：She compiled a list of resources for the project. 她为该项目汇编了一份资源清单。|近义词：assemble, build, compose|反义词:decompile, disassemble|verb

ETYMOLOGY->
From Old French compiler "to collect, gather, heap up," from Latin compilare "to plunder, rob, pillage," later "to collect, gather," from com- "together" + pilare "to compress into a ball." The sense shift likely occurred through the notion of gathering things together.
