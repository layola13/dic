multiton:/ˈmʌltɪtən/
n.|多例|A design pattern in software engineering where a limited number of instances of a class are created and reused, rather than creating a new instance each time. It's a variation of the singleton pattern, but allows for multiple instances (a fixed number).|The application uses a multiton pattern to manage database connections, creating only five instances to handle all database interactions.|近义词:pool, cache|反义词:factory|词性:名词
