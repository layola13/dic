getline:/ˈɡɛtˌlaɪn/ | function (in C++) | 读取一行 | Reads a line of text from an input stream, usually stdin (standard input), until a newline character is encountered.  It is often used to read user input from the console. |
Example 1: `std::string line; std::getline(std::cin, line);`  // Reads a line from standard input and stores it in the string variable 'line'. (从标准输入读取一行并将其存储在字符串变量“line”中。)
Example 2:  `std::ifstream myfile("example.txt"); std::string line; if (myfile.is_open()) {  std::getline(myfile, line); }` // Reads a line from the file "example.txt". (从文件“example.txt”中读取一行。) |
