/ˈɪnˌfaɪl/ | n. | 输入文件 | A file that is used as input to a program or command.  Data is read from an infile. |  The program reads data from the infile and writes results to the outfile. 程序从输入文件读取数据并将结果写入输出文件。| synonyms: input file, data file | antonyms: outfile, output file | noun

/ˈɪnˌfaɪl/ | v. | 将…作为输入文件 | To use a file as input to a program or command. | The script infiles the configuration settings.  该脚本将配置文件作为输入文件。| synonyms: input, read from file | antonyms: output, write to file | verb
