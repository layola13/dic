dereferencing:/diːˈrɛfərənsɪŋ/| n.|解引用|The process of accessing the value stored at a memory location pointed to by a pointer.  It involves using the pointer to retrieve the data that it references.|

Example: In C++, *ptr dereferences the pointer ptr, giving you the value stored at the memory address ptr points to.|

Synonym: indirection, access|

Antonym: referencing, assigning|

Noun
