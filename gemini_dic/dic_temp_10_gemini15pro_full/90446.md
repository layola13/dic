http:/eɪʧ tiː tiː piː/| n. |超文本传输协议|HyperText Transfer Protocol, the underlying protocol used by the World Wide Web. It defines how messages are formatted and transmitted, and what actions web servers and browsers should take in response to various commands.|Example:  Websites use HTTP to transfer data.  网站使用HTTP传输数据。|Example: Secure websites use HTTPS, which encrypts the HTTP traffic. 安全网站使用 HTTPS，它会加密 HTTP 流量。|Synonym: protocol|Antonym: N/A|abbreviation
