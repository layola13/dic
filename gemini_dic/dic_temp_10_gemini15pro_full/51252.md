destructuring:/dɪˈstrʌktʃərɪŋ/ /diːˈstrʌktʃərɪŋ/ | n. |解构，析构| A JavaScript expression that makes it possible to unpack values from arrays, or properties from objects, into distinct variables.|
`const [a, b] = [1, 2]; // a is 1, b is 2`  `const { name, age } = { name: 'John', age: 30 }; // name is 'John', age is 30`|近义词：unpacking, decomposition|反义词:constructing, composing|词性:noun
