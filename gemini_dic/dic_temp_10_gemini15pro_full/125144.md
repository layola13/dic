multi-threaded:/ˈmʌltiˈθrɛdɪd/| adj.|多线程的|Having multiple threads of execution, allowing a program to perform multiple tasks concurrently.|This server uses a multi-threaded architecture to handle multiple client requests simultaneously. 这台服务器使用多线程架构来同时处理多个客户端请求。|Modern operating systems are multi-threaded, enabling them to run multiple programs seemingly at the same time. 现代操作系统都是多线程的，使它们能够看似同时运行多个程序。|concurrent, parallel|single-threaded|adjective
