intellisense:/ɪnˈtɛlɪsɛns/| n.|智能感知;代码补全|A feature of some software development environments that provides automatic code completion, parameter info, quick info, and member lists. It helps developers write code faster and with fewer errors by suggesting possible code completions as they type.  It analyzes the code being written and offers suggestions based on the context.|Example:  IntelliSense in Visual Studio is a great help when coding in C#.  在Visual Studio中，智能感知对于C#编码很有帮助。

Example: The IntelliSense feature saved me a lot of time when I was developing that complex application.  智能感知功能在我开发那个复杂的应用程序时节省了我很多时间。|近义词：autocomplete, code completion, code hinting|反义词:None|词性:noun
