half-dried:/ˌhɑːfˈdraɪd/
| adj. |
| 半干的；未完全干燥的 |
| Partially dried; not completely dry.  It describes something that has lost some moisture but not all, remaining somewhat damp. |
| The half-dried laundry still felt damp.  半干的衣服仍然感觉潮湿。|

| The half-dried fish were preserved using salt. 用盐保存半干的鱼。 |
| partially dried, semi-dried |
| completely dried, saturated, soaked |
| adjective |
