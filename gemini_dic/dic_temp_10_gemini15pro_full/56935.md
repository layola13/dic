.dotm:/ˌdɒtˈɛm/ | n. | Word Open XML Macro-Enabled Document | A file extension used for Word documents that contain macros.  This format allows for the inclusion of VBA (Visual Basic for Applications) code that can automate tasks within the document. | Example: Please send the report as a .dotm file so I can use the embedded macros.  请将报告以 .dotm 文件格式发送，以便我使用嵌入的宏。| Synonym: (no direct synonym, but related to .docm, .docx) | Antonym: (N/A - file formats don't have antonyms) | file extension
