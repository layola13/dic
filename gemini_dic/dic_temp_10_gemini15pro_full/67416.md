externs:/ˈɛkstɜːrnz/
noun
外部变量; 外部函数; 外部声明
A keyword or other notation used to declare that a variable or function is defined externally.  Declaring something as an extern allows the compiler to know that the symbol exists, even if it's defined in a separate file.
The header file declares the functions as `externs`, and the source file defines them.  头文件将这些函数声明为`externs`，而源文件定义它们。
The `extern` keyword tells the compiler that the variable is defined elsewhere.  `extern`关键字告诉编译器该变量在其他地方定义。
external variables, global variables
local variables, internal variables
