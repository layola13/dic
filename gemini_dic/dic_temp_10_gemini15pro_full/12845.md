associativeness:/əˌsoʊʃiəˈtɪvnɪs/
n.|结合性，结合律|The property of an operation whereby the result of applying the operation to a set of operands is independent of how the operands are grouped.|The associativeness of addition means that (a + b) + c = a + (b + c). 加法的结合性意味着 (a + b) + c 等于 a + (b + c).|associativity|N/A|noun
Etymology: From associate + -ive + -ness.  Ultimately from Latin *associare*.
