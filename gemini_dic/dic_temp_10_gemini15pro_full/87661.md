hexadecimally:/ˌheksəˈdesɪməli/| adv.|十六进制地|Relating to or expressed in hexadecimal notation (base 16).  It uses the digits 0-9 and the letters A-F to represent values.|The computer stores the color code hexadecimally.  计算机以十六进制存储颜色代码。|The memory address is displayed hexadecimally. 内存地址以十六进制显示。|in base 16, using hexadecimal notation|N/A|adverb

ETYMOLOGY->
From hexadecimal (adjective), from hexa- (six) + decimal (ten).  Ultimately from Greek *hexa (six) and Latin decem (ten).
