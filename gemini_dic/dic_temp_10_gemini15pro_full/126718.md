namespaces:/ˈneɪmˌspeɪsɪz/| n.|命名空间|A designated area within a program, computer, or network that is assigned a name and used to organize code or data, preventing naming conflicts.  This concept is particularly important in object-oriented programming and distributed systems.|The XML namespaces recommendation provides a way to avoid element name conflicts.|近义词：scope, domain, area|反义词: N/A|noun
