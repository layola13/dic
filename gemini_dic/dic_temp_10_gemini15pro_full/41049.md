copile: /ˈkɒpaɪl/ (British English), /kəˈpaɪl/ (American English) | v. | 编译 | To convert computer code written in a high-level programming language (like Java or C++) into a low-level language (like machine code) that a computer can understand and execute. | The programmer compiled the code into an executable file. 程序员将代码编译成可执行文件。 |  The compiler detected several errors in the code. 编译器在代码中检测到几个错误。 | synonyms: build, assemble | antonyms: interpret | verb
