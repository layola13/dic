hardcode:/ˈhɑːrdkoʊd/ (US) /ˈhɑːdkəʊd/ (UK)| v. n. adj.|硬编码，硬连接|To embed data directly into the source code of a program or configuration file, rather than obtaining it from external sources or generating it at runtime.  It also refers to the result of this process, the hardcoded data itself.|例句：The database connection string is hardcoded into the application. 数据库连接字符串被硬编码到应用程序中。

例句：Changing the server address requires recompiling the application because it's hardcoded.  更改服务器地址需要重新编译应用程序，因为它是硬编码的。|近义词：embed, fixed, built-in|反义词:softcode, configurable, dynamic|词性:verb, noun, adjective (primarily used as a verb)
