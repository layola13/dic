bigquery:/ˈbɪɡˌkwɪri/| n.|BigQuery|A fully-managed, serverless data warehouse provided by Google Cloud Platform that allows for scalable analysis of massive datasets using SQL-like queries.|Example: We use BigQuery to analyze terabytes of website traffic data. 我们使用 BigQuery 来分析数 TB 的网站流量数据。

Example: BigQuery's ability to handle petabytes of data makes it a powerful tool for data scientists.  BigQuery 处理 PB 级数据的能力使其成为数据科学家们强大的工具。|近义词:data warehouse, cloud database|反义词:None|noun
