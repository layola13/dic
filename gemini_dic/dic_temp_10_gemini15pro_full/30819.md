cgi:/siːdʒiːˈaɪ/| n. |通用网关接口 (Common Gateway Interface)|A standard that defines how web servers can interact with external programs to generate dynamic content.|例句：The website uses CGI scripts to process user input.  这个网站使用CGI脚本处理用户输入。

例句：CGI allows web pages to be interactive and dynamic. CGI使得网页可以互动和动态。|近义词：server-side scripting, web application framework|反义词: (No direct antonym, perhaps static content)|词性: noun
