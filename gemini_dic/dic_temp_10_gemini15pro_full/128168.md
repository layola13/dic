netty:/ˈnɛti/ | n. | Netty |  A non-blocking I/O client-server framework for developing Java network applications, such as protocol servers and clients.  It greatly simplifies and streamlines network programming such as TCP and UDP socket server development. | Netty is commonly used in high-performance applications like game servers. Netty被广泛应用于高性能应用，例如游戏服务器。 | synonyms:  NIO framework, asynchronous networking library | antonyms: blocking I/O library | noun
