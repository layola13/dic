compilers:/kəmˈpaɪlərz/| n.|编译器|A program that translates source code written in a high-level programming language into a lower-level language (such as assembly language or machine code) that can be understood and executed by a computer.|The compilers for C++ and Java are different. C++和Java的编译器是不同的。|synonyms: translator, interpreter|antonyms: interpreter (sometimes)|noun
