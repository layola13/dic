PHRASE->
`std::cout << "Hello" << endl;`  打印"Hello"并换行。
`std::cerr << "Error!" << endl;` 打印"Error!"并换行（通常用于错误信息）。
`std::ofstream file("output.txt"); file << "Data" << endl;` 将"Data"写入文件"output.txt"并换行。
`std::cout << "First line" << endl << "Second line" << endl;` 打印两行文本，每行后面都有换行。
`std::cout << "Value: " << x << endl;` 打印变量x的值并换行。


注意事项:  endl 在 C++ 中主要用于输出流，插入换行符并刷新缓冲区。
