autocode:/ˈɔːtəʊˌkəʊd/| n. |自动编码，自动代码|A symbolic assembly language for a specific computer, enabling programmers to write programs using mnemonic codes and symbolic addresses, which are then translated into machine code by a program called an assembler.|The program was written in an early autocode.|近义词：assembly language, symbolic code|反义词:machine code|noun
