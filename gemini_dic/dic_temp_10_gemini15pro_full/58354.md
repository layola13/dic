dtype:/ˈdeɪtaɪp/ | n. |数据类型 |  A classification of data which tells the compiler or interpreter how the programmer intends to use the data.  It defines the operations that can be performed on the data and the way the values of that type can be stored. |  The dtype of this array is integer.  这个数组的数据类型是整数。| synonym: data type, type | antonym: N/A | noun
