linting:/ˈlɪntɪŋ/ | n. | 代码检查，静态代码分析 | The process of running a program that analyzes code for potential errors, bugs, stylistic errors, and suspicious constructs. |  Example: Linting helps to improve code quality and prevent bugs. 代码检查有助于提高代码质量并防止错误。

Example: Before submitting the code, make sure you run the linter. 在提交代码之前，请确保运行代码检查器。 | 近义词: static analysis, code analysis | 反义词:  (No direct antonym, but one could consider "ignoring code quality" a conceptual opposite) | 词性: noun
