delinting:/diːˈlɪntɪŋ/
noun
去除棉绒，代码清理
The process of removing lint, fluff, or debris, especially from fabric or code.  In software development, it refers to the process of removing warning messages from code, improving code style and consistency, and potentially catching some bugs.
The factory has a delinting machine to clean the fabric.  这家工厂有一台去除织物上绒毛的机器。
The developer spent hours delinting the code before submitting it. 开发人员在提交代码之前花了几个小时清理代码。
synonyms: cleaning, tidying, refining (for code)
antonyms: messing, cluttering (for code)
n.
