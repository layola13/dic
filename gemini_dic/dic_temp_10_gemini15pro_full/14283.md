autogrid:/ˈɔːtəʊɡrɪd/ | n. | 自动网格 | A software tool used to create a 3D grid for molecular simulations. It automatically determines the size and spacing of the grid based on the input molecule(s). | AutoGrid is essential for preparing molecular docking simulations.  AutoGrid对于准备分子对接模拟至关重要。 |  AutoGrid helps define the search space for ligand docking. AutoGrid有助于定义配体对接的搜索空间。 |  grid generator, meshing tool | manual grid generation | noun
