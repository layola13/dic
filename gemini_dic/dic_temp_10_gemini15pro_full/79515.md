globing:/ˈɡloʊbɪŋ/| n.|文件名扩展，通配符匹配|The process of expanding wildcards in filenames.  It's like using an asterisk (*) to select multiple files at once.|Example:  Using `*.txt` to select all text files in a directory is an example of globbing.|近义词：wildcard matching, pattern matching|反义词: (No direct antonym, perhaps "explicit file selection")|词性：名词

词源:  Derived from the term "global," referencing the broad scope of selection enabled by wildcard characters.
