accessors:/ˈæksesərz/| n.|访问器，取值器，设值器|Methods that allow retrieval or modification of the state of an object or variable.  They are often used in object-oriented programming to encapsulate data and control access to it.|The class provides accessors for the private member variables.|近义词：getters, setters|反义词:N/A|Noun

ETYMOLOGY->
From "access" + "-or" (agent suffix).  "Access" ultimately comes from Latin "accessus", meaning "approach".

USAGE->
Declaring accessors:
public String getName() { return name; } =  public String getName() { return name; } = 获取名字
public void setName(String newName) { name = newName; } = public void setName(String newName) { name = newName; } = 设置名字

Using accessors:
String studentName = student.getName(); = String studentName = student.getName(); = 获取学生姓名
student.setAge(18); = student.setAge(18); = 设置学生年龄为18岁


PHRASE->
get accessor = get访问器
set accessor = set访问器
property accessors = 属性访问器
data accessors = 数据访问器
