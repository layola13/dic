filename:/ˈfaɪlneɪm/| n. |文件名|The name of a file, usually including an extension that indicates the type of data it contains.  A filename allows the operating system and other software to identify and access specific files. |Please tell me the filename so I can locate it.  请告诉我文件名，以便我找到它。|Synonym: file identifier, file label |Antonym: N/A| noun


ETYMOLOGY->
From "file" + "name". "File" originates from Old French "fil" meaning "thread, string," later evolving to refer to a collection of papers on a string.  "Name" comes from Old English "nama," meaning "name, reputation."
