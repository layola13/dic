callback:/ˈkɔːlbæk/ (US) /ˈkɔːlbæk/ (UK)| n. |回拨，回调函数，回复|A request that someone return a telephone call; a function that is passed as an argument to another function and is executed later.|例句：I left a callback number with the receptionist. 我给接待员留了回拨号码。

例句：The API uses callbacks to handle asynchronous operations.  这个API使用回调函数来处理异步操作。|近义词：return call, response, feedback (in some contexts)|反义词: N/A |词性:noun
