dereferenced:/ˌdiːˈrɛfərənst/ | adj. | 解引用的 | Accessed via a pointer; having its value retrieved.  Used in computer programming to describe when the value stored at the memory location pointed to by a pointer is accessed. | The dereferenced pointer revealed the value of the variable. 解引用的指针显示了变量的值。 |  | N/A | adjective
