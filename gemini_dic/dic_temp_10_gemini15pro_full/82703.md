gui:/ɡiː/ | n. | 图形用户界面 (Graphical User Interface) | A type of user interface that allows users to interact with electronic devices through graphical icons and visual indicators such as secondary notation, instead of text-based user interfaces, typed command labels or text navigation. | Example:  The gui is easy to use and navigate.  这个图形用户界面很容易使用和导航。 | Example: This software has a very intuitive gui.  这个软件有一个非常直观的图形用户界面。| Synonym:  graphical user interface, GUI, user interface | Antonym: command-line interface, CLI | noun
