coroun:/ˈkɔːˌruːtn/ | n. | 协程 | A coroutine is a computer program component that generalizes subroutines for non-preemptive multitasking, by allowing execution to be suspended and resumed. | Example: Coroutines are useful for generators, asynchronous operations, and event handling. 协程对于生成器、异步操作和事件处理非常有用。| Example:  Using coroutines can simplify the code for cooperative tasks. 使用协程可以简化协作任务的代码。 | Synonym: task, fiber, thread (loosely) | Antonym: subroutine, function (in some contexts) | noun
