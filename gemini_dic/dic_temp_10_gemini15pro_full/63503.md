enum:/ˈiːnəm/ |n.|枚举，枚举类型|A set of named values called elements, members, enumeral, or enumerators of the same type.  It is a data type that consists of a set of named values.|  Example:  In C++, `enum Color { RED, GREEN, BLUE };` defines a type `Color` with three possible values. 在 C++ 中，`enum Color { RED, GREEN, BLUE };` 定义了一个名为 `Color` 的类型，它有三个可能的值。|Synonym: listing, enumeration|Antonym: N/A|Noun
