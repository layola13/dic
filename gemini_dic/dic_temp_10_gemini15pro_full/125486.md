multiprogrammed:/ˌmʌltiˈprəʊɡræmd/ | adj. | 多道程序的 | Designed to run multiple programs seemingly simultaneously by switching between them rapidly.  | This server is multiprogrammed to handle numerous user requests. 这台服务器采用多道程序设计来处理大量的用户请求。 | The operating system is multiprogrammed, allowing multiple applications to run concurrently. 该操作系统是多道程序设计的，允许多个应用程序并发运行。 |  concurrent, multitasking | single-tasking | adjective
