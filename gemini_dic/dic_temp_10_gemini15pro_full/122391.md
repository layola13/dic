mmap:/ˈmæp/ (US), /mæp/ (UK)| n. vt.|内存映射|A function that maps a file or device into memory.  It allows you to access the file or device as if it were part of your program's address space.|

例句：
The `mmap` function is useful for manipulating large files.
`mmap`函数对于操作大型文件很有用。

The program uses `mmap` to access the shared memory.
该程序使用`mmap`访问共享内存。

|近义词：map, allocate|反义词: unmap, deallocate|词性: noun, verb|
