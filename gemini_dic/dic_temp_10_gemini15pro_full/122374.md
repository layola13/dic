mkdir: /ˌemˈkeɪˈdɪər/ (often pronounced /ˈmɪkˌdɪər/) | v. | 创建目录 | To create a new directory (folder) in a computer's file system.  This command is common in Unix-like operating systems. | `mkdir new_directory` (creates a directory named "new_directory")  创建一个名为 "new_directory" 的目录 | 近义词: create_directory | 反义词: rmdir | 词性: verb

PHRASE->
`mkdir -p` = create parent directories as needed  如果父目录不存在，则创建父目录
`mkdir -m` = set permissions  设置权限
`mkdir -v` = verbose mode; print a message for each created directory  详细模式；为每个创建的目录打印消息
`mkdir new_directory/sub_directory` = creates a subdirectory within a directory  在一个目录中创建子目录
`mkdir Documents/Project1/Data` = creates a nested directory structure 创建嵌套目录结构
