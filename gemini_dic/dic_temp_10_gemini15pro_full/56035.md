doas:/ˈduːæz/ (US), /ˈduːæs/ (UK)| n.| 代表 "Delegate All Other Systems" | doas is a program designed to allow some programs to be run as another user. It is similar to sudo, but simpler and focused on only allowing specific commands to be run as a different user. |  He used doas to update the system configuration files as root. 他使用doas以root身份更新系统配置文件。 | Example: The system administrator uses doas to perform privileged tasks. 系统管理员使用doas执行特权任务。 | synonym: sudo | antonym:  (No direct antonym, perhaps "restrict access" as a concept) | noun
