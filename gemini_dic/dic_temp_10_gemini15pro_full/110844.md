lintier:/ˈlɪntiər/ | adj. | 更挑剔的，更细致的 | More meticulous or demanding, especially about code quality.  More prone to finding faults or errors, particularly in code. | This new version of the linter is lintier than the last one; it's flagging so many minor style issues. 这个新版本的linter比上一个版本更挑剔；它标记了这么多小的风格问题。|  近义词:  fussier, pickier, more critical, more exacting | 反义词: less critical, more lenient, more forgiving | 词性: 形容词

linter:/ˈlɪntər/ | n. | 代码检查工具，绒毛清除器 | A tool that analyzes source code to flag programming errors, bugs, stylistic errors, and suspicious constructs. Also, a tool for removing lint (small fibers) from clothes. | The linter helped me find a potential bug in my code before it caused any problems.  这个代码检查工具帮助我在代码引起任何问题之前发现了一个潜在的错误。 | 近义词: code analyzer, static analyzer | 反义词:  (For the code analysis meaning, there isn't a direct antonym) | 词性: 名词

词源：lint (n.) + -er (suffix forming agent nouns)

"lint" 本身来源于中古荷兰语 "linte"，指亚麻屑或其他纤维屑。  "linter" 最初指的是清除衣服上绒毛的工具。  在计算机科学领域，"linter" 比喻为清除代码中“绒毛”（错误和风格问题）的工具。
