nulls:/nʌlz/| n.|空值，零值|A value that indicates the absence of data.  A database field is null when it contains no value.  Nulls are not the same as empty strings or zeros.|The database returned nulls for several records.  数据库返回了几个记录的空值。|synonyms: blanks, empty values|antonyms: values, data|noun
