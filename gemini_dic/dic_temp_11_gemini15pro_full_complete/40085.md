containerization:/kənˌteɪnərɪˈzeɪʃən/ | n. | 集装箱化；容器化 | The process of packaging software code with all its dependencies and libraries into a standardized unit for software development. This unit, called a container, can then be run consistently on any infrastructure. | Containerization has revolutionized software deployment. 集装箱化彻底改变了软件部署。 Docker is a popular tool for containerization. Docker是一个流行的集装箱化工具。 | 近义词: virtualization, packaging | 反义词: N/A | 词性: noun

Etymology: From container + -ize + -ation.  "Container" itself comes from the verb "contain," ultimately from Latin "continere" (to hold together).
