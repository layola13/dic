numpy:/ˈnʌmpaɪ/ (US), /ˈnʌmpi/ (UK)| n.| NumPy (Numerical Python) |  A Python library specializing in numerical computation, offering powerful N-dimensional array objects, tools for integrating C/C++ and Fortran code, sophisticated (broadcasting) functions, and useful linear algebra, Fourier transform, and random number capabilities.  It is a fundamental package for scientific computing with Python.|  Example: We used NumPy to perform efficient matrix operations. 我们使用 NumPy 执行高效的矩阵运算。| 近义词: scipy, pandas | 反义词:  (No direct antonym for a library) | Noun
