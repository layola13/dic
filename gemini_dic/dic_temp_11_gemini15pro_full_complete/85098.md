hardcode:/ˈhɑːrdkoʊd/ (US) /ˈhɑːdkəʊd/ (UK)| v. n. adj.|硬编码，硬代码|To embed data directly into the source code of a program, rather than obtaining it from external sources like configuration files or databases.  As a noun, it refers to the data or code that has been hardcoded. As an adjective, it describes something that has been hardcoded.|

例句：
The developer hardcoded the server's IP address into the application.
开发者将服务器的IP地址硬编码到应用程序中。

We should avoid hardcoding values whenever possible.
我们应该尽可能避免硬编码值。

近义词：embed, fixed, built-in|
反义词：softcode, configurable, dynamic|
词性: verb, noun, adjective|
