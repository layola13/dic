arrays:/əˈreɪz/| n. |数组，阵列|A data structure that stores a fixed-size sequential collection of elements of the same type.  It is used to store collections of data, often numbers, but can be used to store any type of data.|Example: The program uses arrays to store the test scores.  这个程序使用数组来存储测试分数。|Example:  We used arrays to organize the large dataset. 我们使用数组来组织大型数据集。|近义词：matrix, table, vector, list|反义词: (Arrays don't have a direct opposite, but conceptually, a non-linear data structure like a linked list or tree could be considered a contrasting concept.)|词性:noun
