classlist:/ˈklɑːslɪst/ | n. | 类别列表 | A list of classes.  It is commonly used in web development to manipulate CSS classes of HTML elements. |  `document.getElementById("myDIV").classList.add("mystyle");` adds the "mystyle" class to the element with id="myDIV". | synonyms: class collection, class group | antonyms: N/A | noun
