cacheable:/ˈkæʃəbl/| adj.|可缓存的|Able to be stored in a cache (a temporary storage area) to speed up future access.|The website's images are cacheable, so they load faster on subsequent visits.  网站的图片是可缓存的，所以在以后访问时加载速度更快。|Synonyms: storable, retrievable|Antonyms: uncacheable, non-cacheable|adjective

cacheable:/ˈkæʃəbl/| adj.|可缓存的|Suitable for caching; designed to be cached.|The data is highly cacheable, reducing server load.  这些数据非常适合缓存，从而减少服务器负载。|Synonyms:  cachable (alternative spelling)|Antonyms: uncacheable, non-cacheable|adjective
