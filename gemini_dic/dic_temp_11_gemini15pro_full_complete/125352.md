multilist:/ˈmʌltɪlɪst/| n.|多重列表，多维列表|A data structure that allows multiple elements to be stored at each position in a list.  It can be visualized as a list of lists.  Allows for more complex data organization than a simple list.|The program uses a multilist to store the user's preferences.|近义词：nested list, list of lists|反义词:single list, array|noun
