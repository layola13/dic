enum:/ˈiːnəm/ (US) /ˈɛnəm/ (UK)| n.|枚举，枚举类型|A set of named values that can be used to represent a set of distinct states or options.|The `enum` type in C# allows you to define a set of named constants. C#中的枚举类型允许你定义一组命名常量。

The status of the order can be one of the following: Pending, Processing, Shipped, or Delivered, all defined in an enum. 订单状态可以是以下几种之一：待处理、处理中、已发货或已交付，所有这些都在枚举中定义。|近义词：listing, enumeration|反义词: N/A|词性:noun
