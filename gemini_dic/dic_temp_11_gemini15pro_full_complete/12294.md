ascii: /ˈæskiː/ | n. | 美国信息交换标准码 | American Standard Code for Information Interchange, a character encoding standard for electronic communication. ASCII codes represent text in computers, telecommunications equipment, and other devices. | Example:  The ASCII code for the letter "A" is 65.  字母“A”的ASCII码是65。

Example:  Many older systems still rely on ASCII for basic text processing. 许多旧系统仍然依赖ASCII进行基本文本处理。 | 近义词: character set, character encoding | 反义词: Unicode (in a broader sense, as it encompasses ASCII) | 词性: noun
