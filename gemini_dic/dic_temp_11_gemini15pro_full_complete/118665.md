meta-language:/ˈmɛtəˌlæŋɡwɪdʒ/| n.|元语言|A language used to describe or analyze another language (the object language).  A metalanguage may be identical to the object language, but is often a distinct language or symbolic system.  For example, linguists use metalanguage to describe the grammar of a language.|Linguists use a metalanguage to describe the grammar of English.|Synonyms: metalanguage (no commonly used true synonyms)|Antonyms: object language|noun
