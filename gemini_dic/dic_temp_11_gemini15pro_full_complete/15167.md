back-to-the-drawing-board:/ˌbæk tə ðə ˈdrɔːɪŋ bɔːrd/ |idiom|重新开始；回到起点|This idiom refers to the idea of having to start a project again from the beginning because the first attempt failed.|The project failed, so it's back to the drawing board.|Synonyms: start over, begin again, return to square one|Antonyms: succeed, complete, finish|phrase
