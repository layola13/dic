decompiler:/diːkəmˈpaɪlər/| n. |反编译器|A program that takes executable code and attempts to create a high-level language source file that can be recompiled to produce the same executable.|The decompiler successfully reconstructed the original source code from the binary file.|近义词:reverse engineer, disassembler|反义词:compiler|词性:noun
