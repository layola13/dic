DCOM:/ˌdiːˌkɒm/ (UK) /ˌdiːˌkɑːm/ (US)| n. | 分布式组件对象模型 (Distributed Component Object Model)| A technology that allows software components to communicate directly over a network, primarily used in Microsoft Windows environments. It's a proprietary technology by Microsoft. |Example: DCOM allows applications to be distributed across multiple computers.  分布式组件对象模型允许应用程序分布在多台计算机上。

Example:  The DCOM configuration on the server needs to be checked.  需要检查服务器上的 DCOM 配置。|近义词:CORBA, RPC |反义词: N/A |词性:noun
