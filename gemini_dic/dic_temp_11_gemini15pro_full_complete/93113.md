iff:/ɪf ænd əʊnli ɪf/ | conj. | 当且仅当 | if and only if, meaning that one thing is true or valid only if another thing is also true or valid. | If and only if it rains, the picnic will be cancelled. = 当且仅当下雨，野餐才会被取消。 | 近义词: only if | 反义词: unless, except if | 词性: conjunction

USAGE->

一般现在时 (Simple Present):
The function is continuous iff its derivative exists. = 当且仅当其导数存在时，该函数才是连续的。
I will go to the party iff you go too. = 当且仅当你也去，我才会去参加派对。
