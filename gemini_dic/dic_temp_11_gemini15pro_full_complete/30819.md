cgi:/siːdʒiːˈaɪ/| n.|通用网关接口 (Common Gateway Interface)|A standard protocol for web servers to execute programs that execute like console applications, which generate web pages dynamically.  It allows a web server to pass requests to external applications and return the results to the user.|The website uses CGI scripts to process user input.|近义词:server-side scripting, web application interface|反义词: (No direct antonym)|abbreviation
