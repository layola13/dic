adt:/ˈædɪtiː/ |n.|抽象数据类型 (Abstract Data Type)|An abstract data type (ADT) is a mathematical model for data types.  An ADT is defined by its behavior (semantics) from the point of view of a *user* of the data, specifically in terms of possible values, possible operations on data of this type, and the behavior of these operations.  It does not specify *how* the data type is implemented.|Example: A stack is an ADT.  We can push and pop elements, but the implementation (array, linked list) is hidden.  栈是一个抽象数据类型。我们可以推入和弹出元素，但实现（数组、链表）是隐藏的。|近义词:data structure, data type|反义词:concrete data type|词性:名词
