nonlocal:/ˌnɒnˈləʊkl/ (UK) /ˌnɑːnˈloʊkl/ (US)| keyword | 非局部的 |  A keyword in Python used to declare that a variable inside a nested function refers to a variable in its enclosing function's scope, rather than creating a new local variable.  This allows modification of the outer variable from within the inner function.|

```python
def outer_function():
    x = 10

    def inner_function():
        nonlocal x
        x = 20

    inner_function()
    print(x)  # Output: 20
```
非局部变量 x 的值在内部函数 inner_function 中被修改。

```python
    x = "hello"

    def inner_function():
        nonlocal x
        x = "world"

    inner_function()

```
