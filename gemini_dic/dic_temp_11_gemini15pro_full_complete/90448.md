https:/ˌeɪʧ tiː tiː piː ˈɛs/ | n. | 超文本传输协议安全 | HyperText Transfer Protocol Secure.  A protocol for secure communication over a computer network.  It encrypts communications between a web browser and a web server, protecting sensitive information like passwords and credit card numbers. | Example:  Most websites now use HTTPS to protect user data.  大多数网站现在使用 HTTPS 来保护用户数据。

Example:  Make sure the website you're visiting uses HTTPS before entering any personal information.  在输入任何个人信息之前，请确保您访问的网站使用 HTTPS。| Synonym: Secure HTTP | Antonym: HTTP | Noun
