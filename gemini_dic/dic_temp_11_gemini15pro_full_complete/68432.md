falsy:/ˈfɔːlsi/| adj.|假的，不真实的，在布尔上下文中被视为假|In JavaScript, a falsy value is a value that is considered false when encountered in a Boolean context.|例句：Empty strings are considered falsy in JavaScript.  在JavaScript中，空字符串被认为是假的。

例句：Zero is a falsy value. 零是一个假值。|近义词：false, untrue, invalid, null, undefined (in programming contexts)|反义词：truthy, true, valid|词性:形容词

词源:  Derived from "false" with the "-y" suffix, indicating a property or characteristic.  源自“false”，带有“-y”后缀，表示属性或特征。
