elif:/ˈɛlɪf/| conj.|否则如果 (用于编程)|A conditional statement in Python used after an initial 'if' statement, and before any 'else' statements, to check for additional conditions.|Example: 
```python
x = 10
if x > 15:
    print("x is greater than 15")
elif x > 5:
    print("x is greater than 5 but not greater than 15")
else:
    print("x is not greater than 5")
```
中文：
```
x = 10
如果 x > 15:
    打印("x 大于 15")
否则如果 x > 5:
    打印("x 大于 5 但不大于 15")
否则:
    打印("x 不大于 5")
```
|近义词: else if|反义词: None|词性: conjunction (连词)
