github: /ˈɡɪthʌb/ | n. | 代码托管平台 | A web-based hosting service for version control using Git. It offers all of the distributed version control and source code management (SCM) functionality of Git as well as adding its own features. | Example:  I pushed my latest code changes to GitHub.  我将我最新的代码更改推送到了GitHub。

Example:  Many open-source projects are hosted on GitHub. 许多开源项目都托管在GitHub上。 | 近义词: GitLab, Bitbucket | 反义词:  None (No direct antonym for a platform) | 词性: noun
