PHRASE->
`mkdir -p` = create parent directories as needed =  如果父目录不存在，则创建父目录
`mkdir -v` = verbose mode, print each directory created = 详细模式，打印每个创建的目录
`mkdir -m` = set file permissions = 设置文件权限
`mkdir directory_name` = creates a directory named "directory_name" = 创建名为 "directory_name" 的目录
`mkdir Documents/Projects` = creates a directory "Projects" inside the "Documents" directory = 在 "Documents" 目录内创建一个 "Projects" 目录
