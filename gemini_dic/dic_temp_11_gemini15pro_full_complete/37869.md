compilers:/kəmˈpaɪlərz/ | n. | 编译器 | A program that translates source code written in a high-level programming language into a low-level language, such as machine code or assembly language, that can be understood and executed by a computer. |  Compilers are essential tools for software development. 编译器是软件开发的重要工具。


Compilers can optimize the code for performance. 编译器可以优化代码以提高性能。


近义词: translator | 反义词: interpreter | 词性: noun
