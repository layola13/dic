/ˈdʒeɪˈɛs/ | n. | JavaScript |  A high-level, often just-in-time compiled language that conforms to the ECMAScript specification. It is characterized by dynamic typing, prototype-based object-orientation, and first-class functions. It is commonly used for client-side scripting in web browsers, but can also be used on the server-side and in other applications.  | Example: The website uses JavaScript to create interactive elements.  该网站使用JavaScript来创建交互元素。 | Synonym: ECMAScript (often used interchangeably, though technically JavaScript is an implementation of ECMAScript) | Antonym: (No direct antonym for a programming language) | noun
