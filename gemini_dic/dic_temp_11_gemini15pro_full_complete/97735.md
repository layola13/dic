initialised:/ɪˈnɪʃəlaɪzd/ | adj. |初始化的，已初始化的 |  Having been set to an initial value or state.  Meaning that a variable or system has been given a starting value or configuration. | The program crashed because a variable was not initialised. 程序崩溃了，因为一个变量没有被初始化。 | The system was initialised with the default settings.  系统用默认设置初始化。 | initialized, preset, preconfigured | uninitialized | adjective
