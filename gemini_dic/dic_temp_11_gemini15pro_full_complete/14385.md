autosaves:/ˈɔːtəʊseɪvz/ (US) /ˈɔːtəʊseɪvz/ (UK)| n.|自动保存|A feature in software that automatically saves changes to a file at regular intervals, preventing data loss in case of crashes or power outages.|The autosaves feature in my word processor saved my work when the computer crashed.|Synonyms: automatic saving, background saving|Antonyms: manual saving|noun

autosaves:/ˈɔːtəʊseɪvz/ (US) /ˈɔːtəʊseɪvz/ (UK)| v.|自动保存|The action of a software program automatically saving changes to a file.|The program autosaves every five minutes.|Synonyms: automatically saves, backs up|Antonyms: manually saves|verb
