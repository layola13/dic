dll:/diː ɛl ɛl/ | n. | dynamic-link library 动态链接库 | A dynamic-link library (DLL) is a collection of small programs, any of which can be called upon when needed by a larger program running in Microsoft Windows or the OS/2 operating system.  | Example: The program crashed because it couldn't find the required DLL.  这个程序崩溃了，因为它找不到所需的动态链接库。 | Example:  Make sure you have the correct DLLs installed for the game to run properly. 确保安装了正确的动态链接库才能使游戏正常运行。 | 近义词: library, module | 反义词:  (No direct antonym for a file type) | 词性: noun
