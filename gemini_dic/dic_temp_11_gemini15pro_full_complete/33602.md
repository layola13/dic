cin:/sɪn/ | n. | 标准输入流 |  In C++, cin is the standard input stream object, used to get input from the user, typically from the keyboard. It's part of the <iostream> library. | `std::cin >> age;`  从键盘读取用户输入并存储到变量 age 中 / Read user input from the keyboard and store it in the variable age. | 近义词: standard input | 反义词: cout (standard output) |  词性: noun (object)


USAGE->

一般现在时 (Simple Present):
`cin >> x;` 读取一个值并存储到 x 中 / Reads a value and stores it in x.
`int age; cin >> age;`  声明一个整数变量 age，然后从输入流读取一个值存入其中 / Declares an integer variable age, then reads a value from the input stream into it.

PHRASE->
`cin >> x >> y;`  读取两个值，分别存储到 x 和 y 中 / Reads two values and stores them in x and y respectively.
`cin.get();` 读取一个字符，包括空格和换行 / Reads a single character, including whitespace and newline.
`cin.ignore();` 忽略输入流中的下一个字符 / Ignores the next character in the input stream.
`cin.clear();` 清除输入流中的错误标志 / Clears the error flags in the input stream.
`cin.fail();` 检查输入流是否处于错误状态 / Checks if the input stream is in an error state.
