coroun:/ˈkɔːruːtiːn/ /ˈkɔːrəˌtiːn/ | n. | 协程 | A coroutine is a generalization of a subroutine.  It is a special type of computer program component that generalizes subroutines for non-preemptive multitasking, by allowing multiple entry points for suspending and resuming execution at certain locations. |  Example: Coroutines can be used to implement cooperative multitasking.  协程可以用来实现协作式多任务处理。

Example: Python supports coroutines using the `async` and `await` keywords. Python 使用 `async` 和 `await` 关键字支持协程。 | 近义词: generator, task, thread | 反义词: subroutine, process | 词性: noun
