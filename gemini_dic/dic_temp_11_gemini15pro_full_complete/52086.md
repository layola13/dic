diamagnetically:/ˌdaɪəˌmæɡˈnɛtɪkli/
| adv. |
| 反磁性地；抗磁性地 |
| In a diamagnetic manner; exhibiting diamagnetism.  This refers to a property of materials that causes them to create a magnetic field in opposition to an externally applied magnetic field. |
|  Bismuth is diamagnetically repelled by a magnetic field.  铋被磁场反磁性地排斥。|

| The pyrolytic carbon exhibits diamagnetically anisotropic behavior.  热解碳表现出反磁性的各向异性行为。 |
| 近义词: nonmagnetically (in some contexts) |
| 反义词: paramagnetically, ferromagnetically |
| 词性: 副词(adverb) |
