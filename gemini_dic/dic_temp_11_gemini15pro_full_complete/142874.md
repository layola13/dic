out-of-line:/ˌaʊt əv ˈlaɪn/
adjective
越界的，不恰当的，过分的
Deviating from what is considered acceptable or appropriate; exceeding limits.
That comment was completely out-of-line. 那个评论完全不恰当。
His behavior was out-of-line and he should apologize. 他的行为过分了，他应该道歉。
近义词：inappropriate, unacceptable, excessive, improper, offensive
反义词：appropriate, acceptable, proper, reasonable

out-of-line:/ˌaʊt əv ˈlaɪn/
adjective
（代码）外联的，非内联的
(of code) Not included directly in the main body of a program but stored separately and accessed when needed.
Out-of-line functions can improve code readability. 外联函数可以提高代码可读性。
The compiler decided to generate out-of-line code for this function. 编译器决定为此函数生成外联代码。
近义词：external, separate
反义词：inline, embedded

词性: adjective
