dereferenced:/diːˈrɛfərənst/ | adj. | 解引用的 | Accessed via a pointer; having the location of data retrieved by following a pointer.  The action of accessing data at the memory location pointed to by a pointer. | The dereferenced pointer gave us the value stored in that memory location. 解引用的指针给了我们存储在该内存位置的值。 | The program crashed when it tried to dereference a null pointer. 程序试图解引用一个空指针时崩溃了。 |  accessed, pointed, resolved | unreferenced, null | adjective
