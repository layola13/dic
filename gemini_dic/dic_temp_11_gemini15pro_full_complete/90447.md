http's:/ˌeɪtʃ tiː piː ˈɛs/ | n. | 超文本传输安全协议 |  Hyper Text Transfer Protocol Secure. It's a protocol for secure communication over a computer network, ensuring data privacy and integrity by encrypting communications between a web server and a web browser. | Example:  Most websites now use http's for added security. 大多数网站现在使用 https 以增强安全性。| Example:  Make sure the website you're visiting uses http's, especially if you're entering personal information. 确保你访问的网站使用 https，尤其是在输入个人信息时。 | Synonym: Secure HTTP | Antonym: HTTP | noun
