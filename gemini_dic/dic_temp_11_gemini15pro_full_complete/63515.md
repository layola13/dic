enums:/ˈiːnʌmz/ | n. | 枚举 | A set of named values representing distinct elements within a particular data type.  Enums are often used to represent a finite set of options or states. |  Java developers often use enums.  Java开发者经常使用枚举。 | synonyms: enumerated type, enumerated list | antonyms: N/A | noun
