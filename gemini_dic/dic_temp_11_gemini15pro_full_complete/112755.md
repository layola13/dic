lte:/ˌɛl tiː ˈiː/|abbr.|Long Term Evolution (长期演进)|A standard for wireless broadband communication for mobile devices and data terminals, based on the GSM/EDGE and UMTS/HSPA network technologies. It increases the capacity and speed using a different radio interface together with core network improvements.|例句：My phone supports LTE, so I can download files very quickly. 我的手机支持LTE，所以我可以很快地下载文件。

例句：The rollout of the LTE network significantly improved mobile internet speeds in the area. LTE网络的推出显著提高了该地区的移动互联网速度。|近义词：4G, mobile broadband|反义词: 3G, 2G|词性:abbreviation
