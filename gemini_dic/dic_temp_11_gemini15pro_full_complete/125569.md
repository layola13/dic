multiton:/ˈmʌltɪtɒn/ (UK) /ˈmʌltɪtɑːn/ (US)| n.|多例|A software design pattern that restricts the instantiation of a class to a small set of pre-defined instances.|The logging system uses a multiton pattern to ensure only one instance per log level.|近义词: enumerated type, pool|反义词: singleton|词性:noun
