ajax:/ˈeɪdʒæks/| n.|异步 JavaScript 和 XML|A group of interrelated web development techniques used on the client-side to create asynchronous web applications.|Example:  Ajax allows web pages to be updated asynchronously by exchanging small amounts of data with the server in the background.  Ajax允许网页在后台与服务器交换少量数据，从而异步更新。

Example: This website uses Ajax to load new content without refreshing the entire page.  这个网站使用Ajax加载新内容，而无需刷新整个页面。|近义词:AJAJ (Asynchronous JavaScript and JSON)|反义词:None|词性:noun
