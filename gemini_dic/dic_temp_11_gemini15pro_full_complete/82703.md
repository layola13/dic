gui:/ɡiː/ /ɡwiː/ | n. |图形用户界面 (Graphical User Interface) |A type of user interface that allows users to interact with electronic devices through graphical icons and visual indicators such as secondary notation, as opposed to text-based user interfaces, typed command labels or text navigation. | Example: This program has a user-friendly GUI.  这个程序有一个用户友好的图形用户界面。| Example:  Many modern applications rely heavily on a well-designed GUI. 许多现代应用程序严重依赖于精心设计的图形用户界面。 | Synonym:  interface, frontend | Antonym: command-line interface (CLI) | noun
