cors:/kɔːrz/| n.|跨域资源共享 (Cross-Origin Resource Sharing)|A mechanism that allows restricted resources on a web page to be requested from another domain outside the domain from which the first resource was served.|The website uses CORS to allow access to its API from different domains.|Synonyms: cross-domain sharing|Antonyms: same-origin policy|Noun
