fallbacks:/ˈfɔːlbæks/| n.|后备方案，备用方案，退路|A fallback is a plan or course of action that is used if the original plan or course of action fails.|We need a fallback in case the primary plan doesn't work. 我们需要一个备用方案，以防主要方案行不通。|synonyms: backup, reserve, alternative, contingency plan|antonyms: primary plan, main strategy|noun

fallbacks:/ˈfɔːlbæks/| n.|（计算机）回退，备选方案|In computer science, a fallback refers to a default option or method that is used if the preferred option or method is unavailable.|The system has a fallback mechanism in case the main server fails.  系统有一个备用机制，以防主服务器出现故障。|synonyms: default, reserve, alternative, contingency|antonyms: primary, main|noun
