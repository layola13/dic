n.
GeoTIFF (Geographic Tagged Image File Format)
地理标记图像文件格式
A public domain metadata standard which allows georeferencing information to be embedded within a TIFF file.
一种公共域元数据标准，允许将地理参考信息嵌入到TIFF文件中。
The satellite imagery was stored in GeoTIFF format for easy integration into GIS software.  卫星图像以GeoTIFF格式存储，以便轻松集成到GIS软件中。
近义词:GeoJPEG, TIF
反义词: (No direct antonym for a file format)
