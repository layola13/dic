一般现在时 (Simple Present):
It is cold today. = 今天很冷。
It seems like a good idea. = 这似乎是个好主意。

现在进行时 (Present Continuous):
It is raining right now. = 现在正在下雨。
It is getting dark. = 天正在变黑。

现在完成时 (Present Perfect):
It has been a long day. = 今天是漫长的一天。
It has rained all day. = 下了一整天的雨。

现在完成进行时 (Present Perfect Continuous):
It has been raining all morning. =  整个早上都在下雨。
It has been snowing for hours. = 已经下雪好几个小时了.

一般过去时 (Simple Past):
It was a beautiful day yesterday. = 昨天天气很好。
It happened so fast. = 它发生得如此之快。

过去进行时 (Past Continuous):
It was raining when I left. = 我离开的时候正在下雨。
It was getting late. = 天渐渐晚了。

过去完成时 (Past Perfect):
It had already happened before I arrived. = 在我到达之前它就已经发生了。
It had been raining for hours. = 雨已经下了好几个小时了.

过去完成进行时 (Past Perfect Continuous):
It had been raining for hours before the sun came out. =  太阳出来之前，雨已经下了好几个小时。
It had been snowing all night. = 整夜都在下雪。

一般将来时 (Simple Future):
It will be cold tomorrow. = 明天会很冷。
It will rain later. = 稍后会下雨。

将来进行时 (Future Continuous):
It will be raining this time tomorrow. = 明天这个时候会下雨。
It will be getting dark soon. = 天很快就会黑了。

将来完成时 (Future Perfect):
It will have been a year by then. = 到那时就一年了。
It will have stopped raining by the time we leave. = 我们离开的时候雨就会停了。

将来完成进行时 (Future Perfect Continuous):
It will have been raining for days by then. = 到那时雨将已经下了好几天。
It will have been snowing for a week by next Friday. = 到下周五，雪将已经下了一个星期。
