cross-platform:/ˌkrɒsˈplæt.fɔːm/ /ˌkrɔːsˈplæt.fɔːrm/| adj.|跨平台的|Able to be used on different types of computer systems.|This software is cross-platform, so you can use it on both Windows and macOS.|近义词：multi-platform, platform-independent|反义词: platform-specific|词性:形容词

cross-platform:/ˌkrɒsˈplæt.fɔːm/ /ˌkrɔːsˈplæt.fɔːrm/| n.|跨平台|The ability of a software application to run on multiple computing platforms.|Developing for cross-platform can save time and resources.|近义词：multi-platform, platform independence|反义词: platform dependence|词性:名词
