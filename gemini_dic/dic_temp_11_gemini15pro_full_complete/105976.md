kml:/keɪɛmˈɛl/| n.| Keyhole Markup Language (密钥孔标记语言)|A file format used to display geographic data in an Earth browser such as Google Earth. It uses a tag-based structure with nested elements and attributes and is based on XML.|Example: The KML file displayed the location of all the hiking trails in the park.  这个KML文件显示了公园里所有徒步旅行路线的位置。|近义词: gpx, geojson|反义词: N/A|词性:noun
