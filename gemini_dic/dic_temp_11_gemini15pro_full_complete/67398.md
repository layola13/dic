extern:/ˈekstɜːrn/ | adj.| 外部的；外面的 | Declares that a variable or function is defined externally.  It is used to give a reference of a global variable that is visible to ALL the program files. When you use 'extern', the variable cannot be initialized however, it points the variable name at a storage location that has been previously defined.|
`extern int x;`  This declares x to be an integer variable defined elsewhere. |近义词: external, outer | 反义词: internal, inner | 词性: keyword (in C/C++)

PHRASE->

`extern "C"`
用于 C++ 代码中，声明接下来的代码块按照 C 的方式进行编译和链接。

`extern "C" int my_c_function();`
在 C++ 中声明一个 C 风格的函数 my_c_function()。

`extern template class std::vector<int>;`
显式实例化一个外部定义的模板类 std::vector<int>。

`extern const int MAX_VALUE;`
声明一个外部定义的常量整型变量 MAX_VALUE。

`extern thread_local int counter;`
声明一个线程局部存储的整型变量 counter。


注意事项:

'extern' 主要用于声明，而不是定义变量或函数。它告诉编译器该变量或函数已在其他地方定义。


The tense examples are not applicable because 'extern' is a keyword used at compile time and doesn't relate to grammatical tenses. Its usage relates to how the compiler links code.
