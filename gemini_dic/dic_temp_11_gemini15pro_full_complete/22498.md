boolean:/ˈbuːliən/| adj.|布尔型的，逻辑的|A data type having two possible values (usually denoted true and false), intended to represent the truth values of logic and Boolean algebra.|Example: The function returns a boolean value, either true or false.  这个函数返回一个布尔值，不是真就是假。|Example: Boolean logic is fundamental to computer programming. 布尔逻辑是计算机编程的基础。|Synonym: logical, binary|Antonym: N/A|adjective

ETYMOLOGY->
Named after George Boole, a 19th-century English mathematician who developed Boolean algebra.
