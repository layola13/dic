crawler:/ˈkrɔːlər/| n.|爬行者，爬虫，履带式车辆|A person or thing that crawls; a creeping animal or plant.  A program that systematically browses the World Wide Web in order to create an index of data.|The baby is a crawler, not a walker yet.  这个婴儿还只会爬，不会走。| The search engine uses a web crawler to index new websites.  这个搜索引擎使用网络爬虫来索引新的网站。|Synonyms: creeper, reptile, insect, spider, bot|Antonyms: walker, runner|noun
