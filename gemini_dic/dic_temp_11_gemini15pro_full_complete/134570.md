nonlocal:/nɒnˈləʊkl/ | adj. | 非局部的 |  In Python, the `nonlocal` keyword is used to indicate that a variable within a nested function refers to a variable in its enclosing scope, but not a global variable. |

Example 1:
```python
def outer_function():
    x = 10
    def inner_function():
        nonlocal x
        x = 20
    inner_function()
    print(x)  # Output: 20
```
在内嵌函数`inner_function`中修改`x`的值会影响外层函数`outer_function`中`x`的值。

Example 2:
```python
def outer_function():
    x = 10
    def inner_function():
        x = 20  # This creates a new local variable x
    inner_function()
    print(x)  # Output: 10
```
如果没有`nonlocal`关键字，内嵌函数会创建一个新的局部变量`x`，而不会修改外层函数的`x`。
