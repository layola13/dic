configured:/kənˈfɪɡərd/ (US), /kənˈfɪɡəd/ (UK)| adj.|已配置的，已组态的|Arranged in a particular form or order; set up for operation in a specific way.  Derived from the verb "configure." From Latin "con-" (together) + "figura" (shape).|The system was configured to automatically back up data every hour. 系统配置为每小时自动备份数据。|The network is configured for high-speed data transfer.  该网络配置为高速数据传输。|近义词：arranged, organized, set up, established, customized|反义词: unconfigured, disorganized|词性:形容词
