functional:/ˈfʌŋkʃənəl/| adj. n.|功能的，实用的；功能性障碍；泛函|Relating to functions or functioning; designed for or adapted to a particular function or use.  (n.) A functional is a function of functions, mapping functions to numbers. |This chair is beautiful, but not very functional. 这把椅子很漂亮，但不太实用。|The patient presented with several functionals, including difficulty walking and speaking. 病人表现出几种功能障碍，包括行走和说话困难。|synonyms: practical, useful, operative, working, running|antonyms: impractical, useless, nonfunctional, broken, dysfunctional
functional:/ˈfʌŋkʃənəl/| n.|泛函|In mathematics, a functional is a function whose domain is a set of functions.  It assigns a numerical value to each function in its domain. |The energy functional in physics assigns a value to each possible configuration of a system.  物理学中的能量泛函为系统的每个可能配置分配一个值。|This functional defines a norm on the vector space. 这个泛函定义了向量空间上的一个范数。|synonyms: operator, mapping|antonyms: N/A

ETYMOLOGY->
From Late Latin *functionalis*, from Latin functio (“performance, execution”), from fungi (“perform, execute”).  The mathematical sense emerged in the early 20th century.
