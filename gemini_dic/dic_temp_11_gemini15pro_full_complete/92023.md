hyperparameter:/ˌhaɪpərˈpəræmɪtər/| n.|超参数|A parameter whose value is used to control the learning process. A hyperparameter is not part of the model and is typically set before training begins.  It influences how a machine learning algorithm learns.|The learning rate is a crucial hyperparameter in training neural networks.|Synonyms: meta-parameter|Antonyms: parameter|noun
