cronet:/ˈkroʊnɛt/| n.|克罗内特，谷歌网络库|A networking library developed by Google, providing a cross-platform transport abstraction.  It offers a higher performance alternative to the default HTTP stack in some operating systems.|Example: The app uses Cronet to improve network performance.|Synonym: network library, HTTP client|Antonym: (No direct antonym)|noun
