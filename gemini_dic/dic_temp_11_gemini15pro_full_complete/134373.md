noninterpolating:/ˌnɑːnɪnˈtɜːrpəleɪtɪŋ/ | adj. | 非插值的，不进行插值的 | Not based on interpolation; not using or involving interpolation.  It refers to a function or data set that does not rely on estimated values derived from surrounding data points. | This algorithm uses a noninterpolating method for image resizing, preserving sharp edges.  这个算法使用非插值方法来调整图像大小，保留了清晰的边缘。

The noninterpolating function accurately represents the discrete data points.  这个非插值函数精确地表示了离散的数据点。 | 近义词: discrete, non-smooth | 反义词: interpolating, smooth | 词性: adjective
