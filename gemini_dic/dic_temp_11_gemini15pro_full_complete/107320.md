lambdas:/ˈlæmbdəz/| n.| 拉姆达表达式，匿名函数 |A lambda expression is a short block of code which takes parameters and returns a value. It's an anonymous function, meaning it doesn't have a name.|Example: Python supports lambdas for creating small, throwaway functions.  Python 支持使用 lambda 表达式来创建小型、一次性的函数。|Example:  Using lambdas can make your code more concise. 使用 lambda 表达式可以使你的代码更简洁。|近义词: anonymous functions, closures (in some contexts)|反义词: named functions|词性:noun
