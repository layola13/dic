csv:/ˌsiːɛsˈviː/| n. |逗号分隔值|A file format used to store tabular data (numbers and text) in plain text. Each line of the file is a data record. Each record consists of one or more fields, separated by commas.|I saved the data as a CSV file so it can be opened in any spreadsheet program.|近义词：TSV, delimited file|反义词: (No direct antonym, but binary formats like .xls or .xlsx could be considered opposites)|名词
