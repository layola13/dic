interactable:/ˌɪntərˈæktəbl/
adjective
可互动的，可交互的
Capable of being interacted with; allowing interaction.
The museum has many interactable exhibits that are fun for both children and adults.  博物馆有许多可以互动的展品，对儿童和成人来说都很有趣。
This app is known for its interactable interface.  这个应用程序以其可交互的界面而闻名。
近义词: interactive, responsive
反义词: static, unresponsive
词性: adjective

interactable:/ˌɪntərˈæktəbl/
noun
可交互元素
An element or object that can be interacted with.
The game designers added more interactables to the virtual world. 游戏设计师在虚拟世界中添加了更多可交互的元素。
The new software update includes several new interactables for users to explore. 新的软件更新包括几个新的可交互元素供用户探索。
近义词: interactive element, interactive object
反义词: static element, non-interactive object
词性: noun
