constructors:/kənˈstrʌktərz/| n.|构造函数，构造器|A special method within a class used to create and initialize objects of that class.  It prepares the new object for use by setting initial values for its members.|The `Car` class has a constructor that takes the make, model, and year as arguments.|近义词:initializer|反义词:destructor|词性:noun
