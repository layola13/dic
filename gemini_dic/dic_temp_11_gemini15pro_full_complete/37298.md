commandlet:/kəˈmændlɪt/| n.|小型命令，cmdlet，指Windows PowerShell中的轻量级命令|A commandlet is a specialized, single-function command-line tool that is built into the Windows PowerShell environment.  |例句：This commandlet gets all running processes.  这个cmdlet获取所有正在运行的进程。

例句：You can use this commandlet to manage user accounts. 你可以使用此cmdlet来管理用户帐户。|近义词：command, script, tool|反义词: (No direct antonym exists.  Consider phrases like "manual process")|词性: noun
