dnx: /ˌdiː ɛnˈɛks/
noun
.NET execution environment
.NET execution environment, a software development kit (SDK) from Microsoft for building and running .NET applications.
DNX is used to build and run ASP.NET 5 applications.  It provides a cross-platform runtime environment for executing .NET code.
Microsoft has replaced DNX with the .NET CLI.  DNX is now obsolete and no longer supported.
Synonym: .NET CLI (modern equivalent)
Antonym: N/A
noun
