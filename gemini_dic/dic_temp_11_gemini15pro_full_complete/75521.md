ftp:/ˌeftiːˈpiː/| n.|文件传输协议 (File Transfer Protocol)|A standard network protocol used to transfer computer files between a client and server on a computer network.|Example: I used FTP to upload my website files to the server. 我使用FTP将我的网站文件上传到服务器。|Example: The FTP server is down, so I can't access the files. FTP服务器宕机了，所以我无法访问文件。|近义词:SFTP, SCP|反义词:None|noun
