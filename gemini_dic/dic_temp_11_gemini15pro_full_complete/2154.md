add-on:/ˈæd ɑːn/ (US), /ˈæd ɒn/ (UK)| n. adj.|附加组件，附加物；附加的|Something added to an existing object or system, often to improve or enhance it.  An add-on can be a physical object or a software program.|This car has a lot of expensive add-ons. 这辆车有很多昂贵的附加装置。|synonyms:  supplement, addition, accessory, attachment, extension|antonyms: base, core, essential|noun

add-on:/ˈæd ɑːn/ (US), /ˈæd ɒn/ (UK)| adj.|附加的|Relating to or being something that is added to an existing object or system. |The add-on features of this software are very useful. 这个软件的附加功能非常有用。|synonyms:  supplementary, additional, auxiliary|antonyms:  built-in, integrated, inherent|adjective
