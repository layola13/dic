finalizer:/ˈfaɪnəlaɪzər/| n.|终结器，终结方法|An object that is responsible for releasing resources or performing other cleanup operations when an object is no longer in use, especially in garbage-collected programming environments.|Example: The finalizer ensures that the file handle is closed when the object is garbage collected. 终结器确保在对象被垃圾回收时关闭文件句柄。|Example: In Java, the finalize() method acts as a finalizer. 在Java中，finalize() 方法充当终结器。|近义词：destructor, cleaner|反义词:constructor, initializer|词性:noun
