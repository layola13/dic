initialized: /ɪˈnɪʃəlaɪzd/ | adj. | 初始化的，已初始化的 |  Something that is initialized has been set to an initial value or state, especially in computer programming. It means the system or program has been set up and is ready to use. | The program initialized the variables to zero. 程序将变量初始化为零。| initialized variables have been assigned an initial value. 已初始化的变量已被赋予初始值。 | initialized, set up, configured | uninitialized, unset | adjective
