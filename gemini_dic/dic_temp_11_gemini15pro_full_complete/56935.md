.dotm:/ˈdɒtˌɛm/ | n. | Word Open XML Macro-Enabled Document | A file extension used by Microsoft Word for documents that contain macros.  It is based on the Open XML format. |  Example: Please send the report as a .dotm file so I can use the embedded macros. 请以 .dotm 文件格式发送报告，以便我使用嵌入的宏。 | Synonym: (No direct synonym, but similar to .docm, .dotx, .dot) | Antonym: (No antonym) | abbreviation
