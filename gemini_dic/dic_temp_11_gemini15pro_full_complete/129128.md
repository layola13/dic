nlink:/ˈɛnlɪŋk/ | n. | 文件链接数 | The number of hard links to a file.  This count keeps track of how many directories have entries for this file.  |  The `nlink` value of the file was 2, meaning it was present in two directories.  该文件的 `nlink` 值为 2，意味着它存在于两个目录中。 | synonyms: link count, hard link count | antonyms: N/A | noun
