executable:/ˈɛksɪkjuːtəbəl/| adj. n.|可执行的；可执行程序|A file or program that can be run by a computer. |The downloaded file is an executable that installs the software.  下载的文件是一个安装软件的可执行文件。|近义词:runnable, operable|反义词:inexecutable, non-executable|adjective

executable:/ˈɛksɪkjuːtəbəl/| n.|可执行程序|A program file that can be run by a computer. |Please send me the executable for the latest version of the software. 请把最新版软件的可执行程序发给我。|近义词:program, application|反义词: (No direct antonym for the noun form)|noun


ETYMOLOGY->
From Late Middle English *executabyll*, from Late Latin *executabilis*, from Latin *execut-* (past participle stem of *exequi*, "to follow out, perform") + *-abilis* (suffix forming adjectives meaning "capable of being").
