combinator:/ˈkɒmbɪneɪtə(r)/ (US) /ˈkɑːmbɪneɪtər/ (UK)| n.|组合子，运算符|A function with no free variables. In functional programming, combinators are higher-order functions that use other functions as arguments and return a new function as a result, without referencing any variables from the surrounding scope.|The Y combinator is a famous example of a fixed-point combinator.|近义词: operator, function|反义词: N/A|词性: noun

combinatory:/kəmˈbɪnətɔːri/ (US) /kəmˈbɪnətri/ (UK)| adj.|组合的|Relating to combinations, or the ways in which different things can be combined.|Combinatory logic is a system of logic that deals with combinators.|近义词: combined, permutation|反义词: N/A|词性: adjective
