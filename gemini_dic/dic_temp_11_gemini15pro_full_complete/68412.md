falsey:/ˈfɔːlsi/| adj.|假的，虚假的，不真实的 (在编程中指会被强制转换为false的值)|In JavaScript, 0, null, undefined, NaN, '' (empty string), and false itself are all falsey values. 在JavaScript中，0、null、undefined、NaN、''（空字符串）和false本身都是falsey值。|If the variable is falsey, the code inside the if statement will not execute. 如果变量是falsey，if语句中的代码将不会执行。|近义词:falsy, zero, null, empty|反义词:truthy, non-zero, non-null, non-empty|词性:形容词
