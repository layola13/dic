locales:/ləʊˈkɑːlz/ (UK), /loʊˈkælz/ (US)| n.|场所，地点；区域设置|A locale is a set of parameters that defines the user's language, region and any special variant preferences that the user wants to see in their user interface. |We need to adapt our software to different locales. 我们需要使我们的软件适应不同的地区设置。|synonyms: places, regions, settings|antonyms: global, universal|noun

locales:/ləʊˈkɑːlz/ (UK), /loʊˈkælz/ (US)| n.|地点|The festival takes place in various locales around the city. 这个节日在城市周围的各个地方举行。|synonyms: venues, spots, areas|antonyms: nowhere, noplace|noun
