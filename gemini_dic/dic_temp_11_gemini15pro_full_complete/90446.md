http:/eɪtʃ tiː tiː piː/| n.|超文本传输协议|HyperText Transfer Protocol, the underlying protocol used by the World Wide Web. It defines how messages are formatted and transmitted, and what actions Web servers and browsers should take in response to various commands.|Example:  Websites use HTTP to transmit data.  网站使用 HTTP 传输数据。

Example: Secure websites use HTTPS, an encrypted version of HTTP. 安全的网站使用HTTPS，一个加密版本的HTTP。|近义词:protocol|反义词:None|Noun
