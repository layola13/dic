nonatomic:/ˌnɒn əˈtɒmɪk/| adj.|非原子的|Not atomic; not guaranteeing that a value will not be accessed by multiple threads simultaneously.  Used in Objective-C and other programming contexts to indicate that access to a variable is not thread-safe.|`@property (nonatomic, strong) NSString *name;` (Objective-C) This declares a string property 'name' with non-atomic access.  这意味着字符串属性 'name' 的访问是非原子的。|近义词: (In some contexts) thread-unsafe|反义词: atomic|词性: adjective
