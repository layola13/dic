doas:/ˈduːəz/ (US), /ˈduːæs/ (UK)| n.| 代表 "delegated operator as" |A Unix utility that allows a user to execute commands as another user, typically root. It is similar to sudo but with a different configuration and focus on specific users and commands.|Example:  "doas pacman -Syu" updates the system using the package manager as root. |近义词：sudo, su|反义词: N/A|词性:名词 (Although it functions as a command, it's fundamentally a utility represented by a noun.)
