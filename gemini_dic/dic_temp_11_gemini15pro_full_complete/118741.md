metalanguages:/ˈmetəlˌlæŋɡwɪdʒɪz/
noun
元语言
a language used to describe or analyze another language (the object language), often used in logic, linguistics, and computer science.
Metalanguages are used to describe the syntax and semantics of programming languages. 元语言用于描述编程语言的语法和语义。
Formal logic employs metalanguages to discuss the properties of logical systems. 形式逻辑使用元语言来讨论逻辑系统的特性。
synonyms: meta-language
antonyms: object language
n.
