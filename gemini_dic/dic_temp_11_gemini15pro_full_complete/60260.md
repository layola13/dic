efs:/iːɛfˈɛs/| n. |加密文件系统 (Encrypted File System)|A file system that provides cryptographic security for data on disk.  It transparently encrypts and decrypts files as they are read and written.|Encrypted File System (EFS) is a feature of Windows that allows you to store information on your hard drive in an encrypted format. | synonym: encrypted filesystem| antonym: unencrypted filesystem| noun
