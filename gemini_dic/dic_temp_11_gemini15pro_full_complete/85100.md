hardcoding:/ˈhɑːrdˌkoʊdɪŋ/| n.|硬编码|The practice of embedding data directly into the source code of a program instead of obtaining the data from external sources like a configuration file or a database.|Example: Hardcoding the server's IP address makes the application less flexible.  将服务器的IP地址硬编码使得应用程序缺乏灵活性。

Example:  Avoid hardcoding values; use variables instead. 避免硬编码值；使用变量代替。|近义词: embedding, fixed value|反义词: softcoding, dynamic coding|词性: noun
