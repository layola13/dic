nonmodally:/ˌnɑːnˈmoʊdəli/| adv.|非模态地|Presenting information or interacting with a user interface in a way that doesn't block other interactions.  It allows the user to continue working with other parts of the application while the nonmodal window or element is open.  This contrasts with modal interactions, which require the user to address the modal window before continuing.|The settings panel appeared nonmodally, allowing me to continue editing my document. 设置面板以非模态方式出现，允许我继续编辑文档。|The software update notification displayed nonmodally, so I could finish my work before restarting. 软件更新通知以非模态方式显示，因此我可以在重新启动之前完成我的工作。|近义词：asynchronously, concurrently|反义词：modally|词性:副词


ETYMOLOGY->
"non-" (prefix meaning "not") + "modal" (referring to a mode or manner of operation) + "-ly" (adverbial suffix).  "Modal" ultimately derives from Latin "modus" (manner, measure).
