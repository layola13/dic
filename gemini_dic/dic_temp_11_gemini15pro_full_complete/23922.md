breakpoint:/ˈbreɪkpɔɪnt/| n.|断点; 临界点; 转折点|A point in a program at which execution may be interrupted to allow the programmer to examine the state of the program or to change variables before continuing execution. A moment when significant change occurs or can be made.|例句：The debugger allows you to set breakpoints in your code.  调试器允许你在代码中设置断点。|近义词：stopping point, milestone, turning point|反义词: continuation point|noun
