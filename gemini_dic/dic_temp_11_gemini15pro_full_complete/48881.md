defragmenter:/diːfræɡˈmɛntər/| n.|碎片整理程序|A utility that reorganizes the files on a hard disk so that they occupy contiguous storage locations, thus speeding up file access.|I ran the disk defragmenter to improve my computer's performance. 我运行了磁盘碎片整理程序来提高我的电脑性能。|Synonyms: optimizer, defragger|Antonyms: fragmenter|noun

defragment:/diːfræɡˈmɛnt/| vt.|整理碎片|To reorganize files on a disk so that they occupy contiguous storage locations.|I need to defragment my hard drive. 我需要整理我的硬盘碎片。|Synonyms: optimize, consolidate|Antonyms: fragment|verb
