keymap:/ˈkiːmæp/| n. |键位图，键盘映射|A keymap is a file or data structure that defines the mapping between keyboard keys and the actions or characters they produce.  It allows users to customize their keyboard layout and assign specific functions to different keys.|I customized my keymap to improve my coding efficiency. 我自定义了我的键位图以提高我的编码效率。|synonyms: keyboard layout, key bindings|antonyms: N/A|noun
