debugger: /diːˈbʌɡər/ | n. | 调试器 | A computer program used to test and debug other programs.  A debugger can be used to step through code line by line, inspect variables, set breakpoints, and more. |  I used the debugger to find the cause of the crash. 我使用调试器找到了崩溃的原因。 |  The debugger allowed me to step through the code and see what was happening.  调试器允许我逐步执行代码并查看发生了什么。 | synonym: debugging tool | antonym: N/A | noun
