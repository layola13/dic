multi-threading:/ˌmʌltiˈθrɛdɪŋ/ |n.|多线程|A technique in software programming where multiple threads of execution run concurrently within a single process, allowing for parallel processing and improved performance.|The application uses multi-threading to speed up complex calculations. 该应用程序使用多线程来加速复杂的计算。|parallel processing, concurrency|single-threading|noun
Etymology: "multi-" (from Latin "multus" meaning "many") + "threading" (from "thread" meaning a sequence of instructions within a program).
