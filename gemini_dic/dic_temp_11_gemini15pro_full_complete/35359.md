co-infinite:/ˌkoʊˈɪnfɪnɪt/
adjective
共同无限的，余有限的
Having a finite complement (with respect to some universal set).  A subset A of a set X is co-infinite in X if its complement X\A is infinite.
The set of even numbers is co-infinite in the set of all integers.  偶数集在所有整数集中是余有限的。

The set of prime numbers is co-infinite in the set of all natural numbers. 质数集在所有自然数集中是余有限的。
近义词:cofinite
反义词:infinite
