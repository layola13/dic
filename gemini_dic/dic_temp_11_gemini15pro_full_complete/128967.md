nio:/ˈnaɪoʊ/
noun
非阻塞式输入/输出 (Java)
Non-blocking Input/Output (in Java).  A Java API that allows for non-blocking I/O operations.
The server uses NIO to handle multiple client connections concurrently.  服务器使用NIO来同时处理多个客户端连接。
Netty is built on top of Java NIO. Netty构建于Java NIO之上。
asynchronous I/O, non-blocking I/O
blocking I/O
n.
