cout:/kaʊt/| n.| (computing) Character output; an output stream object typically used to display text on a computer screen or write text to a file. | It's similar to the `printf` function in C, but is type-safe.  | `std::cout << "Hello, world!" << std::endl;`  输出 "Hello, world!" 到控制台。| synonyms: output, display, write (in context of programming) | antonyms: cin, input (in context of programming) |  noun

虚拟语气:
If the value were valid, the program would cout it. 如果值有效，程序就会输出它。
I wish the system would cout more detailed error messages. 我希望系统能输出更详细的错误信息。


被动语态:
(Cout is rarely used in the passive voice because it describes an action performed by the program, not something done to the output.)


疑问句:
What does the program cout when it encounters an error? 当程序遇到错误时，它会输出什么？
Why is the system cout'ing so many warning messages? 为什么系统会输出这么多警告信息？

否定句:
The program does not cout the user's password. 程序不会输出用户的密码。
The system didn't cout any error messages, which made debugging difficult.  系统没有输出任何错误信息，这使得调试变得困难。



PHRASE->
`std::cout << ... << std::endl;`  输出到控制台并换行。
`std::cout << std::fixed << std::setprecision(2) << ... ;`  设置输出精度。
`std::cout << std::setw(10) << ... ;`  设置输出宽度。
`std::cout << std::left << ... ;`  左对齐输出。
`std::cout << std::hex << ... ;`  以十六进制输出。
