kotlin:/ˈkɒtlɪn/ | n. | 科特林（一种编程语言） |  A cross-platform, statically typed, general-purpose programming language with type inference. Kotlin is designed to interoperate fully with Java, and the JVM version of its standard library depends on the Java Class Library, but type inference allows its syntax to be more concise. | Kotlin is used by many Android developers. 科特林被许多安卓开发者使用。| 近义词: Java, Swift, C# | 反义词: (无) | noun
