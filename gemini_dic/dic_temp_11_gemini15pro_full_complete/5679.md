alua:/əˈluːə/ | n. |  非对称逻辑单元访问 (Asymmetric Logical Unit Access) | Asymmetric Logical Unit Access (ALUA) is a set of SCSI commands that define the preferred path to a particular logical unit.  It allows multipathing software to differentiate between paths to a storage device based on their characteristics, such as performance and availability. |  The storage array supports ALUA, which allows for optimized path selection. 存储阵列支持 ALUA，从而可以优化路径选择。|  synonyms:  multipathing, path failover | antonyms: single path | noun
