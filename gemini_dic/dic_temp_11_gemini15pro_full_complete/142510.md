oss:/ɒs/ | n. | 开源软件 (Open Source Software) | Open Source Software refers to software whose source code is made available for anyone to study, modify, and distribute. | Example:  Many popular web servers run on oss. 许多流行的网络服务器都在开源软件上运行。

Example:  The oss community is very active and supportive. 开源软件社区非常活跃，而且互相支持。 | 近义词: free software, open-source software | 反义词: proprietary software, closed-source software | 词性: noun
