functor:/ˈfʌŋktər/ | n. |函子；算子|A mapping between categories. In programming, it's often a type class that abstracts over type constructors, allowing them to be mapped over. |Example 1: In Haskell, the `Functor` type class allows you to apply a function to the values inside a data structure without changing the structure itself.  中文：在 Haskell 中，`Functor` 类型类允许您将函数应用于数据结构内的值，而无需更改结构本身。

Example 2:  Functors are crucial in functional programming for manipulating data within containers. 中文：函子在函数式编程中对于操作容器内的数据至关重要。 |Near Synonyms:  mapping, transformation, operator |Antonyms:  N/A | Noun
