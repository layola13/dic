PHRASE->
`std::cout << "Error!" << std::endl;`  输出: Error!(换行)
`std::cerr << "Warning: File not found." << std::endl;` 输出: Warning: File not found.(换行)
`std::ofstream outfile; outfile.open("output.txt"); outfile << "Data written" << std::endl; outfile.close();`  将"Data written"写入文件 output.txt, 并换行.
`std::cout << "First line" << std::endl << "Second line" << std::endl;` 输出：First line(换行) Second line(换行)
`std::cout << "Value: " << x << std::endl;` 输出：Value: (x的值) (换行)
