dma:/ˌdiː ɛm ˈeɪ/| n.|直接内存访问; 动态机械分析|Direct Memory Access (a feature of computers that allows certain hardware subsystems to access main system memory (RAM) independently of the CPU); Dynamic Mechanical Analysis (a technique used to study the viscoelastic properties of materials).|

The DMA controller allows the hard drive to transfer data directly to RAM without involving the CPU.  DMA控制器允许硬盘直接向RAM传输数据，而无需CPU参与。

Dynamic Mechanical Analysis (DMA) is used to measure the stiffness and damping properties of polymers. 动态力学分析 (DMA) 用于测量聚合物的刚度和阻尼特性。

|近义词：(For DMA as Direct Memory Access) channel I/O, bus mastering|反义词: (No direct antonym for DMA)|词性:noun
