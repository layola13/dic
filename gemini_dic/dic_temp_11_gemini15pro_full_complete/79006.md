git:/ɡɪt/ | n. | Git（分布式版本控制系统） | A distributed version control system for tracking changes in source code during software development. It is designed for coordinating work among programmers, but it can be used to track changes in any set of files. |  I use git to manage my coding projects. 我使用 Git 管理我的编码项目。 |  Version control system, VCS, repository |  None | Noun
