compiler:/kəmˈpaɪlər/ (US) /kəmˈpaɪlə(r)/ (UK)| n. |编译器|A program that translates source code written in a high-level language (like Java or C++) into a low-level language (like machine code) that a computer can understand and execute.|The compiler detected several errors in my code.|近义词:translator|反义词:interpreter|词性:noun

ETYMOLOGY->
From the verb "compile," which comes from the Old French "compiler" meaning "to collect, gather, heap up."  Ultimately from the Latin "compilare," formed from "com-" (together) and "pilare" (to plunder, heap up).
