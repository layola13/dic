literals:/ˈlɪtərəlz/| n.|字面量，文字，常量|A literal is a notation for representing a fixed value in source code. It is a way of expressing a specific value within a program, such as a number, character, or string.  In essence, it's a value written directly into the code.|Example:  The number "5" in the code `x = 5` is a literal.  代码`x = 5`中的数字“5”就是一个字面量。|Synonyms: constants, values|Antonyms: variables|Noun
