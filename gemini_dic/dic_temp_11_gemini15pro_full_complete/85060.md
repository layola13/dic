hard-code:/ˌhɑːrd ˈkoʊd/| v. adj.|硬编码，硬连接|To embed data directly into the source code of a program or other executable object rather than obtain it from external sources or generate it at runtime.  The practice of hard-coding data can make software less flexible and more difficult to maintain.|Example: The server's IP address is hard-coded into the application, so if the server changes, we'll have to update the code. 服务器的IP地址硬编码到应用程序中，所以如果服务器更改，我们将不得不更新代码。|近义词: embed, fix, integrate|反义词: soft-code, parameterize|词性:动词 & 形容词

hard-coded:/ˌhɑːrd ˈkoʊdɪd/| adj.|硬编码的|Embedded directly into the source code of a program or other executable object.|Example: The database connection string is hard-coded into the configuration file. 数据库连接字符串硬编码到配置文件中。|近义词: fixed, embedded, built-in|反义词: parameterized, configurable|词性:形容词
