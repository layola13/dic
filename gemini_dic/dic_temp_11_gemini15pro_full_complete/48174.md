decompiler:/diːkəmˈpaɪlər/| n.|反编译器|A program that takes compiled code (like machine code or bytecode) and attempts to reconstruct the original source code (like C++ or Java) from which it was compiled.|The decompiler helped me understand how the closed-source software worked. 反编译器帮助我理解了这个闭源软件是如何工作的。|近义词:reverse engineer, disassembler|反义词:compiler|词性:noun
