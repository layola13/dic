json:/ˈdʒeɪsən/ | n. |  JSON (JavaScript Object Notation) | A lightweight data-interchange format that is easy for humans to read and write and easy for machines to parse and generate. It is based on a subset of the JavaScript Programming Language, Standard ECMA-262 3rd Edition - December 1999. | Example: The API returns data in JSON format.  这个API以JSON格式返回数据。 | Synonym: data format, text format | Antonym: binary format | noun
