multi-threaded:/ˌmʌltiˈθrɛdɪd/| adj.|多线程的|Designed to execute multiple threads concurrently, typically within a single process.  It allows a program to perform multiple tasks seemingly at the same time, improving efficiency and responsiveness.|This application is multi-threaded, allowing it to handle multiple user requests simultaneously. 这个应用程序是多线程的，可以同时处理多个用户请求。|The multi-threaded design significantly improved the server's performance. 多线程设计显著提高了服务器的性能。|concurrent, parallel|single-threaded|adjective
