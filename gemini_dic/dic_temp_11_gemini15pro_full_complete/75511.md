fstream:/ˈɛfˌstriːm/| n.|文件流|A type of stream in C++ used for reading and writing to files.  It combines the functionalities of ifstream (input file stream) and ofstream (output file stream).|`#include <fstream>

int main() {
  std::fstream file("my_file.txt", std::ios::in | std::ios::out); 
  if (file.is_open()) {
    file << "Hello, file!";
    file.close();
  }
}`
C++中用于读写文件的一种流类型，它结合了ifstream（输入文件流）和ofstream（输出文件流）的功能。|近义词:iostream (for general stream operations), ifstream (for input), ofstream (for output)|反义词: N/A|词性:名词
