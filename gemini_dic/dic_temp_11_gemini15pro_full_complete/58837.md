duplexed:/ˈduːplɛkst/ /ˈdjuːplɛkst/
adjective
双工的，双倍的，双重的
equipped with two parts to perform the same function, so that if one part fails, the other can continue to operate.
The server has a duplexed hard drive, so data is mirrored to prevent loss. 服务器有一个双工硬盘，所以数据被镜像以防止丢失。
The system employs a duplexed communication link for enhanced reliability. 该系统采用双工通信链路以增强可靠性。
Synonyms: redundant, duplicated, doubled
Antonyms: single, simplex
词性: 形容词
