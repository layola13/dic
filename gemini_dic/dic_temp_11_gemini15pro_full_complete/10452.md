applet:/ˈæplɪt/| n.|小型应用程序，小程序|A small application, especially a Java application that runs within a web browser.|Example: This website uses Java applets to display interactive graphics.  这个网站使用Java小程序来显示交互式图形。|Example:  He created a simple applet to calculate mortgage payments. 他创建了一个简单的小程序来计算抵押贷款还款。|近义词:application, program, widget|反义词: (No direct antonym, perhaps "main application")|词性:noun
