IIFE:/ˈɪfi/| n.|立即执行函数表达式 (Immediately Invoked Function Expression)|A JavaScript function that runs as soon as it is defined.|
```javascript
(function() {
  console.log("Hello from an IIFE!");
})(); // Output: Hello from an IIFE!
```

```javascript
(function(name) {
  console.log("Hello " + name + " from an IIFE!");
})("World"); // Output: Hello World from an IIFE!
```
|近义词：self-executing anonymous function|反义词: 無|名词
