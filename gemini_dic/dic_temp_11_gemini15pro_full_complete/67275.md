.exr:/ˌɪːˌɛksˈɑːr/| n.|OpenEXR 文件|A high-dynamic-range (HDR) image file format developed by Industrial Light & Magic (ILM).|The VFX artist saved the rendered image as an .exr file to preserve the wide color gamut.  视觉特效艺术家将渲染的图像保存为 .exr 文件以保留宽色域。|近义词：.hdr, .tif (sometimes)|反义词: .jpg, .png (generally)|词性:noun

.exr:/ˌɪːˌɛksˈɑːr/| n.|OpenEXR 图像|An image saved in the OpenEXR file format. |The .exr image contained a lot more detail in the highlights and shadows. .exr 图像在高光和阴影部分包含更多细节。|近义词：HDR image|反义词:LDR image|词性:noun
