bytecode:/ˈbaɪtkəʊd/ (US) /ˈbaɪtkoʊd/| n. |字节码|Instructions for a computer, in an intermediate form between source code and machine code.  It is designed to be easily executed by a virtual machine.|The Java compiler converts Java source code into bytecode, which can then be run on any platform with a Java Virtual Machine.|近义词:intermediate code, p-code|反义词:source code|Noun
