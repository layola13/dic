combinator:/ˈkɒmbɪneɪtə(r)/ (US) /ˈkɑːmbɪneɪtər/| n. |组合子, 合成器|A function with no free variables. A combinator is a higher-order function that uses only function application and earlier defined combinators to define a result from its arguments.  It is used in combinatory logic, a system of logic that avoids the use of variables.|The programmer used combinators to create a new function. 程序员使用组合子来创建一个新函数。|Synonym: operator, function|Antonym:  N/A (The concept doesn't lend itself to direct opposites.)|Noun
