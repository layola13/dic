multi-access:/ˈmʌltiˈækses/| adj.|多路访问的，多址访问的|Allowing several users to access a computer system simultaneously.|This server uses a multi-access network to handle multiple connections at the same time. 这台服务器使用多路访问网络来同时处理多个连接。|Synonyms: multiple-access, multiuser|Antonyms: single-access|adjective
