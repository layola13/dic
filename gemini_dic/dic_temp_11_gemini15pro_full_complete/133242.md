nonexecutable:/ˌnɒnˌɛksɪˈkjuːtəbl/ | adj. | 不可执行的 | Not able to be executed, as a computer file that does not contain instructions that a computer can directly run. | The configuration file is nonexecutable and only contains settings.  配置文件是不可执行的，只包含设置。 | The image file is nonexecutable and requires a separate program to view it.  图像文件是不可执行的，需要单独的程序来查看。 | 近义词: unexecutable, non-runnable | 反义词: executable, runnable | 词性: 形容词

Etymology:
non- (prefix) + executable

non-: from Latin *non, from Old Latin noenum "not one," from ne "not" + oinos "one"
executable: from execute + -able

execute: from Latin executare, frequentative of exsequi "follow out, carry out," from ex- "out" + sequi "to follow"
-able: suffix forming adjectives meaning "capable of, fit for, tending to, given to"
